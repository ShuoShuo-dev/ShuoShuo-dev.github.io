📦 如何发布笔记
  1. 在本页面新增 / 编辑笔记（保存至 notes.json）
  2. 将笔记 Markdown 文件放入 md-notes/ 目录
  3. 在博客根目录执行 python build.py
  4. 执行 python push.py 推送至 GitHub
  5. 等待 1~2 分钟，GitHub Pages 自动部署完成
⚡ 快速命令
构建博客
python build.py
构建并推送
python push.py
查看线上博客
