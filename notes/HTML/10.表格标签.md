# 表格标签

## 表格标签概述

HTML表格用于展示行列格式的二维数据。

## 基本结构

```html
<table>
    <tr>
        <td>单元格1</td>
        <td>单元格2</td>
    </tr>
    <tr>
        <td>单元格3</td>
        <td>单元格4</td>
    </tr>
</table>
```

### 标签说明

| 标签 | 说明 |
|------|------|
| `<table>` | 表格容器 |
| `<tr>` | 表格行（table row） |
| `<td>` | 单元格（table data） |
| `<th>` | 表头单元格（table header） |

## 完整表格示例

```html
<table>
    <thead>
        <tr>
            <th>姓名</th>
            <th>年龄</th>
            <th>职业</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>张三</td>
            <td>25</td>
            <td>前端工程师</td>
        </tr>
        <tr>
            <td>李四</td>
            <td>28</td>
            <td>后端工程师</td>
        </tr>
        <tr>
            <td>王五</td>
            <td>30</td>
            <td>产品经理</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td colspan="3">共3条记录</td>
        </tr>
    </tfoot>
</table>
```

## 表格结构化标签

| 标签 | 作用 |
|------|------|
| `<thead>` | 包裹表头行 |
| `<tbody>` | 包裹主体内容 |
| `<tfoot>` | 包裹表尾行 |

**注意**：结构化标签是可选的，但推荐使用，有利于SEO和样式控制。

## 单元格合并

### 横向合并（colspan）

```html
<tr>
    <td colspan="2">合并两个单元格</td>
    <!-- 少写一个td -->
</tr>
```

### 纵向合并（rowspan）

```html
<tr>
    <td rowspan="2">合并两个单元格</td>
    <td>内容1</td>
</tr>
<tr>
    <!-- 少写一个td，因为被上面的td占用了 -->
    <td>内容2</td>
</tr>
```

### 合并示例

```html
<table border="1">
    <tr>
        <th colspan="3">学生成绩表</th>
    </tr>
    <tr>
        <th>姓名</th>
        <th>科目</th>
        <th>分数</th>
    </tr>
    <tr>
        <td rowspan="2">张三</td>
        <td>语文</td>
        <td>90</td>
    </tr>
    <tr>
        <!-- 姓名列被张三占用 -->
        <td>数学</td>
        <td>95</td>
    </tr>
</table>
```

## 常用属性

| 属性 | 说明 | 值 |
|------|------|-----|
| `border` | 边框宽度 | 像素值 |
| `cellpadding` | 单元格内边距 | 像素值 |
| `cellspacing` | 单元格间距 | 像素值 |
| `width` | 表格宽度 | 像素值/百分比 |
| `height` | 表格高度 | 像素值/百分比 |

```html
<table border="1" cellpadding="10" cellspacing="5" width="500">
```

**注意**：表格样式属性建议使用CSS实现。

## 注意事项

1. **使用语义化结构**：`thead`、`tbody`、`tfoot` 有助于SEO和样式控制
2. **合并单元格要准确**：确保被合并的单元格被正确移除
3. **不要用表格布局**：现代网页应该使用CSS Flexbox或Grid布局
4. **设置边框样式**：默认没有边框，记得添加

## 总结

- `<table>` 创建表格
- `<tr>` 创建行
- `<td>` 创建单元格
- `<th>` 创建表头（自动加粗居中）
- `<thead>`、`<tbody>`、`<tfoot>` 结构化表格
- `colspan` 横向合并
- `rowspan` 纵向合并
