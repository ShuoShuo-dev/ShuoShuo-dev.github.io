# 标题标签

## 标题标签介绍

HTML提供了六个级别的标题标签：`<h1>` 到 `<h6>`

```html
<h1>一级标题</h1>
<h2>二级标题</h2>
<h3>三级标题</h3>
<h4>四级标题</h4>
<h5>五级标题</h5>
<h6>六级标题</h6>
```

## 标题标签特点

1. **语义化**：标题标签代表文档的层级结构
2. **加粗**：标题文字默认加粗显示
3. **字号**：从h1到h6字号依次减小
4. **独占一行**：标题标签是块级元素

## 标题标签的使用场景

```html
<body>
    <!-- 网站logo或主标题 -->
    <h1>我的网站</h1>

    <!-- 文章大标题 -->
    <h2>第一章：HTML基础</h2>
    <h3>1.1 标题标签</h3>

    <!-- 段落标题 -->
    <h4>重要说明</h4>
</body>
```

## 注意事项

- **h1标签**：一个页面最好只有一个，用于页面主标题，SEO很重要
- **h2-h6标签**：可以多次使用，但要保持层级清晰
- 不要用标题标签来改变文字大小，应该用CSS

## 标题标签与SEO

搜索引擎非常重视h1标签，它告诉搜索引擎页面的主题是什么。

```html
<!-- 推荐写法：一个页面一个h1 -->
<body>
    <h1>文章标题</h1>
    <article>
        <h2>章节一</h2>
        <h3>小节1.1</h3>
    </article>
</body>

<!-- 不推荐：一个页面多个h1 -->
<body>
    <h1>标题1</h1>
    <h1>标题2</h1>
    <h1>标题3</h1>
</body>
```

## 总结

标题标签是HTML中最基本的标签之一，用于构建文档的大纲结构。使用时要遵循语义化原则，保持层级清晰。
