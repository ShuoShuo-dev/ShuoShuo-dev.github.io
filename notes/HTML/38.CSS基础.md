# CSS基础

## 一、CSS简介

CSS（Cascading Style Sheets，层叠样式表）用于控制网页的**外观和布局**。

### 1.1 CSS与HTML的关系

```
HTML = 网页的结构（骨架）
CSS = 网页的样式（皮肤）
JavaScript = 网页的行为（灵魂）
```

### 1.2 CSS的基本语法

```css
/* 选择器 { 属性: 值; } */
selector {
    property: value;
    property: value;
}

/* 示例 */
h1 {
    color: red;
    font-size: 24px;
}
```

### 1.3 CSS的引入方式

```html
<!-- 1. 内联样式（不推荐） -->
<h1 style="color: red;">内联样式</h1>

<!-- 2. 内部样式表 -->
<head>
    <style>
        h1 { color: red; }
    </style>
</head>

<!-- 3. 外部样式表（推荐） -->
<head>
    <link rel="stylesheet" href="style.css">
</head>

<!-- 4. @import -->
<style>
    @import url("style.css");
</style>
```

## 二、CSS选择器

### 2.1 基本选择器

```css
/* 元素选择器 */
p { color: blue; }

/* 类选择器 */
.highlight { background: yellow; }

/* ID选择器 */
#header { background: #333; }

/* 通配符选择器 */
* { margin: 0; padding: 0; }

/* 属性选择器 */
input[type="text"] { border: 1px solid #ddd; }
a[href^="https"] { color: green; }    /* 以https开头 */
a[href$=".pdf"] { color: red; }       /* 以.pdf结尾 */
a[href*="example"] { color: blue; }  /* 包含example */
```

### 2.2 组合选择器

```css
/* 后代选择器 */
div p { color: blue; }

/* 子选择器 */
div > p { color: blue; }

/* 相邻兄弟选择器 */
h1 + p { color: blue; }

/* 通用兄弟选择器 */
h1 ~ p { color: blue; }

/* 分组选择器 */
h1, h2, h3 { color: blue; }
```

### 2.3 伪类选择器

```css
/* 链接状态 */
a:link { color: blue; }        /* 未访问 */
a:visited { color: purple; }  /* 已访问 */
a:hover { color: red; }        /* 悬停 */
a:active { color: orange; }    /* 点击中 */

/* 表单伪类 */
input:focus { border-color: blue; }
input:valid { border-color: green; }
input:invalid { border-color: red; }
input:disabled { background: #eee; }
input:checked + label { color: blue; }

/* 结构伪类 */
li:first-child { color: red; }
li:last-child { color: blue; }
li:nth-child(2) { color: green; }      /* 第2个 */
li:nth-child(odd) { background: #f5f5f5; }  /* 奇数 */
li:nth-child(even) { background: #fff; }  /* 偶数 */
li:nth-child(3n) { color: red; }           /* 3的倍数 */

/* 否定伪类 */
p:not(.intro) { color: blue; }
input:not(:disabled) { border: 1px solid #ddd; }
```

### 2.4 伪元素选择器

```css
/* 伪元素 */
p::before { content: "前面"; }
p::after { content: "后面"; }
p::first-line { color: red; }    /* 首行 */
p::first-letter { font-size: 2em; } /* 首字母 */

/* 选中文本 */
::selection { background: yellow; color: black; }

/* 占位符 */
input::placeholder { color: #999; }
```

## 三、CSS常用属性

### 3.1 文本属性

```css
/* 颜色 */
color: #333;           /* 十六进制 */
color: rgb(255,0,0);   /* RGB */
color: rgba(0,0,0,0.5);/* 带透明度 */
color: red;            /* 颜色名 */

/* 字体 */
font-family: Arial, "Microsoft YaHei", sans-serif;
font-size: 16px;
font-weight: normal;    /* normal, bold, 100-900 */
font-style: normal;     /* normal, italic, oblique */
font: italic bold 16px/1.5 Arial;  /* 简写 */

/* 文本对齐 */
text-align: left;       /* left, center, right, justify */
vertical-align: middle; /* top, middle, bottom */

/* 文本装饰 */
text-decoration: none;         /* none, underline, overline, line-through */
text-decoration: underline wavy red; /* 多值写法 */

/* 行高 */
line-height: 1.5;      /* 数字 */
line-height: 24px;    /* 固定值 */

/* 文字间距 */
letter-spacing: 2px;   /* 字符间距 */
word-spacing: 4px;     /* 单词间距 */

/* 首行缩进 */
text-indent: 2em;

/* 文本转换 */
text-transform: uppercase;  /* uppercase, lowercase, capitalize */

/* 文字阴影 */
text-shadow: 2px 2px 4px rgba(0,0,0,0.3);

/* 文本溢出 */
white-space: nowrap;        /* nowrap, pre, pre-wrap */
overflow: hidden;
text-overflow: ellipsis;    /* ellipsis, clip */
```

### 3.2 背景属性

```css
/* 背景色 */
background-color: #f5f5f5;

/* 背景图 */
background-image: url('bg.jpg');
background-repeat: no-repeat;  /* repeat, repeat-x, repeat-y, no-repeat */
background-position: center;   /* 位置 */

/* 背景尺寸 */
background-size: cover;       /* cover, contain, 100px 200px */

/* 背景固定 */
background-attachment: fixed; /* scroll, fixed */

/* 渐变背景 */
background: linear-gradient(to right, red, blue);
background: linear-gradient(45deg, red, blue);
background: radial-gradient(circle, red, blue);

/* 多重背景 */
background:
    url('img1.png') no-repeat left top,
    url('img2.png') no-repeat right bottom,
    #f5f5f5;

/* 简写 */
background: #f5f5f5 url('bg.jpg') no-repeat center/cover;
```

### 3.3 盒子模型

```css
/* 尺寸 */
width: 200px;
height: 100px;
max-width: 100%;
min-height: 50px;

/* 内边距 */
padding: 10px;           /* 四个方向 */
padding: 10px 20px;      /* 上下 左右 */
padding: 10px 20px 30px; /* 上 左右 下 */
padding: 10px 20px 30px 40px; /* 上 右 下 左 */

/* 外边距 */
margin: 10px;
margin: 0 auto;           /* 居中 */

/* 边框 */
border: 1px solid #333;
border-width: 2px;
border-style: solid;     /* solid, dashed, dotted, double */
border-color: #333;
border-radius: 8px;      /* 圆角 */

/* 边框各边 */
border-top: 2px solid red;
border-right: none;
border-bottom: 1px dashed #999;
border-left: 3px solid blue;

/* 阴影 */
box-shadow: 2px 2px 8px rgba(0,0,0,0.2);
box-shadow: inset 0 0 10px rgba(0,0,0,0.1); /* 内阴影 */

/* 盒子-sizing */
box-sizing: content-box; /* 默认：width = 内容 */
box-sizing: border-box; /* width = 内容 + padding + border */
```

### 3.4 布局属性

```css
/* 显示 */
display: block;          /* block, inline, inline-block */
display: none;          /* 隐藏 */
display: flex;          /* 弹性盒 */
display: grid;          /* 网格布局 */

/* 定位 */
position: static;       /* static, relative, absolute, fixed, sticky */
position: relative;
position: absolute;
position: fixed;        /* 固定定位 */
position: sticky;       /* 粘性定位 */
top: 0;
right: 0;
bottom: 0;
left: 0;
z-index: 1;

/* 浮动 */
float: left;    /* left, right, none */
float: right;

/* 清除浮动 */
clear: both;

/* 溢出 */
overflow: hidden;   /* hidden, auto, scroll, visible */
overflow-x: hidden;
overflow-y: auto;
```

## 四、Flexbox布局

### 4.1 容器属性

```css
.container {
    display: flex;

    /* 方向 */
    flex-direction: row;           /* row, column, row-reverse, column-reverse */

    /* 换行 */
    flex-wrap: nowrap;             /* nowrap, wrap, wrap-reverse */

    /* 主轴对齐 */
    justify-content: flex-start;   /* flex-start, flex-end, center, space-between, space-around, space-evenly */

    /* 交叉轴对齐 */
    align-items: stretch;          /* stretch, flex-start, flex-end, center, baseline */

    /* 多行对齐 */
    align-content: stretch;       /* stretch, flex-start, flex-end, center, space-between, space-around */
}
```

### 4.2 项目属性

```css
.item {
    /* 放大 */
    flex-grow: 0;     /* 默认0，不放大 */

    /* 缩小 */
    flex-shrink: 1;   /* 默认1，可缩小 */

    /* 基准尺寸 */
    flex-basis: auto; /* auto 或具体值 */

    /* 简写 */
    flex: 1;          /* flex-grow: 1; flex-shrink: 1; flex-basis: 0%; */

    /* 单个项目对齐 */
    align-self: center;

    /* 顺序 */
    order: 0;
}
```

### 4.3 示例

```css
/* 水平居中 */
.flex-center {
    display: flex;
    justify-content: center;
    align-items: center;
}

/* 两端对齐 */
.space-between {
    display: flex;
    justify-content: space-between;
}

/* 三栏布局 */
.three-columns {
    display: flex;
    gap: 20px;
}
.three-columns > :nth-child(1) { flex: 1; }
.three-columns > :nth-child(2) { flex: 2; }
.three-columns > :nth-child(3) { flex: 1; }
```

## 五、Grid布局

### 5.1 基本语法

```css
.container {
    display: grid;

    /* 定义列 */
    grid-template-columns: 100px 100px 100px;
    grid-template-columns: repeat(3, 100px);
    grid-template-columns: 1fr 1fr 1fr;    /* fr单位 */
    grid-template-columns: 200px 1fr 2fr;

    /* 定义行 */
    grid-template-rows: 100px 200px;

    /* 间距 */
    gap: 20px;
    row-gap: 10px;
    column-gap: 20px;

    /* 自动列数 */
    grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
}
```

### 5.2 网格项目

```css
.item {
    /* 位置 */
    grid-column: 1 / 3;     /* 开始 / 结束 */
    grid-row: 1 / 2;

    /* 跨度 */
    grid-column: 1 / span 2;

    /* 简写 */
    grid-area: 1 / 1 / 2 / 3;

    /* 命名 */
    grid-area: header;
}
```

### 5.3 响应式网格

```css
.grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    gap: 20px;
}
```

## 六、CSS动画

### 6.1 transition

```css
.box {
    transition: property duration timing-function delay;
    transition: all 0.3s ease;
    transition: transform 0.3s, background-color 0.3s;
}

.box:hover {
    transform: scale(1.1);
    background-color: red;
}
```

### 6.2 animation

```css
@keyframes slideIn {
    from {
        transform: translateX(-100%);
        opacity: 0;
    }
    to {
        transform: translateX(0);
        opacity: 1;
    }
}

.element {
    animation: slideIn 0.5s ease forwards;
    animation-duration: 0.5s;
    animation-timing-function: ease;
    animation-delay: 0s;
    animation-iteration-count: 1;  /* 次数 */
    animation-direction: normal;     /* 方向 */
    animation-fill-mode: forwards;   /* 结束时状态 */
}
```

## 七、CSS变量

### 7.1 定义和使用

```css
:root {
    --primary-color: #007bff;
    --border-radius: 8px;
    --spacing: 16px;
}

.button {
    background-color: var(--primary-color);
    border-radius: var(--border-radius);
    padding: var(--spacing);
}

.button-secondary {
    --primary-color: #6c757d;  /* 覆盖变量 */
}
```

### 7.2 动态更新

```javascript
document.documentElement.style.setProperty('--primary-color', 'red');
```
