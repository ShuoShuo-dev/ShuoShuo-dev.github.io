# 表单标签（上）

## 表单简介

表单用于收集用户输入的信息，是网页与用户交互的重要方式。

## form标签

表单的容器，所有表单元素都放在 `<form>` 标签内。

```html
<form action="提交地址" method="提交方式">
    <!-- 表单元素 -->
</form>
```

### form常用属性

| 属性 | 说明 | 值 |
|------|------|-----|
| `action` | 表单提交的地址 | URL |
| `method` | 提交方式 | GET / POST |
| `name` | 表单名称 | 字符串 |
| `target` | 提交后打开方式 | _self / _blank |

```html
<form action="/submit" method="POST">
    <!-- 表单内容 -->
</form>
```

## input标签

`<input>` 是最常用的表单元素，是一个单标签。

### 基本语法

```html
<input type="类型" name="名称" value="默认值">
```

### type属性值

| type值 | 说明 | 示例 |
|--------|------|------|
| `text` | 单行文本输入 | 用户名 |
| `password` | 密码输入（隐藏） | 密码 |
| `radio` | 单选按钮 | 性别 |
| `checkbox` | 多选按钮 | 爱好 |
| `file` | 文件上传 | 上传文件 |
| `submit` | 提交按钮 | 提交表单 |
| `reset` | 重置按钮 | 重置表单 |
| `button` | 普通按钮 | 自定义 |
| `hidden` | 隐藏字段 | 传值 |
| `email` | 邮箱输入 | 邮箱验证 |
| `number` | 数字输入 | 数量 |
| `tel` | 电话输入 | 手机号 |
| `url` | URL输入 | 网址 |
| `date` | 日期选择 | 日期 |
| `search` | 搜索框 | 搜索 |

## 文本输入框

```html
<form>
    <label>用户名：</label>
    <input type="text" name="username" placeholder="请输入用户名">
</form>
```

### 常用属性

| 属性 | 说明 |
|------|------|
| `name` | 字段名称 |
| `value` | 默认值 |
| `placeholder` | 占位提示文字 |
| `maxlength` | 最大字符数 |
| `readonly` | 只读 |
| `disabled` | 禁用 |

## 密码输入框

```html
<form>
    <label>密码：</label>
    <input type="password" name="password" placeholder="请输入密码">
</form>
```

## 单选按钮 radio

```html
<form>
    <p>性别：</p>
    <input type="radio" name="gender" value="male" id="male">
    <label for="male">男</label>

    <input type="radio" name="gender" value="female" id="female">
    <label for="female">女</label>
</form>
```

### 注意事项

1. 同一组单选按钮必须有相同的 `name` 属性
2. 建议使用 `<label>` 关联文字
3. `checked` 属性设置默认选中

```html
<input type="radio" name="gender" value="male" checked> 男
```

## 多选按钮 checkbox

```html
<form>
    <p>爱好：</p>
    <input type="checkbox" name="hobby" value="reading" id="reading">
    <label for="reading">阅读</label>

    <input type="checkbox" name="hobby" value="music" id="music">
    <label for="music">音乐</label>

    <input type="checkbox" name="hobby" value="sports" id="sports">
    <label for="sports">运动</label>
</form>
```

### 常用属性

```html
<input type="checkbox" name="agree" value="yes" checked disabled>
```

## label标签

用于关联文本和表单元素，提高可访问性。

```html
<!-- 方式1：通过for属性关联 -->
<label for="username">用户名：</label>
<input type="text" id="username">

<!-- 方式2：嵌套包裹 -->
<label>
    <input type="checkbox"> 同意协议
</label>
```

## 完整登录表单示例

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>登录表单</title>
</head>
<body>
    <h2>用户登录</h2>
    <form action="/login" method="POST">
        <p>
            <label for="username">用户名：</label>
            <input type="text" id="username" name="username" placeholder="请输入用户名">
        </p>
        <p>
            <label for="password">密码：</label>
            <input type="password" id="password" name="password" placeholder="请输入密码">
        </p>
        <p>
            <label>
                <input type="checkbox" name="remember"> 记住密码
            </label>
        </p>
        <p>
            <input type="submit" value="登录">
            <input type="reset" value="重置">
        </p>
    </form>
</body>
</html>
```

## 总结

- `<form>` 是表单容器
- `<input>` 是核心表单元素
- `type` 属性决定输入类型
- `name` 属性用于数据提交
- `<label>` 关联文本和表单元素
