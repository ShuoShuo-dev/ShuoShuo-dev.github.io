# 特殊符号

## 为什么需要特殊符号

HTML中某些字符有特殊含义，不能直接使用，需要通过字符实体来表示。

## 需要转义的字符

### 小于号和大于号

| 显示 | 字符实体 | 说明 |
|------|----------|------|
| `<` | `&lt;` | 小于号 |
| `>` | `&gt;` | 大于号 |

```html
<!-- 错误：会被浏览器解析为标签 -->
<p>5 < 10 > 3</p>

<!-- 正确：使用实体 -->
<p>5 &lt; 10 &gt; 3</p>
```

### 和号

| 显示 | 字符实体 | 说明 |
|------|----------|------|
| `&` | `&amp;` | 和号 |

```html
<!-- 错误：可能被解析 -->
<p>Tom & Jerry</p>

<!-- 正确 -->
<p>Tom &amp; Jerry</p>
```

### 引号

| 显示 | 字符实体 | 说明 |
|------|----------|------|
| `"` | `&quot;` | 双引号 |
| `'` | `&apos;` | 单引号（IE不支持） |

```html
<!-- 属性值中的引号 -->
<input type="text" value="Hello &quot;World&quot;">

<!-- 特殊情况 -->
<p>她说：&apos;你好&apos;</p>
```

## 实用场景

### 显示HTML代码

```html
<pre>
<code>
&amp;lt;div class="container"&amp;gt;
    &amp;lt;p&amp;gt;Hello World&amp;lt;/p&amp;gt;
&amp;lt;/div&amp;gt;
</code>
</pre>

<!-- 或者直接用转义 -->
&lt;div&gt;内容&lt;/div&gt;
```

### 显示JavaScript代码

```html
<script>
    if (x &lt; 10) {
        console.log(&quot;Hello &quot; + name);
    }
</script>
```

### 数学公式

```html
<p>勾股定理：a&sup2; + b&sup2; = c&sup2;</p>
<p>一元二次方程：ax&sup2; + bx + c = 0</p>
<p>圆面积：S = &pi;r&sup2;</p>
<p>求根公式：x = (&minus;b &plusmn; &radic;(b&sup2; &minus; 4ac)) / 2a</p>
```

### 价格显示

```html
<p>原价：&yen;299.00</p>
<p>折扣价：&yen;199.00</p>
<p>节省：&yen;100.00</p>
```

### 版权和商标

```html
<footer>
    <p>&copy; 2024 公司名称. All Rights Reserved.</p>
    <p>HTML&amp;CSS&amp;JavaScript are registered trademarks.</p>
</footer>
```

### 列表项目

```html
<ul>
    <li>&bull; 列表项一</li>
    <li>&bull; 列表项二</li>
```

## 字符实体vs Unicode

### 字符实体

```html
<!-- 使用名称 -->
&copy;  &lt;  &gt;
```

### Unicode编码

```html
<!-- 使用编号 -->
&#169;  &#60;  &#62;
```

### 常用Unicode范围

| 范围 | 说明 |
|------|------|
| `&#x0000;` - `&#x00FF;` | Latin-1 |
| `&#x4E00;` - `&#x9FA5;` | 中文 |
| `&#x3040;` - `&#x30FF;` | 日文 |
| `&#xAC00;` - `&#xD7AF;` | 韩文 |

## 注意事项

1. **字符实体以 `&` 开头，以 `;` 结尾**
2. **名称区分大小写**
3. **某些浏览器不支持某些实体**
4. **Unicode编号更通用可靠**
5. **现代HTML推荐使用UTF-8编码**

## 总结

- `<` 和 `>` 必须转义：`&lt;` 和 `&gt;`
- `&` 必须转义：`&amp;`
- 引号在属性中需要转义
- 特殊符号可用实体名称或Unicode编号
- 推荐使用UTF-8编码减少转义需求
