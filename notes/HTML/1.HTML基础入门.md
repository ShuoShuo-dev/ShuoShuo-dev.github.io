# HTML基础入门

## 什么是HTML

HTML（Hyper Text Markup Language）超文本标记语言，是一种用于创建网页的标准标记语言。

- **超文本**：指不只是普通文本，还可以包含图片、链接、音乐、视频等非文本内容
- **标记语言**：由标签组成，用于描述网页结构

## HTML的作用

HTML负责**网页的结构和内容**，CSS负责**样式和美化**，JavaScript负责**行为和交互**。

```
HTML = 结构（骨骼）
CSS = 样式（皮肤）
JavaScript = 行为（灵魂）
```

## 第一个HTML页面

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>我的第一个网页</title>
</head>
<body>
    <h1>Hello World!</h1>
    <p>这是我的第一个网页</p>
</body>
</html>
```

## HTML基本结构

| 标签 | 作用 |
|------|------|
| `<!DOCTYPE html>` | 声明文档类型为HTML5 |
| `<html>` | 根元素，包含所有HTML内容 |
| `<head>` | 头部，存放页面元信息 |
| `<body>` | 主体，包含可见的页面内容 |
| `<meta>` | 元数据标签 |
| `<title>` | 页面标题 |
