# HTML注释

## 注释语法

HTML注释用于在代码中添加说明，不会显示在浏览器页面中。

```html
<!-- 这是注释内容 -->
```

## 注释的写法

### 基本语法

```html
<!-- 单行注释 -->

<!-- 这是一段注释 -->
<p>这是可见的段落</p>
```

### 多行注释

```html
<!--
    这是多行注释
    可以写很多行
    不会显示在页面上
-->
```

### 行内注释

```html
<p>普通文本 <!-- 行内注释 --> 继续文本</p>
```

## 注释的用途

### 1. 代码说明

```html
<!-- 页面主导航 -->
<nav>
    <a href="/">首页</a>
    <a href="/about">关于我们</a>
</nav>

<!-- 产品列表 -->
<section class="products">
    <!-- 产品项目 -->
    <div class="product">...</div>
</section>
```

### 2. 临时禁用代码

```html
<!-- 暂时禁用这段代码
<div class="deprecated">
    这是旧功能
</div>
-->

<!-- 禁用单个标签 -->
<div>
    <p>显示的内容</p>
    <!-- <p>这段被注释掉了</p> -->
</div>
```

### 3. 调试和问题标记

```html
<!-- TODO: 添加更多导航项 -->
<!-- FIXME: 这个样式在IE下有问题 -->
<!-- BUG: 这里的链接失效 -->
<!-- NOTE: 需要优化这段代码 -->
```

### 4. 结构标记

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <!-- ========== 页面元数据 ========== -->
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>页面标题</title>

    <!-- ========== 样式表 ========== -->
    <link rel="stylesheet" href="style.css">

    <!-- ========== 脚本 ========== -->
    <script src="main.js"></script>
</head>
<body>
    <!-- ========== 页面头部 ========== -->
    <header>
        <nav>...</nav>
    </header>

    <!-- ========== 主内容 ========== -->
    <main>
        <!-- 文章内容 -->
        <article>...</article>

        <!-- 侧边栏 -->
        <aside>...</aside>
    </main>

    <!-- ========== 页脚 ========== -->
    <footer>...</footer>
</body>
</html>
```

### 5. 条件注释（IE兼容）

```html
<!--[if IE 8]>
    <link href="ie8.css" rel="stylesheet">
<![endif]-->

<!--[if lt IE 9]>
    <script src="https://cdnjs.cloudflare.com/ajax/libs/html5shiv/3.7.3/html5shiv.min.js"></script>
<![endif]-->
```

## 注释的规范

### 1. 清晰简洁

```html
<!-- 推荐：清晰的注释 -->
<!-- 用户登录表单 -->
<form>...</form>

<!-- 不推荐：过于简单 -->
<!-- 表单 -->
<form>...</form>
```

### 2. 适当使用

```html
<!-- 推荐：必要的注释 -->
<!-- 点击跳转到首页 -->
<a href="/">首页</a>

<!-- 不推荐：过多的注释 -->
<!-- 段落开始 -->
<p>第一段文字</p>
<!-- 段落结束 -->
<!-- 段落开始 -->
<p>第二段文字</p>
<!-- 段落结束 -->
```

### 3. 使用约定标记

```html
<!-- TODO: 待完成的功能 -->
<!-- FIXME: 需要修复的问题 -->
<!-- NOTE: 说明或提示 -->
<!-- DEPRECATED: 已废弃的功能 -->
<!-- XXX: 需要改进的代码 -->
```

## 注释与SEO

### 避免过度使用

```html
<!-- 不推荐：大量注释影响SEO和加载 -->
<!-- 关键词1 -->
<!-- 关键词2 -->
<!-- 关键词3 -->
<p>实际内容</p>
```

### 搜索引擎会忽略注释

```html
<!-- 搜索引擎会忽略这段 -->
<p>搜索引擎会读取这段</p>
```

## 注释的快捷键

### VS Code

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + /` | 注释/取消单行注释 |
| `Shift + Alt + A` | 注释/取消块注释 |
| `Ctrl + K Ctrl + C` | 添加行注释 |

## 注释示例

### 页面结构注释

```html
<!--
    文件名: index.html
    作者: 张三
    日期: 2024-01-01
    描述: 首页
-->
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <!-- 公共头部 -->
    <meta charset="UTF-8">
    <title>首页</title>
</head>
<body>
    <!-- 顶部导航栏 -->
    <header class="header">
        <!-- Logo -->
        <div class="logo">
            <img src="logo.png" alt="Logo">
        </div>

        <!-- 导航菜单 -->
        <nav class="nav">
            <a href="/">首页</a>
            <a href="/about">关于</a>
            <a href="/contact">联系</a>
        </nav>
    </header>

    <!-- 主要内容区域 -->
    <main class="main">
        <!-- 轮播图 -->
        <section class="banner">
            <!-- 暂时禁用
            <div class="carousel">...</div>
            -->
        </section>

        <!-- 产品列表 -->
        <section class="products">
            <!-- 产品项目开始 -->
            <div class="product">产品1</div>
            <div class="product">产品2</div>
            <!-- 产品项目结束 -->
        </section>
    </main>

    <!-- 页脚 -->
    <footer class="footer">
        <!-- 版权信息 -->
        <p>&copy; 2024 公司名称</p>
    </footer>
</body>
</html>
```

### 功能模块注释

```html
<!-- ========== 用户认证模块 ========== -->
<section class="auth">
    <!-- 登录表单 -->
    <div class="login-form">
        <h2>用户登录</h2>
        <form action="/login" method="POST">
            <!-- 用户名输入 -->
            <input type="text" name="username" placeholder="用户名">
            <!-- 密码输入 -->
            <input type="password" name="password" placeholder="密码">
            <!-- 提交按钮 -->
            <button type="submit">登录</button>
        </form>
    </div>
</section>
```

## 注释与其他语言对比

### HTML注释

```html
<!-- HTML注释 -->
```

### CSS注释

```css
/* CSS注释 */
/* 多行
   CSS注释 */
```

### JavaScript注释

```javascript
// 单行注释
/* 多行注释 */
```

## 注意事项

1. **注释不会隐藏敏感信息**：用户查看源码可以看到注释内容
2. **不要在注释中放密码或密钥**
3. **删除无用注释**：保持代码整洁
4. **使用标准化标记**：如TODO、FIXME等
5. **合理使用条件注释**：现代浏览器已不需要

## 总结

- HTML注释格式：`<!-- 注释内容 -->`
- 注释不会被浏览器渲染
- 用于代码说明、临时禁用、调试标记
- 保持注释简洁、清晰、必要
- 不要在注释中放敏感信息
