# 超链接标签

## a标签介绍

`<a>` 标签用于创建超链接，实现页面之间的跳转。

```html
<a href="链接地址">链接文本</a>
```

## 基本用法

### 外部链接

```html
<a href="https://www.baidu.com">百度</a>
<a href="https://github.com">GitHub</a>
```

### 内部链接

```html
<a href="about.html">关于我们</a>
<a href="contact.html">联系我们</a>
```

### 空链接

```html
<a href="#">空链接</a>
<a href="javascript:;">阻止跳转</a>
```

## href属性详解

### 绝对URL

```html
<a href="https://www.example.com">访问网站</a>
```

### 相对URL

```html
<a href="page.html">相对路径</a>
<a href="./images/photo.jpg">查看图片</a>
<a href="../index.html">上级目录</a>
```

### 锚点链接

```html
<!-- 定义锚点 -->
<h2 id="section1">第一章</h2>
<h2 id="section2">第二章</h2>

<!-- 链接到锚点 -->
<a href="#section1">跳转到第一章</a>
<a href="#section2">跳转到第二章</a>
<a href="page.html#section1">跳转到其他页面的锚点</a>
```

### 邮件链接

```html
<a href="mailto:example@email.com">发送邮件</a>
```

### 电话链接

```html
<a href="tel:13800138000">拨打热线</a>
```

## target属性

控制链接打开方式：

| 属性值 | 说明 |
|--------|------|
| `_self` | 默认，在当前窗口打开 |
| `_blank` | 在新窗口/标签页打开 |
| `_parent` | 在父窗口打开 |
| `_top` | 在顶层窗口打开 |

```html
<a href="https://www.baidu.com">当前窗口打开</a>
<a href="https://www.baidu.com" target="_blank">新窗口打开</a>
```

## 下载链接

```html
<a href="file.pdf" download>下载PDF</a>
<a href="archive.zip" download="打包文件.zip">下载压缩包</a>
```

## 完整示例

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>超链接示例</title>
</head>
<body>
    <h1>超链接标签示例</h1>

    <h2>外部链接</h2>
    <a href="https://github.com">GitHub</a> |
    <a href="https://www.npmjs.com" target="_blank" rel="noopener">NPM（新窗口）</a>

    <h2>内部链接</h2>
    <a href="about.html">关于我们</a>

    <h2>锚点链接</h2>
    <a href="#top">返回顶部</a> |
    <a href="#bottom">跳到底部</a>

    <h2>功能链接</h2>
    <a href="mailto:help@example.com">发送邮件</a> |
    <a href="tel:400-888-8888">联系电话</a>

    <h2>下载链接</h2>
    <a href="document.pdf" download>下载文档</a>

    <h2>图片链接</h2>
    <a href="https://www.example.com">
        <img src="logo.png" alt="点击进入">
    </a>

    <div id="bottom">
        <p>页面底部</p>
    </div>

    <div id="top">
        <p><a href="#">返回顶部</a></p>
    </div>
</body>
</html>
```

## rel属性

指定当前文档与链接文档的关系：

```html
<!-- 防止新页面访问当前页面window对象 -->
<a href="https://external.com" target="_blank" rel="noopener">安全链接</a>

<!-- nofollow告诉搜索引擎不追踪此链接 -->
<a href="https://external.com" rel="nofollow">不追踪的链接</a>

<!-- 预加载 -->
<link rel="prefetch" href="next-page.html">
```

## 链接状态样式（CSS）

```css
/* 未访问的链接 */
a:link {
    color: blue;
    text-decoration: none;
}

/* 已访问的链接 */
a:visited {
    color: purple;
}

/* 鼠标悬停 */
a:hover {
    color: red;
    text-decoration: underline;
}

/* 点击时 */
a:active {
    color: orange;
}
```

## 注意事项

1. **链接必须有href属性**：否则不是有效的链接
2. **添加target="_blank"时**：建议添加 `rel="noopener"` 防止安全风险
3. **语义化链接文本**：避免使用"点击这里"、"这里"等模糊文本
4. **外部链接提示**：如果链接到外部网站，可以添加图标或文字提示

```html
<!-- 推荐写法 -->
<a href="article.html">阅读完整文章</a>
<a href="https://external.com" target="_blank" rel="noopener">访问外部网站 ↗</a>

<!-- 不推荐写法 -->
<a href="article.html">点击这里</a>
<a href="article.html">这里</a>
```

## 总结

- `<a>` 标签用于创建超链接
- `href` 属性指定链接目标
- `target` 属性控制打开方式，`_blank` 为新窗口
- 合理使用锚点链接提升用户体验
- 外部链接添加 `rel="noopener"` 提高安全性
