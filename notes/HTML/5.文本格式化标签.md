# 文本格式化标签

## 简介

HTML提供了一些用于格式化文本的标签，可以使文字呈现不同的样式，如加粗、斜体、下划线等。

## 加粗标签

### `<strong>` 标签（推荐）

```html
<strong>这是加粗的文字</strong>
```

### `<b>` 标签

```html
<b>这也是加粗的文字</b>
```

**区别**：`strong` 标签有语义，表示重要性；`b` 标签只是视觉上加粗。

## 斜体标签

### `<em>` 标签（推荐）

```html
<em>这是斜体的文字</em>
```

### `<i>` 标签

```html
<i>这也是斜体的文字</i>
```

**区别**：`em` 标签有语义，表示强调；`i` 标签只是视觉上斜体。

## 下划线标签

### `<ins>` 标签（推荐）

```html
<ins>这是带下划线的文字</ins>
```

### `<u>` 标签

```html
<u>这也是带下划线的文字</u>
```

**区别**：`ins` 标签有语义，表示插入的内容；`u` 标签只是视觉上下划线。

## 删除线标签

### `<del>` 标签（推荐）

```html
<del>这是带删除线的文字</del>
```

### `<s>` 标签

```html
<s>这也是带删除线的文字</s>
```

**区别**：`del` 标签有语义，表示被删除的内容；`s` 标签只是视觉上删除线。

## 综合示例

```html
<body>
    <p><strong>加粗标签</strong>用于强调重要的内容</p>
    <p><em>斜体标签</em>用于强调或引用</p>
    <p><ins>下划线标签</ins>用于表示新增的内容</p>
    <p><del>删除线标签</del>用于表示已删除的内容</p>

    <hr>

    <p>原价格：<del>¥199</del>，优惠价：<strong>¥99</strong></p>
    <p><em>限时优惠，先到先得！</em></p>
</body>
```

## 常用文本标签汇总

| 标签 | 作用 | 语义 |
|------|------|------|
| `<strong>` | 加粗 | 表示重要 |
| `<b>` | 加粗 | 无语义 |
| `<em>` | 斜体 | 表示强调 |
| `<i>` | 斜体 | 无语义 |
| `<ins>` | 下划线 | 表示插入 |
| `<u>` | 下划线 | 无语义 |
| `<del>` | 删除线 | 表示删除 |
| `<s>` | 删除线 | 无语义 |

## 最佳实践

1. **优先使用有语义的标签**：`strong` 代替 `b`，`em` 代替 `i`
2. **不要用标签来改变样式**：样式应该用CSS来控制
3. **保持语义清晰**：选择合适的标签表达文本的含义

```html
<!-- 推荐写法 -->
<p>本课程原价 <del>¥299</del>，现价 <strong>¥199</strong></p>
<p><em>注意</em>：报名截止日期为本周五</p>

<!-- 不推荐：只用样式标签 -->
<p>本课程原价 <s>¥299</s>，现价 <b>¥199</b></p>
```

## 总结

文本格式化标签分为两类：
- **有语义**：推荐使用，如 `strong`、`em`、`ins`、`del`
- **无语义**：仅用于视觉样式，如 `b`、`i`、`u`、`s`

实际开发中，样式应该交给CSS，HTML标签应该保持语义清晰。
