# 表单标签（下）

## 按钮标签

### input按钮

```html
<input type="submit" value="提交">
<input type="reset" value="重置">
<input type="button" value="普通按钮">
```

### button按钮（推荐）

```html
<button type="submit">提交</button>
<button type="reset">重置</button>
<button type="button">普通按钮</button>
```

**区别**：
- `input` 是行内块元素
- `button` 默认是行内块元素，且可以在内部包含其他内容

```html
<!-- button可以包含HTML -->
<button type="button">
    <strong>*</strong> 必填项
</button>
```

## 文件上传 file

```html
<input type="file" name="avatar">

<!-- 允许上传多个文件 -->
<input type="file" name="photos" multiple>

<!-- 限制文件类型 -->
<input type="file" name="document" accept=".pdf,.doc,.docx">
<input type="file" name="image" accept="image/*">
<input type="file" name="video" accept="video/*">
```

## 隐藏字段 hidden

```html
<input type="hidden" name="token" value="abc123">
<input type="hidden" name="action" value="delete">
```

用于传递不需要用户看到但需要提交的数据。

## HTML5新增input类型

### email邮箱

```html
<input type="email" name="email" placeholder="请输入邮箱">
```

### number数字

```html
<input type="number" name="age" min="1" max="120" step="1">
```

### tel电话

```html
<input type="tel" name="phone" placeholder="请输入手机号">
```

### url网址

```html
<input type="url" name="website" placeholder="请输入网址">
```

### 日期相关

```html
<input type="date">              <!-- 日期 -->
<input type="time">              <!-- 时间 -->
<input type="datetime-local">     <!-- 本地日期时间 -->
<input type="month">              <!-- 月份 -->
<input type="week">               <!-- 周 -->
```

### range滑动条

```html
<input type="range" name="volume" min="0" max="100" value="50">
```

### color颜色选择

```html
<input type="color" name="theme" value="#ff0000">
```

### search搜索框

```html
<input type="search" name="keyword" placeholder="搜索...">
```

## textarea多行文本

```html
<textarea
    name="description"
    cols="30"
    rows="5"
    placeholder="请输入描述..."
></textarea>
```

### 常用属性

| 属性 | 说明 |
|------|------|
| `cols` | 列数（宽度） |
| `rows` | 行数（高度） |
| `placeholder` | 占位文字 |
| `maxlength` | 最大字符数 |
| `readonly` | 只读 |
| `disabled` | 禁用 |

```html
<textarea name="message" rows="4" maxlength="200"></textarea>
```

## select下拉框

```html
<select name="city">
    <option value="">请选择城市</option>
    <option value="beijing">北京</option>
    <option value="shanghai">上海</option>
    <option value="guangzhou">广州</option>
    <option value="shenzhen">深圳</option>
</select>
```

### selected设置默认选中

```html
<option value="beijing" selected>北京</option>
```

### optgroup分组

```html
<select name="district">
    <optgroup label="北京">
        <option value="chaoyang">朝阳区</option>
        <option value="haidian">海淀区</option>
    </optgroup>
    <optgroup label="上海">
        <option value="pudong">浦东新区</option>
        <option value="huangpu">黄浦区</option>
    </optgroup>
</select>
```

## 表单属性

### required必填

```html
<input type="text" name="username" required>
```

### autofocus自动聚焦

```html
<input type="text" name="username" autofocus>
```

### autocomplete自动完成

```html
<input type="text" name="username" autocomplete="on">
<input type="password" name="password" autocomplete="off">
```

### novalidate禁止验证

```html
<form action="/submit" novalidate>
    <input type="email" name="email">
    <button type="submit">提交</button>
</form>
```

## enctype属性

表单编码类型：

| 值 | 说明 |
|----|------|
| `application/x-www-form-urlencoded` | 默认，编码所有字符 |
| `multipart/form-data` | 用于文件上传 |
| `text/plain` | 纯文本（不编码） |

```html
<!-- 文件上传必须使用multipart/form-data -->
<form action="/upload" method="POST" enctype="multipart/form-data">
    <input type="file" name="file">
    <button type="submit">上传</button>
</form>
```

## 总结

- 按钮推荐使用 `<button>` 标签
- HTML5新增多种input类型提供更好的用户体验
- `textarea` 用于多行文本
- `select` 用于下拉选择
- 注意 `enctype="multipart/form-data"` 用于文件上传
