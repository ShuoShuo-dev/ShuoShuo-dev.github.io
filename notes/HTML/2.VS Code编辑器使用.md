# VS Code编辑器使用

## VS Code简介

VS Code（Visual Studio Code）是微软开发的免费开源代码编辑器，支持Windows、Mac、Linux系统。

## 下载安装

官网地址：https://code.visualstudio.com/

选择对应的系统版本下载安装即可。

## 常用快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + S` | 保存文件 |
| `Ctrl + C` | 复制 |
| `Ctrl + V` | 粘贴 |
| `Ctrl + X` | 剪切 |
| `Ctrl + Z` | 撤销 |
| `Ctrl + Shift + Z` | 反撤销 |
| `Ctrl + A` | 全选 |
| `Ctrl + D` | 选中相同内容 |

## 快速生成HTML骨架

输入 `!` 或 `html:5`，按 `Tab` 键自动生成HTML5模板：

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>

</body>
</html>
```

## 常用插件

1. **Chinese (Simplified)** - 中文语言包
2. **Live Server** - 本地服务器预览
3. **Prettier** - 代码格式化
4. **Auto Rename Tag** - 自动重命名标签

## Live Server使用

1. 安装Live Server插件
2. 右键HTML文件
3. 选择"Open with Live Server"
4. 自动在浏览器中打开并支持热更新

## 总结

熟练使用VS Code和快捷键可以大大提高开发效率。
