# 列表标签

## 列表标签概述

HTML提供了三种类型的列表标签：
- **无序列表**：没有特定顺序
- **有序列表**：有编号顺序
- **自定义列表**：带有标题和描述

## 无序列表 `<ul>`

### 基本语法

```html
<ul>
    <li>列表项1</li>
    <li>列表项2</li>
    <li>列表项3</li>
</ul>
```

### 无序列表特点

1. 列表项前会有项目符号（圆点、方块、圆形等）
2. `<ul>` 内部只能包含 `<li>` 标签
3. `<li>` 内部可以包含任意内容

### type属性（CSS控制更推荐）

```html
<ul type="disc">圆点（默认）</ul>
<ul type="square">方块</ul>
<ul type="circle">空心圆</ul>
<ul type="none">无符号</ul>
```

### 应用场景

```html
<!-- 网站导航 -->
<nav>
    <ul>
        <li><a href="/">首页</a></li>
        <li><a href="/about">关于我们</a></li>
        <li><a href="/products">产品中心</a></li>
        <li><a href="/contact">联系我们</a></li>
    </ul>
</nav>

<!-- 功能列表 -->
<ul>
    <li>用户注册</li>
    <li>订单管理</li>
    <li>支付功能</li>
    <li>物流跟踪</li>
</ul>
```

## 有序列表 `<ol>`

### 基本语法

```html
<ol>
    <li>第一步</li>
    <li>第二步</li>
    <li>第三步</li>
</ol>
```

### 有序列表特点

1. 列表项前会有序号（数字、字母、罗马数字等）
2. `<ol>` 内部只能包含 `<li>` 标签
3. 支持 `start` 和 `reversed` 属性

### type属性

```html
<ol type="1">数字（默认）</ol>
<ol type="A">大写字母</ol>
<ol type="a">小写字母</ol>
<ol type="I">大写罗马数字</ol>
<ol type="i">小写罗马数字</ol>
```

### start属性

```html
<ol start="3">
    <li>第三项</li>
    <li>第四项</li>
    <li>第五项</li>
</ol>
```

### reversed属性

```html
<ol reversed>
    <li>倒数第一</li>
    <li>倒数第二</li>
    <li>倒数第三</li>
</ol>
```

### 应用场景

```html
<!-- 步骤说明 -->
<h3>注册步骤</h3>
<ol>
    <li>填写基本信息</li>
    <li>验证手机号码</li>
    <li>设置登录密码</li>
    <li>注册成功</li>
</ol>

<!-- 排行榜 -->
<ol>
    <li>第一名：张三</li>
    <li>第二名：李四</li>
    <li>第三名：王五</li>
</ol>
```

## 自定义列表 `<dl>`

### 基本语法

```html
<dl>
    <dt>术语标题</dt>
    <dd>术语描述1</dd>
    <dd>术语描述2</dd>
</dl>
```

### 标签说明

- `<dl>`：定义列表容器
- `<dt>`：定义术语/标题
- `<dd>`：定义描述

### 应用场景

```html
<!-- 名词解释 -->
<dl>
    <dt>HTML</dt>
    <dd>超文本标记语言，用于构建网页结构</dd>

    <dt>CSS</dt>
    <dd>层叠样式表，用于控制网页外观</dd>

    <dt>JavaScript</dt>
    <dd>一种脚本语言，用于实现网页交互</dd>
</dl>

<!-- 联系我们 -->
<dl>
    <dt>地址</dt>
    <dd>北京市朝阳区某某路123号</dd>

    <dt>电话</dt>
    <dd>400-888-8888</dd>

    <dt>邮箱</dt>
    <dd>contact@example.com</dd>
</dl>
```

## 列表嵌套

列表可以无限嵌套，但通常建议不超过三层：

```html
<ul>
    <li>水果
        <ul>
            <li>苹果
                <ul>
                    <li>红富士</li>
                    <li>青苹果</li>
                </ul>
            </li>
            <li>香蕉</li>
            <li>橙子</li>
        </ul>
    </li>
    <li>蔬菜
        <ul>
            <li>白菜</li>
            <li>萝卜</li>
        </ul>
    </li>
</ul>
```

## 完整示例

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>列表标签示例</title>
</head>
<body>
    <h1>前端技术栈</h1>

    <h2>无序列表 - 常用框架</h2>
    <ul>
        <li>Vue.js</li>
        <li>React</li>
        <li>Angular</li>
    </ul>

    <h2>有序列表 - 学习步骤</h2>
    <ol>
        <li>学习HTML基础</li>
        <li>学习CSS样式</li>
        <li>学习JavaScript</li>
        <li>学习前端框架</li>
    </ol>

    <h2>自定义列表 - 技术名词</h2>
    <dl>
        <dt>DOM</dt>
        <dd>文档对象模型，是HTML和XML文档的编程接口</dd>

        <dt>BOM</dt>
        <dd>浏览器对象模型，提供了与浏览器交互的对象</dd>

        <dt>AJAX</dt>
        <dd>异步JavaScript和XML，用于异步数据交换</dd>
    </dl>
</body>
</html>
```

## CSS去除列表样式

```css
/* 去除默认样式 */
ul, ol {
    list-style: none;
    padding: 0;
    margin: 0;
}

/* 横向菜单 */
.nav ul {
    display: flex;
    list-style: none;
}

.nav li {
    margin-right: 20px;
}
```

## 总结

| 类型 | 标签 | 用途 |
|------|------|------|
| 无序列表 | `<ul>` + `<li>` | 用于没有顺序的列表项 |
| 有序列表 | `<ol>` + `<li>` | 用于有顺序/步骤的列表项 |
| 自定义列表 | `<dl>` + `<dt>` + `<dd>` | 用于名词解释、术语表 |
