# HTML实体与编码

## 一、什么是HTML实体

HTML实体是在HTML中表示特殊字符的字符串，以 `&` 开头，以 `;` 结尾。

### 1.1 为什么需要HTML实体

```html
<!-- ❌ 错误：浏览器会把 < 和 > 解析为标签 -->
<p>5 < 10</p>
<p><div></p>

<!-- ✅ 正确：使用HTML实体 -->
<p>5 &lt; 10</p>
<p>&lt;div&gt;</p>
```

### 1.2 HTML实体分为两种

| 类型 | 格式 | 示例 |
|------|------|------|
| 命名实体 | `&name;` | `&lt;` `&gt;` `&amp;` |
| 数字实体 | `&#number;` | `&#60;` `&#62;` |

## 三、HTML字符编码

### 3.1 常见的字符编码

| 编码 | 说明 | 特点 |
|------|------|------|
| ASCII | 美国标准信息码 | 7位，128个字符 |
| ISO-8859-1 | Latin-1 | 8位，256个字符，西欧语言 |
| GB2312 | 中文简体 | 简体中文 |
| GBK | 中文扩展 | 简体中文+繁体中文 |
| **UTF-8** | 万国码 | 1-4字节，推荐使用 |
| UTF-16 | Unicode | 2或4字节 |

### 3.2 在HTML中指定字符编码

```html
<!-- 方式1：meta标签（推荐） -->
<meta charset="UTF-8">

<!-- 方式2：HTTP响应头 -->
<!-- Content-Type: text/html; charset=UTF-8 -->

<!-- 方式3：HTML5简写 -->
<meta charset="utf-8">
```

### 3.3 编码声明的位置

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <!-- 必须在前512字节内声明 -->
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>页面标题</title>
</head>
```

## 四、Unicode字符

### 4.1 使用Unicode

```html
<!-- 直接使用Unicode码点 -->
<p>版权符号：&#169; 或 &#x00A9;</p>
<p>心形：&#10084; 或 &#x2764;</p>
<p>emoji：&#128513; 或 &#x1F601;</p>
```

### 4.2 常用Unicode范围

| 范围 | 说明 |
|------|------|
| `\u0000-\u007F` | ASCII |
| `\u0080-\u00FF` | Latin-1 扩展 |
| `\u4E00-\u9FFF` | 中文常用字 |
| `\u3040-\u309F` | 日文平假名 |
| `\u30A0-\u30FF` | 日文片假名 |
| `\uAC00-\uD7AF` | 韩文音节 |

## 五、实际应用

### 5.1 代码展示

```html
<p>要在HTML中显示代码，使用&lt;code&gt;标签</p>
<pre><code>&lt;div class="container"&gt;
    &lt;p&gt;Hello World!&lt;/p&gt;
&lt;/div&gt;</code></pre>
```

### 5.2 空白字符

```html
<!-- 多个空格只显示一个 -->
<p>这   是   很   多   空   格</p>

<!-- 使用&nbsp;保持空格 -->
<p>公&nbsp;&nbsp;司&nbsp;&nbsp;名&nbsp;&nbsp;称</p>

<!-- 使用<pre>保持原始格式 -->
<pre>
function hello() {
    console.log('Hello');
}
</pre>
```

### 5.3 显示HTML标签

```html
<!-- 方法1：使用 &lt; 和 &gt; -->
<p>使用 &lt;strong&gt; 标签加粗文字</p>

<!-- 方法2：使用 <code> 和实体 -->
<p>使用 <code>&lt;strong&gt;</code> 标签加粗文字</p>

<!-- 方法3：使用 textarea 只读 -->
<textarea readonly>
<div>不会被解析的HTML</div>
</textarea>
```

## 六、注意事项

1. **HTML实体区分大小写**：`&lt;` 正确，`&LT;` 可能不支持
2. **数字实体建议使用10进制**：`&#60;` 更通用
3. **推荐使用UTF-8编码**，可以正确显示世界上几乎所有字符
4. **HTML5默认编码是UTF-8**
5. **XHTML中必须使用实体**：`&` 必须写成 `&amp;`
