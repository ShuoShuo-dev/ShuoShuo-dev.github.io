# HTML规范与DOCTYPE

## DOCTYPE声明

DOCTYPE（Document Type Declaration）声明告诉浏览器文档的类型和版本。

### HTML5声明

```html
<!DOCTYPE html>
```

这是现代网页的标准写法，简洁明了。

### HTML4.01声明

```html
<!-- 严格型：不含弃用元素 -->
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">

<!-- 过渡型：含弃用元素 -->
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">

<!-- 框架型：含框架集 -->
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Frameset//EN" "http://www.w3.org/TR/html4/frameset.dtd">
```

### XHTML声明

```html
<!-- XHTML 1.0 -->
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-loose.dtd">

<!-- XHTML 1.1 -->
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.1//EN" "http://www.w3.org/MarkUp/DTD/xhtml11.dtd">
```

## 渲染模式

### 标准模式（Standards Mode）

```html
<!DOCTYPE html>
<html>
```

浏览器按照W3C标准渲染页面。

### 混杂模式（Quirks Mode）

没有DOCTYPE或DOCTYPE不正确时，浏览器使用混杂模式，模拟旧版本浏览器的行为。

### 差异对比

| 特性 | 标准模式 | 混杂模式 |
|------|----------|----------|
| 盒模型 | W3C标准 | IE旧版本 |
| width/height | content | content+padding+border |
| 行内元素宽高 | 不能设置 | 可以设置 |
| 图片对齐 | 基线对齐 | 底部对齐 |
| 字体继承 | inherit | 不同浏览器不同 |

## HTML规范

### 1. 基本结构

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>页面标题</title>
</head>
<body>
</body>
</html>
```

### 2. 标签闭合

```html
<!-- 双标签：必须闭合 -->
<p>段落</p>
<div>容器</div>
<span>文本</span>

<!-- 单标签（自闭合）：建议加上斜杠 -->
<br>
<br/>
<hr>
<hr/>
<img src="image.jpg">
<img src="image.jpg" />
```

### 3. 标签嵌套规则

```html
<!-- 正确：块级元素包含块级或行内元素 -->
<div>
    <p>段落</p>
    <span>文本</span>
</div>

<!-- 错误：行内元素不能包含块级元素 -->
<span>
    <div>错误</div>
</span>

<!-- 错误：p标签不能包含块级元素 -->
<p>
    <div>错误</div>
</p>
```

### 4. 属性规范

```html
<!-- 属性值用引号包裹 -->
<input type="text">
<input type='text'>

<!-- 多单词属性值 -->
<input type="text" placeholder="请输入">

<!-- 布尔属性 -->
<input type="text" disabled>
<input type="text" disabled="disabled">
```

### 5. 命名规范

```html
<!-- 推荐：小写字母 -->
<div class="header-content">
    <nav id="main-nav">
    </nav>
</div>

<!-- 不推荐：大小写混用 -->
<div class="HeaderContent">
</div>

<!-- 不推荐：中文（某些情况除外） -->
<div class="头部">
</div>
```

### 6. lang属性

```html
<!-- 必须设置lang属性 -->
<html lang="zh-CN">

<!-- 可选值 -->
<html lang="zh">          <!-- 中文 -->
<html lang="zh-TW">       <!-- 繁体中文 -->
<html lang="en">          <!-- 英文 -->
<html lang="ja">          <!-- 日语 -->
<html lang="ko">          <!-- 韩语 -->
```

### 常见错误

```html
<!-- 错误1：缺少DOCTYPE -->
<html>
<head>
    <title>页面</title>
</head>
<body>
</body>
</html>

<!-- 错误2：标签未闭合 -->
<div>
    <p>未闭合的段落</div>

<!-- 错误3：错误的属性 -->
<input type="text" disabled="true">  <!-- 应为disabled或disabled="disabled" -->

<!-- 错误4：使用了弃用标签 -->
<center>居中内容</center>
<font size="5">字体</font>
<marquee>滚动文字</marquee>

<!-- 错误5：img缺少alt属性 -->
<img src="photo.jpg">
```

## HTML最佳实践

### 1. 使用语义化标签

```html
<!-- 推荐 -->
<header>导航</header>
<main>
    <article>
        <section>内容</section>
    </article>
    <aside>侧边栏</aside>
</main>
<footer>底部</footer>

<!-- 不推荐 -->
<div class="header">导航</div>
<div class="main">
    <div class="content">内容</div>
</div>
```

### 2. 结构与表现分离

```html
<!-- 推荐：样式用CSS -->
<div class="highlight">重要内容</div>

<!-- 不推荐：用标签属性 -->
<div align="center">居中内容</div>
<font color="red">红色文字</font>
```

### 3. 可访问性

```html
<!-- 图片添加alt -->
<img src="logo.png" alt="网站Logo">

<!-- 表单关联label -->
<label for="username">用户名：</label>
<input type="text" id="username">

<!-- 语义化链接 -->
<a href="article.html">阅读"如何学习HTML"</a>
```

### 4. 性能优化

```html
<!-- CSS放在head -->
<head>
    <link rel="stylesheet" href="style.css">
</head>

<!-- JS放在body末尾 -->
<body>
    <script src="main.js"></script>
</body>

<!-- 图片指定尺寸 -->
<img src="photo.jpg" width="300" height="200" alt="描述">

<!-- 使用懒加载 -->
<img src="photo.jpg" loading="lazy" alt="描述">
```

## 完整HTML5模板

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">

    <!-- SEO -->
    <title>页面标题 - 网站名称</title>
    <meta name="description" content="页面描述">
    <meta name="keywords" content="关键词1, 关键词2">
    <meta name="author" content="作者">

    <!-- 图标 -->
    <link rel="icon" href="favicon.ico">

    <!-- 样式 -->
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <!-- 内容 -->
    <script src="main.js"></script>
</body>
</html>
```

## 常见问题

### 1. 页面显示混乱

可能原因：
- 缺少 `<!DOCTYPE html>`
- 缺少字符编码声明
- 标签未正确闭合

### 2. 样式不生效

可能原因：
- CSS文件路径错误
- 选择器优先级问题
- CSS语法错误

### 3. 移动端显示异常

可能原因：
- 缺少 `<meta name="viewport">`
- 视口设置不正确

## 总结

- `<!DOCTYPE html>` 是HTML5的标准声明
- DOCTYPE影响浏览器渲染模式
- 遵循HTML规范：标签闭合、嵌套正确、属性规范
- 保持语义化、结构与表现分离
- 使用W3C验证器检查代码
- 注意可访问性和性能优化
