# head头部内容

## head标签简介

`<head>` 标签是HTML文档的头部，包含了文档的元数据，不会直接显示在页面上。

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <!-- 元数据标签 -->
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>页面标题</title>
    <link rel="icon" href="favicon.ico">
</head>
<body>
    <!-- 页面内容 -->
</body>
</html>
```

## head中可包含的元素

| 标签 | 说明 |
|------|------|
| `<title>` | 页面标题（必需） |
| `<meta>` | 元数据 |
| `<link>` | 外部资源链接 |
| `<style>` | 内联样式 |
| `<script>` | 脚本代码 |
| `<base>` | 基准URL |
| `<noscript>` | 无脚本时的内容 |

## title标签

页面标题，必需元素。

```html
<head>
    <title>页面标题 - 网站名称</title>
</head>
```

### title的最佳实践

```html
<!-- 推荐：清晰的标题 -->
<title>HTML教程 - Web前端学习网</title>

<!-- 不推荐：过于简单 -->
<title>首页</title>

<!-- 不推荐：所有页面相同 -->
<title>我的网站</title>
```

### SEO优化

```html
<!-- 首页 -->
<title>网站名称 - 网站描述</title>

<!-- 内页 -->
<title>文章标题 - 网站名称</title>

<!-- 产品页 -->
<title>产品名称 - 分类 - 网站名称</title>
```

## base标签

定义页面上所有链接的基准URL。

```html
<head>
    <base href="https://example.com/" target="_blank">
</head>
<body>
    <a href="about.html">关于我们</a>
    <!-- 实际链接：https://example.com/about.html -->
</body>
```

### base标签属性

| 属性 | 说明 |
|------|------|
| `href` | 基准URL |
| `target` | 默认打开方式 |

### 注意事项

- 一个页面只能有一个 `<base>` 标签
- 应放在其他含URL的元素之前
- 最好放在 `<head>` 最前面

## style标签

定义内联CSS样式。

```html
<head>
    <style>
        body {
            font-family: Arial, sans-serif;
            margin: 0;
        }
        h1 {
            color: #333;
        }
    </style>
</head>
```

### type属性

```html
<!-- 默认，可以省略 -->
<style type="text/css">
    /* CSS代码 */
</style>
```

### media属性

```html
<style media="print">
    /* 打印样式 */
    body { font-size: 12pt; }
    nav, footer { display: none; }
</style>
```

## script标签

引入外部脚本或定义内联脚本。

### 外部脚本

```html
<head>
    <!-- 同步加载（阻塞渲染） -->
    <script src="jquery.js"></script>

    <!-- 异步加载（不阻塞） -->
    <script src="analytics.js" async></script>

    <!-- 延迟执行 -->
    <script src="plugin.js" defer></script>
</head>
```

### async vs defer

| 特性 | 无属性 | async | defer |
|------|--------|-------|-------|
| 加载时机 | 立即加载执行 | 立即加载，完成后执行 | 加载后等待HTML解析完成 |
| 执行时机 | HTML解析暂停 | 下载完成即执行 | HTML解析完成后执行 |
| 执行顺序 | 按出现顺序 | 按下载完成顺序 | 按出现顺序 |

```html
<head>
    <!-- 推荐：保持顺序又不阻塞 -->
    <script src="plugin.js" defer></script>
</head>
```

### 内联脚本

```html
<head>
    <script>
        // JavaScript代码
        console.log('页面加载中...');
    </script>
</head>
```

## noscript标签

当浏览器不支持或禁用JavaScript时显示的内容。

```html
<head>
    <noscript>
        <meta http-equiv="refresh" content="0;url=nojs.html">
    </noscript>
</head>
<body>
    <noscript>
        <p>您的浏览器不支持JavaScript或已禁用，请启用JavaScript或升级浏览器。</p>
    </noscript>

    <!-- JavaScript可用的内容 -->
    <script>
        document.write('JavaScript已启用');
    </script>
</body>
```

## link标签的常见用法

### 1. 引入CSS

```html
<link rel="stylesheet" href="style.css">
```

### 2. 引入图标

```html
<link rel="icon" href="favicon.ico" type="image/x-icon">
<link rel="shortcut icon" href="favicon.ico">
<link rel="apple-touch-icon" href="apple-touch-icon.png">
```

### 3. 预连接优化

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="dns-prefetch" href="https://cdn.example.com">
```

### 4. RSS订阅

```html
<link rel="alternate" type="application/rss+xml" href="feed.xml" title="RSS Feed">
```

## meta标签详解

### 1. 字符编码

```html
<meta charset="UTF-8">
```

### 2. 视口设置

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

### 3. SEO标签

```html
<meta name="description" content="页面描述，150字以内">
<meta name="keywords" content="关键词1, 关键词2">
<meta name="author" content="作者名称">
<meta name="robots" content="index, follow">
```

### 4. 浏览器兼容性

```html
<meta http-equiv="X-UA-Compatible" content="IE=edge,chrome=1">
```

### 5. 移动端优化

```html
<meta name="mobile-web-app-capable" content="yes">
<meta name="apple-mobile-web-app-capable" content="yes">
<meta name="apple-mobile-web-app-status-bar-style" content="black">
<meta name="theme-color" content="#4CAF50">
```

### 6. 社交分享

```html
<!-- Open Graph (Facebook) -->
<meta property="og:title" content="标题">
<meta property="og:description" content="描述">
<meta property="og:image" content="图片URL">
<meta property="og:url" content="页面URL">

<!-- Twitter Cards -->
<meta name="twitter:card" content="summary">
<meta name="twitter:title" content="标题">
<meta name="twitter:description" content="描述">
<meta name="twitter:image" content="图片URL">
```

## 完整head示例

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <!-- 基础设置 -->
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">

    <!-- SEO -->
    <title>页面标题 - 网站名称</title>
    <meta name="description" content="页面描述">
    <meta name="keywords" content="关键词1, 关键词2">
    <meta name="author" content="作者">
    <meta name="robots" content="index, follow">

    <!-- 移动端 -->
    <meta name="theme-color" content="#ffffff">
    <meta name="apple-mobile-web-app-capable" content="yes">
    <meta name="apple-mobile-web-app-status-bar-style" content="black">

    <!-- 图标 -->
    <link rel="icon" href="favicon.ico">
    <link rel="apple-touch-icon" href="apple-touch-icon.png">

    <!-- 性能优化 -->
    <link rel="preconnect" href="https://fonts.googleapis.com">
    <link rel="dns-prefetch" href="https://cdn.example.com">

    <!-- 样式 -->
    <link rel="stylesheet" href="css/reset.css">
    <link rel="stylesheet" href="css/style.css">
    <style>
        /* 内联样式（少量时） */
    </style>

    <!-- 脚本（推荐放在body末尾，但meta等head元素可在此） -->
</head>
<body>
    <!-- 页面内容 -->
</body>
</html>
```

## 最佳实践

1. **始终指定字符编码**：`<meta charset="UTF-8">`
2. **视口设置**：`<meta name="viewport">` 对移动端必需
3. **合理的title**：每个页面有独特的标题
4. **添加favicon**：提升网站专业度
5. **样式放head**：CSS在head中加载
6. **脚本放底部或加defer**：避免阻塞页面渲染

## 总结

- `<head>` 包含页面的元数据
- `<title>` 是必需元素，影响SEO
- `<meta>` 提供各种元信息
- `<link>` 链接外部资源
- `<script>` 引入脚本，注意加载时机
- `<base>` 设置基准URL（谨慎使用）
- `<noscript>` 提供无JS时的降级方案
