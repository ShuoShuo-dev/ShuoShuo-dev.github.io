# link标签

## link标签简介

`<link>` 标签用于链接外部资源到HTML文档，最常用于引入CSS文件。

```html
<head>
    <link rel="stylesheet" href="style.css">
</head>
```

## rel属性

定义当前文档与被链接文档的关系。

### stylesheet样式表

```html
<link rel="stylesheet" href="style.css">
```

### icon图标

```html
<link rel="icon" href="favicon.ico" type="image/x-icon">
<link rel="icon" href="icon.png" type="image/png">

<!-- Apple设备图标 -->
<link rel="apple-touch-icon" href="apple-touch-icon.png">
```

### alternate备选版本

```html
<!-- RSS订阅 -->
<link rel="alternate" type="application/rss+xml" href="feed.xml">

<!-- 打印样式 -->
<link rel="stylesheet" href="print.css" media="print">
```

### preconnect预连接

```html
<!-- 预连接到字体服务器 -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
```

### prefetch预取

```html
<!-- 预取下一个页面需要的资源 -->
<link rel="prefetch" href="next-page.html">

<!-- 预加载CSS -->
<link rel="preload" href="styles.css" as="style">

<!-- 预加载字体 -->
<link rel="preload" href="font.woff2" as="font" type="font/woff2" crossorigin>
```

## href属性

指定被链接资源的地址。

```html
<link rel="stylesheet" href="./css/style.css">
<link rel="stylesheet" href="https://cdn.example.com/style.css">
```

## type属性

指定被链接资源的MIME类型。

```html
<link rel="stylesheet" href="style.css" type="text/css">
<link rel="icon" href="favicon.ico" type="image/x-icon">
```

## media属性

指定资源生效的媒体类型。

```html
<!-- 屏幕设备 -->
<link rel="stylesheet" href="screen.css" media="screen">

<!-- 打印设备 -->
<link rel="stylesheet" href="print.css" media="print">

<!-- 所有设备 -->
<link rel="stylesheet" href="all.css" media="all">

<!-- 最大宽度768px -->
<link rel="stylesheet" href="tablet.css" media="screen and (max-width: 768px)">

<!-- 最小宽度769px -->
<link rel="stylesheet" href="desktop.css" media="screen and (min-width: 769px)">
```

### 常用媒体查询值

| 值 | 说明 |
|----|------|
| `all` | 所有设备 |
| `screen` | 电脑屏幕 |
| `print` | 打印预览 |
| `speech` | 语音合成器 |
| `braille` | 盲文触觉设备 |
| `handheld` | 手持设备 |

### 媒体查询语法

```html
<!-- 横屏 -->
<link rel="stylesheet" href="landscape.css" media="screen and (orientation: landscape)">

<!-- 竖屏 -->
<link rel="stylesheet" href="portrait.css" media="screen and (orientation: portrait)">

<!-- 彩色屏幕 -->
<link rel="stylesheet" href="color.css" media="screen and (color)">

<!-- 高分辨率屏幕 -->
<link rel="stylesheet" href="retina.css" media="screen and (-webkit-min-device-pixel-ratio: 2)">
```

## crossorigin属性

跨域资源共享设置。

```html
<!-- 匿名请求 -->
<link rel="preload" href="font.woff2" as="font" type="font/woff2" crossorigin>

<!-- 带凭据的请求 -->
<link rel="preload" href="https://example.com/resource" crossorigin="use-credentials">
```

## 常用场景

### 1. 引入CSS文件

```html
<head>
    <link rel="stylesheet" href="css/reset.css">
    <link rel="stylesheet" href="css/base.css">
    <link rel="stylesheet" href="css/style.css">
</head>
```

### 2. 引入网站图标

```html
<head>
    <link rel="icon" href="favicon.ico" sizes="any">
    <link rel="icon" href="icon.svg" type="image/svg+xml">
</head>
```

### 3. 引入字体

```html
<head>
    <!-- Google Fonts -->
    <link rel="preconnect" href="https://fonts.googleapis.com">
    <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
    <link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet">
</head>
```

### 4. 性能优化

```html
<head>
    <!-- DNS预解析 -->
    <link rel="dns-prefetch" href="//cdn.example.com">

    <!-- 预连接 -->
    <link rel="preconnect" href="https://cdn.example.com" crossorigin>

    <!-- 预加载关键资源 -->
    <link rel="preload" href="main.css" as="style">
    <link rel="preload" href="hero.webp" as="image">

    <!-- 预取下一页资源 -->
    <link rel="prefetch" href="next-page.html">

    <!-- 预渲染页面 -->
    <link rel="prerender" href="next-page.html">
</head>
```

### 5. RSS订阅

```html
<head>
    <link rel="alternate" type="application/rss+xml" title="博客订阅" href="/feed.xml">
</head>
```

## 完整示例

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">

    <!-- 基础样式 -->
    <link rel="stylesheet" href="css/reset.css">
    <link rel="stylesheet" href="css/base.css">

    <!-- 响应式样式 -->
    <link rel="stylesheet" href="css/tablet.css" media="screen and (max-width: 768px)">
    <link rel="stylesheet" href="css/mobile.css" media="screen and (max-width: 480px)">

    <!-- 打印样式 -->
    <link rel="stylesheet" href="css/print.css" media="print">

    <!-- 图标 -->
    <link rel="icon" href="favicon.ico">
    <link rel="apple-touch-icon" href="apple-touch-icon.png">

    <!-- 字体 -->
    <link rel="preconnect" href="https://fonts.googleapis.com">
    <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
    <link href="https://fonts.googleapis.com/css2?family=Noto+Sans+SC:wght@400;700&display=swap" rel="stylesheet">

    <!-- 性能优化 -->
    <link rel="dns-prefetch" href="//cdn.example.com">
    <link rel="preload" href="fonts/custom.woff2" as="font" type="font/woff2" crossorigin>

    <title>页面标题</title>
</head>
<body>
    <!-- 内容 -->
</body>
</html>
```

## link vs @import

| 特性 | link | @import |
|------|------|---------|
| 加载方式 | 并行加载 | 串行加载 |
| 加载时机 | HTML解析时加载 | CSS解析时加载 |
| 兼容性 | 所有浏览器 | IE5+ |
| DOM控制 | 可通过JS控制 | 不能通过JS控制 |
| 权重 | 较低 | 较高 |

```html
<!-- 推荐：使用link -->
<link rel="stylesheet" href="style.css">

<!-- 不推荐：使用@import -->
<style>
    @import url("style.css");
</style>
```

## 总结

- `<link>` 是单标签，放在 `<head>` 中
- `rel` 属性定义链接类型，最常用 `stylesheet`
- `href` 指定资源地址
- `media` 属性设置响应式条件
- 用于引入CSS、图标、字体等多种资源
- 性能优化：使用 `preload`、`prefetch`、`preconnect`
