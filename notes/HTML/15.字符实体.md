# 字符实体

## 什么是字符实体

字符实体是用于在HTML中显示特殊字符的代码。因为HTML中某些字符有特殊含义，不能直接使用。

## 字符实体语法

```html
&实体名称;      <!-- 如 &lt; -->
&#实体编号;     <!-- 如 &#60; -->
```

## 常用字符实体

### 空格

| 实体 | 说明 | 示例 |
|------|------|------|
| `&nbsp;` | 不断行空格 | 单词&nbsp;之间 |
| `&ensp;` | 半角空格（EN空格） | 单词&ensp;之间 |
| `&emsp;` | 全角空格（EM空格） | 单词&emsp;之间 |

```html
<!-- 普通空格会被合并 -->
<p>单词    单词</p>

<!-- 不间断空格保持间距 -->
<p>单词&nbsp;&nbsp;&nbsp;&nbsp;单词</p>

<!-- 不同宽度的空格 -->
<p>单词&ensp;单词&empsp;单词</p>
```

## 实用场景

### 显示HTML代码

```html
<p>如何在HTML中显示&lt;p&gt;标签：</p>
<pre>
&lt;p&gt;这是一个段落&lt;/p&gt;
</pre>
```

### 数学表达式

```html
<p>公式：a &lt; b &amp;&amp; c &gt; d</p>
<p>计算：5 &times; 6 = 30</p>
<p>除法：20 &divide; 4 = 5</p>
```

### 价格显示

```html
<p>商品价格：&yen;199.00</p>
<p>原价：&yen;<s>299</s></p>
```

### 版权信息

```html
<footer>
    <p>&copy; 2024 公司名称. All rights reserved.</p>
</footer>
```

## 完整示例

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>字符实体示例</title>
</head>
<body>
    <h1>HTML字符实体</h1>

    <h2>特殊符号</h2>
    <ul>
        <li>小于号 &lt; &amp; 大于号 &gt;</li>
        <li>版权符号 &copy;</li>
        <li>注册商标 &reg;</li>
        <li>商标符号 &trade;</li>
    </ul>

    <h2>货币符号</h2>
    <ul>
        <li>美元：&dollar;100</li>
        <li>欧元：&euro;100</li>
        <li>英镑：&pound;100</li>
        <li>人民币/日元：&yen;100</li>
    </ul>

    <h2>数学符号</h2>
    <ul>
        <li>乘号：5 &times; 6 = 30</li>
        <li>除号：20 &divide; 4 = 5</li>
        <li>加减号：&plusmn;</li>
        <li>平方根：&radic;</li>
        <li>无穷大：&infin;</li>
    </ul>

    <h2>箭头和图形</h2>
    <ul>
        <li>箭头：&larr; &uarr; &rarr; &darr;</li>
        <li>扑克牌：&spades; &hearts; &diams; &clubs;</li>
    </ul>

    <h2>代码示例</h2>
    <pre>
if (a &lt; b &amp;&amp; c &gt; d) {
    console.log(&quot;Hello World!&quot;);
}
    </pre>
</body>
</html>
```

## 注意事项

1. **实体名称区分大小写**：`&nbsp;` 而非 `&NBSP;`
2. **实体后要加分号**：`&lt;` 而非 `&lt`
3. **使用实体编号更可靠**：因为有些浏览器不支持某些实体名称
4. **字符编码**：现代HTML推荐使用UTF-8编码，可以直接使用Unicode字符

## 总结

- 字符实体以 `&` 开头，`;` 结尾
- `&lt;` 显示 `<`，`&gt;` 显示 `>`
- `&amp;` 显示 `&`（常用于链接和属性中）
- `&nbsp;` 显示不间断空格
- `&copy;` 显示版权符号
