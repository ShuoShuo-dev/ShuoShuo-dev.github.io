# 段落和换行标签

## 段落标签 `<p>`

段落标签用于定义文本的段落，是HTML中最常用的标签之一。

```html
<p>这是一个段落，用于表达一段完整的意思。</p>
<p>这是另一个段落，与上面的段落相互独立。</p>
```

### 段落标签特点

1. **语义化**：表示一个独立的段落
2. **块级元素**：独占一行
3. **上下外边距**：段落之间会有一定的间距
4. **自动换行**：根据容器宽度自动换行

### 示例

```html
<body>
    <h1>欢迎来到我的博客</h1>
    <p>这是一篇关于HTML基础知识的文章。HTML是超文本标记语言，用于构建网页的结构和内容。</p>
    <p>学习HTML是Web开发的第一步。掌握好基础知识，才能更好地学习CSS和JavaScript。</p>
</body>
```

## 换行标签 `<br>`

换行标签用于在文本中强制换行，是单标签。

```html
<p>第一行文字<br>第二行文字<br>第三行文字</p>
```

### 换行标签特点

1. **单标签**：不需要闭合标签
2. **简单换行**：只是简单地换到下一行
3. **不产生间距**：不像p标签那样有上下间距

### 示例

```html
<p>
    床前明月光，<br>
    疑是地上霜。<br>
    举头望明月，<br>
    低头思故乡。
</p>
```

## p标签和br标签的区别

| 特性 | `<p>` | `<br>` |
|------|-------|--------|
| 标签类型 | 双标签 | 单标签 |
| 是否独占一行 | 是 | 否（仅换行） |
| 上下间距 | 有 | 无 |
| 语义 | 段落 | 换行 |

## 错误写法示例

```html
<!-- 错误：用多个br模拟段落 -->
<p>第一段文字<br><br><br>第二段文字</p>

<!-- 错误：在p标签内嵌套块级元素 -->
<p>
    <div>这是不规范的写法</div>
</p>
```

## 实际应用

```html
<article>
    <h1>如何学习HTML</h1>
    <p>HTML是网页开发的基础语言。学习HTML需要掌握各种标签的用法和语义。</p>
    <p>首先，要理解HTML的基本结构：doctype声明、html根元素、head头部和body主体。</p>
    <p>其次，要熟悉常用的标签：<br>
       - 标题标签：h1到h6<br>
       - 段落标签：p<br>
       - 换行标签：br<br>
       - 文本格式化标签等
    </p>
</article>
```

## 总结

- `<p>` 标签用于定义段落，表示一段完整的内容
- `<br>` 标签用于强制换行，适合诗歌、地址等需要换行但不分段的情况
- 不要用多个 `<br>` 来代替 `<p>` 段落标签
