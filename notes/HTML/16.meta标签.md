# meta标签

## meta标签简介

`<meta>` 标签用于提供HTML文档的元数据，不会显示在页面上，但对搜索引擎和浏览器很重要。

```html
<head>
    <meta charset="UTF-8">
    <meta name="description" content="页面描述">
    <meta name="keywords" content="关键词">
    <title>页面标题</title>
</head>
```

## charset字符编码

```html
<meta charset="UTF-8">
```

常用编码：
- `UTF-8`：支持全球所有语言字符（推荐）
- `GB2312`：简体中文
- `GBK`：简体中文扩展
- `ISO-8859-1`：拉丁字母字符

## name属性

### description页面描述

```html
<meta name="description" content="这是一个关于Web开发的博客，分享HTML、CSS、JavaScript技术文章。">
```

### keywords关键词

```html
<meta name="keywords" content="HTML, CSS, JavaScript, Web开发, 前端">
```

### author作者

```html
<meta name="author" content="张三">
```

### robots搜索引擎

```html
<meta name="robots" content="index,follow">        <!-- 允许抓取和追踪 -->
<meta name="robots" content="noindex,nofollow">    <!-- 禁止抓取和追踪 -->
<meta name="robots" content="index,nofollow">      <!-- 允许抓取，禁止追踪 -->
<meta name="robots" content="noindex,follow">     <!-- 禁止抓取，允许追踪 -->
```

### viewport视口

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

常用属性值：

| 属性值 | 说明 |
|--------|------|
| `width=device-width` | 视口宽度等于设备宽度 |
| `initial-scale=1.0` | 初始缩放比例 |
| `maximum-scale=1.0` | 最大缩放比例 |
| `minimum-scale=1.0` | 最小缩放比例 |
| `user-scalable=no` | 禁止用户缩放 |

```html
<!-- 移动端常用 -->
<meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=no">
```

### format-detection格式检测

```html
<meta name="format-detection" content="telephone=no">      <!-- 禁止识别手机号 -->
<meta name="format-detection" content="email=no">          <!-- 禁止识别邮箱 -->
```

### renderer渲染引擎

```html
<meta name="renderer" content="webkit">              <!-- 优先webkit内核 -->
<meta name="renderer" content="ie-comp">             <!-- 优先IE兼容模式 -->
<meta name="renderer" content="ie-stand">             <!-- 优先IE标准模式 -->
```

## http-equiv属性

### X-UA-Compatible浏览器兼容性

```html
<meta http-equiv="X-UA-Compatible" content="IE=edge,chrome=1">
```

### expires过期时间

```html
<meta http-equiv="expires" content="Wed, 20 Jun 2024 12:00:00 GMT">
```

### pragma缓存控制

```html
<meta http-equiv="pragma" content="no-cache">  <!-- 禁用缓存 -->
```

### refresh页面刷新

```html
<meta http-equiv="refresh" content="30">                    <!-- 30秒后刷新 -->
<meta http-equiv="refresh" content="5;url=https://example.com"> <!-- 5秒后跳转到其他页面 -->
```

### content-type文档类型

```html
<meta http-equiv="content-type" content="text/html; charset=UTF-8">
```

## SEO相关meta标签

### 完整SEO配置示例

```html
<head>
    <!-- 字符编码 -->
    <meta charset="UTF-8">

    <!-- 视口设置 -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0">

    <!-- SEO基础 -->
    <title>页面标题 - 网站名称</title>
    <meta name="description" content="页面描述，150字以内">
    <meta name="keywords" content="关键词1, 关键词2, 关键词3">
    <meta name="author" content="作者名称">

    <!-- 搜索引擎 -->
    <meta name="robots" content="index, follow">

    <!-- 浏览器 -->
    <meta http-equiv="X-UA-Compatible" content="IE=edge,chrome=1">

    <!-- 移动端 -->
    <meta name="theme-color" content="#ffffff">
    <meta name="apple-mobile-web-app-capable" content="yes">
    <meta name="apple-mobile-web-app-status-bar-style" content="black">

    <!-- 社交分享 -->
    <meta property="og:title" content="分享标题">
    <meta property="og:description" content="分享描述">
    <meta property="og:image" content="分享图片URL">
    <meta property="og:url" content="页面URL">
    <meta property="og:type" content="website">

    <!-- Twitter -->
    <meta name="twitter:card" content="summary_large_image">
    <meta name="twitter:title" content="标题">
    <meta name="twitter:description" content="描述">
    <meta name="twitter:image" content="图片URL">
</head>
```

## 移动端相关

### Apple移动端标签

```html
<meta name="apple-mobile-web-app-capable" content="yes">                    <!-- 是否启用WebApp全屏模式 -->
<meta name="apple-mobile-web-app-status-bar-style" content="black">           <!-- 状态栏样式 -->
<meta name="apple-mobile-web-app-title" content="标题">                       <!-- 添加到主屏幕标题 -->
<meta name="apple-touch-icon" content="apple-touch-icon.png">                  <!-- 应用图标 -->
<meta name="apple-touch-icon-precomposed" content="apple-touch-icon.png">     <!-- 默认图标 -->
```

### Windows移动端标签

```html
<meta name="msapplication-tap-highlight" content="no">           <!-- 禁止高亮 -->
<meta name="msapplication-starturl" content="/">                  <!-- 起始URL -->
<meta name="msapplication-navbutton-color" content="#FF0000">     <!-- 导航按钮颜色 -->
```

### 主题颜色

```html
<meta name="theme-color" content="#4CAF50">
```

## 完整HTML模板

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="IE=edge,chrome=1">

    <title>页面标题 - 网站名称</title>
    <meta name="description" content="页面描述">
    <meta name="keywords" content="关键词1, 关键词2">
    <meta name="author" content="作者">

    <!-- 图标 -->
    <link rel="icon" href="favicon.ico">

    <!-- 样式 -->
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <!-- 页面内容 -->
</body>
</html>
```

## 总结

- `<meta>` 标签放在 `<head>` 中
- `charset` 设置字符编码，UTF-8是标准
- `viewport` 对移动端开发必不可少
- `description` 和 `keywords` 对SEO很重要
- `http-equiv` 模拟HTTP响应头
- 移动端需要额外的meta标签优化体验
