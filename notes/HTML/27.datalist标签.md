# datalist标签

## 基本介绍

`<datalist>` 用于为输入框提供自动补全建议列表，需要与 `<input>` 配合使用。

```html
<input list="suggestions">
<datalist id="suggestions">
    <option value="选项1">
    <option value="选项2">
</datalist>
```

## 基本语法

```html
<label>
    选择编程语言：
    <input type="text" list="languages" placeholder="请输入或选择">
</label>

<datalist id="languages">
    <option value="JavaScript">
    <option value="Python">
    <option value="Java">
    <option value="C++">
    <option value="Go">
</datalist>
```

## option标签

`<option>` 用于定义datalist中的选项。

```html
<datalist id="fruits">
    <!-- 简单值 -->
    <option value="苹果">

    <!-- 带标签的选项 -->
    <option value="香蕉" label="新鲜的香蕉">

    <!-- 仅标签（有些浏览器支持） -->
    <option label="橙子">橙子</option>
</datalist>
```

### option属性

| 属性 | 说明 |
|------|------|
| `value` | 选项的值 |
| `label` | 显示的标签 |
| `disabled` | 禁用该选项 |

## JavaScript交互

### 获取选中的值

```javascript
const input = document.getElementById('language');
const datalist = document.getElementById('languages');

input.addEventListener('change', function() {
    console.log('选择了：', this.value);

    // 检查是否为有效选项
    const options = Array.from(datalist.options);
    const isValid = options.some(opt => opt.value === this.value);
    console.log('是有效选项：', isValid);
});
```

### 动态添加选项

```javascript
const datalist = document.getElementById('dynamic-list');

// 添加单个选项
const option = document.createElement('option');
option.value = 'PHP';
datalist.appendChild(option);

// 批量添加
const languages = ['Ruby', 'Swift', 'Kotlin', 'Rust'];
languages.forEach(lang => {
    const opt = document.createElement('option');
    opt.value = lang;
    datalist.appendChild(opt);
});
```

### 实时搜索过滤

```javascript
const input = document.getElementById('search');
const datalist = document.getElementById('suggestions');

input.addEventListener('input', function() {
    const searchText = this.value.toLowerCase();

    // 隐藏/显示选项
    Array.from(datalist.options).forEach(option => {
        const text = option.value.toLowerCase();
        option.hidden = !text.includes(searchText);
    });
});
```

## datalist vs select

| 特性 | datalist | select |
|------|----------|--------|
| 输入方式 | 可输入可选择 | 仅能选择 |
| 下拉行为 | 显示匹配建议 | 显示所有选项 |
| 输入过滤 | 自动过滤 | 需手动选择 |
| 自由度 | 用户可输入任意值 | 只能是选项值 |
| 多选 | 不支持 | 支持 |
| 空值 | 可留空 | 可设置空选项 |

### 选择建议

```html
<!-- 推荐用datalist：需要用户输入 -->
<label for="country">国家：</label>
<input type="text" id="country" list="countries">
<datalist id="countries">
    <option value="中国">
    <option value="美国">
    <option value="日本">
</datalist>

<!-- 推荐用select：固定选项 -->
<label for="gender">性别：</label>
<select id="gender">
    <option value="">请选择</option>
    <option value="male">男</option>
    <option value="female">女</option>
</select>
```

## 注意事项

1. **兼容性**：IE不支持datalist，需要考虑降级方案
2. **值可以是任意值**：用户输入的值不一定在datalist中
3. **无法直接样式化**：datalist下拉样式由浏览器控制
4. **可与input type配合**：支持text、email、url、number等

## 总结

- `<datalist>` 为 `<input>` 提供建议列表
- 用户可以输入也可以选择
- `value` 是选项的值
- 支持JavaScript动态操作
- 适用于搜索建议、邮箱后缀等场景
- 与 `<select>` 的区别在于是否允许用户输入
