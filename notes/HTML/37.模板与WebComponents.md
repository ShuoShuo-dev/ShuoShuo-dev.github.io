# 模板与WebComponents

## 一、HTML模板元素

### 1.1 `<template>` 元素

`<template>` 元素用于存储HTML片段，不会被渲染，但可以被JavaScript访问和克隆。

```html
<!-- 模板不会被渲染 -->
<template id="card-template">
    <div class="card">
        <h2 class="title"></h2>
        <p class="content"></p>
        <button>点击</button>
    </div>
</template>

<!-- 页面内容 -->
<div id="container"></div>

<script>
const template = document.getElementById('card-template');
const container = document.getElementById('container');

// 克隆模板内容
const clone = template.content.cloneNode(true);

// 修改内容
clone.querySelector('.title').textContent = '动态标题';
clone.querySelector('.content').textContent = '动态内容';

// 添加到页面
container.appendChild(clone);
</script>
```

### 1.2 模板 vs 隐藏内容

```html
<!-- ❌ 不推荐：隐藏内容会占用DOM -->
<div id="template" style="display: none;">
    <p>模板内容</p>
</div>

<!-- ✅ 推荐：template不占用DOM -->
<template id="template">
    <p>模板内容</p>
</template>
```

### 1.3 使用表格行模板

```html
<table id="myTable">
    <thead>
        <tr>
            <th>姓名</th>
            <th>年龄</th>
        </tr>
    </thead>
    <tbody></tbody>
</table>

<template id="row-template">
    <tr>
        <td class="name"></td>
        <td class="age"></td>
    </tr>
</template>

<script>
const users = [
    { name: '张三', age: 18 },
    { name: '李四', age: 20 },
    { name: '王五', age: 22 }
];

const template = document.getElementById('row-template');
const tbody = document.querySelector('#myTable tbody');

users.forEach(user => {
    const clone = template.content.cloneNode(true);
    clone.querySelector('.name').textContent = user.name;
    clone.querySelector('.age').textContent = user.age;
    tbody.appendChild(clone);
});
</script>
```

## 二、Web Components 基础

Web Components 是一组web原生API，允许创建可复用的自定义元素。

### 2.1 三个核心概念

1. **Custom Elements** - 自定义HTML元素
2. **Shadow DOM** - 独立的DOM树
3. **HTML Templates** - `<template>` 和 `<slot>`

### 2.2 生命周期钩子

```javascript
class MyElement extends HTMLElement {
    constructor() {
        super();
        // 元素创建时调用
    }

    connectedCallback() {
        // 元素添加到DOM时调用
    }

    disconnectedCallback() {
        // 元素从DOM移除时调用
    }

    attributeChangedCallback(name, oldValue, newValue) {
        // 属性变化时调用
    }

    adoptedCallback() {
        // 元素移动到新文档时调用
    }

    static get observedAttributes() {
        return ['title', 'count']; // 监听这些属性变化
    }
}
```

## 三、创建自定义元素

### 3.1 基本自定义元素

```html
<!-- 使用自定义元素 -->
<my-element></my-element>

<script>
// 定义自定义元素
class MyElement extends HTMLElement {
    constructor() {
        super();
        this.innerHTML = '<p>Hello, World!</p>';
    }
}

// 注册元素
customElements.define('my-element', MyElement);
</script>
```

### 3.2 带属性和方法的元素

```javascript
class ProgressBar extends HTMLElement {
    constructor() {
        super();
        this._value = 0;
    }

    connectedCallback() {
        this.render();
    }

    static get observedAttributes() {
        return ['value', 'color'];
    }

    attributeChangedCallback(name, oldValue, newValue) {
        if (oldValue !== newValue) {
            if (name === 'value') {
                this._value = Number(newValue) || 0;
            } else if (name === 'color') {
                this._color = newValue;
            }
            this.render();
        }
    }

    // 属性同步到property
    get value() {
        return this._value;
    }

    set value(val) {
        this._value = Number(val) || 0;
        this.setAttribute('value', this._value);
        this.render();
    }

    render() {
        this.innerHTML = `
            <style>
                .progress {
                    width: 200px;
                    height: 20px;
                    background: #eee;
                    border-radius: 10px;
                    overflow: hidden;
                }
                .bar {
                    width: ${this._value}%;
                    height: 100%;
                    background: ${this._color || '#4CAF50'};
                    transition: width 0.3s;
                }
            </style>
            <div class="progress">
                <div class="bar" style="width: ${this._value}%"></div>
            </div>
            <span>${this._value}%</span>
        `;
    }
}

customElements.define('progress-bar', ProgressBar);
```

### 3.3 使用

```html
<progress-bar value="75" color="#2196F3"></progress-bar>

<script>
const progress = document.querySelector('progress-bar');
progress.value = 90; // 通过property设置
</script>
```

## 四、Shadow DOM

Shadow DOM 允许创建独立的DOM树，样式和逻辑不会影响外部。

### 4.1 基本用法

```javascript
class MyCard extends HTMLElement {
    constructor() {
        super();

        // 创建Shadow DOM
        const shadow = this.attachShadow({ mode: 'open' });

        // 添加内容
        shadow.innerHTML = `
            <style>
                .card {
                    border: 1px solid #ddd;
                    padding: 20px;
                    border-radius: 8px;
                }
                h2 { color: #333; }
            </style>
            <div class="card">
                <h2>卡片标题</h2>
                <slot></slot>
            </div>
        `;
    }
}

customElements.define('my-card', MyCard);
```

```html
<!-- 使用 -->
<my-card>
    <p>这是卡片内容</p>
</my-card>
```

### 4.2 Slot 插槽

```javascript
class TabCard extends HTMLElement {
    constructor() {
        super();
        const shadow = this.attachShadow({ mode: 'open' });
        shadow.innerHTML = `
            <style>
                .container { border: 1px solid #ddd; }
                .header { display: flex; background: #f5f5f5; }
                .tab {
                    padding: 10px 20px;
                    cursor: pointer;
                    border: none;
                    background: transparent;
                }
                .tab.active {
                    background: #fff;
                    border-bottom: 2px solid #007bff;
                }
                .content {
                    padding: 20px;
                    display: none;
                }
                .content.active { display: block; }
            </style>
            <div class="container">
                <div class="header">
                    <button class="tab active" data-tab="1">标签1</button>
                    <button class="tab" data-tab="2">标签2</button>
                </div>
                <div class="content active" data-tab="1">
                    <slot name="tab1"></slot>
                </div>
                <div class="content" data-tab="2">
                    <slot name="tab2"></slot>
                </div>
            </div>
            <script>
                // 注意：shadow内的script不会执行
            </script>
        `;

        // 添加交互
        shadow.querySelectorAll('.tab').forEach(tab => {
            tab.addEventListener('click', () => {
                shadow.querySelectorAll('.tab').forEach(t => t.classList.remove('active'));
                shadow.querySelectorAll('.content').forEach(c => c.classList.remove('active'));
                tab.classList.add('active');
                shadow.querySelector(\`.content[data-tab="\${tab.dataset.tab}"]\`).classList.add('active');
            });
        });
    }
}

customElements.define('tab-card', TabCard);
```

```html
<tab-card>
    <p slot="tab1">标签1的内容</p>
    <p slot="tab2">标签2的内容</p>
</tab-card>
```

## 五、样式隔离

### 5.1 :host 伪类

```javascript
class MyElement extends HTMLElement {
    constructor() {
        super();
        const shadow = this.attachShadow({ mode: 'open' });
        shadow.innerHTML = `
            <style>
                /* :host 选择自定义元素本身 */
                :host {
                    display: block;
                    padding: 20px;
                    border: 1px solid #ddd;
                }

                /* :host(selector) 条件选择 */
                :host([theme="dark"]) {
                    background: #333;
                    color: #fff;
                }

                :host([center]) {
                    text-align: center;
                }
            </style>
            <slot></slot>
        `;
    }
}
```

### 5.2 CSS 自定义属性穿透

```javascript
class MyButton extends HTMLElement {
    constructor() {
        super();
        const shadow = this.attachShadow({ mode: 'open' });
        shadow.innerHTML = `
            <style>
                button {
                    /* 使用CSS自定义属性 */
                    background: var(--button-bg, #007bff);
                    color: var(--button-color, white);
                    padding: var(--button-padding, 10px 20px);
                    border-radius: var(--button-radius, 4px);
                }
            </style>
            <button>
                <slot></slot>
            </button>
        `;
    }
}
```

```html
<my-button>默认按钮</my-button>
<my-button style="--button-bg: red;">红色按钮</my-button>
```

## 六、完整示例：可复用的卡片组件

```javascript
class ArticleCard extends HTMLElement {
    constructor() {
        super();
        const shadow = this.attachShadow({ mode: 'open' });
    }

    connectedCallback() {
        this.render();
    }

    static get observedAttributes() {
        return ['title', 'image', 'author', 'date'];
    }

    attributeChangedCallback() {
        this.render();
    }

    get title() { return this.getAttribute('title') || '无标题'; }
    get image() { return this.getAttribute('image') || 'placeholder.jpg'; }
    get author() { return this.getAttribute('author') || '匿名'; }
    get date() { return this.getAttribute('date') || ''; }

    render() {
        this.shadowRoot.innerHTML = `
            <style>
                :host {
                    display: block;
                    font-family: system-ui, sans-serif;
                }
                .card {
                    border: 1px solid #e0e0e0;
                    border-radius: 12px;
                    overflow: hidden;
                    transition: transform 0.2s, box-shadow 0.2s;
                }
                .card:hover {
                    transform: translateY(-4px);
                    box-shadow: 0 8px 24px rgba(0,0,0,0.12);
                }
                .image {
                    width: 100%;
                    height: 180px;
                    object-fit: cover;
                }
                .content {
                    padding: 16px;
                }
                .title {
                    font-size: 18px;
                    font-weight: 600;
                    margin: 0 0 12px 0;
                    color: #1a1a1a;
                }
                .meta {
                    font-size: 14px;
                    color: #666;
                    display: flex;
                    justify-content: space-between;
                }
            </style>
            <article class="card">
                <img class="image" src="${this.image}" alt="${this.title}">
                <div class="content">
                    <h3 class="title">${this.title}</h3>
                    <div class="meta">
                        <span class="author">${this.author}</span>
                        <time class="date">${this.date}</time>
                    </div>
                </div>
            </article>
        `;
    }
}

customElements.define('article-card', ArticleCard);
```

```html
<article-card
    title="Web Components 入门"
    image="https://picsum.photos/400/200"
    author="张三"
    date="2024-01-15">
</article-card>

<article-card
    title="CSS Grid 布局教程"
    image="https://picsum.photos/400/201"
    author="李四"
    date="2024-01-16">
</article-card>
```

## 七、浏览器支持与兼容性

### 7.1 支持情况

| 特性 | Chrome | Firefox | Safari | Edge |
|------|--------|---------|--------|------|
| Custom Elements | ✅ | ✅ | ✅ | ✅ |
| Shadow DOM | ✅ | ✅ | ✅ | ✅ |
| HTML Templates | ✅ | ✅ | ✅ | ✅ |
| Slots | ✅ | ✅ | ✅ | ✅ |

### 7.2 兼容性处理

```javascript
// 检测支持
if ('customElements' in window) {
    // 支持 Web Components
    class MyElement extends HTMLElement { ... }
    customElements.define('my-element', MyElement);
} else {
    // 使用Polyfill或降级
    document.write('<div class="my-element">...</div>');
}
```

### 7.3 使用Polyfill

```html
<script src="https://unpkg.com/@webcomponents/webcomponentsjs@2/index.js"></script>
```
