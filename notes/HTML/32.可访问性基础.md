# 可访问性基础

## 一、什么是可访问性

网页可访问性（Accessibility，简称 a11y）是指让所有人都能平等地访问和使用网页内容，包括：

- 视力障碍用户
- 听力障碍用户
- 行动不便用户
- 认知障碍用户
- 老年人
- 使用低速网络的手机用户

## 二、WAI-ARIA 基础

### 2.1 ARIA 角色（role）

```html
<!-- 导航区域 -->
<nav role="navigation">
    <ul>...</ul>
</nav>

<!-- 主要内容区域 -->
<main role="main">
    <article>...</article>
</main>

<!-- 侧边栏 -->
<aside role="complementary">
    ...
</aside>

<!-- 搜索表单 -->
<form role="search">
    <input type="search" aria-label="搜索">
</form>
```

### 2.2 ARIA 属性

```html
<!-- 标记必填字段 -->
<input type="text" aria-required="true">

<!-- 标记只读字段 -->
<input type="text" aria-readonly="true" value="不可编辑">

<!-- 标记禁用状态 -->
<button aria-disabled="true">不可点击</button>

<!-- 标记展开/折叠状态 -->
<button aria-expanded="false" aria-controls="menu">
    展开菜单
</button>
<div id="menu" hidden>...</div>

<!-- 标记当前选中项 -->
<li aria-selected="true">当前选中的项</li>
```

## 三、表单可访问性

### 3.1 标签关联

```html
<!-- 方式1：label的for属性 -->
<label for="username">用户名</label>
<input type="text" id="username">

<!-- 方式2：包装在label内 -->
<label>
    密码
    <input type="password">
</label>

<!-- 方式3：aria-label -->
<input type="text" aria-label="请输入邮箱地址">
```

### 3.2 错误提示

```html
<!-- 使用 aria-describedby 关联错误提示 -->
<label for="email">邮箱</label>
<input type="email" id="email" aria-describedby="email-error" aria-invalid="true">
<span id="email-error" style="color: red;">请输入有效的邮箱地址</span>

<!-- 使用 aria-live 动态提示 -->
<div aria-live="polite" id="status"></div>
```

### 3.3 占位符与标签

```html
<!-- ❌ 不好：placeholder不能替代label -->
<input type="text" placeholder="请输入用户名">

<!-- ✅ 好：label + placeholder -->
<label for="username" class="sr-only">用户名</label>
<input type="text" id="username" placeholder="请输入用户名">

<!-- .sr-only 让label视觉上隐藏但屏幕阅读器可读 -->
<style>
.sr-only {
    position: absolute;
    width: 1px;
    height: 1px;
    padding: 0;
    margin: -1px;
    overflow: hidden;
    clip: rect(0, 0, 0, 0);
    border: 0;
}
</style>
```

## 四、图片可访问性

### 4.1 alt 文本

```html
<!-- 有意义的图片：描述图片内容 -->
<img src="user-avatar.jpg" alt="张三的头像">

<!-- 装饰性图片：alt留空 -->
<img src="decoration.png" alt="">

<!-- 功能性图片：说明链接去向 -->
<a href="/home">
    <img src="home-icon.png" alt="返回首页">
</a>
```

### 4.2 复杂图片

```html
<!-- 使用 longdesc 或 figure/figcaption -->
<figure>
    <img src="chart.png" alt="销售趋势图">
    <figcaption>
        2024年Q1-Q4销售额趋势图
        <details>
            <summary>详细数据</summary>
            <table>
                <tr><th>季度</th><th>销售额</th></tr>
                <tr><td>Q1</td><td>100万</td></tr>
                <tr><td>Q2</td><td>150万</td></tr>
            </table>
        </details>
    </figcaption>
</figure>
```

## 五、键盘可访问性

### 5.1 可聚焦元素

```html
<!-- 原生可聚焦元素：自动支持tab -->
<a href="#">链接</a>
<button>按钮</button>
<input type="text">
<select>...</select>
<textarea>...</textarea>

<!-- 非交互元素：需要tabindex才能聚焦 -->
<div tabindex="0">可聚焦的div</div>

<!-- 不应该聚焦的元素 -->
<div tabindex="-1">只能通过JS聚焦</div>
```

### 5.2 焦点样式

```css
/* ❌ 移除焦点样式不好 */
:focus {
    outline: none;
}

/* ✅ 保持或自定义焦点样式 */
:focus-visible {
    outline: 2px solid #007bff;
    outline-offset: 2px;
}

/* 或者 */
:focus {
    outline: 2px solid #007bff;
    outline-offset: 2px;
}
```

### 5.3 跳转到内容

```html
<!-- 为键盘用户提供跳过链接 -->
<body>
    <a href="#main-content" class="skip-link">跳转到主要内容</a>
    <nav>...</nav>
    <main id="main-content">...</main>
</body>

<style>
.skip-link {
    position: absolute;
    top: -40px;
    left: 0;
    padding: 8px 16px;
    background: #000;
    color: #fff;
    z-index: 100;
}
.skip-link:focus {
    top: 0;
}
</style>
```

## 六、颜色与对比度

### 6.1 对比度要求

| 级别 | 文字与背景对比度 | 适用场景 |
|------|------------------|----------|
| AAA | 7:1 | 重要文字 |
| AA | 4.5:1 | 普通文字 |
| AA | 3:1 | 大字号(18px+)或粗体(14px+) |

### 6.2 检查工具

- Chrome DevTools > Lighthouse
- WebAIM Contrast Checker
- Contrast Ratio Calculator

```css
/* 满足AA标准 */
.text {
    color: #333;
    background: #fff;
    /* 对比度约12.6:1 ✅ */
}

/* 不满足AA标准 */
.text {
    color: #999;
    background: #fff;
    /* 对比度约2.5:1 ❌ */
}
```

## 七、其他最佳实践

### 7.1 语义化HTML

```html
<!-- ❌ 无语义 -->
<div onclick="toggle()">点击展开</div>

<!-- ✅ 语义化 -->
<button type="button" aria-expanded="false" aria-controls="content">
    点击展开
</button>
<div id="content">...</div>
```

### 7.2 语言声明

```html
<html lang="zh-CN">
    <p lang="en">This is English text.</p>
</html>
```

### 7.3 页面标题

```html
<!-- ✅ 描述性标题 -->
<title>首页 - 公司名称</title>
<title>联系方式 - 公司名称</title>

<!-- ❌ 模糊标题 -->
<title>首页</title>
<title>欢迎</title>
```

### 7.4 表格标题

```html
<table>
    <caption>2024年销售报表</caption>
    <thead>
        <tr>
            <th scope="col">季度</th>
            <th scope="col">销售额</th>
        </tr>
    </thead>
    <tbody>...</tbody>
</table>
```

## 八、快速检查清单

1. [ ] 所有图片都有alt属性
2. [ ] 表单都有关联的label
3. [ ] 颜色对比度达到4.5:1
4. [ ] 可以纯键盘操作
5. [ ] 焦点样式清晰可见
6. [ ] 页面有语义化结构
7. [ ] 有跳过导航链接
8. [ ] 动态内容有ARIA标注
