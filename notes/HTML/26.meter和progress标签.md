# meter和progress标签

## 简介

这两个标签都用于显示某种度量或进度，但使用场景不同：
- `<meter>`：用于已知的、有具体范围的值
- `<progress>`：用于不确定值的任务进度

## progress标签

表示任务的完成进度，常用于文件上传、下载、加载等场景。

### 基本语法

```html
<progress value="50" max="100">50%</progress>
```

### 属性

| 属性 | 说明 | 默认值 |
|------|------|--------|
| `value` | 当前进度值 | 0 |
| `max` | 最大值 | 1 |

### 示例

```html
<!-- 默认样式 -->
<progress value="30" max="100"></progress>

<!-- 百分比表示 -->
<progress value="0.75"></progress>

<!-- 显示进度文本 -->
<progress value="60" max="100">60%</progress>
```

### 常见应用场景

#### 1. 文件上传进度

```html
<label for="upload">上传文件：</label>
<progress id="upload" value="0" max="100">0%</progress>

<script>
    // 模拟上传进度
    const progress = document.getElementById('upload');
    let value = 0;
    const interval = setInterval(() => {
        value += 10;
        progress.value = value;
        if (value >= 100) {
            clearInterval(interval);
        }
    }, 500);
</script>
```

#### 2. 页面加载进度

```html
<div id="loading">
    <p>加载中...</p>
    <progress id="page-load" value="0" max="100"></progress>
</div>
```

#### 3. 下载进度

```html
<p>正在下载更新包...</p>
<progress value="45" max="100"></progress>
<p>45%</p>
```

## meter标签

表示在已知范围内的标量值或分数。

### 基本语法

```html
<meter value="3" min="0" max="10">3 out of 10</meter>
```

### 属性

| 属性 | 说明 |
|------|------|
| `value` | 当前值（必需） |
| `min` | 最小值（默认0） |
| `max` | 最大值（默认1） |
| `low` | "低"的阈值 |
| `high` | "高"的阈值 |
| `optimum` | 最优值 |
| `form` | 关联的表单 |

### 完整示例

```html
<!-- 完整属性 -->
<meter value="60" min="0" max="100" low="30" high="70" optimum="50">
    60%
</meter>
```

### meter颜色变化

根据值的大小自动显示不同颜色：

```html
<!-- 值过低（红色） -->
<meter value="10" min="0" max="100" low="30" high="70" optimum="50"></meter>

<!-- 值正常（绿色） -->
<meter value="50" min="0" max="100" low="30" high="70" optimum="50"></meter>

<!-- 值过高（橙色） -->
<meter value="80" min="0" max="100" low="30" high="70" optimum="50"></meter>
```

### 常见应用场景

#### 1. 磁盘使用量

```html
<p>磁盘使用：</p>
<meter value="250" min="0" max="500" low="400" high="450">
    250GB / 500GB
</meter>
```

#### 2. 投票结果

```html
<p>投票结果</p>
<p>选项A: <meter value="35" max="100">35%</meter> 35%</p>
<p>选项B: <meter value="45" max="100">45%</meter> 45%</p>
<p>选项C: <meter value="20" max="100">20%</meter> 20%</p>
```

#### 3. 评分系统

```html
<p>用户评分：</p>
<meter value="4.2" min="0" max="5" low="2" high="4" optimum="5">
    4.2 / 5
</meter>
```

#### 4. 电池电量

```html
<p>电池电量：</p>
<meter value="80" min="0" max="100" low="20" high="90"></meter>
```

#### 5. 温度/湿度

```html
<p>当前温度：<meter value="25" min="0" max="40" low="10" high="35" optimum="22"></meter> 25°C</p>
<p>当前湿度：<meter value="60" min="0" max="100" low="30" high="70" optimum="50"></meter> 60%</p>
```

## progress vs meter 对比

| 特性 | progress | meter |
|------|----------|-------|
| 用途 | 任务进度 | 已知范围的度量 |
| 值的变化 | 从起点到终点 | 在范围内任意位置 |
| 典型场景 | 上传、下载、加载 | 评分、容量、百分比 |
| 颜色变化 | 无（需要CSS） | 自动根据阈值变色 |
| 不确定值 | 支持 | 不支持 |

## 样式自定义

### progress样式

```css
/* Chrome/Safari/Edge */
progress {
    -webkit-appearance: none;
    appearance: none;
    width: 100%;
    height: 20px;
    border-radius: 10px;
    overflow: hidden;
}

progress::-webkit-progress-bar {
    background: #f0f0f0;
    border-radius: 10px;
}

progress::-webkit-progress-value {
    background: linear-gradient(to right, #4CAF50, #8BC34A);
    border-radius: 10px;
}

/* Firefox */
progress {
    background: #f0f0f0;
    border-radius: 10px;
}

progress::-moz-progress-bar {
    background: linear-gradient(to right, #4CAF50, #8BC34A);
    border-radius: 10px;
}
```

### meter样式

```css
/* Chrome/Safari/Edge */
meter {
    -webkit-appearance: none;
    appearance: none;
    width: 100%;
    height: 20px;
    border-radius: 10px;
    overflow: hidden;
}

meter::-webkit-meter-bar {
    background: #f0f0f0;
    border-radius: 10px;
}

meter::-webkit-meter-optimum-value {
    background: #4CAF50;
}

meter::-webkit-meter-suboptimum-value {
    background: #FFC107;
}

meter::-webkit-meter-even-less-good-value {
    background: #F44336;
}

/* Firefox */
meter::-moz-meter-bar {
    border-radius: 10px;
}

meter:indeterminate::-moz-meter-bar {
    background: #2196F3;
}
```

## 总结

- `<progress>` 用于显示任务进度，如上传、下载、加载
- `<meter>` 用于显示已知范围内的值，如评分、容量、百分比
- progress没有value时显示不确定状态（动画）
- meter支持low/high/optimum实现颜色自动变化
- 都支持CSS样式自定义
- 现代浏览器支持良好
