# 全局属性

## 什么是全局属性

全局属性是所有HTML元素都可以使用的属性。

## 常用全局属性

### 1. id属性

元素的唯一标识符。

```html
<div id="header">页面头部</div>
```

**规则**：
- 页面中必须唯一
- 必须以字母开头
- 可包含字母、数字、下划线、连字符

```javascript
// 获取元素
const element = document.getElementById('header');
```

### 2. class属性

元素的类名，用于CSS样式和JavaScript操作。

```html
<div class="container main-content">多个类用空格分隔</div>
```

```css
/* 通过class选择 */
.container { ... }
.main-content { ... }
```

```javascript
// 获取元素
const elements = document.querySelectorAll('.container');
```

### 3. style属性

内联样式，直接在元素上写CSS。

```html
<div style="color: red; font-size: 16px;">红色文字</div>
```

### 4. title属性

鼠标悬停时显示的提示文字。

```html
<a href="#" title="查看完整信息">链接</a>
<p title="这是一段描述">段落</p>
```

### 5. lang属性

指定元素内容的语言。

```html
<html lang="zh-CN">
<p lang="en">This is English text.</p>
```

### 6. dir属性

文本方向。

```html
<p dir="ltr">从左到右（默认）</p>
<p dir="rtl">从右到左（阿拉伯语、希伯来语）</p>
```

## HTML5新增全局属性

### 1. data-*属性

用于存储页面或应用私有的数据。

```html
<div id="product" data-product-id="123" data-price="99.99">
    产品名称
</div>
```

```javascript
const product = document.getElementById('product');

// 获取data属性
console.log(product.dataset.productId);    // "123"
console.log(product.dataset.price);         // "99.99"

// 设置data属性
product.dataset.category = 'electronics';
```

**命名规则**：
- `data-user-id` → `dataset.userId`（驼峰转换）
- `data-product-id` → `dataset.productId`

### 2. contenteditable属性

规定元素内容是否可编辑。

```html
<p contenteditable="true">这段文字可以编辑</p>

<!-- 只读 -->
<p contenteditable="false">这段文字不可编辑</p>
```

```javascript
// 检测编辑状态
element.addEventListener('input', function() {
    console.log('内容已更改：', this.textContent);
});
```

### 3. spellcheck属性

规定是否对元素进行拼写检查。

```html
<!-- 启用拼写检查 -->
<textarea spellcheck="true">This is a tset</textarea>

<!-- 禁用拼写检查 -->
<textarea spellcheck="false">代码片段</textarea>
```

### 4. draggable属性

规定元素是否可拖拽。

```html
<div draggable="true">拖动我</div>
<img src="image.jpg" draggable="true">
```

```javascript
element.addEventListener('dragstart', function(e) {
    e.dataTransfer.setData('text/plain', '拖拽数据');
});
```

### 5. hidden属性

隐藏元素（相当于 `display: none`）。

```html
<div hidden>被隐藏的内容</div>
```

```css
/* hidden的默认样式 */
[hidden] {
    display: none;
}

/* 覆盖默认样式使其可见但不交互 */
[hidden] {
    display: block;
    visibility: hidden;
}
```

### 6. tabindex属性

规定元素的Tab键导航顺序。

```html
<!-- 可以通过Tab获取焦点 -->
<input type="text" tabindex="1">

<!-- 跳过一个元素 -->
<input type="text" tabindex="3">
<input type="text" tabindex="2">

<!-- 移除Tab焦点（可点击但Tab跳过） -->
<div tabindex="-1">不能Tab访问</div>

<!-- 0：按文档顺序 -->
<!-- 正数：按数字顺序 -->
<!-- -1：不参与Tab顺序 -->
```

### 7. accesskey属性

定义激活元素的快捷键。

```html
<!-- Windows: Alt + 快捷键 -->
<!-- Mac: Ctrl + Alt + 快捷键 -->
<a href="/" accesskey="h">首页(h)</a>
<a href="/about" accesskey="a">关于(a)</a>
<a href="/contact" accesskey="c">联系(c)</a>
```

## 可访问性相关属性

### 1. role属性

定义元素的ARIA角色。

```html
<nav role="navigation">
    <ul role="list">
        <li role="listitem">导航项</li>
    </ul>
</nav>

<button role="button">点击</button>
<div role="alert">重要信息</div>
```

### 2. aria-*属性

提供可访问性信息。

```html
<!-- 标签 -->
<label id="name-label">姓名</label>
<input type="text" aria-labelledby="name-label">

<!-- 描述 -->
<input type="email" aria-describedby="email-help">
<p id="email-help">请输入有效的邮箱地址</p>

<!-- 状态 -->
<button aria-pressed="false">未选中</button>
<div aria-expanded="true">展开状态</div>

<!-- 其他常用 -->
aria-hidden="true"           <!-- 隐藏 -->
aria-disabled="true"         <!-- 禁用 -->
aria-selected="true"          <!-- 选中 -->
aria-checked="true"           <!-- 选中 -->
aria-required="true"          <!-- 必填 -->
aria-invalid="true"           <!-- 无效 -->
```

## 事件处理属性

### 常用事件属性

```html
<div onclick="alert('点击')">点击我</div>
<div onmouseover="this.style.color='red'">鼠标悬停</div>
<input onfocus="this.style.borderColor='blue'" onblur="this.style.borderColor='gray'">
<form onsubmit="return confirm('确认提交?')">
```

### 常用事件列表

| 属性 | 触发时机 |
|------|----------|
| `onclick` | 点击 |
| `ondblclick` | 双击 |
| `onmouseover` | 鼠标移入 |
| `onmouseout` | 鼠标移出 |
| `onfocus` | 获取焦点 |
| `onblur` | 失去焦点 |
| `onchange` | 值改变并失去焦点 |
| `oninput` | 输入时 |
| `onsubmit` | 表单提交 |
| `onkeydown` | 键盘按下 |
| `onkeyup` | 键盘释放 |
| `onload` | 加载完成 |

## 自定义元素属性

### is属性（HTML模板）

```html
<!-- 声明自定义元素 -->
<script>
    class MyElement extends HTMLElement {
        constructor() {
            super();
        }
    }
    customElements.define('my-element', MyElement);
</script>

<!-- 使用 -->
<my-element is="my-element"></my-element>
```

### slot属性

用于Web Components的插槽。

```html
<!-- 定义插槽 -->
<template id="my-template">
    <slot name="content">默认内容</slot>
</template>

<!-- 使用 -->
<my-element>
    <span slot="content">自定义内容</span>
</my-element>
```

## 完整示例

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>全局属性示例</title>
    <style>
        body { font-family: Arial, sans-serif; padding: 20px; }
        .demo { margin: 20px 0; padding: 15px; border: 1px solid #ddd; }
        .highlight { background: yellow; }
        .card { background: #f5f5f5; padding: 20px; margin: 10px 0; }
    </style>
</head>
<body>
    <h1>全局属性示例</h1>

    <!-- data-* 属性 -->
    <div class="demo">
        <h2>1. data-* 属性</h2>
        <div id="product" class="card" data-id="123" data-category="electronics" data-price="2999">
            产品信息 - 点击查看数据
        </div>
        <button onclick="showData()">显示产品数据</button>
    </div>

    <!-- contenteditable -->
    <div class="demo">
        <h2>2. contenteditable 属性</h2>
        <div contenteditable="true" class="card">
            这段文字可以编辑，试试看！
        </div>
    </div>

    <!-- draggable -->
    <div class="demo">
        <h2>3. draggable 属性</h2>
        <div id="drag1" class="card" draggable="true"
             style="display:inline-block; cursor:move;">
            拖动我！
        </div>
        <div id="dropzone" class="card" style="min-height:100px;">
            拖放到这里
        </div>
    </div>

    <!-- tabindex -->
    <div class="demo">
        <h2>4. tabindex 属性</h2>
        <p>按Tab键切换焦点：</p>
        <input type="text" placeholder="第3个" tabindex="3">
        <input type="text" placeholder="第1个" tabindex="1">
        <input type="text" placeholder="第2个" tabindex="2">
        <span tabindex="0">可通过Tab聚焦的span</span>
    </div>

    <!-- title -->
    <div class="demo">
        <h2>5. title 属性</h2>
        <button title="点击后将删除数据">删除按钮</button>
    </div>

    <!-- hidden -->
    <div class="demo">
        <h2>6. hidden 属性</h2>
        <p>这是一段可见的文字</p>
        <p hidden>这段被隐藏了</p>
        <button onclick="toggleHidden()">切换显示</button>
    </div>

    <script>
        // data-* 属性
        function showData() {
            const product = document.getElementById('product');
            alert(`ID: ${product.dataset.id}\n分类: ${product.dataset.category}\n价格: ¥${product.dataset.price}`);
        }

        // draggable
        const drag = document.getElementById('drag1');
        const drop = document.getElementById('dropzone');

        drag.addEventListener('dragstart', e => {
            e.dataTransfer.setData('text', '来自拖动元素');
        });

        drop.addEventListener('dragover', e => e.preventDefault());
        drop.addEventListener('drop', e => {
            e.preventDefault();
            drop.textContent = '已放置：' + e.dataTransfer.getData('text');
        });

        // hidden
        function toggleHidden() {
            const hidden = document.querySelector('[hidden]');
            if (hidden.hasAttribute('hidden')) {
                hidden.removeAttribute('hidden');
            } else {
                hidden.setAttribute('hidden', '');
            }
        }
    </script>
</body>
</html>
```

## 全局属性速查表

| 属性 | 说明 |
|------|------|
| `id` | 唯一标识 |
| `class` | 类名 |
| `style` | 内联样式 |
| `title` | 提示文字 |
| `lang` | 语言 |
| `dir` | 文本方向 |
| `data-*` | 自定义数据 |
| `contenteditable` | 可编辑 |
| `spellcheck` | 拼写检查 |
| `draggable` | 可拖拽 |
| `hidden` | 隐藏 |
| `tabindex` | Tab顺序 |
| `accesskey` | 快捷键 |
| `role` | ARIA角色 |
| `aria-*` | ARIA属性 |
| `slot` | 插槽 |

## 总结

- 全局属性是所有HTML元素都可以使用的
- `id` 必须唯一，`class` 可以重复
- `data-*` 用于存储自定义数据
- `contenteditable` 实现原地编辑
- `tabindex` 控制键盘导航
- `aria-*` 用于可访问性支持
