# iframe框架

## iframe简介

`<iframe>` 是内联框架，用于在当前页面中嵌入另一个HTML页面。

```html
<iframe src="页面地址"></iframe>
```

## 基本属性

| 属性 | 说明 |
|------|------|
| `src` | 嵌入页面的URL |
| `width` | 框架宽度 |
| `height` | 框架高度 |
| `name` | 框架名称（用于target） |
| `frameborder` | 边框宽度（0或1） |
| `scrolling` | 滚动条设置 |
| `allow` | 权限设置 |
| `sandbox` | 安全沙箱 |

## 基本用法

### 嵌入网页

```html
<iframe src="https://www.example.com" width="100%" height="400"></iframe>
```

### 设置边框

```html
<!-- 有边框 -->
<iframe src="page.html" frameborder="1"></iframe>

<!-- 无边框 -->
<iframe src="page.html" frameborder="0"></iframe>

<!-- 推荐：使用CSS -->
<iframe src="page.html" style="border: none;"></iframe>
```

### 滚动条设置

```html
<!-- 自动显示滚动条 -->
<iframe src="page.html" scrolling="auto"></iframe>

<!-- 始终显示滚动条 -->
<iframe src="page.html" scrolling="yes"></iframe>

<!-- 从不显示滚动条 -->
<iframe src="page.html" scrolling="no"></iframe>
```

## 常用场景

### 1. 嵌入地图

```html
<iframe
    src="https://maps.google.com/maps?q=北京&output=embed"
    width="100%"
    height="400"
    frameborder="0"
    style="border: 0;"
    allowfullscreen>
</iframe>
```

### 2. 嵌入视频

```html
<!-- YouTube -->
<iframe
    width="560"
    height="315"
    src="https://www.youtube.com/embed/VIDEO_ID"
    frameborder="0"
    allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
    allowfullscreen>
</iframe>

<!-- Bilibili -->
<iframe
    src="//player.bilibili.com/player.html?bvid=BV1xxxxxxx&page=1"
    width="560"
    height="315"
    scrolling="no"
    frameborder="0"
    allowfullscreen>
</iframe>
```

### 3. 嵌入PDF

```html
<iframe src="document.pdf" width="100%" height="600"></iframe>
```

### 4. 作为链接目标

```html
<!-- 定义iframe -->
<iframe name="content-frame" src="default.html" width="100%" height="500"></iframe>

<!-- 通过target指定链接在iframe中打开 -->
<a href="page1.html" target="content-frame">页面1</a>
<a href="page2.html" target="content-frame">页面2</a>
<a href="page3.html" target="content-frame">页面3</a>
```

## sandbox属性

提供额外的安全限制。

```html
<!-- 最安全：禁用所有功能 -->
<iframe src="page.html" sandbox></iframe>

<!-- 允许特定功能 -->
<iframe
    src="page.html"
    sandbox="allow-scripts allow-same-origin allow-forms allow-popups"
></iframe>
```

### sandbox可选值

| 值 | 说明 |
|----|------|
| 无值 | 最严格的限制 |
| `allow-forms` | 允许表单提交 |
| `allow-modals` | 允许打开弹窗 |
| `allow-orientation-lock` | 允许屏幕方向锁定 |
| `allow-pointer-lock` | 允许指针锁定 |
| `allow-popups` | 允许弹窗 |
| `allow-popups-to-escape-sandbox` | 允许弹窗新窗口 |
| `allowPresentation` | 允许演示模式 |
| `allow-same-origin` | 允许同源访问 |
| `allow-scripts` | 允许执行脚本 |
| `allow-top-navigation` | 允许顶部导航 |
| `allow-top-navigation-by-user-activation` | 允许用户触发的顶部导航 |
| `allow-storage-access-by-user-activation` | 允许存储访问 |
| `allow-downloads` | 允许下载 |

## loading属性

控制加载方式。

```html
<!-- 立即加载（默认） -->
<iframe src="page.html" loading="eager"></iframe>

<!-- 懒加载 -->
<iframe src="page.html" loading="lazy"></iframe>
```

## allow属性

指定iframe的功能策略。

```html
<iframe
    src="page.html"
    allow="camera; microphone; geolocation"
></iframe>
```

## referrerpolicy属性

控制引用来源。

```html
<iframe src="page.html" referrerpolicy="no-referrer"></iframe>
```

可选值：
- `no-referrer`
- `no-referrer-when-downgrade`
- `origin`
- `origin-when-cross-origin`
- `same-origin`
- `strict-origin-when-cross-origin`
- `unsafe-url`

## 完整示例

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>iframe示例</title>
    <style>
        body {
            margin: 20px;
            font-family: Arial, sans-serif;
        }
        .iframe-container {
            width: 100%;
            margin: 20px 0;
            border: 1px solid #ddd;
        }
        iframe {
            display: block;
            width: 100%;
        }
    </style>
</head>
<body>
    <h1>iframe使用示例</h1>

    <!-- 链接导航 -->
    <nav>
        <a href="demo1.html" target="content-frame">示例1</a> |
        <a href="demo2.html" target="content-frame">示例2</a> |
        <a href="demo3.html" target="content-frame">示例3</a>
    </nav>

    <!-- iframe容器 -->
    <div class="iframe-container">
        <iframe
            name="content-frame"
            src="demo1.html"
            height="400"
            frameborder="0"
            loading="lazy">
        </iframe>
    </div>

    <!-- 嵌入YouTube视频 -->
    <h2>嵌入视频</h2>
    <div class="iframe-container">
        <iframe
            width="560"
            height="315"
            src="https://www.youtube.com/embed/dQw4w9WgXcQ"
            frameborder="0"
            allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
            allowfullscreen>
        </iframe>
    </div>

    <!-- 嵌入地图 -->
    <h2>嵌入地图</h2>
    <div class="iframe-container">
        <iframe
            src="https://maps.google.com/maps?q=北京大学&output=embed"
            height="400"
            frameborder="0"
            style="border: 0;"
            allowfullscreen>
        </iframe>
    </div>
</body>
</html>
```

## iframe与SEO

iframe对SEO有一定影响：
- 搜索引擎可能无法抓取iframe内的内容
- 建议为iframe添加描述性标题

```html
<iframe
    src="content.html"
    title="主要内容"
    aria-label="页面主要内容"
></iframe>
```

## 响应式iframe

```css
.video-container {
    position: relative;
    width: 100%;
    padding-bottom: 56.25%; /* 16:9比例 */
    height: 0;
}

.video-container iframe {
    position: absolute;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
}
```

```html
<div class="video-container">
    <iframe
        src="https://www.youtube.com/embed/VIDEO_ID"
        frameborder="0"
        allowfullscreen>
    </iframe>
</div>
```

## 总结

- `<iframe>` 用于在页面中嵌入其他页面
- 常用属性：`src`、`width`、`height`、`name`、`frameborder`
- `sandbox` 属性提供安全限制
- `loading="lazy"` 实现懒加载
- 注意SEO和可访问性问题
- 现代开发中可以考虑使用 `<object>` 或其他方案
