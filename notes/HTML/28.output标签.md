# output标签

## 基本介绍

`<output>` 是HTML5新增的语义化标签，用于显示计算结果或用户操作的输出。

```html
<output name="result">默认值</output>
```

## 基本属性

| 属性 | 说明 |
|------|------|
| `name` | 输出区域的名称 |
| `for` | 关联的输入元素ID |
| `form` | 关联的表单ID |

## for属性

`for` 属性指定哪些输入元素影响了输出值。

```html
<label>数值1：</label>
<input type="number" id="num1" value="0">
<br>
<label>数值2：</label>
<input type="number" id="num2" value="0">
<br>
<output for="num1 num2" name="sum">0</output>
```

## 基本示例

### 1. 简单加法计算

```html
<form oninput="result.value=parseInt(a.value)+parseInt(b.value)">
    <input type="number" id="a" value="0"> +
    <input type="number" id="b" value="0"> =
    <output name="result">0</output>
</form>
```

### 2. 数字范围滑块

```html
<label>
    评分：<span id="rating-value">5</span>/10
</label>
<input type="range" id="rating" min="0" max="10" value="5"
       oninput="document.getElementById('rating-value').textContent = this.value">
<output for="rating" name="rating-output">5</output>
```

### 3. 购物车计算

```html
<form id="cart">
    <label>单价：¥<input type="number" id="price" value="100" min="0"></label>
    <label>数量：<input type="number" id="quantity" value="1" min="1"></label>
    <output name="total" for="price quantity">¥100</output>
</form>

<script>
    const cart = document.getElementById('cart');
    const price = document.getElementById('price');
    const quantity = document.getElementById('quantity');
    const total = cart.elements.namedItem('total');

    function updateTotal() {
        total.textContent = '¥' + (price.value * quantity.value);
    }

    price.addEventListener('input', updateTotal);
    quantity.addEventListener('input', updateTotal);
</script>
```

## 常用场景

### 1. 计算器

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>简单计算器</title>
    <style>
        body { font-family: Arial, sans-serif; padding: 20px; }
        .calculator { max-width: 300px; }
        input, select, output { font-size: 18px; margin: 5px; padding: 8px; }
        output { display: block; margin-top: 10px; background: #f0f0f0; text-align: center; }
    </style>
</head>
<body>
    <h2>简单计算器</h2>
    <div class="calculator">
        <form oninput="result.value = eval(num1.value + op.value + num2.value)">
            <input type="number" id="num1" value="0">
            <select id="op">
                <option value="+">+</option>
                <option value="-">-</option>
                <option value="*">×</option>
                <option value="/">÷</option>
            </select>
            <input type="number" id="num2" value="0">
            <output name="result">0</output>
        </form>
    </div>
</body>
</html>
```

### 2. BMI计算器

```html
<form oninput="bmi.value = (weight.value / (height.value/100) ** 2).toFixed(1)">
    <label>身高 (cm)：<input type="number" id="height" value="170" min="50" max="250"></label>
    <label>体重 (kg)：<input type="number" id="weight" value="65" min="20" max="300"></label>
    <p>BMI：<output name="bmi">22.5</output></p>
</form>
```

### 3. 面积计算器

```html
<form id="area-form">
    <label>长度 (cm)：<input type="number" id="length" value="10" min="0"></label>
    <label>宽度 (cm)：<input type="number" id="width" value="5" min="0"></label>
    <p>矩形面积：<output name="area">50</output> cm²</p>
</form>

<script>
    const form = document.getElementById('area-form');
    const length = document.getElementById('length');
    const width = document.getElementById('width');
    const area = form.elements.namedItem('area');

    function calculate() {
        area.value = (parseFloat(length.value) || 0) * (parseFloat(width.value) || 0);
    }

    length.addEventListener('input', calculate);
    width.addEventListener('input', calculate);
</script>
```

### 4. 进度指示器

```html
<form>
    <label>当前进度 (%)：<input type="number" id="current" value="50" min="0" max="100"></label>
    <label>目标值：<input type="number" id="target" value="100" min="1"></label>
    <p>已完成：<output for="current target" name="done">50</output></p>
    <progress id="progress-bar" value="50" max="100"></progress>
</form>

<script>
    const current = document.getElementById('current');
    const target = document.getElementById('target');
    const progressBar = document.getElementById('progress-bar');
    const doneOutput = document.querySelector('output[name="done"]');

    function updateProgress() {
        const percentage = (current.value / target.value * 100).toFixed(1);
        progressBar.value = percentage;
        doneOutput.value = percentage + '%';
    }

    current.addEventListener('input', updateProgress);
    target.addEventListener('input', updateProgress);
</script>
```

## 与表单关联

### form属性

```html
<form id="myForm">
    <input type="number" id="x" value="10">
    <input type="number" id="y" value="20">
    <output name="sum" for="x y">30</output>
</form>

<!-- 放在表单外部，但关联到表单 -->
<output name="product" form="myForm" for="x y">200</output>
```

### name属性

```html
<form id="calc">
    <input type="number" name="a" value="5">
    <input type="number" name="b" value="3">
    <output name="result">8</output>
</form>

<script>
    const form = document.getElementById('calc');
    // 通过name获取output
    const result = form.elements.namedItem('result');
    console.log(result.value); // 8
</script>
```

## 注意事项

1. **IE不支持**：需要考虑降级方案
2. **语义化标签**：`output` 有语义意义，表示计算结果
3. **可访问性**：屏幕阅读器可以识别output元素
4. **常与表单结合**：适合实时计算的场景

## 总结

- `<output>` 用于显示计算或操作的结果
- `for` 属性关联输入元素
- `name` 属性用于表单提交
- 常用于计算器、实时统计等场景
- 支持CSS样式定制
- IE不支持，需要考虑兼容性
