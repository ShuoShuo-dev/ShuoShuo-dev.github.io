# details和summary标签

## 基本语法

`<details>` 用于创建可折叠的内容区域，`<summary>` 用于定义折叠区域的标题。

```html
<details>
    <summary>点击展开/收起</summary>
    <p>这是折叠的内容</p>
</details>
```

## 基本示例

### 简单折叠

```html
<details>
    <summary>什么是HTML？</summary>
    <p>HTML（超文本标记语言）是用于创建网页的标准标记语言。</p>
</details>
```

### 默认展开

```html
<details open>
    <summary>默认展开的内容</summary>
    <p>这个内容默认是展开的。</p>
</details>
```

## 样式自定义

### 默认小箭头样式

```html
<style>
    details {
        border: 1px solid #ddd;
        border-radius: 4px;
        padding: 10px;
        margin: 10px 0;
    }

    summary {
        cursor: pointer;
        font-weight: bold;
        outline: none;
    }

    summary::-webkit-details-marker {
        color: #666;
    }
</style>

<details>
    <summary>点击查看更多</summary>
    <p>这里是隐藏的内容...</p>
</details>
```

### 替换默认箭头

```html
<style>
    details {
        border: 1px solid #ddd;
        border-radius: 4px;
        padding: 10px;
    }

    summary {
        cursor: pointer;
        list-style: none; /* 隐藏默认箭头 */
        position: relative;
        padding-left: 25px;
    }

    summary::before {
        content: "+";
        position: absolute;
        left: 5px;
        font-weight: bold;
        color: #333;
    }

    details[open] summary::before {
        content: "-";
    }
</style>

<details>
    <summary>展开查看</summary>
    <p>内容区域...</p>
</details>
```

### 动画效果

```html
<style>
    details {
        overflow: hidden;
    }

    summary {
        cursor: pointer;
    }

    details[open] summary {
        margin-bottom: 10px;
        border-bottom: 1px solid #eee;
        padding-bottom: 10px;
    }

    /* 注意：CSS无法直接动画展开/收起
       需要使用JS或第三方库实现动画 */
</style>
```

## JavaScript交互

### 切换事件

```javascript
document.querySelector('details').addEventListener('toggle', function(e) {
    if (this.open) {
        console.log('展开了');
    } else {
        console.log('收起了');
    }
});
```

### 程序控制

```javascript
// 展开
document.querySelector('details').setAttribute('open', '');

// 收起
document.querySelector('details').removeAttribute('open');

// 切换
const details = document.querySelector('details');
if (details.open) {
    details.removeAttribute('open');
} else {
    details.setAttribute('open', '');
}
```

### 点击其他关闭

```javascript
const allDetails = document.querySelectorAll('details');

allDetails.forEach(detail => {
    detail.addEventListener('toggle', function() {
        if (this.open) {
            allDetails.forEach(d => {
                if (d !== this) d.removeAttribute('open');
            });
        }
    });
});
```

## 总结

- `<details>` 创建可折叠区域
- `<summary>` 定义折叠标题
- `open` 属性控制默认展开状态
- 可以自定义样式和箭头图标
- 支持JavaScript事件控制
- 是实现FAQ、手风琴等功能的原生方案
