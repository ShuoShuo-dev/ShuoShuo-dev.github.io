# div和span标签

## div标签

`<div>` 是HTML中最常用的容器标签，用于组织和布局页面内容。

```html
<div>
    <h2>这是div内的标题</h2>
    <p>这是div内的段落</p>
</div>
```

### div标签特点

1. **块级元素**：独占一行
2. **无语义**：本身没有特殊含义
3. **容器**：可以包含任何内容
4. **布局利器**：常用于页面布局

### 常见用途

```html
<body>
    <div class="header">
        <h1>网站Logo</h1>
        <nav>导航菜单</nav>
    </div>

    <div class="main">
        <div class="sidebar">侧边栏</div>
        <div class="content">主内容区</div>
    </div>

    <div class="footer">
        <p>版权信息</p>
    </div>
</body>
```

## span标签

`<span>` 是行内元素，用于在文本中添加样式或标记部分内容。

```html
<p>今天天气<span class="highlight">非常晴朗</span>，适合外出。</p>
```

### span标签特点

1. **行内元素**：不独占一行，与其他内容共处一行
2. **无语义**：本身没有特殊含义
3. **文本级容器**：只能包含文本或行内元素
4. **用于点缀**：常用于给文本添加样式

### 常见用途

```html
<p>
    优惠价：<span class="price">¥199</span>
</p>

<p>
    状态：<span class="status success">已发货</span>
</p>

<p>
    关键词：<span class="keyword">HTML</span>、
    <span class="keyword">CSS</span>、
    <span class="keyword">JavaScript</span>
</p>
```

## div和span对比

| 特性 | `<div>` | `<span>` |
|------|---------|----------|
| 元素类型 | 块级元素 | 行内元素 |
| 宽度 | 默认100% | 由内容决定 |
| 是否独占一行 | 是 | 否 |
| 常用于 | 布局、结构 | 文本修饰 |
| 能否包含块级元素 | 可以 | 不可以 |

## 布局示例

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>div和span布局示例</title>
    <style>
        .header { background: #333; color: white; padding: 20px; }
        .content { display: flex; }
        .sidebar { width: 200px; background: #f4f4f4; padding: 20px; }
        .main { flex: 1; padding: 20px; }
        .footer { background: #666; color: white; padding: 10px; text-align: center; }
    </style>
</head>
<body>
    <div class="header">
        <h1>我的网站</h1>
    </div>

    <div class="content">
        <div class="sidebar">
            <h3>侧边栏</h3>
            <p>这是侧边栏内容</p>
        </div>
        <div class="main">
            <h2>主内容区</h2>
            <p>这是主内容区，使用<span class="highlight">div</span>进行布局。</p>
            <p>重要词汇用<span style="color: red;">span</span>标记。</p>
        </div>
    </div>

    <div class="footer">
        <p>© 2024 我的网站 | <span>联系邮箱</span></p>
    </div>
</body>
</html>
```

## 语义化标签（HTML5新增）

HTML5提供了更多语义化的布局标签：

```html
<body>
    <header>页头</header>
    <nav>导航</nav>
    <main>
        <section>章节</section>
        <article>文章</article>
        <aside>侧边栏</aside>
    </main>
    <footer>页脚</footer>
</body>
```

## 总结

- `<div>` 是块级元素，用于页面布局和大区块划分
- `<span>` 是行内元素，用于文本中的小范围标记
- 现代开发推荐使用语义化标签，但div和span仍是基础
- 布局时应优先考虑语义化标签，再用div做容器
