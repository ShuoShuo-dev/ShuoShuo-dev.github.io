# 图像标签

## img标签介绍

`<img>` 标签用于在网页中插入图片，是一个单标签（自闭合标签）。

```html
<img src="图片路径" alt="图片描述">
```

## 基本属性

### src属性（必填）

指定图片的路径或URL。

```html
<img src="logo.png">
<img src="https://example.com/image.jpg">
```

### alt属性（必填）

图片的替代文本，当图片无法显示时显示此文字。

```html
<img src="photo.jpg" alt="风景照片">
```

### title属性

鼠标悬停时显示的提示文字。

```html
<img src="photo.jpg" alt="风景照片" title="点击查看大图">
```

## 其他常用属性

### width和height

设置图片的宽度和高度。

```html
<!-- 像素值 -->
<img src="photo.jpg" width="300" height="200">

<!-- 百分比（相对于父容器） -->
<img src="photo.jpg" width="50%">
```

### loading

控制图片加载方式。

```html
<img src="photo.jpg" loading="lazy"> <!-- 懒加载 -->
<img src="photo.jpg" loading="eager"> <!-- 立即加载 -->
```

### align（已废弃）

图片对齐方式，建议使用CSS。

## 图片格式

| 格式 | 特点 | 适用场景 |
|------|------|----------|
| JPEG/JPG | 支持1600万色彩，适合照片 | 商品图、照片 |
| PNG | 支持透明背景，适合图标 | 图标、Logo |
| GIF | 支持动画 | 动图、小动画 |
| SVG | 矢量图，可缩放 | 图标、矢量图形 |
| WebP | 新格式，压缩率高 | 现代网页 |

## 路径

### 相对路径

相对于当前文件的位置：

```html
<!-- 同一目录 -->
<img src="photo.jpg">

<!-- 子目录 -->
<img src="images/photo.jpg">

<!-- 上一级目录 -->
<img src="../photo.jpg">

<!-- 上两级目录 -->
<img src="../../photo.jpg">
```

### 绝对路径

完整的网络地址或本地路径：

```html
<img src="https://example.com/images/photo.jpg">
<img src="C:/images/photo.jpg">
```

## 完整示例

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>图片标签示例</title>
</head>
<body>
    <!-- 基本用法 -->
    <h2>基本图片</h2>
    <img src="photo.jpg" alt="示例图片" title="示例图片">

    <!-- 带尺寸的图片 -->
    <h2>设置尺寸</h2>
    <img src="photo.jpg" alt="示例图片" width="300">

    <!-- 相对路径示例 -->
    <h2>相对路径</h2>
    <img src="images/logo.png" alt="Logo">

    <!-- 网络图片 -->
    <h2>网络图片</h2>
    <img src="https://via.placeholder.com/300x200" alt="占位图">
</body>
</html>
```

## 最佳实践

1. **总是添加alt属性**：对搜索引擎优化（SEO）和可访问性很重要
2. **指定width和height**：避免页面加载时布局跳动
3. **使用相对路径**：便于项目迁移和维护
4. **压缩图片**：使用工具压缩图片大小，提高加载速度
5. **使用WebP格式**：在支持的情况下优先使用WebP格式

```html
<!-- 推荐写法 -->
<img
    src="images/product.webp"
    alt="笔记本电脑"
    title="查看详情"
    width="400"
    height="300"
    loading="lazy"
>

<!-- 不推荐：缺少alt属性 -->
<img src="photo.jpg">
```

## 响应式图片

```html
<!-- 根据屏幕大小加载不同图片 -->
<picture>
    <source media="(min-width: 768px)" srcset="large.jpg">
    <source media="(min-width: 480px)" srcset="medium.jpg">
    <img src="small.jpg" alt="响应式图片">
</picture>
```

## 总结

- `<img>` 是自闭合标签，用于插入图片
- `src` 和 `alt` 是必填属性
- 注意图片路径的正确性（相对路径或绝对路径）
- 合理设置图片尺寸，避免页面布局跳动
