# 媒体标签

## video视频标签

### 基本语法

```html
<video src="视频路径" controls></video>
```

### 常用属性

| 属性 | 说明 |
|------|------|
| `src` | 视频文件路径 |
| `controls` | 显示播放控件 |
| `width` | 视频宽度 |
| `height` | 视频高度 |
| `autoplay` | 自动播放（需配合muted） |
| `muted` | 静音播放 |
| `loop` | 循环播放 |
| `poster` | 封面图片 |
| `preload` | 预加载策略 |

### 完整示例

```html
<video
    src="movie.mp4"
    controls
    width="600"
    poster="poster.jpg"
    autoplay
    muted
    loop
>
</video>
```

### 多格式视频源

由于浏览器兼容性问题，建议提供多种格式的视频：

```html
<video controls width="600">
    <source src="movie.mp4" type="video/mp4">
    <source src="movie.webm" type="video/webm">
    <source src="movie.ogg" type="video/ogg">
    您的浏览器不支持video标签
</video>
```

### autoplay和muted

现代浏览器要求自动播放必须静音：

```html
<!-- 正确：静音自动播放 -->
<video src="ad.mp4" autoplay muted></video>

<!-- 错误：不会自动播放 -->
<video src="ad.mp4" autoplay></video>
```

## audio音频标签

### 基本语法

```html
<audio src="音频路径" controls></audio>
```

### 常用属性

| 属性 | 说明 |
|------|------|
| `src` | 音频文件路径 |
| `controls` | 显示播放控件 |
| `autoplay` | 自动播放 |
| `muted` | 静音 |
| `loop` | 循环播放 |
| `preload` | 预加载策略 |

### 多格式音频源

```html
<audio controls>
    <source src="music.mp3" type="audio/mpeg">
    <source src="music.ogg" type="audio/ogg">
    您的浏览器不支持audio标签
</audio>
```

## embed和object

### embed标签

用于嵌入外部内容，如Flash（已淘汰）或PDF：

```html
<!-- 嵌入PDF -->
<embed src="document.pdf" type="application/pdf" width="600" height="800">

<!-- 嵌入Flash（已不推荐） -->
<embed src="banner.swf" type="application/x-shockwave-flash">
```

### object和param标签

用于嵌入多种类型的内容：

```html
<object data="document.pdf" type="application/pdf" width="600" height="800">
    <param name="quality" value="high">
    您的浏览器不支持object标签
</object>
```

## 兼容性处理

由于不同浏览器支持的音视频格式不同，建议提供多种格式：

```html
<!-- 视频示例 -->
<video controls poster="poster.jpg" width="640">
    <!-- 按优先级排列 -->
    <source src="video.mp4" type="video/mp4">
    <source src="video.webm" type="video/webm">
    <source src="video.ogg" type="video/ogg">
    <!-- 兼容旧浏览器 -->
    <object data="video.swf" type="application/x-shockwave-flash">
        <param name="movie" value="video.swf">
    </object>
    <!-- 降级文本 -->
    <p>您的浏览器不支持视频播放，请 <a href="video.mp4">下载视频</a></p>
</video>

<!-- 音频示例 -->
<audio controls>
    <source src="music.mp3" type="audio/mpeg">
    <source src="music.ogg" type="audio/ogg">
    <p>您的浏览器不支持音频播放，请 <a href="music.mp3">下载音频</a></p>
</audio>
```

## 常用音视频格式

### 视频格式

| 格式 | 特点 | 浏览器支持 |
|------|------|-----------|
| MP4 | 压缩率高，兼容性好 | 广泛支持 |
| WebM | 开源格式，体积小 | 较好支持 |
| OGG | 开源格式 | 有限支持 |

### 音频格式

| 格式 | 特点 | 浏览器支持 |
|------|------|-----------|
| MP3 | 压缩率高，兼容性好 | 广泛支持 |
| WAV | 无压缩，音质好 | 广泛支持 |
| OGG | 开源格式 | 有限支持 |

## 响应式媒体

```css
video, audio {
    max-width: 100%;
    height: auto;
}
```

## 完整示例

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>媒体标签示例</title>
    <style>
        .media-container {
            max-width: 800px;
            margin: 20px auto;
        }
        video, audio {
            width: 100%;
            margin: 10px 0;
        }
    </style>
</head>
<body>
    <div class="media-container">
        <h2>视频示例</h2>
        <video controls poster="poster.jpg" width="640">
            <source src="sample.mp4" type="video/mp4">
            <source src="sample.webm" type="video/webm">
            <p>您的浏览器不支持视频播放</p>
        </video>

        <h2>音频示例</h2>
        <audio controls>
            <source src="music.mp3" type="audio/mpeg">
            <source src="music.ogg" type="audio/ogg">
            <p>您的浏览器不支持音频播放</p>
        </audio>

        <h2>PDF示例</h2>
        <embed src="document.pdf" type="application/pdf" width="100%" height="500">
    </div>
</body>
</html>
```

## 总结

- `<video>` 用于嵌入视频，支持多种属性控制播放
- `<audio>` 用于嵌入音频
- 提供多种格式以确保跨浏览器兼容
- 使用 `<source>` 标签指定多个媒体源
- 始终提供降级方案（如下载链接）
