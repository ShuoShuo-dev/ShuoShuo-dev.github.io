# 28.CSS性能优化

## 选择器优化

### 避免过深嵌套
```css
/* 不好 */
.nav ul li a span { }

/* 好 */
.nav-link { }
```

### 避免通配符
```css
/* 不好 */
* { margin: 0; }

/* 好 */
body, h1, h2, p, ul { margin: 0; }
```

### 使用具体选择器
```css
/* 不好 */
div p { }

/* 好 */
.article-text { }
```

## 渲染优化

### 减少重排和重绘
```css
/* 好：使用 transform 和 opacity */
.good {
  transform: translateX(100px);
  opacity: 0.5;
}

/* 不好：触发布局 */
.bad {
  left: 100px;
  background: red;
}
```

### will-change 属性
```css
.animated {
  will-change: transform, opacity;
}
```

### 使用 GPU 加速
```css
.gpu {
  transform: translateZ(0);
  backface-visibility: hidden;
}
```

## 其他优化

### 使用 inherit
```css
.child {
  color: inherit;  /* 减少代码 */
}
```

### 避免 @import
```css
/* 不好 */
@import url("other.css");

/* 好 */
<link rel="stylesheet" href="other.css">
```

### 使用简写属性
```css
/* 不好 */
margin-top: 10px;
margin-right: 20px;
margin-bottom: 10px;
margin-left: 20px;

/* 好 */
margin: 10px 20px;
```
