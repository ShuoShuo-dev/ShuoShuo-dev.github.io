# 32.CSS层叠和继承

## 层叠规则

当多个样式规则作用于同一元素时，按以下顺序决定：

1. 来源和重要性
2. 优先级（Specificity）
3. 源码顺序

## 来源和重要性

```css
/* 重要性：user agent < user < author < author !important < user !important */
```

### 优先级计算

| 选择器 | 优先级 |
|--------|--------|
| 内联样式 `style=""` | 1000 |
| ID `#header` | 100 |
| 类 `.nav` / 属性 `[type]` / 伪类 `:hover` | 10 |
| 元素 `div` / 伪元素 `::before` | 1 |
| 通配符 `*` | 0 |

```css
/* 计算示例 */
#header .nav li:first-child  /* 100 + 10 + 1 + 10 = 121 */
.nav a:hover                  /* 10 + 10 = 20 */
nav                            /* 1 */
```

## !important

```css
/* 最高优先级，慎用 */
.element {
  color: red !important;
}
```

## 继承

```css
/* 可继承的属性 */
color, font-size, font-family, text-align, visibility, cursor

/* 不可继承的属性 */
width, height, margin, padding, border, background
```

### 强制继承

```css
.child {
  inherit: color;        /* 继承父元素 */
  initial: color;        /* 设为默认值 */
  unset: color;         /* 有值则继承，无则初始值 */
}
```

### inherit

```css
.parent {
  color: #3498db;
}

.child {
  color: inherit;  /* 继承父元素的颜色 */
}
```

### initial

```css
.element {
  color: initial;  /* 设为浏览器默认值 */
}
```

### unset

```css
.element {
  color: unset;  /* 有继承值则继承，否则使用初始值 */
}
```

## all 属性

```css
/* 重置所有属性 */
.element {
  all: initial;   /* 重置为初始值 */
  all: inherit;   /* 继承父元素 */
  all: unset;     /* 混合效果 */
}
```

## 实用技巧

### 1. 利用继承
```css
body {
  font-family: 'Microsoft YaHei', sans-serif;
  color: #333;
}
```

### 2. 重置样式
```css
*, *::before, *::after {
  box-sizing: inherit;
}
```

### 3. 控制继承
```css
.reset-list {
  all: unset;
  list-style: none;
}
```
