# 10.CSS过渡

## 什么是过渡？

过渡（transition）用于在元素的样式状态改变时添加平滑的动画效果。

## transition 属性

```css
.element {
  transition: 属性 时长 缓动 延迟;
}
```

## transition-property 指定属性

```css
/* 单个属性 */
.box {
  transition-property: width;
}

/* 多个属性 */
.box {
  transition-property: width, height, background-color;
}

/* 所有可过渡的属性 */
.box {
  transition-property: all;
}

/* 不指定（默认） */
.box {
  transition-property: none;
}
```

## transition-duration 时长

```css
/* 秒 */
.box {
  transition-duration: 1s;
}

/* 毫秒 */
.box {
  transition-duration: 500ms;
}

/* 多个时长 */
.box {
  transition-duration: 0.5s, 1s;  /* 对应多个属性 */
}
```

## transition-timing-function 缓动函数

```css
.box {
  /* 关键词 */
  transition-timing-function: ease;         /* 慢-快-慢（默认） */
  transition-timing-function: linear;        /* 匀速 */
  transition-timing-function: ease-in;       /* 慢开始 */
  transition-timing-function: ease-out;   /* 慢结束 */
  transition-timing-function: ease-in-out;  /* 慢开始和结束 */
  transition-timing-function: step-start;   /* 直接跳到终点 */
  transition-timing-function: step-end;     /* 在开始前跳一下 */
  transition-timing-function: steps(4);     /* 分4步 */
}

/* 贝塞尔曲线 */
.box {
  transition-timing-function: cubic-bezier(0.25, 0.1, 0.25, 1); /* 等同于ease */
  transition-timing-function: cubic-bezier(0.68, -0.55, 0.27, 1.55); /* 回弹效果 */
}
```

**常用贝塞尔曲线：**
```css
.ease { transition-timing-function: ease; }
.ease-in { transition-timing-function: ease-in; }
.ease-out { transition-timing-function: ease-out; }
.ease-in-out { transition-timing-function: ease-in-out; }
.linear { transition-timing-function: linear; }
.bounce { transition-timing-function: cubic-bezier(0.68, -0.55, 0.27, 1.55); }
.smooth { transition-timing-function: cubic-bezier(0.4, 0, 0.2, 1); }
```

## transition-delay 延迟

```css
.box {
  transition-delay: 0s;      /* 无延迟（默认） */
  transition-delay: 1s;      /* 延迟1秒开始 */
  transition-delay: -0.5s;   /* 提前0.5秒开始（相当于跳过前一半） */
}
```

## transition 简写

```css
/* 完整写法 */
.box {
  transition-property: all;
  transition-duration: 0.3s;
  transition-timing-function: ease;
  transition-delay: 0s;
}

/* 简写 */
.box {
  transition: all 0.3s ease;
}

/* 多属性 */
.box {
  transition: width 0.3s ease, height 0.5s ease-in-out, background-color 0.2s linear;
}

/* all + 其他属性 */
.box {
  transition: all 0.3s ease, width 0.5s ease-in-out;
}
```

## 常用过渡效果

### 1. 按钮悬停效果
```css
.btn {
  background: #3498db;
  color: white;
  padding: 12px 30px;
  border: none;
  border-radius: 4px;
  transition: all 0.3s ease;
  cursor: pointer;
}

.btn:hover {
  background: #2980b9;
  transform: translateY(-2px);
  box-shadow: 0 4px 10px rgba(0, 0, 0, 0.2);
}
```

### 2. 链接悬停效果
```css
a {
  color: #3498db;
  text-decoration: none;
  transition: color 0.3s ease;
}

a:hover {
  color: #e74c3c;
}
```

### 3. 图片放大效果
```css
.img-wrapper {
  overflow: hidden;
  border-radius: 8px;
}

.img-wrapper img {
  width: 100%;
  transition: transform 0.5s ease;
}

.img-wrapper:hover img {
  transform: scale(1.1);
}
```

### 4. 卡片阴影效果
```css
.card {
  background: white;
  border-radius: 8px;
  box-shadow: 0 2px 8px rgba(0, 0, 0, 0.1);
  transition: box-shadow 0.3s ease, transform 0.3s ease;
}

.card:hover {
  box-shadow: 0 8px 25px rgba(0, 0, 0, 0.15);
  transform: translateY(-5px);
}
```

### 5. 菜单展开效果
```css
.menu-item .submenu {
  max-height: 0;
  overflow: hidden;
  transition: max-height 0.3s ease-out;
}

.menu-item:hover .submenu {
  max-height: 500px; /* 设置足够大的值 */
}
```

### 6. 渐变背景切换
```css
.btn-gradient {
  background: linear-gradient(90deg, #667eea, #764ba2);
  background-size: 200% 100%;
  transition: background-position 0.5s ease;
}

.btn-gradient:hover {
  background-position: 100% 0;
}
```

### 7. 边框颜色动画
```css
.input {
  border: 2px solid #ddd;
  transition: border-color 0.3s ease;
}

.input:focus {
  border-color: #3498db;
}
```

### 8. 滑块效果
```css
.slider {
  width: 100px;
  background: #ddd;
  height: 10px;
  border-radius: 5px;
  position: relative;
}

.slider::after {
  content: '';
  position: absolute;
  left: 0;
  top: 0;
  height: 100%;
  width: 0%;
  background: #3498db;
  border-radius: 5px;
  transition: width 0.3s ease;
}

.slider:hover::after {
  width: 50%;
}
```

### 9. 旋转效果
```css
.icon {
  transition: transform 0.3s ease;
}

.icon:hover {
  transform: rotate(180deg);
}
```

### 10. 透明度变化
```css
.overlay {
  opacity: 0;
  transition: opacity 0.3s ease;
}

.parent:hover .overlay {
  opacity: 1;
}
```

## 注意事项

### 1. 不要过渡 display
```css
/* 不推荐：display 过渡无效 */
.box {
  transition: display 0.3s; /* 无效 */
}

/* 正确做法：使用 opacity + visibility */
.box {
  transition: opacity 0.3s, visibility 0.3s;
}
```

### 2. 性能优化
```css
/* 优先过渡不触发重排/重绘的属性 */
.good {
  transition: transform 0.3s, opacity 0.3s; /* 好 */
}

.bad {
  transition: width 0.3s, height 0.3s, background 0.3s; /* 可能触发重排 */
}
```

### 3. 触发时机
```css
/* :hover 触发 */
.box:hover { }

/* :focus 触发 */
.input:focus { }

/* :active 触发 */
.button:active { }

/* :checked 触发（配合label） */
.checkbox:checked + label { }

/* JS class 切换触发 */
.active .box { }
```
