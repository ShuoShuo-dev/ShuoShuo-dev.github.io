# 22.CSS渐变

## 线性渐变

```css
/* 方向 */
background: linear-gradient(to right, red, blue);
background: linear-gradient(to bottom, red, blue);
background: linear-gradient(to top, red, blue);
background: linear-gradient(to left, red, blue);

/* 角度 */
background: linear-gradient(45deg, red, blue);
background: linear-gradient(90deg, red, blue, green);
background: linear-gradient(180deg, red, orange, yellow, green);
```

## 径向渐变

```css
/* 从中心向外 */
background: radial-gradient(circle, red, blue);

/* 椭圆 */
background: radial-gradient(ellipse, red, blue);

/* 指定位置 */
background: radial-gradient(circle at 30% 70%, red, blue);
background: radial-gradient(ellipse at top, red, blue);
background: radial-gradient(ellipse at bottom right, red, blue);
```

## 多色渐变

```css
/* 多颜色 */
background: linear-gradient(to right, red, orange, yellow, green, blue);

/* 彩虹效果 */
background: linear-gradient(
  to right,
  red, orange, yellow, green, blue, indigo, violet
);
```

## 色标渐变

```css
/* 指定位置 */
background: linear-gradient(
  to right,
  red 0%,
  blue 50%,
  green 100%
);

/* 色块效果 */
background: linear-gradient(
  to bottom,
  #3498db 0%,
  #3498db 25%,
  #2ecc71 25%,
  #2ecc71 50%,
  #e74c3c 50%,
  #e74c3c 75%,
  #f39c12 75%,
  #f39c12 100%
);
```

## 重复渐变

```css
/* 条纹 */
background: repeating-linear-gradient(
  45deg,
  #f5f5f5,
  #f5f5f5 10px,
  #fff 10px,
  #fff 20px
);

/* 格子 */
background: 
  repeating-linear-gradient(
    0deg,
    #333 0px,
    #333 10px,
    #fff 10px,
    #fff 20px
  ),
  repeating-linear-gradient(
    90deg,
    #333 0px,
    #333 10px,
    #fff 10px,
    #fff 20px
  );
background-size: 20px 20px;

/* 径向重复 */
background: repeating-radial-gradient(
  circle at center,
  #3498db 0px,
  #3498db 10px,
  #fff 10px,
  #fff 20px
);
```

## 实用渐变

### 1. 按钮渐变
```css
.btn-gradient {
  background: linear-gradient(135deg, #667eea, #764ba2);
  color: white;
  border: none;
}
```

### 2. 卡片顶部渐变
```css
.card-header {
  background: linear-gradient(135deg, #3498db, #2ecc71);
  color: white;
  padding: 20px;
  border-radius: 8px 8px 0 0;
}
```

### 3. 背景渐变
```css
.page-bg {
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  min-height: 100vh;
}
```

### 4. 文字渐变
```css
.gradient-text {
  background: linear-gradient(90deg, #3498db, #2ecc71);
  -webkit-background-clip: text;
  background-clip: text;
  -webkit-text-fill-color: transparent;
}
```

### 5. 光晕效果
```css
.glow {
  background: radial-gradient(
    circle,
    rgba(52, 152, 219, 0.3) 0%,
    transparent 70%
  );
}
```
