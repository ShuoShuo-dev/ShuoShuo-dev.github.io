## 知识点总结

| 技能点 | 应用 |
|--------|------|
| CSS变量 | 主题颜色、间距系统 |
| Flexbox | 导航、卡片布局 |
| Grid | 作品集网格 |
| 伪类选择器 | 悬停效果、动画 |
| 伪元素 | 装饰、图标 |
| CSS动画 | 过渡、变形 |
| 媒体查询 | 响应式设计 |
| 固定定位 | 导航栏、模态框 |
