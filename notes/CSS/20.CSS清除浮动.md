# 20.CSS清除浮动

## 为什么要清除浮动？

浮动元素会脱离文档流，导致父元素高度塌陷。

```html
<div class="parent">
  <div class="child" style="float: left;">子元素1</div>
  <div class="child" style="float: left;">子元素2</div>
  <!-- 父元素高度变为0！ -->
</div>
```

## 清除浮动的方法

### 方法1：空元素清除（不推荐）

```html
<div class="parent">
  <div class="child" style="float: left;">子元素1</div>
  <div class="child" style="float: left;">子元素2</div>
  <div style="clear: both;"></div>  <!-- 空元素 -->
</div>
```

```css
.clear {
  clear: both;
}
```

**缺点：** 增加无意义的HTML元素

### 方法2：伪元素清除（推荐）

```css
.parent::after {
  content: '';
  display: block;
  clear: both;
}
```

**完整写法：**
```css
.clearfix::after {
  content: '';
  display: block;
  clear: both;
  height: 0;
  visibility: hidden;
}

.clearfix {
  *zoom: 1;  /* IE6/7 兼容 */
}
```

### 方法3：overflow清除

```css
.parent {
  overflow: hidden;  /* 或 auto */
  *zoom: 1;  /* IE6/7 兼容 */
}
```

**缺点：** 可能隐藏溢出的内容

### 方法4：display: flow-root（现代方法）

```css
.parent {
  display: flow-root;  /* 新的BFC创建方式 */
}
```

**优点：** 语义清晰，不会有副作用

## 完整清除浮动示例

```css
/* 清除浮动工具类 */
.clearfix::before,
.clearfix::after {
  content: '';
  display: table;
}

.clearfix::after {
  clear: both;
}

.clearfix {
  *zoom: 1;  /* IE6/7 */
}

/* 或更简洁的版本 */
.clearfix::after {
  content: '';
  display: block;
  clear: both;
}
```

## 什么时候需要清除浮动？

### 1. 父元素没有固定高度

```css
/* 需要清除 */
.parent {
  overflow: auto;
}

/* 不需要清除 */
.parent {
  height: 500px;  /* 固定高度 */
}
```

### 2. 父元素使用 background

```css
.parent {
  background: #f5f5f5;
  overflow: auto;  /* 需要清除浮动 */
}
```

### 3. 父元素需要边框

```css
.parent {
  border: 1px solid #ddd;
  overflow: auto;  /* 需要清除浮动 */
}
```

## 现代布局替代方案

如果可以，优先使用 Flexbox 或 Grid 布局，它们不需要清除浮动：

### Flexbox

```css
.parent {
  display: flex;
  flex-wrap: wrap;
  gap: 20px;
}

.child {
  flex: 1 1 200px;
}
```

### Grid

```css
.parent {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  gap: 20px;
}
```

## 清除浮动的实际应用

### 1. 卡片网格

```html
<div class="card-grid clearfix">
  <div class="card" style="float: left; width: 33.33%;">卡片1</div>
  <div class="card" style="float: left; width: 33.33%;">卡片2</div>
  <div class="card" style="float: left; width: 33.33%;">卡片3</div>
</div>
```

```css
.card-grid::after {
  content: '';
  display: block;
  clear: both;
}
```

### 2. 导航栏

```html
<div class="navbar clearfix">
  <div class="logo" style="float: left;">Logo</div>
  <div class="nav-links" style="float: right;">
    <a href="#">链接</a>
  </div>
</div>
```

### 3. 两栏布局

```html
<div class="layout clearfix">
  <div class="sidebar" style="float: left; width: 250px;">侧边栏</div>
  <div class="content" style="margin-left: 250px;">主内容</div>
</div>
```

### 4. 列表布局

```html
<ul class="product-list clearfix">
  <li style="float: left; width: 25%;">产品1</li>
  <li style="float: left; width: 25%;">产品2</li>
  <li style="float: left; width: 25%;">产品3</li>
  <li style="float: left; width: 25%;">产品4</li>
</ul>
```

## BFC 与清除浮动

创建新的 Block Formatting Context 可以包含浮动：

```css
.parent {
  display: flow-root;  /* 创建新的BFC */
  /* 或 */
  overflow: hidden;    /* 创建新的BFC */
  /* 或 */
  position: absolute;  /* 创建新的BFC */
  /* 或 */
  float: left;         /* 创建新的BFC，但会改变布局 */
}
```

## 最佳实践

```css
/* 推荐：使用伪元素清除浮动 */
.clearfix::after {
  content: '';
  display: block;
  clear: both;
}

/* 推荐：使用 Flexbox 替代浮动布局 */
.flex-container {
  display: flex;
  flex-wrap: wrap;
  gap: 20px;
}

/* 推荐：使用 Grid 替代浮动布局 */
.grid-container {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  gap: 20px;
}
```
