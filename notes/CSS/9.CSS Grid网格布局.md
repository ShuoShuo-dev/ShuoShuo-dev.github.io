# 9.CSS Grid网格布局

## 什么是 Grid？

Grid（网格布局）是一种二维布局系统，可以同时处理行和列。

## 基本概念

```
┌─────────────────────────────────────┐
│  Grid Container                      │
│  ┌───────────┬───────────┬─────────┐│
│  │  grid-    │  grid-    │  grid- ││
│  │  item     │  item     │  item  ││
│  │           │           │        ││
│  │           │           │        ││
│  ├───────────┼───────────┼─────────┤│
│  │  grid-    │  grid-    │  grid- ││
│  │  item     │  item     │  item  ││
│  │           │           │        ││
│  └───────────┴───────────┴─────────┘│
└─────────────────────────────────────┘
```

## 启用 Grid

```css
.container {
  display: grid;       /* 块级网格 */
  display: inline-grid; /* 行内网格 */
}
```

## 容器属性

### 1. grid-template-columns 定义列

```css
.container {
  /* 具体值 */
  grid-template-columns: 100px 100px 100px;

  /* 百分比 */
  grid-template-columns: 33.33% 33.33% 33.33%;

  /* repeat() 函数 */
  grid-template-columns: repeat(3, 1fr);  /* 3列，平均分配 */

  /* auto-fit + minmax */
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));

  /* 命名网格线 */
  grid-template-columns: [start] 1fr [middle] 1fr [end];
}
```

### 2. grid-template-rows 定义行

```css
.container {
  grid-template-rows: 100px 200px 100px;
  grid-template-rows: repeat(3, 1fr);
  grid-template-rows: auto 1fr auto;
}
```

### 3. grid-template-areas 区域命名

```css
.container {
  grid-template-areas:
    "header header header"
    "sidebar main main"
    "footer footer footer";
}

.header {
  grid-area: header;
}

.sidebar {
  grid-area: sidebar;
}

.main {
  grid-area: main;
}

.footer {
  grid-area: footer;
}
```

### 4. gap 间距

```css
.container {
  gap: 20px;           /* 行和列间距 */
  gap: 10px 20px;      /* 行间距 列间距 */
  row-gap: 10px;       /* 行间距 */
  column-gap: 20px;     /* 列间距 */
}
```

### 5. justify-items 水平对齐

```css
.container {
  justify-items: stretch;   /* 拉伸填满（默认） */
  justify-items: start;     /* 左对齐 */
  justify-items: end;       /* 右对齐 */
  justify-items: center;   /* 居中 */
}
```

### 6. align-items 垂直对齐

```css
.container {
  align-items: stretch;   /* 拉伸填满（默认） */
  align-items: start;     /* 顶部对齐 */
  align-items: end;       /* 底部对齐 */
  align-items: center;   /* 垂直居中 */
}
```

### 7. place-items 简写

```css
.container {
  place-items: center;        /* align-items + justify-items */
  place-items: start end;
}
```

### 8. justify-content 水平对齐（网格小于容器）

```css
.container {
  justify-content: stretch;      /* 拉伸（默认） */
  justify-content: start;        /* 左对齐 */
  justify-content: end;          /* 右对齐 */
  justify-content: center;       /* 居中 */
  justify-content: space-between;/* 两端对齐 */
  justify-content: space-around; /* 间距相等 */
  justify-content: space-evenly; /* 完全相等 */
}
```

### 9. align-content 垂直对齐

```css
.container {
  align-content: stretch;
  align-content: start;
  align-content: end;
  align-content: center;
  align-content: space-between;
  align-content: space-around;
  align-content: space-evenly;
}
```

### 10. grid-auto-flow 自动放置

```css
.container {
  grid-auto-flow: row;           /* 按行放置（默认） */
  grid-auto-flow: column;       /* 按列放置 */
  grid-auto-flow: dense;        /* 填补空白 */
  grid-auto-flow: row dense;    /* 按行 + 填补 */
}
```

### 11. grid-auto-columns/rows 自动生成轨道

```css
.container {
  grid-auto-columns: 100px;   /* 自动生成的列宽 */
  grid-auto-rows: minmax(100px, auto); /* 自动生成的行高 */
}
```

## 项目属性

### 1. grid-column-start/end 列开始/结束位置

```css
.item {
  grid-column-start: 1;      /* 从第1条列线开始 */
  grid-column-end: 3;        /* 到第3条列线结束 */
  grid-column-end: span 2;   /* 跨2列 */

  /* 简写 */
  grid-column: 1 / 3;
  grid-column: 1 / span 2;
  grid-column: 1;            /* 只写开始位置 */
}
```

### 2. grid-row-start/end 行开始/结束位置

```css
.item {
  grid-row-start: 1;
  grid-row-end: 3;

  /* 简写 */
  grid-row: 1 / 3;
  grid-row: 1 / span 2;
}
```

### 3. grid-area 区域

```css
.item {
  grid-area: header;  /* 使用命名区域 */
  grid-area: 1 / 1 / 3 / 2;  /* row-start / col-start / row-end / col-end */
}
```

### 4. justify-self 单项水平对齐

```css
.item {
  justify-self: stretch;
  justify-self: start;
  justify-self: end;
  justify-self: center;
}
```

### 5. align-self 单项垂直对齐

```css
.item {
  align-self: stretch;
  align-self: start;
  align-self: end;
  align-self: center;
}
```

### 6. place-self 简写

```css
.item {
  place-self: center;       /* align-self + justify-self */
  place-self: start end;
}
```

## fr 单位

`fr` 表示"分数"（fraction），自动分配可用空间：

```css
.container {
  /* 三列，各占1/3 */
  grid-template-columns: 1fr 1fr 1fr;

  /* 简写 */
  grid-template-columns: repeat(3, 1fr);

  /* 混合使用 */
  grid-template-columns: 200px 1fr 1fr;
  /* 第一列固定200px，剩余空间平分 */
}
```

## minmax() 函数

```css
.container {
  /* 最小200px，最大1fr */
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
}
```

## auto-fill vs auto-fit

```css
/* auto-fill：保持最小宽度，允许空白 */
.container {
  grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
}

/* auto-fit：扩展填满，压缩项目 */
.container {
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
}
```

## 常用布局案例

### 1. 响应式网格
```css
.grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 20px;
}
```

### 2. 圣杯布局
```css
.container {
  display: grid;
  grid-template-rows: auto 1fr auto;
  grid-template-columns: 200px 1fr 200px;
  min-height: 100vh;
}

.header { grid-area: 1 / 1 / 2 / 4; }
.sidebar-left { grid-area: 2 / 1 / 3 / 2; }
.main { grid-area: 2 / 2 / 3 / 3; }
.sidebar-right { grid-area: 2 / 3 / 3 / 4; }
.footer { grid-area: 3 / 1 / 4 / 4; }
```

### 3. 瀑布流
```css
.masonry {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
  grid-auto-rows: 10px;
}

.item:nth-child(1) { grid-row: span 25; }
.item:nth-child(2) { grid-row: span 15; }
.item:nth-child(3) { grid-row: span 30; }
```

### 4. 经典页面布局
```css
.page {
  display: grid;
  grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer";
  grid-template-columns: 250px 1fr;
  grid-template-rows: auto 1fr auto;
  min-height: 100vh;
}
```

### 5. 居中布局
```css
.center {
  display: grid;
  place-items: center;  /* 完美居中！ */
}
```

## Grid vs Flexbox

| 特性 | Grid | Flexbox |
|------|------|---------|
| 维度 | 二维（行+列） | 一维（行或列） |
| 适用场景 | 整体布局 | 组件内部布局 |
| 控制方式 | 父容器控制 | 父容器 + 子项控制 |
| 定位 | 基于网格线 | 基于内容流 |
