# 4.CSS盒模型

## 什么是盒模型？

CSS中每个元素都被看作一个矩形盒子，包含：
- Content（内容）
- Padding（内边距）
- Border（边框）
- Margin（外边距）

## 盒模型的组成

```
┌─────────────────────────────────┐
│           Margin                │
│  ┌───────────────────────────┐  │
│  │         Border            │  │
│  │  ┌─────────────────────┐  │  │
│  │  │       Padding       │  │  │
│  │  │  ┌───────────────┐  │  │  │
│  │  │  │    Content     │  │  │  │
│  │  │  └───────────────┘  │  │  │
│  │  └─────────────────────┘  │  │
│  └───────────────────────────┘  │
└─────────────────────────────────┘
```

## width 和 height

```css
/* 具体数值 */
.box {
  width: 300px;
  height: 200px;
}

/* 百分比（相对于父元素） */
.box {
  width: 50%;
  height: 100%;
}

/* auto（自动计算） */
.box {
  width: auto;
  height: auto;
}

/* 最大最小限制 */
.box {
  min-width: 100px;
  max-width: 500px;
  min-height: 50px;
  max-height: 300px;
}
```

## padding（内边距）

```css
/* 四边相同 */
.box {
  padding: 20px;
}

/* 上下 左右 */
.box {
  padding: 10px 20px;
}

/* 上 左右 下 */
.box {
  padding: 10px 20px 30px;
}

/* 上 右 下 左（顺时针） */
.box {
  padding: 10px 15px 20px 25px;
}

/* 分别设置 */
.box {
  padding-top: 10px;
  padding-right: 15px;
  padding-bottom: 20px;
  padding-left: 25px;
}
```

## border（边框）

### 边框宽度
```css
.box {
  border-width: 1px;
  border-top-width: 2px;
}
```

### 边框样式
```css
.box {
  border-style: solid;    /* 实线 */
  border-style: dashed;    /* 虚线 */
  border-style: dotted;    /* 点线 */
  border-style: double;    /* 双线 */
  border-style: groove;    /* 3D凹槽 */
  border-style: ridge;     /* 3D凸槽 */
  border-style: inset;     /* 3D凹边 */
  border-style: outset;    /* 3D凸边 */
  border-style: none;      /* 无边框 */
}
```

### 边框颜色
```css
.box {
  border-color: red;
  border-color: #ff0000;
  border-color: rgb(255, 0, 0);
}
```

### 边框简写
```css
/* 完整写法 */
.box {
  border-width: 1px;
  border-style: solid;
  border-color: #333;
}

/* 简写 */
.box {
  border: 1px solid #333;
}

/* 单边设置 */
.box {
  border-top: 2px solid red;
  border-bottom: 1px dashed blue;
}

/* 圆角边框 */
.box {
  border-radius: 10px;
  border-radius: 50%; /* 圆形 */
}
```

## margin（外边距）

```css
/* 四边相同 */
.box {
  margin: 20px;
}

/* 上下 左右 */
.box {
  margin: 10px 20px;
}

/* 上 左右 下 */
.box {
  margin: 10px 20px 30px;
}

/* 上 右 下 左 */
.box {
  margin: 10px 15px 20px 25px;
}

/* 分别设置 */
.box {
  margin-top: 10px;
  margin-right: 15px;
  margin-bottom: 20px;
  margin-left: 25px;
}

/* 居中（常用） */
.box {
  margin: 0 auto;
}
```

### margin 合并（折叠）

**垂直外边距会合并：**
```css
/* 两个元素上下排列，下面的元素 margin-top 会与上面的 margin-bottom 合并 */
.box1 {
  margin-bottom: 20px;
}

.box2 {
  margin-top: 30px;
}
/* 最终间距是 30px，不是 50px */

/* 合并规则：
 * 1. 两者都是正数 → 取最大值
 * 2. 一正一负 → 取两者之和
 * 3. 都是负数 → 取绝对值最大的
 */
```

**阻止合并：**
```css
/* 给父元素加边框或内边距 */
.parent {
  border-top: 1px solid #ccc;
  /* 或 */
  padding-top: 1px;
}

/* 给父元素加 overflow */
.parent {
  overflow: hidden;
}

/* 给父元素加 display: flex */
.parent {
  display: flex;
}
```

## box-sizing（盒尺寸计算）

### content-box（默认）
```css
/* width = 内容宽度 */
.box {
  width: 200px;
  padding: 20px;
  border: 5px solid #333;
  /* 最终宽度 = 200 + 20*2 + 5*2 = 250px */
  box-sizing: content-box;
}
```

### border-box（推荐）
```css
/* width = 内容 + padding + border */
.box {
  width: 200px;
  padding: 20px;
  border: 5px solid #333;
  /* 最终宽度 = 200px（内容自动压缩） */
  box-sizing: border-box;
}
```

### 全局设置
```css
/* 常用全局重置 */
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

/* 继承 box-sizing */
html {
  box-sizing: border-box;
}

*, *::before, *::after {
  box-sizing: inherit;
}
```

## 盒模型计算示例

```html
<style>
.box {
  width: 200px;
  height: 100px;
  padding: 20px;
  border: 5px solid #333;
  margin: 10px;
  box-sizing: border-box;
}
</style>

<div class="box">
  内容区域
</div>
```

**content-box 下的计算：**
- 内容宽高：200 × 100px
- 内边距：20px × 2
- 边框：5px × 2
- 最终占位：250 × 150px

**border-box 下的计算：**
- 内容宽高：150 × 50px（自动压缩）
- 加上 padding 和 border
- 最终占位：200 × 100px

## display 与盒模型

```css
/* 块级元素 */
div {
  display: block;
  width: 100%; /* 默认占满整行 */
}

/* 行内元素 */
span {
  display: inline;
  /* width/height 设置无效 */
  /* padding/margin 水平有效，垂直不占位 */
}

/* 行内块元素 */
input {
  display: inline-block;
  /* 保持 inline 特性，同时支持宽高 */
}

/* 弹性盒子 */
.container {
  display: flex;
}

/* 网格布局 */
.container {
  display: grid;
}
```

## 实用技巧

### 元素居中
```css
/* 水平居中 */
.center-h {
  margin-left: auto;
  margin-right: auto;
}

/* 已知宽高的垂直居中 */
.center {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}
```

### 创建三角形
```css
.triangle {
  width: 0;
  height: 0;
  border-left: 50px solid transparent;
  border-right: 50px solid transparent;
  border-bottom: 100px solid red;
}
```
