# 3.CSS字体和文本样式

## 字体属性

### 1. font-family 字体族
```css
/* 具体字体名称 */
body {
  font-family: "Microsoft YaHei", "PingFang SC", sans-serif;
}

/* 字体类别 */
p {
  font-family: serif, sans-serif, monospace, cursive, fantasy;
}
```

### 2. font-size 字体大小
```css
/* 具体数值 */
h1 {
  font-size: 32px;
}

/* 相对单位 */
p {
  font-size: 1.2em;
}

small {
  font-size: 0.8rem;
}

/* 关键词 */
.caption {
  font-size: small;
}

/* 百分比 */
.text {
  font-size: 120%;
}
```

### 3. font-weight 字体粗细
```css
.normal {
  font-weight: normal; /* 400 */
}

.bold {
  font-weight: bold; /* 700 */
}

.bolder {
  font-weight: bolder; /* 比继承值粗 */
}

.lighter {
  font-weight: lighter; /* 比继承值细 */
}

/* 数值 */
.thin {
  font-weight: 100;
}

.light {
  font-weight: 300;
}

.normal {
  font-weight: 400;
}

.medium {
  font-weight: 500;
}

.semibold {
  font-weight: 600;
}

.bold {
  font-weight: 700;
}

.extrabold {
  font-weight: 800;
}

.black {
  font-weight: 900;
}
```

### 4. font-style 字体样式
```css
.normal {
  font-style: normal;
}

.italic {
  font-style: italic; /* 斜体 */
}

.oblique {
  font-style: oblique; /* 倾斜的文字 */
}
```

### 5. font-variant 小型大写字母
```css
.variant {
  font-variant: small-caps;
}
```

### 6. font 简写属性
```css
/* 按顺序: style variant weight size family */
p {
  font: italic small-caps bold 16px/1.5 "Microsoft YaHei", sans-serif;
}

/* 最简写法 */
body {
  font: 16px "Microsoft YaHei", sans-serif;
}
```

## 文本属性

### 1. color 文本颜色
```css
/* 颜色名称 */
p {
  color: red;
}

/* HEX */
h1 {
  color: #ff0000;
}

/* 简写 */
h2 {
  color: #f00;
}

/* RGB */
span {
  color: rgb(255, 0, 0);
}

/* RGBA（带透明） */
div {
  color: rgba(255, 0, 0, 0.5);
}

/* HSL */
em {
  color: hsl(0, 100%, 50%);
}
```

### 2. text-align 文本对齐
```css
.left {
  text-align: left;
}

.center {
  text-align: center;
}

.right {
  text-align: right;
}

.justify {
  text-align: justify; /* 两端对齐 */
}
```

### 3. line-height 行高
```css
/* 具体数值 */
p {
  line-height: 1.8;
}

/* 百分比 */
p {
  line-height: 150%;
}

/* 无单位数字（相对于字体大小） */
p {
  line-height: 1.5;
}

/* 常用写法：垂直居中文字 */
.box {
  height: 100px;
  line-height: 100px;
}
```

### 4. text-decoration 文本装饰
```css
.none {
  text-decoration: none; /* 无装饰 */
}

.underline {
  text-decoration: underline; /* 下划线 */
}

.overline {
  text-decoration: overline; /* 上划线 */
}

.line-through {
  text-decoration: line-through; /* 删除线 */
}

/* 多重装饰 */
.combined {
  text-decoration: underline overline;
}

/* 样式变体 */
.solid {
  text-decoration: underline solid red;
}

.dashed {
  text-decoration: underline dashed red;
}

.dotted {
  text-decoration: underline dotted red;
}

.wavy {
  text-decoration: underline wavy red;
}
```

### 5. text-transform 文本转换
```css
.uppercase {
  text-transform: uppercase; /* 全大写 */
}

.lowercase {
  text-transform: lowercase; /* 全小写 */
}

.capitalize {
  text-transform: capitalize; /* 首字母大写 */
}

.full-width {
  text-transform: full-width; /* 全角字符 */
}
```

### 6. letter-spacing 字符间距
```css
.normal {
  letter-spacing: normal;
}

.wide {
  letter-spacing: 2px;
}

.tight {
  letter-spacing: -1px;
}
```

### 7. word-spacing 单词间距
```css
.normal {
  word-spacing: normal;
}

.wide {
  word-spacing: 5px;
}
```

### 8. text-indent 首行缩进
```css
p {
  text-indent: 2em; /* 首行缩进2个字 */
}

/* 悬挂缩进 */
.indent {
  text-indent: -2em;
  padding-left: 2em;
}
```

### 9. white-space 空白处理
```css
.normal {
  white-space: normal;
}

.pre {
  white-space: pre; /* 保留所有空白和换行 */
}

.nowrap {
  white-space: nowrap; /* 不换行 */
}

.pre-wrap {
  white-space: pre-wrap; /* 保留空白，但正常换行 */
}

.pre-line {
  white-space: pre-line; /* 合并空白，保留换行 */
}
```

### 10. word-break 词内断行
```css
.normal {
  word-break: normal;
}

.break-all {
  word-break: break-all; /* 任意字符间断行 */
}

.keep-all {
  word-break: keep-all; /* 整词断行（中文不停） */
}

.break-word {
  word-break: break-word; /* 长单词断行 */
}
```

### 11. overflow-wrap 换行
```css
.normal {
  overflow-wrap: normal;
}

.break-word {
  overflow-wrap: break-word; /* 长单词断行 */
}
```

## 文本阴影

```css
/* 基本阴影 */
.shadow {
  text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.3);
}

/* 多层阴影 */
.multi {
  text-shadow: 
    2px 2px 0 #ff0000,
    4px 4px 0 #00ff00,
    6px 6px 0 #0000ff;
}

/* 发光效果 */
.glow {
  text-shadow: 0 0 10px #ff6600, 0 0 20px #ff6600;
}
```

## 文本溢出处理

```css
/* 单行文本溢出 */
.single-line {
  overflow: hidden;
  text-overflow: ellipsis;
  white-space: nowrap;
}

/* 多行文本溢出 */
.multi-line {
  display: -webkit-box;
  -webkit-box-orient: vertical;
  -webkit-line-clamp: 3; /* 显示3行 */
  overflow: hidden;
  text-overflow: ellipsis;
}
```

## 常用文本样式组合

```css
/* 正文样式 */
.article {
  font-family: "Microsoft YaHei", sans-serif;
  font-size: 16px;
  line-height: 1.8;
  color: #333;
  text-align: justify;
}

/* 标题样式 */
.heading {
  font-weight: bold;
  line-height: 1.3;
  margin-bottom: 0.5em;
}

h1 {
  font-size: 2em;
}

h2 {
  font-size: 1.5em;
}

h3 {
  font-size: 1.2em;
}
```
