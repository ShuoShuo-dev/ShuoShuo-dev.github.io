# 31.CSS单位

## 绝对单位

| 单位 | 说明 |
|------|------|
| px | 像素，相对显示器分辨率 |
| cm | 厘米 |
| mm | 毫米 |
| in | 英寸 (1in = 96px) |
| pt | 点 (1pt = 1/72in) |
| pc | 派卡 (1pc = 12pt) |

## 相对单位

### 1. em

```css
/* 相对于父元素的 font-size */
.parent {
  font-size: 16px;
}

.child {
  font-size: 1.5em;  /* 24px */
}
```

### 2. rem

```css
/* 相对于根元素 html 的 font-size */
html {
  font-size: 16px;
}

.element {
  font-size: 1.5rem;  /* 24px */
}

/* 推荐使用 rem */
```

### 3. %

```css
/* 相对于父元素 */
.parent {
  width: 200px;
}

.child {
  width: 50%;  /* 100px */
}
```

### 4. vw / vh

```css
/* vw: 视口宽度的1% */
.full-width {
  width: 100vw;
}

/* vh: 视口高度的1% */
.full-height {
  height: 100vh;
}
```

### 5. vmin / vmax

```css
/* vmin: vw 和 vh 中较小值 */
.square {
  width: 100vmin;
  height: 100vmin;
}

/* vmax: vw 和 vh 中较大值 */
.square {
  width: 100vmax;
  height: 100vmax;
}
```

### 6. ch

```css
/* 相对于 "0" 的宽度 */
.input {
  width: 20ch;
}
```

### 7. ex

```css
/* 相对于字母 "x" 的高度 */
.box {
  font-size: 2ex;
}
```

## calc() 计算

```css
/* 混合单位计算 */
.box {
  width: calc(100% - 40px);
  height: calc(100vh - 200px);
}

/* 运算 */
.margin {
  margin: calc(10px + 20px);
}
```

## clamp() 限制

```css
/* clamp(最小值, 首选值, 最大值) */
.text {
  font-size: clamp(14px, 2vw, 24px);
}
```

## 常用场景

```css
/* 响应式字体 */
.text {
  font-size: clamp(1rem, 2.5vw, 2rem);
}

/* 响应式间距 */
.section {
  padding: clamp(20px, 5vw, 60px);
}

/* 固定宽高比 */
.video {
  width: min(100%, 800px);
  aspect-ratio: 16/9;
}

/* 流体间距 */
.grid {
  gap: clamp(10px, 2vw, 30px);
}
```

## 单位对比

| 单位 | 相对性 | 适用场景 |
|------|--------|----------|
| px | 绝对 | 边框、阴影 |
| rem | 根元素 | 字体、间距 |
| em | 父元素 | 组件内字体 |
| % | 父元素 | 宽度、高度 |
| vw/vh | 视口 | 全屏布局 |
| vmin/vmax | 视口 | 响应式正方形 |
