# 12.CSS transform

## 什么是 transform？

transform 属性允许你旋转、缩放、倾斜或平移元素。

## transform 值

```css
.element {
  transform: none;                  /* 默认，无变换 */
  transform: rotate(45deg);         /* 旋转 */
  transform: scale(1.5);            /* 缩放 */
  transform: translate(50px, 100px); /* 平移 */
  transform: skew(20deg, 10deg);    /* 倾斜 */
  transform: matrix(1, 0, 0, 1, 0, 0); /* 矩阵变换 */
}
```

## 1. rotate 旋转

```css
.box {
  transform: rotate(45deg);        /* 顺时针旋转45度 */
  transform: rotate(-45deg);       /* 逆时针旋转45度 */
  transform: rotateX(180deg);      /* 绕X轴旋转（3D） */
  transform: rotateY(180deg);       /* 绕Y轴旋转（3D） */
  transform: rotateZ(90deg);        /* 绕Z轴旋转（3D） */
  transform: rotate3d(1, 1, 1, 45deg); /* 3D旋转 */
}
```

## 2. scale 缩放

```css
.box {
  transform: scale(2);              /* 宽高都放大2倍 */
  transform: scale(0.5);            /* 宽高都缩小一半 */
  transform: scale(1.5, 2);        /* 宽1.5倍，高2倍 */
  transform: scaleX(2);            /* 只缩放宽度 */
  transform: scaleY(0.5);          /* 只缩放高度 */
  transform: scaleZ(2);            /* Z轴缩放（3D） */
  transform: scale3d(1, 2, 3);     /* X、Y、Z轴缩放 */
}
```

## 3. translate 平移

```css
.box {
  transform: translate(50px);              /* 沿X轴移动50px */
  transform: translate(50px, 100px);        /* 沿X轴50px，Y轴100px */
  transform: translateX(50px);              /* 只沿X轴移动 */
  transform: translateY(100px);            /* 只沿Y轴移动 */
  transform: translateZ(50px);             /* Z轴移动（3D） */
  transform: translate3d(50px, 100px, 50px); /* 3D移动 */

  /* 百分比（相对于自身尺寸） */
  transform: translateX(50%);   /* 向右移动自身宽度的50% */
  transform: translateY(-100%); /* 向上移动自身高度的100% */
}
```

## 4. skew 倾斜

```css
.box {
  transform: skew(20deg);               /* X轴倾斜20度 */
  transform: skew(20deg, 10deg);        /* X轴20度，Y轴10度 */
  transform: skewX(20deg);              /* 只X轴倾斜 */
  transform: skewY(10deg);              /* 只Y轴倾斜 */
}
```

## 5. matrix 矩阵变换

```css
/* matrix(a, b, c, d, e, f) */
/* a,d: 缩放 */
/* b,c: 倾斜 */
/* e,f: 平移 */
.box {
  transform: matrix(1, 0, 0, 1, 0, 0);     /* 默认 */
  transform: matrix(1.5, 0, 0, 1.5, 0, 0); /* 放大1.5倍 */
}
```

## 6. perspective 透视

```css
.container {
  perspective: 1000px; /* 3D透视距离 */
}

.box {
  transform: rotateX(45deg);
}
```

## 7. transform-origin 变换原点

```css
.box {
  transform-origin: center center;      /* 默认：中心 */
  transform-origin: top left;          /* 左上 */
  transform-origin: bottom right;     /* 右下 */
  transform-origin: 50% 50%;          /* 百分比 */
  transform-origin: 100px 100px;      /* 具体数值 */
  transform-origin: 0 0;              /* 左上角 */
  transform-origin: 100% 100%;         /* 右下角 */
}
```

## 8. transform-style 3D呈现

```css
.container {
  transform-style: flat;       /* 平面呈现（默认） */
  transform-style: preserve-3d; /* 保持3D效果 */
}
```

## 9. backface-visibility 背面可见性

```css
.box {
  backface-visibility: visible;  /* 可见（默认） */
  backface-visibility: hidden;   /* 隐藏背面 */
}
```

## 多重变换

```css
.box {
  /* 多个变换可以一起写 */
  transform: rotate(45deg) scale(1.5) translate(100px, 50px);
}

/* 变换顺序会影响结果！ */
/* 先平移后旋转 vs 先旋转后平移效果不同 */
```

## 常用效果

### 1. 居中元素
```css
.center {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}
```

### 2. 卡片翻转
```css
.card {
  position: relative;
  transform-style: preserve-3d;
}

.card-front,
.card-back {
  position: absolute;
  backface-visibility: hidden;
}

.card-back {
  transform: rotateY(180deg);
}

.card:hover {
  transform: rotateY(180deg);
}
```

### 3. 3D立方体
```css
.cube {
  width: 100px;
  height: 100px;
  position: relative;
  transform-style: preserve-3d;
  transform: rotateX(-30deg) rotateY(30deg);
}

.face {
  position: absolute;
  width: 100px;
  height: 100px;
  backface-visibility: hidden;
}

.front  { transform: translateZ(50px); }
.back   { transform: rotateY(180deg) translateZ(50px); }
.right  { transform: rotateY(90deg) translateZ(50px); }
.left   { transform: rotateY(-90deg) translateZ(50px); }
.top    { transform: rotateX(90deg) translateZ(50px); }
.bottom { transform: rotateX(-90deg) translateZ(50px); }
```

### 4. 按钮悬停效果
```css
.btn {
  transition: transform 0.3s ease;
}

.btn:hover {
  transform: scale(1.1);
}

.btn:active {
  transform: scale(0.95);
}
```

### 5. 图片放大
```css
.img-container {
  overflow: hidden;
}

.img-container img {
  transition: transform 0.5s ease;
}

.img-container:hover img {
  transform: scale(1.2);
}
```

### 6. 旋转加载
```css
.loading {
  width: 50px;
  height: 50px;
  border: 4px solid #ddd;
  border-top-color: #3498db;
  border-radius: 50%;
  animation: spin 1s linear infinite;
}

@keyframes spin {
  to { transform: rotate(360deg); }
}
```

### 7. 3D倾斜效果
```css
.tilt-card {
  transition: transform 0.3s ease;
}

.tilt-card:hover {
  transform: perspective(1000px) rotateX(5deg) rotateY(-5deg);
}
```

### 8. 下拉菜单箭头
```css
.dropdown::after {
  content: '';
  display: inline-block;
  margin-left: 5px;
  border: solid #333;
  border-width: 0 2px 2px 0;
  padding: 3px;
  transform: rotate(45deg);
  transition: transform 0.3s ease;
}

.dropdown.open::after {
  transform: rotate(225deg);
}
```
