# 5.CSS背景

## background-color 背景颜色

```css
.box {
  background-color: red;
  background-color: #ff0000;
  background-color: rgb(255, 0, 0);
  background-color: rgba(255, 0, 0, 0.5);
  background-color: transparent; /* 默认透明 */
}
```

## background-image 背景图片

```css
.box {
  background-image: url('bg.jpg');
  background-image: url('pattern.png');
}

/* 多张背景图 */
.multi-bg {
  background-image: 
    url('foreground.png'),
    url('middleground.png'),
    url('background.jpg');
}
```

## background-repeat 背景平铺

```css
.box {
  background-repeat: repeat;      /* 默认：水平和垂直都平铺 */
  background-repeat: no-repeat;    /* 不平铺 */
  background-repeat: repeat-x;     /* 只水平平铺 */
  background-repeat: repeat-y;     /* 只垂直平铺 */
  background-repeat: space;        /* 平铺，图片间距自动调整 */
  background-repeat: round;        /* 平铺，图片尺寸自动调整 */
}
```

## background-position 背景定位

```css
.box {
  /* 关键词 */
  background-position: top;
  background-position: bottom;
  background-position: left;
  background-position: right;
  background-position: center;

  /* 两个值 */
  background-position: top center;      /* 上中 */
  background-position: bottom right;   /* 右下 */
  background-position: center left;    /* 左中 */

  /* 具体数值 */
  background-position: 20px 30px;      /* x偏移 y偏移 */
  background-position: -10px 50px;    /* 负值也有效 */
}

/* 百分比定位（相对于容器） */
.box {
  background-position: 50% 50%;       /* 居中 */
  background-position: 0% 0%;          /* 左上 */
  background-position: 100% 100%;     /* 右下 */
}
```

## background-size 背景尺寸

```css
.box {
  /* 具体尺寸 */
  background-size: 200px 100px;
  background-size: 50%;                 /* 相对于容器宽度，高度自动 */
  background-size: 80% 60%;

  /* 关键词 */
  background-size: auto;               /* 保持原始尺寸 */
  background-size: cover;               /* 覆盖整个容器，可能被裁剪 */
  background-size: contain;            /* 完整显示，可能留白 */
}

/* cover vs contain */
.box-cover {
  background-size: cover;
  /* 扩展图片到完全覆盖容器
   * 图片可能超出容器被裁剪
   * 保持宽高比
   */
}

.box-contain {
  background-size: contain;
  /* 完整显示整张图片
   * 可能出现空白
   * 保持宽高比
   */
}
```

## background-attachment 背景固定

```css
.box {
  background-attachment: scroll;        /* 随元素滚动（默认） */
  background-attachment: fixed;        /* 固定，相对于视口 */
  background-attachment: local;        /* 随元素内容滚动 */
}

/* 视差效果示例 */
.parallax {
  background-image: url('mountains.jpg');
  background-attachment: fixed;
  background-size: cover;
  background-position: center;
}
```

## background-origin 背景定位参考

```css
.box {
  background-origin: padding-box;      /* 从 padding 开始（默认） */
  background-origin: border-box;       /* 从 border 开始 */
  background-origin: content-box;      /* 从 content 开始 */
}
```

## background-clip 背景裁剪

```css
.box {
  background-clip: border-box;         /* 显示到 border（默认） */
  background-clip: padding-box;        /* 显示到 padding */
  background-clip: content-box;        /* 只显示 content */
  background-clip: text;               /* 只显示文字内（需要 -webkit-） */
}
```

## background 简写

```css
/* 按顺序：color image repeat position/size attachment */
.box {
  background: 
    #f5f5f5                          /* 颜色 */
    url('image.jpg')                  /* 图片 */
    no-repeat                         /* 平铺 */
    center center                     /* 定位 */
    / 100% 100%                       /* 尺寸 */
    fixed;                            /* 固定 */
}

/* 常用简写 */
.hero {
  background: url('hero.jpg') no-repeat center / cover;
}

.button {
  background: #3498db;
}
```

## 多重背景

```css
.multi-bg {
  background: 
    /* 第1层：渐变 */
    linear-gradient(135deg, #667eea 0%, #764ba2 100%),
    /* 第2层：图片 */
    url('photo.jpg'),
    /* 第3层：颜色 */
    #333;
  
  background-size: 
    100% 100%,
    cover,
    auto;
  
  background-position: 
    center,
    center,
    center;
}
```

## 渐变背景

### 线性渐变
```css
.linear {
  /* 方向 */
  background: linear-gradient(to right, red, blue);
  background: linear-gradient(to bottom right, red, blue);
  background: linear-gradient(45deg, red, blue);
  background: linear-gradient(180deg, red, orange, yellow, green);
}

/* 多色渐变 */
.rainbow {
  background: linear-gradient(
    to right,
    red 0%,
    orange 25%,
    yellow 50%,
    green 75%,
    blue 100%
  );
}
```

### 径向渐变
```css
.radial {
  /* 从中心向外 */
  background: radial-gradient(circle, red, blue);
  
  /* 指定位置开始 */
  background: radial-gradient(circle at 30% 70%, red, blue);
  
  /* 椭圆 */
  background: radial-gradient(ellipse, red, blue);
}
```

### 重复渐变
```css
/* 条纹背景 */
.stripe {
  background: repeating-linear-gradient(
    45deg,
    #f5f5f5,
    #f5f5f5 10px,
    #fff 10px,
    #fff 20px
  );
}

/* 格子图案 */
.checker {
  background: 
    repeating-linear-gradient(
      0deg,
      #333 0px,
      #333 10px,
      #fff 10px,
      #fff 20px
    ),
    repeating-linear-gradient(
      90deg,
      #333 0px,
      #333 10px,
      #fff 10px,
      #fff 20px
    );
  background-size: 20px 20px;
}
```

## 实用背景效果

### 半透明背景
```css
.overlay {
  background: rgba(0, 0, 0, 0.5);  /* 黑色50%透明 */
}

.glass {
  background: rgba(255, 255, 255, 0.2);
  backdrop-filter: blur(10px);
}
```

### 噪点纹理
```css
.noise {
  background-image: url('noise.png');
  background-color: #f5f5f5;
}
```

### 网点图案
```css
.dots {
  background-image: radial-gradient(#333 1px, transparent 1px);
  background-size: 20px 20px;
}
```

### 渐变阴影
```css
.gradient-shadow {
  position: relative;
}

.gradient-shadow::after {
  content: '';
  position: absolute;
  bottom: -20px;
  left: 10%;
  width: 80%;
  height: 20px;
  background: radial-gradient(ellipse, rgba(0,0,0,0.3), transparent);
}
```
