# 16.CSS变量

## 什么是CSS变量？

CSS变量（自定义属性）允许你存储值以便重复使用。

## 定义变量

```css
/* 在 :root 中定义（全局变量） */
:root {
  --primary-color: #3498db;
  --font-size: 16px;
  --spacing: 20px;
  --border-radius: 8px;
}

/* 在选择器中定义（局部变量） */
.card {
  --card-bg: #fff;
  --card-padding: 20px;
}
```

## 使用变量

```css
.button {
  background: var(--primary-color);
  font-size: var(--font-size);
  padding: var(--spacing);
  border-radius: var(--border-radius);
}
```

## 变量默认值

```css
.element {
  color: var(--text-color, #333);  /* 变量不存在时使用默认值 */
}

.element {
  padding: var(--card-padding, 20px);
}
```

## 变量继承

```css
:root {
  --base-color: #3498db;
}

.card {
  background: var(--base-color);  /* 继承全局变量 */
}

.special-card {
  --base-color: #e74c3c;  /* 覆盖局部变量 */
  background: var(--base-color);  /* 使用新的局部变量 */
}
```

## calc() 与变量

```css
:root {
  --spacing: 10px;
}

.element {
  margin: calc(var(--spacing) * 2);  /* 20px */
  padding: calc(var(--spacing) + 5px);  /* 15px */
}
```

## 响应式变量

```css
:root {
  --font-size: 16px;
}

@media (min-width: 768px) {
  :root {
    --font-size: 18px;
  }
}

@media (min-width: 1200px) {
  :root {
    --font-size: 20px;
  }
}

body {
  font-size: var(--font-size);
}
```

## JavaScript 操作变量

```javascript
// 获取变量
const root = document.documentElement;
const color = getComputedStyle(root).getPropertyValue('--primary-color');

// 设置变量
root.style.setProperty('--primary-color', '#e74c3c');

// 设置变量（带单位）
root.style.setProperty('--spacing', '30px');

// 删除变量
root.style.removeProperty('--primary-color');
```

## 实际应用案例

### 1. 主题切换

```css
:root {
  --bg-color: #ffffff;
  --text-color: #333333;
  --primary-color: #3498db;
}

[data-theme="dark"] {
  --bg-color: #1a1a1a;
  --text-color: #ffffff;
  --primary-color: #5dade2;
}

body {
  background: var(--bg-color);
  color: var(--text-color);
}

.button {
  background: var(--primary-color);
}
```

### 2. 组件样式

```css
.btn {
  --btn-bg: #3498db;
  --btn-color: white;
  --btn-padding: 12px 24px;
  --btn-radius: 4px;

  background: var(--btn-bg);
  color: var(--btn-color);
  padding: var(--btn-padding);
  border-radius: var(--btn-radius);
  border: none;
  cursor: pointer;
}

.btn:hover {
  filter: brightness(1.1);
}

.btn-primary {
  --btn-bg: #3498db;
}

.btn-success {
  --btn-bg: #2ecc71;
}

.btn-danger {
  --btn-bg: #e74c3c;
}
```

### 3. 间距系统

```css
:root {
  --space-xs: 5px;
  --space-sm: 10px;
  --space-md: 20px;
  --space-lg: 40px;
  --space-xl: 80px;
}

.section {
  padding: var(--space-lg) 0;
}

.card {
  padding: var(--space-md);
  margin-bottom: var(--space-sm);
}
```

### 4. 阴影系统

```css
:root {
  --shadow-sm: 0 1px 2px rgba(0, 0, 0, 0.1);
  --shadow-md: 0 4px 6px rgba(0, 0, 0, 0.1);
  --shadow-lg: 0 10px 25px rgba(0, 0, 0, 0.15);
  --shadow-xl: 0 20px 50px rgba(0, 0, 0, 0.2);
}

.card {
  box-shadow: var(--shadow-md);
}

.card:hover {
  box-shadow: var(--shadow-lg);
}
```

### 5. 动画配置

```css
:root {
  --transition-fast: 0.15s;
  --transition-normal: 0.3s;
  --transition-slow: 0.5s;
}

.button {
  transition: all var(--transition-normal) ease;
}

.modal {
  transition: opacity var(--transition-normal) ease,
              transform var(--transition-normal) ease;
}
```
