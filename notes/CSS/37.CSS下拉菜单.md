# 37.CSS下拉菜单

## 基本下拉菜单

```css
.dropdown {
  position: relative;
  display: inline-block;
}

.dropdown-content {
  display: none;
  position: absolute;
  background: white;
  min-width: 200px;
  box-shadow: 0 4px 10px rgba(0, 0, 0, 0.1);
  z-index: 1000;
}

.dropdown:hover .dropdown-content {
  display: block;
}

.dropdown-content a {
  display: block;
  padding: 12px 15px;
  color: #333;
  text-decoration: none;
}

.dropdown-content a:hover {
  background: #f5f5f5;
}
```

## 带箭头下拉

```css
.dropdown-btn::after {
  content: '▼';
  font-size: 10px;
  margin-left: 5px;
}

.dropdown-content {
  border-top: 3px solid #3498db;
}
```

## 多级下拉

```css
.sub-dropdown {
  position: relative;
}

.sub-dropdown .submenu {
  display: none;
  position: absolute;
  left: 100%;
  top: 0;
  background: white;
  min-width: 180px;
  box-shadow: 0 4px 10px rgba(0, 0, 0, 0.1);
}

.sub-dropdown:hover .submenu {
  display: block;
}
```

## 动画下拉

```css
.dropdown-content {
  opacity: 0;
  visibility: hidden;
  transform: translateY(-10px);
  transition: all 0.3s ease;
}

.dropdown:hover .dropdown-content {
  opacity: 1;
  visibility: visible;
  transform: translateY(0);
}
```

## 点击触发下拉

```css
.dropbtn {
  background: #3498db;
  color: white;
  padding: 12px 20px;
  border: none;
  cursor: pointer;
}

.dropbtn:focus + .dropdown-content {
  display: block;
}
```

## 实用技巧

```css
/* 带分隔线 */
.dropdown-content a:nth-child(4) {
  border-top: 1px solid #eee;
  margin-top: 5px;
  padding-top: 17px;
}

/* 图标+文字 */
.dropdown-content a::before {
  content: '→';
  margin-right: 10px;
  color: #3498db;
}
```
