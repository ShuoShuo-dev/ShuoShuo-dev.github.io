# 30.CSS新特性

## Container Queries（容器查询）

```css
/* 定义容器 */
.card-container {
  container-type: inline-size;
}

/* 根据容器大小应用样式 */
@container (min-width: 400px) {
  .card {
    display: flex;
  }
}
```

## :has() 选择器

```css
/* 父元素选择器 */
.parent:has(.child) {
  background: blue;
}

/* 表单验证 */
form:has(input:invalid) {
  border-color: red;
}
```

## @layer 级联层

```css
@layer base {
  h1 { color: blue; }
}

@layer components {
  h1 { color: red; }
}
```

## :is() 和 :where()

```css
/* :is() - 匹配任一选择器 */
:is(h1, h2, h3) {
  font-weight: bold;
}

/* :where() - 优先级为0 */
:where(h1, h2, h3) {
  font-weight: bold;
}
```

## :focus-visible

```css
/* 仅键盘焦点时显示焦点环 */
:focus:not(:focus-visible) {
  outline: none;
}

:focus-visible {
  outline: 2px solid #3498db;
}
```

## color-mix() 颜色混合

```css
.mixed {
  color: color-mix(in srgb, #3498db 50%, #2ecc71);
}
```

## 颜色相对语法

```css
.element {
  background: hsl(from blue 30deg saturation +20% lightness -10%);
}
```

## 滚动条样式

```css
::-webkit-scrollbar {
  width: 10px;
  height: 10px;
}

::-webkit-scrollbar-track {
  background: #f1f1f1;
}

::-webkit-scrollbar-thumb {
  background: #888;
  border-radius: 5px;
}
```

## aspect-ratio 宽高比

```css
.video {
  aspect-ratio: 16 / 9;
}
```

## gap 属性扩展

```css
/* gap 现在可用于 flex 和 grid 以外 */
.flex-container {
  display: flex;
  gap: 20px;
}
```

## logical 属性

```css
.element {
  /* 而不是 margin-left/margin-right */
  margin-inline: 20px;
  /* 而不是 margin-top/margin-bottom */
  margin-block: 20px;
}
```

## 环境变量

```css
.safe-area {
  padding-top: env(safe-area-inset-top);
  padding-bottom: env(safe-area-inset-bottom);
}
```

## 触摸操作

```css
.touchable {
  touch-action: manipulation;
}
```

## 其他实用特性

```css
/* 粘性定位更稳定 */
.sticky {
  position: sticky;
  top: 0;
}

/* backdrop-filter 背景模糊 */
.glass {
  backdrop-filter: blur(10px);
  background: rgba(255, 255, 255, 0.5);
}

/* 插入符号样式 */
.editable::selection {
  background: #3498db;
  color: white;
}
```
