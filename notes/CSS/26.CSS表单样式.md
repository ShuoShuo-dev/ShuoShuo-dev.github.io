# 26.CSS表单样式

## 输入框样式

```css
input[type="text"],
input[type="email"],
input[type="password"],
input[type="number"],
textarea {
  width: 100%;
  padding: 12px 15px;
  border: 2px solid #ddd;
  border-radius: 4px;
  font-size: 16px;
  transition: border-color 0.3s;
}

input:focus,
textarea:focus {
  outline: none;
  border-color: #3498db;
}
```

## 输入框状态

```css
/* 默认 */
.input {
  border: 2px solid #ddd;
}

/* 聚焦 */
.input:focus {
  border-color: #3498db;
  box-shadow: 0 0 0 3px rgba(52, 152, 219, 0.2);
}

/* 禁用 */
.input:disabled {
  background: #f5f5f5;
  cursor: not-allowed;
  opacity: 0.7;
}

/* 只读 */
.input:read-only {
  background: #f9f9f9;
}

/* 错误 */
.input.error {
  border-color: #e74c3c;
}

/* 成功 */
.input.success {
  border-color: #2ecc71;
}
```

## 按钮样式

```css
.btn {
  padding: 12px 24px;
  border: none;
  border-radius: 4px;
  font-size: 16px;
  cursor: pointer;
  transition: all 0.3s;
}

.btn-primary {
  background: #3498db;
  color: white;
}

.btn-primary:hover {
  background: #2980b9;
}

.btn-outline {
  background: transparent;
  border: 2px solid #3498db;
  color: #3498db;
}
```

## 复选框和单选框

```css
/* 隐藏默认样式 */
.checkbox input,
.radio input {
  display: none;
}

/* 自定义样式 */
.checkbox label,
.radio label {
  display: flex;
  align-items: center;
  gap: 10px;
  cursor: pointer;
}

.checkbox label::before {
  content: '';
  width: 20px;
  height: 20px;
  border: 2px solid #ddd;
  border-radius: 4px;
  transition: all 0.3s;
}

.checkbox input:checked + label::before {
  background: #3498db;
  border-color: #3498db;
  background-image: url('check.svg');
}

.radio label::before {
  border-radius: 50%;
}

.radio input:checked + label::before {
  background: #3498db;
}
```

## 下拉框样式

```css
select {
  width: 100%;
  padding: 12px 15px;
  border: 2px solid #ddd;
  border-radius: 4px;
  background: white;
  cursor: pointer;
  appearance: none;
  background-image: url('arrow.svg');
  background-repeat: no-repeat;
  background-position: right 15px center;
}

select:focus {
  outline: none;
  border-color: #3498db;
}
```

## 标签样式

```css
.form-group {
  margin-bottom: 20px;
}

label {
  display: block;
  margin-bottom: 5px;
  font-weight: bold;
  color: #333;
}

.required::after {
  content: ' *';
  color: #e74c3c;
}
```

## 错误提示

```css
.error-message {
  color: #e74c3c;
  font-size: 14px;
  margin-top: 5px;
}

.has-error input {
  border-color: #e74c3c;
}
```

## 表单布局

```css
/* 水平布局 */
.form-horizontal .form-group {
  display: flex;
  align-items: center;
  margin-bottom: 15px;
}

.form-horizontal label {
  flex: 0 0 100px;
  margin-bottom: 0;
}

/* 网格布局 */
.form-grid {
  display: grid;
  grid-template-columns: repeat(2, 1fr);
  gap: 20px;
}
```
