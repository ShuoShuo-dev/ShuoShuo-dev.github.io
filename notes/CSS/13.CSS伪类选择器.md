# 13.CSS伪类选择器

## 什么是伪类？

伪类用于选择元素的特定状态，不需要添加class。

```css
selector:pseudo-class {
  property: value;
}
```

## 链接伪类

```css
/* 未访问的链接 */
a:link {
  color: #3498db;
}

/* 已访问的链接 */
a:visited {
  color: #9b59b6;
}

/* 鼠标悬停 */
a:hover {
  color: #e74c3c;
}

/* 鼠标按下 */
a:active {
  color: #2ecc71;
}

/* 获取焦点的链接 */
a:focus {
  outline: 2px solid #3498db;
}

/* LVHA 顺序 */
a:link {}
a:visited {}
a:hover {}
a:active {}
```

## 用户操作伪类

```css
/* 悬停 */
button:hover {
  background: #3498db;
}

/* 激活（按下） */
button:active {
  transform: scale(0.98);
}

/* 焦点 */
input:focus {
  border-color: #3498db;
}

/* 获取焦点的元素 */
:focus-visible {
  outline: 2px solid #3498db;
}

/* 未获取焦点的元素 */
:not(:focus) {
  opacity: 0.8;
}
```

## 表单伪类

```css
/* 启用状态 */
input:enabled {
  background: white;
}

/* 禁用状态 */
input:disabled {
  background: #eee;
  cursor: not-allowed;
}

/* 只读 */
input:read-only {
  background: #f5f5f5;
}

/* 可读写 */
input:read-write {
  background: white;
}

/* 必填 */
input:required {
  border-color: #e74c3c;
}

/* 可选 */
input:optional {
  border-color: #ddd;
}

/* 输入有效 */
input:valid {
  border-color: #2ecc71;
}

/* 输入无效 */
input:invalid {
  border-color: #e74c3c;
}

/* 选中 */
input:checked {
  width: 20px;
}

/* 不确定（进度条等） */
input:indeterminate {
  opacity: 0.5;
}

/* 占位符显示时 */
input:placeholder-shown {
  background: #f5f5f5;
}
```

## 结构伪类

```css
/* 第一个子元素 */
li:first-child {
  border-top: none;
}

/* 最后一个子元素 */
li:last-child {
  border-bottom: none;
}

/* 唯一子元素 */
div:only-child {
  background: yellow;
}

/* 唯一类型 */
p:only-of-type {
  font-style: italic;
}

/* 第n个子元素 */
li:nth-child(2) {
  background: #f0f0f0;
}

/* 倒数第n个 */
li:nth-last-child(1) {
  font-weight: bold;
}

/* 奇数行 */
li:nth-child(odd) {
  background: #f9f9f9;
}

/* 偶数行 */
li:nth-child(even) {
  background: #fff;
}

/* 每3个中的第1个 */
li:nth-child(3n+1) {
  font-weight: bold;
}

/* 不是某个元素 */
div:not(.special) {
  background: white;
}

/* 空元素 */
div:empty {
  display: none;
}

/* 根元素 */
:root {
  --primary-color: #3498db;
}

/* 根元素（HTML） */
html:root {}
```

## nth-child 详解

```css
/* 常见用法 */
:nth-child(1)              /* 第1个 */
:nth-child(2)              /* 第2个 */
:nth-child(3)              /* 第3个 */
:nth-child(n)              /* 所有 */
:nth-child(odd)            /* 奇数 1,3,5... */
:nth-child(even)          /* 偶数 2,4,6... */
:nth-child(3n)            /* 3,6,9,12... */
:nth-child(3n+1)          /* 1,4,7,10... */
:nth-child(3n+2)          /* 2,5,8,11... */
:nth-child(-n+3)          /* 前3个 1,2,3 */
:nth-child(n+4):nth-child(-n+8)  /* 4到8 */
```

## UI状态伪类

```css
/* 默认选项 */
option:default {
  background: #f0f0f0;
}

/* 当前目标（URL hash） */
:target {
  background: #fff3cd;
}

/* 全屏模式 */
:fullscreen {
  background: black;
}

/* 悬停的占位符 */
::placeholder:hover {
  color: #999;
}
```

## 语言相关伪类

```css
/* 中文 */
:lang(zh) {
  font-family: "Microsoft YaHei", sans-serif;
}

/* 英文 */
:lang(en) {
  font-family: Arial, sans-serif;
}
```

## 动画相关伪类

```css
/* 动画暂停 */
video:paused {
  opacity: 0.8;
}

/* 播放中 */
video:playing {
  opacity: 1;
}
```

## 组合使用

```css
/* 悬停的父元素的子元素 */
li:hover span {
  color: white;
}

/* 悬停的li的第一个span */
li:hover span:first-child {
  text-decoration: underline;
}

/* 悬停且是偶数行 */
tr:nth-child(even):hover {
  background: #f5f5f5;
}

/* 悬停且有特定class */
.btn:hover.special {
  transform: scale(1.1);
}

/* 悬停时非特殊元素 */
button:hover:not(.disabled) {
  background: #3498db;
}
```

## 实用案例

### 1. 斑马条纹表格
```css
tbody tr:nth-child(odd) {
  background: #f9f9f9;
}

tbody tr:hover {
  background: #f0f0f0;
}
```

### 2. 卡片网格响应式
```css
.card:nth-child(3n+1) { background: #3498db; }
.card:nth-child(3n+2) { background: #2ecc71; }
.card:nth-child(3n) { background: #e74c3c; }
```

### 3. 第一个和最后一个特殊样式
```css
.nav-item:first-child {
  margin-left: 0;
}

.nav-item:last-child {
  margin-right: 0;
}

.nav-item:first-child a {
  border-radius: 4px 0 0 4px;
}

.nav-item:last-child a {
  border-radius: 0 4px 4px 0;
}
```

### 4. 表单验证样式
```css
.input-group {
  margin-bottom: 20px;
}

.input-group input {
  border: 2px solid #ddd;
  transition: border-color 0.3s;
}

.input-group input:focus {
  border-color: #3498db;
}

.input-group input:valid:not(:placeholder-shown) {
  border-color: #2ecc71;
}

.input-group input:invalid:not(:placeholder-shown) {
  border-color: #e74c3c;
}
```

### 5. 图片画廊
```css
.gallery img {
  opacity: 0.7;
  transition: opacity 0.3s;
}

.gallery img:hover {
  opacity: 1;
  transform: scale(1.05);
}

.gallery img:nth-child(3n) {
  border: 3px solid #3498db;
}
```

### 6. 悬停显示下拉菜单
```css
.menu-item {
  position: relative;
}

.submenu {
  display: none;
  position: absolute;
  top: 100%;
  left: 0;
}

.menu-item:hover > .submenu {
  display: block;
}
```

### 7. 段落首字母
```css
.article p:first-of-type::first-letter {
  float: left;
  font-size: 3em;
  line-height: 1;
  padding-right: 10px;
  color: #3498db;
  font-weight: bold;
}
```

### 8. 标签云
```css
.tag:nth-child(1) { font-size: 12px; }
.tag:nth-child(2) { font-size: 14px; }
.tag:nth-child(3) { font-size: 16px; }
.tag:nth-child(4) { font-size: 18px; }
.tag:nth-child(5) { font-size: 20px; }
```
