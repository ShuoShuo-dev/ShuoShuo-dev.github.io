# 2.CSS选择器

## 基本选择器

### 1. 元素选择器
```css
p {
  color: #333;
}

div {
  margin: 10px;
}
```

### 2. 类选择器
```css
.box {
  width: 100px;
  height: 100px;
}

.highlight {
  background: yellow;
}
```

### 3. ID选择器
```css
#header {
  background: #333;
  color: white;
}
```

### 4. 通配符选择器
```css
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}
```

## 组合选择器

### 1. 后代选择器（空格）
```css
/* 选择 article 内部的所有 p */
article p {
  line-height: 1.8;
}

/* 选择 div 内部的 .box */
div .box {
  background: #f5f5f5;
}
```

### 2. 子选择器（>）
```css
/* 只选择直接子元素 */
ul > li {
  list-style: none;
}
```

### 3. 相邻兄弟选择器（+）
```css
/* 选择 h1 后紧跟的 p */
h1 + p {
  font-weight: bold;
}
```

### 4. 通用兄弟选择器（~）
```css
/* 选择 h1 后所有同级的 p */
h1 ~ p {
  color: #666;
}
```

## 属性选择器

```css
/* 拥有指定属性 */
[disabled] {
  opacity: 0.5;
}

/* 属性等于指定值 */
[type="text"] {
  border: 1px solid #ccc;
}

/* 属性包含指定值 */
[class*="icon"] {
  font-size: 14px;
}

/* 属性以指定值开头 */
[href^="https"] {
  color: green;
}

/* 属性以指定值结尾 */
[href$=".pdf"] {
  color: red;
}
```

## 伪类选择器

### 1. 链接伪类
```css
a:link {
  color: #333;
}

a:visited {
  color: #666;
}

a:hover {
  color: red;
}

a:active {
  color: blue;
}
```

### 2. 结构伪类
```css
/* 第一个子元素 */
:first-child {
  margin-top: 0;
}

/* 最后一个子元素 */
:last-child {
  margin-bottom: 0;
}

/* 第 n 个子元素 */
:nth-child(2) {
  background: #f0f0f0;
}

/* 奇数行 */
:nth-child(odd) {
  background: #f9f9f9;
}

/* 偶数行 */
:nth-child(even) {
  background: #fff;
}

/* 倒数第 n 个 */
:nth-last-child(1) {
  border-bottom: none;
}
```

### 3. 表单伪类
```css
input:focus {
  border-color: blue;
  outline: none;
}

input:disabled {
  background: #eee;
}

input:checked + label {
  color: blue;
}
```

## 伪元素选择器

```css
/* 在元素前插入内容 */
::before {
  content: "→";
  color: #999;
}

/* 在元素后插入内容 */
::after {
  content: "（完）";
  color: #999;
}

/* 首字母 */
::first-letter {
  font-size: 2em;
  font-weight: bold;
}

/* 首行 */
::first-line {
  color: blue;
}

/* 选中的文本 */
::selection {
  background: yellow;
  color: black;
}
```

## 选择器优先级（ specificity ）

| 优先级 | 选择器 |
|--------|--------|
| 最低 | 通配符 `*` |
| 低 | 元素 `div` |
| 中 | 类 `.box` |
| 中高 | 属性 `[type="text"]` |
| 高 | 伪类 `:hover` |
| 最高 | ID `#header` |
| 极高 | 内联样式 `style=""` |

### 优先级计算
```css
/* 内联样式: 1000 */
/* ID选择器: 100 */
/* 类/属性/伪类: 10 */
/* 元素/伪元素: 1 */

#header .nav li:first-child  /* 100 + 10 + 1 + 10 = 121 */
.nav a:hover                 /* 10 + 10 = 20 */
```

## !important 规则

```css
/* 最高优先级，慎用 */
.element {
  color: red !important;
}
```

## 选择器命名规范（BEM）

```css
/* Block */
.card { }

/* Element */
.card__title { }
.card__content { }

/* Modifier */
.card--featured { }
.card__title--large { }
```
