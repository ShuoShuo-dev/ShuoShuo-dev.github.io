# 15.CSS媒体查询

## 什么是媒体查询？

媒体查询允许我们根据设备特性（如屏幕宽度）应用不同的样式。

## 基本语法

```css
@media media-type and (media-feature) {
  /* CSS规则 */
}
```

## 媒体类型

```css
/* 所有设备 */
@media all { }

/* 屏幕设备 */
@media screen { }

/* 打印机 */
@media print { }

/* 语音合成器 */
@media speech { }

/* 电视 */
@media tv { }
```

## 媒体特性

### 1. width 和 height

```css
/* 宽度正好是 768px */
@media (width: 768px) { }

/* 宽度小于等于 768px */
@media (max-width: 768px) { }

/* 宽度大于等于 768px */
@media (min-width: 768px) { }

/* 高度 */
@media (max-height: 500px) { }
```

### 2. orientation（方向）

```css
/* 竖屏 */
@media (orientation: portrait) { }

/* 横屏 */
@media (orientation: landscape) { }
```

### 3. aspect-ratio（宽高比）

```css
@media (aspect-ratio: 16/9) { }
@media (min-aspect-ratio: 16/9) { }
```

### 4. resolution（分辨率）

```css
@media (resolution: 2dppx) { }
@media (min-resolution: 192dpi) { }
```

### 5. color（颜色深度）

```css
@media (color) { }                    /* 支持颜色 */
@media (min-color: 8) { }            /* 至少8位色 */
```

## 常用断点

```css
/* 超小屏幕（手机） */
@media (max-width: 575.98px) { }

/* 小屏幕（平板竖屏） */
@media (min-width: 576px) and (max-width: 767.98px) { }

/* 中等屏幕（平板横屏） */
@media (min-width: 768px) and (max-width: 991.98px) { }

/* 大屏幕（桌面） */
@media (min-width: 992px) and (max-width: 1199.98px) { }

/* 超大屏幕 */
@media (min-width: 1200px) { }

/* Bootstrap 5 断点参考：
 * - sm: 576px
 * - md: 768px
 * - lg: 992px
 * - xl: 1200px
 * - xxl: 1400px
 */
```

## 逻辑运算符

### 1. and（与）

```css
@media (min-width: 768px) and (max-width: 992px) {
  /* 宽度在768px到992px之间 */
}
```

### 2. not（非）

```css
@media not screen {
  /* 非屏幕设备 */
}

@media (not (max-width: 768px)) {
  /* 宽度大于768px */
}
```

### 3. only（仅）

```css
@media only screen {
  /* 仅屏幕设备（防止老浏览器误解） */
}
```

### 4. 逗号（或）

```css
@media (max-width: 768px), (orientation: portrait) {
  /* 宽度小于768px 或 竖屏 */
}
```

## 引入方式

### 1. link 标签

```html
<link rel="stylesheet" media="screen and (max-width: 768px)" href="mobile.css">
```

### 2. @import

```css
@import url("mobile.css") screen and (max-width: 768px);
```

### 3. @media

```css
/* 推荐方式 */
@media (max-width: 768px) {
  /* CSS规则 */
}
```

## 响应式布局实战

### 1. 响应式容器

```css
.container {
  width: 1200px;
  max-width: 100%;
  margin: 0 auto;
  padding: 0 15px;
}
```

### 2. 响应式网格

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  gap: 20px;
}

@media (max-width: 992px) {
  .grid {
    grid-template-columns: repeat(2, 1fr);
  }
}

@media (max-width: 576px) {
  .grid {
    grid-template-columns: 1fr;
  }
}
```

### 3. 响应式Flexbox

```css
.cards {
  display: flex;
  flex-wrap: wrap;
  gap: 20px;
}

.card {
  flex: 1 1 300px;
}
```

### 4. 响应式字体

```css
html {
  font-size: 16px;
}

@media (min-width: 768px) {
  html {
    font-size: 18px;
  }
}

@media (min-width: 1200px) {
  html {
    font-size: 20px;
  }
}
```

### 5. 响应式间距

```css
.section {
  padding: 60px 0;
}

@media (max-width: 768px) {
  .section {
    padding: 40px 0;
  }
}

@media (max-width: 576px) {
  .section {
    padding: 30px 0;
  }
}
```

### 6. 响应式隐藏/显示

```css
/* 默认显示 */
.sidebar {
  display: block;
}

/* 小屏幕隐藏 */
@media (max-width: 768px) {
  .sidebar {
    display: none;
  }
}

/* 小屏幕显示 */
.mobile-menu {
  display: none;
}

@media (max-width: 768px) {
  .mobile-menu {
    display: block;
  }
}
```

### 7. 响应式导航

```css
/* 桌面端：水平导航 */
.nav {
  display: flex;
  gap: 20px;
}

/* 移动端：垂直导航 */
@media (max-width: 768px) {
  .nav {
    flex-direction: column;
    gap: 0;
  }

  .nav a {
    display: block;
    padding: 15px;
    border-bottom: 1px solid #ddd;
  }
}
```

### 8. 响应式图片

```css
img {
  max-width: 100%;
  height: auto;
}

.responsive-img {
  width: 100%;
  max-width: 500px;
  height: auto;
}

/* picture 元素 */
<picture>
  <source srcset="large.jpg" media="(min-width: 992px)">
  <source srcset="medium.jpg" media="(min-width: 576px)">
  <img src="small.jpg" alt="响应式图片">
</picture>
```

### 9. 响应式列布局

```css
.columns {
  column-count: 4;
  column-gap: 30px;
}

@media (max-width: 992px) {
  .columns {
    column-count: 3;
  }
}

@media (max-width: 768px) {
  .columns {
    column-count: 2;
  }
}

@media (max-width: 576px) {
  .columns {
    column-count: 1;
  }
}
```

### 10. 响应式卡片

```css
.card-container {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
  gap: 20px;
}
```

## prefers-color-scheme（偏好颜色方案）

```css
/* 浅色模式 */
@media (prefers-color-scheme: light) {
  body {
    background: white;
    color: black;
  }
}

/* 深色模式 */
@media (prefers-color-scheme: dark) {
  body {
    background: #1a1a1a;
    color: white;
  }
}
```

## prefers-reduced-motion（减少动画）

```css
@media (prefers-reduced-motion: reduce) {
  * {
    animation: none !important;
    transition: none !important;
  }
}
```

## 最佳实践

1. **使用 mobile-first**：先写移动端样式，再逐步添加大屏幕样式
2. **避免固定宽度**：使用相对单位（%、rem、em）
3. **测试真实设备**：不同设备表现可能不同
4. **合理设置断点**：根据内容而非特定设备设置
5. **使用 viewport meta**：`<meta name="viewport" content="width=device-width, initial-scale=1.0">`
