# 11.CSS动画

## 什么是CSS动画？

CSS动画允许元素在样式之间平滑过渡，包括@keyframes定义的复杂动画序列。

## @keyframes 规则

```css
@keyframes animationName {
  from {
    property: value;
  }
  to {
    property: value;
  }
}

/* 或者使用百分比 */
@keyframes animationName {
  0% {
    property: value;
  }
  50% {
    property: value;
  }
  100% {
    property: value;
  }
}
```

## animation 属性

```css
.element {
  animation: name duration timing-function delay count direction fill-mode;
}
```

## 动画属性详解

### 1. animation-name 动画名称

```css
.box {
  animation-name: slideIn;
  animation-name: fadeIn, moveRight;
}
```

### 2. animation-duration 时长

```css
.box {
  animation-duration: 1s;
  animation-duration: 500ms;
  animation-duration: 2s, 1s;
}
```

### 3. animation-timing-function 缓动

```css
.box {
  animation-timing-function: ease;
  animation-timing-function: linear;
  animation-timing-function: ease-in;
  animation-timing-function: ease-out;
  animation-timing-function: ease-in-out;
  animation-timing-function: step-start;
  animation-timing-function: step-end;
  animation-timing-function: cubic-bezier(0.68, -0.55, 0.27, 1.55);
}
```

### 4. animation-delay 延迟

```css
.box {
  animation-delay: 0s;
  animation-delay: 1s;
  animation-delay: -0.5s; /* 立即开始，跳过前一半 */
}
```

### 5. animation-iteration-count 播放次数

```css
.box {
  animation-iteration-count: 1;       /* 播放1次 */
  animation-iteration-count: 3;       /* 播放3次 */
  animation-iteration-count: infinite; /* 无限循环 */
}
```

### 6. animation-direction 方向

```css
.box {
  animation-direction: normal;            /* 正常播放 */
  animation-direction: reverse;           /* 反向播放 */
  animation-direction: alternate;         /* 轮流播放（正→反→正...） */
  animation-direction: alternate-reverse;  /* 反向开始轮流（反→正→反...） */
}
```

### 7. animation-fill-mode 填充模式

```css
.box {
  animation-fill-mode: none;         /* 默认，动画前不应用样式 */
  animation-fill-mode: forwards;      /* 动画结束后保持最后状态 */
  animation-fill-mode: backwards;     /* 动画开始前应用第一帧 */
  animation-fill-mode: both;          /* 同时应用 forwards + backwards */
}
```

### 8. animation-play-state 播放状态

```css
.box {
  animation-play-state: running;   /* 播放中 */
  animation-play-state: paused;    /* 暂停 */
}

/* 悬停暂停 */
.box:hover {
  animation-play-state: paused;
}
```

## animation 简写

```css
.box {
  /* name duration timing-function delay iteration-count direction fill-mode */
  animation: slideIn 1s ease-in-out 0s infinite alternate both;
}
```

## 常用动画效果

### 1. 淡入淡出
```css
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}

@keyframes fadeOut {
  from { opacity: 1; }
  to { opacity: 0; }
}
```

### 2. 滑入滑出
```css
@keyframes slideInLeft {
  from {
    transform: translateX(-100%);
    opacity: 0;
  }
  to {
    transform: translateX(0);
    opacity: 1;
  }
}

@keyframes slideInUp {
  from {
    transform: translateY(100%);
    opacity: 0;
  }
  to {
    transform: translateY(0);
    opacity: 1;
  }
}
```

### 3. 缩放
```css
@keyframes pulse {
  0%, 100% { transform: scale(1); }
  50% { transform: scale(1.1); }
}

@keyframes bounce {
  0%, 100% { transform: translateY(0); }
  50% { transform: translateY(-20px); }
}
```

### 4. 旋转
```css
@keyframes spin {
  from { transform: rotate(0deg); }
  to { transform: rotate(360deg); }
}

@keyframes wiggle {
  0%, 100% { transform: rotate(0deg); }
  25% { transform: rotate(-5deg); }
  75% { transform: rotate(5deg); }
}
```

### 5. 颜色变化
```css
@keyframes colorChange {
  0% { background: #3498db; }
  50% { background: #2ecc71; }
  100% { background: #e74c3c; }
}
```

### 6. 闪烁
```css
@keyframes blink {
  0%, 50%, 100% { opacity: 1; }
  25%, 75% { opacity: 0; }
}
```

### 7. 打字机效果
```css
.typing {
  width: 0;
  overflow: hidden;
  white-space: nowrap;
  animation: typing 3s steps(20) forwards;
}

@keyframes typing {
  from { width: 0; }
  to { width: 200px; }
}
```

## 动画案例

### 加载动画
```css
.loader {
  width: 50px;
  height: 50px;
  border: 4px solid #f3f3f3;
  border-top: 4px solid #3498db;
  border-radius: 50%;
  animation: spin 1s linear infinite;
}

@keyframes spin {
  0% { transform: rotate(0deg); }
  100% { transform: rotate(360deg); }
}
```

### 呼吸灯效果
```css
.glow {
  animation: glow 2s ease-in-out infinite alternate;
}

@keyframes glow {
  from {
    box-shadow: 0 0 5px #3498db, 0 0 10px #3498db;
  }
  to {
    box-shadow: 0 0 20px #3498db, 0 0 30px #3498db, 0 0 40px #3498db;
  }
}
```

### 心跳效果
```css
.heart {
  animation: heartbeat 1s ease-in-out infinite;
}

@keyframes heartbeat {
  0% { transform: scale(1); }
  14% { transform: scale(1.3); }
  28% { transform: scale(1); }
  42% { transform: scale(1.3); }
  70% { transform: scale(1); }
}
```

### 浮动效果
```css
.float {
  animation: float 3s ease-in-out infinite;
}

@keyframes float {
  0%, 100% { transform: translateY(0); }
  50% { transform: translateY(-15px); }
}
```

## 动画性能优化

### 1. 使用 transform 和 opacity
```css
/* 推荐：使用 GPU 加速 */
.good {
  animation: slideIn 1s ease;
}

@keyframes slideIn {
  from { transform: translateX(-100%); opacity: 0; }
  to { transform: translateX(0); opacity: 1; }
}

/* 避免：触发布局变化 */
.bad {
  animation: changeWidth 1s ease;
}
```

### 2. 使用 will-change
```css
.animated {
  will-change: transform, opacity;
  /* 提前告诉浏览器要动画 */
}
```

### 3. 减少动画属性
```css
/* 不好：动画多个属性 */
@keyframes bad {
  0% { width: 0; height: 0; margin: 0; padding: 0; }
  100% { width: 100px; height: 100px; margin: 10px; padding: 10px; }
}

/* 好：只动画 transform 和 opacity */
@keyframes good {
  from { transform: scale(0); opacity: 0; }
  to { transform: scale(1); opacity: 1; }
}
```
