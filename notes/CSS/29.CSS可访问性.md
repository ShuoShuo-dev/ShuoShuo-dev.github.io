# 29.CSS可访问性

## 焦点样式

```css
/* 确保焦点样式清晰可见 */
:focus {
  outline: 2px solid #3498db;
  outline-offset: 2px;
}

/* 移除默认焦点样式（谨慎） */
:focus {
  outline: none;
  box-shadow: 0 0 0 2px #3498db;
}
```

## 对比度

```css
/* 确保文字对比度足够（至少4.5:1） */
.text {
  color: #333;  /* 深色文字配浅色背景 */
  background: #fff;
}

/* 大文本（至少3:1） */
.large-text {
  color: #666;
  background: #fff;
  font-size: 18px;
}
```

## 文字大小

```css
/* 使用相对单位 */
html {
  font-size: 100%;  /* 允许用户缩放 */
}

/* 不要使用 px 设置字体大小 */
.text {
  font-size: 1rem;  /* 好 */
  font-size: 16px;  /* 不好，可能无法缩放 */
}
```

## 点击区域

```css
/* 确保可点击区域足够大（至少44x44px） */
button,
a {
  min-width: 44px;
  min-height: 44px;
  padding: 12px 20px;
}
```

## 隐藏内容

```css
/* 屏幕阅读器可见 */
.sr-only {
  position: absolute;
  width: 1px;
  height: 1px;
  padding: 0;
  margin: -1px;
  overflow: hidden;
  clip: rect(0, 0, 0, 0);
  white-space: nowrap;
  border: 0;
}

/* 视觉隐藏但保持可访问性 */
.hidden {
  visibility: hidden;
  opacity: 0;
  pointer-events: none;
}
```

## 动画和动作

```css
/* 尊重用户减少动画偏好 */
@media (prefers-reduced-motion: reduce) {
  *,
  *::before,
  *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
  }
}
```

## 颜色和色盲

```css
/* 不要仅用颜色传达信息 */
.error {
  color: red;
  /* 添加图标或文字 */
}

.error::before {
  content: '⚠ ';
}
```

## 表单标签

```css
/* 确保标签和输入框关联 */
label {
  display: block;
  cursor: pointer;
}

input:focus + label {
  color: #3498db;
}
```

## 链接样式

```css
/* 下划线链接 */
a {
  text-decoration: underline;
}

/* 悬停样式 */
a:hover {
  text-decoration-thickness: 2px;
}
```
