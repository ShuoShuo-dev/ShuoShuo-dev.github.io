# 14.CSS伪元素

## 什么是伪元素？

伪元素用于创建不在文档树中的元素，可以像真实元素一样样式化。

```css
selector::pseudo-element {
  property: value;
}
```

## 常用伪元素

| 伪元素 | 说明 |
|--------|------|
| `::before` | 元素内容之前插入 |
| `::after` | 元素内容之后插入 |
| `::first-line` | 元素的第一行 |
| `::first-letter` | 元素的第一个字母 |
| `::selection` | 被选中的文本 |
| `::placeholder` | 输入框占位符 |
| `::backdrop` | 全屏元素背景 |

## 1. ::before 和 ::after

```css
/* ::before 在元素内容前插入 */
.element::before {
  content: "";           /* 必须设置content属性 */
  content: "★";         /* 插入文本 */
  content: url("icon.png"); /* 插入图片 */
}

/* ::after 在元素内容后插入 */
.element::after {
  content: "";
  content: "→";
}
```

### 插入内容

```css
/* 文本 */
.title::before {
  content: "标题：";
  color: #999;
}

/* 图片 */
.icon::before {
  content: url("star.png");
}

/* 空内容（用于装饰） */
.box::before {
  content: "";
  position: absolute;
  top: 0;
  left: 0;
  width: 10px;
  height: 10px;
  background: red;
}

/* 属性值 */
.name::before {
  content: attr(data-title);
}
```

### 计数器

```css
ol {
  counter-reset: item;  /* 初始化计数器 */
}

li::before {
  counter-increment: item;  /* 递增 */
  content: counter(item) ".";  /* 显示 */
}
```

## 2. ::first-line

```css
p::first-line {
  color: #3498db;
  font-weight: bold;
  text-transform: uppercase;
}
```

**可用属性：**
- font 属性
- color 属性
- background 属性
- word-spacing
- letter-spacing
- text-decoration
- text-transform
- line-height

## 3. ::first-letter

```css
p::first-letter {
  float: left;
  font-size: 3em;
  line-height: 1;
  padding-right: 10px;
  color: #3498db;
  font-weight: bold;
}
```

**可用属性：**
- font 属性
- color 属性
- background 属性
- margin 属性
- padding 属性
- border 属性
- float
- line-height
- text-transform
- text-decoration

## 4. ::selection

```css
::selection {
  background: #3498db;
  color: white;
}

/* Firefox */
::-moz-selection {
  background: #3498db;
  color: white;
}
```

## 5. ::placeholder

```css
input::placeholder {
  color: #999;
  font-style: italic;
}

/* Firefox */
input::-webkit-input-placeholder {
  color: #999;
}
```

## 6. ::backdrop（全屏）

```css
video:fullscreen::backdrop {
  background: black;
}

dialog::backdrop {
  background: rgba(0, 0, 0, 0.5);
}
```

## 实用案例

### 1. 清除浮动
```css
.clearfix::after {
  content: "";
  display: block;
  clear: both;
}
```

### 2. 卡片装饰角
```css
.card {
  position: relative;
}

.card::before {
  content: "";
  position: absolute;
  top: 0;
  left: 0;
  border-left: 20px solid #3498db;
  border-bottom: 20px solid transparent;
}
```

### 3. 链接箭头
```css
a::after {
  content: "→";
  margin-left: 5px;
  color: #3498db;
}
```

### 4. 图片覆盖层
```css
.image-wrapper {
  position: relative;
}

.image-wrapper::after {
  content: "查看详情";
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  background: rgba(0, 0, 0, 0.5);
  color: white;
  display: flex;
  align-items: center;
  justify-content: center;
  opacity: 0;
  transition: opacity 0.3s;
}

.image-wrapper:hover::after {
  opacity: 1;
}
```

### 5. 输入框图标
```css
.input-icon {
  position: relative;
}

.input-icon input {
  padding-left: 40px;
}

.input-icon::before {
  content: "🔍";
  position: absolute;
  left: 15px;
  top: 50%;
  transform: translateY(-50%);
}
```

### 6. 引用样式
```css
blockquote {
  position: relative;
  padding-left: 30px;
}

blockquote::before {
  content: """;
  position: absolute;
  left: 0;
  top: 0;
  font-size: 3em;
  color: #3498db;
  font-family: Georgia, serif;
}
```

### 7. 渐变下划线
```css
.link {
  position: relative;
  text-decoration: none;
}

.link::after {
  content: "";
  position: absolute;
  bottom: 0;
  left: 0;
  width: 100%;
  height: 2px;
  background: linear-gradient(90deg, #3498db, #2ecc71);
  transform: scaleX(0);
  transition: transform 0.3s;
}

.link:hover::after {
  transform: scaleX(1);
}
```

### 8. 面包屑导航分隔符
```css
.breadcrumb a::after {
  content: "/";
  margin: 0 10px;
  color: #999;
}

.breadcrumb a:last-child::after {
  content: "";
}
```

### 9. 消息提示框
```css
.alert {
  position: relative;
  padding-left: 50px;
}

.alert::before {
  content: "⚠";
  position: absolute;
  left: 15px;
  top: 50%;
  transform: translateY(-50%);
  font-size: 1.5em;
}

.alert.success::before { content: "✓"; color: #2ecc71; }
.alert.error::before { content: "✗"; color: #e74c3c; }
.alert.info::before { content: "ℹ"; color: #3498db; }
```

### 10. 文字首字下沉
```css
.article p:first-of-type::first-letter {
  float: left;
  font-size: 4em;
  line-height: 1;
  padding-right: 15px;
  padding-top: 5px;
  color: #3498db;
  font-weight: bold;
}
```

### 11. 卡片背景图案
```css
.feature-card {
  position: relative;
}

.feature-card::before {
  content: "";
  position: absolute;
  top: 0;
  left: 0;
  right: 0;
  height: 100px;
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  border-radius: 8px 8px 0 0;

```

### 12. 加载动画
```css
.loading::after {
  content: "";
  display: inline-block;
  width: 4px;
  height: 4px;
  margin-left: 5px;
  background: #3498db;
  animation: dotPulse 1.5s infinite;
}

@keyframes dotPulse {
  0%, 20% { content: "."; }
  40% { content: ".."; }
  60%, 100% { content: "..."; }
}
```

## 注意事项

1. **`content` 属性必须设置**：`::before` 和 `::after` 必须设置 `content`
2. **默认是行内元素**：`::before` 和 `::after` 默认是行内元素
3. **不存在于DOM中**：伪元素不在DOM树中，无法用JS直接操作
4. **旧语法用单冒号**：为了兼容性，`:before` 和 `:after` 仍有效，但推荐双冒号
