# 8.Flexbox弹性盒子布局

## 什么是 Flexbox？

Flexbox（弹性盒子）是一种一维布局模型，用于在容器中分配空间和对齐元素。

## 基本概念

```
┌─────────────────────────────────────┐
│              Flex Container          │
│  ┌─────────────────────────────────┐│
│  │           Main Axis             ││
│  │  →→→→→→→→→→→→→→→→→→→→→→→→→→→→→→ ││
│  │  ┌───┐  ┌───┐  ┌───┐  ┌───┐    ││
│  │  │ 1 │  │ 2 │  │ 3 │  │ 4 │    ││
│  │  └───┘  └───┘  └───┘  └───┘    ││
│  │        ↑                        ││
│  │     Cross Axis                  ││
│  └─────────────────────────────────┘│
└─────────────────────────────────────┘
```

- **主轴（Main Axis）**：默认水平从左到右
- **交叉轴（Cross Axis）**：默认垂直从上到下
- **Flex Container**：弹性容器
- **Flex Item**：弹性项目

## 启用 Flexbox

```css
.container {
  display: flex;      /* 块级弹性容器 */
  display: inline-flex; /* 行内弹性容器 */
}
```

## 容器属性

### 1. flex-direction 主轴方向

```css
.container {
  flex-direction: row;               /* 左到右（默认） */
  flex-direction: row-reverse;       /* 右到左 */
  flex-direction: column;            /* 上到下 */
  flex-direction: column-reverse;    /* 下到上 */
}
```

### 2. flex-wrap 换行

```css
.container {
  flex-wrap: nowrap;        /* 不换行（默认） */
  flex-wrap: wrap;           /* 换行，第一行在上 */
  flex-wrap: wrap-reverse;   /* 换行，第一行在下 */
}
```

### 3. flex-flow 简写

```css
.container {
  flex-flow: row wrap;  /* direction + wrap */
  flex-flow: column nowrap;
}
```

### 4. justify-content 主轴对齐

```css
.container {
  justify-content: flex-start;    /* 左对齐（默认） */
  justify-content: flex-end;      /* 右对齐 */
  justify-content: center;        /* 居中 */
  justify-content: space-between; /* 两端对齐，间距相等 */
  justify-content: space-around;  /* 每个项目两侧间距相等 */
  justify-content: space-evenly;  /* 项目间距完全相等 */
}
```

### 5. align-items 交叉轴对齐（单行）

```css
.container {
  align-items: stretch;      /* 拉伸占满（默认） */
  align-items: flex-start;    /* 顶部对齐 */
  align-items: flex-end;      /* 底部对齐 */
  align-items: center;        /* 垂直居中 */
  align-items: baseline;     /* 基线对齐 */
}
```

### 6. align-content 交叉轴对齐（多行）

```css
.container {
  align-content: stretch;        /* 拉伸（默认） */
  align-content: flex-start;      /* 顶部对齐 */
  align-content: flex-end;        /* 底部对齐 */
  align-content: center;          /* 居中 */
  align-content: space-between;   /* 两端对齐 */
  align-content: space-around;    /* 间距相等 */
  align-content: space-evenly;    /* 完全相等 */
}
```

### 7. gap 间距

```css
.container {
  gap: 20px;           /* 行和列间距都是20px */
  gap: 10px 20px;      /* 行间距10px，列间距20px */
  row-gap: 10px;       /* 行间距 */
  column-gap: 20px;    /* 列间距 */
}
```

## 项目属性

### 1. flex-grow 放大比例

```css
.item {
  flex-grow: 0;  /* 不放大（默认） */
  flex-grow: 1;  /* 平均分配剩余空间 */
  flex-grow: 2;  /* 获得比其他项目多一倍的份额 */
}
```

### 2. flex-shrink 缩小比例

```css
.item {
  flex-shrink: 1;  /* 空间不足时缩小（默认） */
  flex-shrink: 0;  /* 不缩小 */
  flex-shrink: 2;  /* 缩小的更快 */
}
```

### 3. flex-basis 初始大小

```css
.item {
  flex-basis: auto;  /* 自动（默认） */
  flex-basis: 200px; /* 固定宽度 */
  flex-basis: 50%;   /* 百分比 */
}
```

### 4. flex 简写

```css
.item {
  flex: 1;           /* flex-grow: 1; flex-shrink: 1; flex-basis: 0%; */
  flex: auto;        /* flex-grow: 1; flex-shrink: 1; flex-basis: auto; */
  flex: none;       /* flex-grow: 0; flex-shrink: 0; flex-basis: auto; */
  flex: 2 1 200px;   /* grow shrink basis */
}
```

### 5. align-self 单独对齐

```css
.item {
  align-self: auto;      /* 继承父元素 */
  align-self: flex-start;
  align-self: flex-end;
  align-self: center;
  align-self: stretch;
  align-self: baseline;
}
```

### 6. order 排列顺序

```css
.item {
  order: 0;   /* 顺序（默认） */
  order: -1;  /* 排在最前面 */
  order: 1;   /* 排在后面 */
}
```

## 常用布局案例

### 1. 水平居中
```css
.center {
  display: flex;
  justify-content: center;
}
```

### 2. 垂直居中
```css
.center {
  display: flex;
  align-items: center;
}
```

### 3. 水平垂直居中
```css
.center {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

### 4. 两端对齐，中间自适应
```css
.toolbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
}
```

### 5. 导航栏
```css
.nav {
  display: flex;
  gap: 20px;
}

.nav-item:first-child {
  margin-right: auto; /* 第一个元素靠左，其余靠右 */
}
```

### 6. 卡片网格
```css
.cards {
  display: flex;
  flex-wrap: wrap;
  gap: 20px;
}

.card {
  flex: 1 1 300px; /* 最小宽度300px */
}
```

### 7. 圣杯布局
```css
.holy-grail {
  display: flex;
  min-height: 100vh;
}

header, footer {
  flex: 1;
}

main {
  flex: 1;
}

aside {
  flex: 0 0 200px;
}
```

### 8. 底部固定，内容滚动
```css
.page {
  display: flex;
  flex-direction: column;
  min-height: 100vh;
}

.content {
  flex: 1;
  overflow-y: auto;
}

.footer {
  flex-shrink: 0;
}
```

## 响应式布局

### 媒体查询 + Flexbox
```css
.container {
  display: flex;
  flex-wrap: wrap;
  gap: 20px;
}

.item {
  flex: 1 1 300px;
}

/* 小屏幕改为单列 */
@media (max-width: 768px) {
  .container {
    flex-direction: column;
  }
}
```

### 自动适应列数
```css
.grid {
  display: flex;
  flex-wrap: wrap;
}

.item {
  flex: 1 1 calc(33.333% - 20px);
}

/* clamp() 方式 */
.item {
  flex: 1 1 clamp(250px, 33.333%, 400px);
}
```
