# 1.CSS基础入门

## 什么是CSS？

CSS（层叠样式表）用于描述HTML文档的外观和格式。

## CSS的三种引入方式

### 1. 行内样式（不推荐）
```html
<p style="color: red; font-size: 16px;">红色文字</p>
```

### 2. 内部样式表
```html
<style>
  p {
    color: red;
    font-size: 16px;
  }
</style>
```

### 3. 外部样式表（推荐）
```html
<link rel="stylesheet" href="styles.css">
```

## CSS基本语法

```css
选择器 {
  属性名: 属性值;
  属性名: 属性值;
}
```

### 示例
```css
h1 {
  color: blue;
  font-size: 24px;
  text-align: center;
}
```

## CSS注释
```css
/* 这是单行注释 */

/*
 * 这是多行注释
 * 用于说明样式规则
 */
```

## CSS特性

1. **继承性**：子元素继承父元素的某些样式
2. **层叠性**：多个样式规则作用于同一元素
3. **优先级**：不同来源的样式有优先级之分
