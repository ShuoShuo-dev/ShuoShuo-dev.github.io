# 39.CSS模态框

## 基础模态框

```css
.modal-overlay {
  display: none;
  position: fixed;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  background: rgba(0, 0, 0, 0.5);
  z-index: 9999;
}

.modal {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
  background: white;
  padding: 30px;
  border-radius: 8px;
  min-width: 400px;
  max-width: 90%;
}

.modal-header {
  display: flex;
  justify-content: space-between;
  align-items: center;
  margin-bottom: 20px;
}

.modal-close {
  background: none;
  border: none;
  font-size: 24px;
  cursor: pointer;
}
```

## 显示模态框

```css
.modal-overlay.show {
  display: flex;
  align-items: center;
  justify-content: center;
}

.modal-overlay.show .modal {
  animation: modalIn 0.3s ease;
}

@keyframes modalIn {
  from {
    opacity: 0;
    transform: translate(-50%, -50%) scale(0.9);
  }
  to {
    opacity: 1;
    transform: translate(-50%, -50%) scale(1);
  }
}
```

## 侧滑模态框

```css
.sidebar-modal {
  position: fixed;
  top: 0;
  right: -400px;
  width: 400px;
  height: 100%;
  background: white;
  transition: right 0.3s ease;
  z-index: 9999;
}

.sidebar-modal.show {
  right: 0;
}
```

## 底部抽屉

```css
.bottom-drawer {
  position: fixed;
  bottom: -300px;
  left: 0;
  width: 100%;
  height: 300px;
  background: white;
  transition: bottom 0.3s ease;
  z-index: 9999;
  border-radius: 16px 16px 0 0;
}

.bottom-drawer.show {
  bottom: 0;
}
```

.confirm-dialog {
