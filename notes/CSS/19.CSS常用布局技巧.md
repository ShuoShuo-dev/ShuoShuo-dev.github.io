# 19.CSS常用布局技巧

## 居中布局

### 1. 水平居中

```css
/* 行内元素 */
.text-center {
  text-align: center;
}

/* 块级元素 */
.block-center {
  margin-left: auto;
  margin-right: auto;
}

/* Flexbox */
.flex-center-h {
  display: flex;
  justify-content: center;
}
```

### 2. 垂直居中

```css
/* Flexbox */
.flex-center-v {
  display: flex;
  align-items: center;
}

/* Grid */
.grid-center-v {
  display: grid;
  align-content: center;
}

/* 绝对定位 */
.parent {
  position: relative;
}

.child {
  position: absolute;
  top: 50%;
  transform: translateY(-50%);
}

/* 行内元素 */
.parent {
  height: 100px;
  line-height: 100px;
}
```

### 3. 水平垂直居中

```css
/* Flexbox */
.flex-center {
  display: flex;
  justify-content: center;
  align-items: center;
}

/* Grid */
.grid-center {
  display: grid;
  place-items: center;
}

/* 绝对定位 */
.parent {
  position: relative;
}

.child {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}

/* 负 margin */
.child {
  position: absolute;
  top: 50%;
  left: 50%;
  width: 200px;
  height: 100px;
  margin-left: -100px;
  margin-top: -50px;
}

/* calc */
.child {
  position: absolute;
  top: calc(50% - 50px);
  left: calc(50% - 100px);
}
```

## 两栏布局

### 1. 左固定右自适应

```css
/* Flexbox */
.layout {
  display: flex;
}

.sidebar {
  width: 250px;
  flex-shrink: 0;
}

.content {
  flex: 1;
}

/* Grid */
.layout {
  display: grid;
  grid-template-columns: 250px 1fr;
}

/* 浮动 */
.sidebar {
  float: left;
  width: 250px;
}

.content {
  margin-left: 250px;
}
```

### 2. 右固定左自适应

```css
.layout {
  display: flex;
}

.content {
  flex: 1;
}

.sidebar {
  width: 250px;
  flex-shrink: 0;
  order: 1;
}
```

### 3. 两侧固定中间自适应

```css
.layout {
  display: flex;
}

.left, .right {
  width: 200px;
  flex-shrink: 0;
}

.center {
  flex: 1;
}

/* 或 */
.layout {
  display: grid;
  grid-template-columns: 200px 1fr 200px;
}
```

## 三栏布局

### 1. 圣杯布局

```css
.container {
  padding-left: 200px;
  padding-right: 200px;
}

.left, .right, .center {
  float: left;
  position: relative;
}

.left {
  width: 200px;
  margin-left: -100%;
  left: -200px;
}

.right {
  width: 200px;
  margin-left: -200px;
  right: -200px;
}

.center {
  width: 100%;
}

/* 清除浮动 */
.container::after {
  content: '';
  display: block;
  clear: both;
}
```

### 2. 双飞翼布局

```css
.center-wrap {
  float: left;
  width: 100%;
}

.center {
  margin: 0 200px;
}

.left {
  float: left;
  width: 200px;
  margin-left: -100%;
}

.right {
  float: left;
  width: 200px;
  margin-left: -200px;
}
```

### 3. Flexbox 实现

```css
.layout {
  display: flex;
}

.center {
  flex: 1;
  order: 2;
}

.left {
  width: 200px;
  order: 1;
}

.right {
  width: 200px;
  order: 3;
}
```

### 4. Grid 实现

```css
.layout {
  display: grid;
  grid-template-columns: 200px 1fr 200px;
}

.center {
  order: 2;
}

.left {
  order: 1;
}

.right {
  order: 3;
}
```

## 等高布局

### 1. Flexbox 等高

```css
.cards {
  display: flex;
  gap: 20px;
}

.card {
  flex: 1;  /* 默认等高 */
}
```

### 2. Grid 等高

```css
.cards {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
}
```

### 3. Table 等高

```css
.cards {
  display: table;
  width: 100%;
}

.card {
  display: table-cell;
  width: 33.33%;
}
```

## Sticky Footer（底部固定）

### 1. Flexbox

```css
html, body {
  height: 100%;
}

body {
  display: flex;
  flex-direction: column;
}

.content {
  flex: 1;
}
```

### 2. Grid

```css
body {
  display: grid;
  grid-template-rows: auto 1fr auto;
  min-height: 100vh;
}
```

### 3. 负 margin

```css
.wrapper {
  min-height: 100vh;
}

.content {
  padding-bottom: 60px;
}

.footer {
  height: 60px;
  margin-top: -60px;
}
```

## 等宽列

### 1. 浮动

```css
.cols {
  overflow: hidden;
}

.col {
  float: left;
  width: 25%;
}
```

### 2. Flexbox

```css
.cols {
  display: flex;
}

.col {
  flex: 1;
}
```

### 3. Grid

```css
.cols {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
}
```

### 4. calc

```css
.col {
  width: calc(25% - 20px);
  margin-right: 20px;
}

.col:last-child {
  margin-right: 0;
}
```

## 响应式布局

### 1. 断点策略

```css
/* 移动优先 */
.container {
  padding: 0 15px;
}

@media (min-width: 576px) {
  .container { max-width: 540px; }
}

@media (min-width: 768px) {
  .container { max-width: 720px; }
}

@media (min-width: 992px) {
  .container { max-width: 960px; }
}

@media (min-width: 1200px) {
  .container { max-width: 1140px; }
}
```

### 2. 列重排

```css
/* 移动端：单列 */
/* 平板：两列 */
/* 桌面：三列 */

.items {
  display: grid;
  grid-template-columns: 1fr;
  gap: 20px;
}

@media (min-width: 768px) {
  .items {
    grid-template-columns: repeat(2, 1fr);
  }
}

@media (min-width: 1024px) {
  .items {
    grid-template-columns: repeat(3, 1fr);
  }
}
```

## 溢出处理

### 1. 单行省略

```css
.ellipsis {
  overflow: hidden;
  text-overflow: ellipsis;
  white-space: nowrap;
}
```

### 2. 多行省略

```css
.multi-line {
  display: -webkit-box;
  -webkit-box-orient: vertical;
  -webkit-line-clamp: 3;
  overflow: hidden;
}
```

### 3. 滚动

```css
.scroll {
  overflow: auto;
}

/* 隐藏滚动条但可滚动 */
.hide-scrollbar {
  overflow: auto;
  scrollbar-width: none; /* Firefox */
}

.hide-scrollbar::-webkit-scrollbar {
  display: none; /* Chrome/Safari */
}
```

## 实用布局代码

```css

  display: flex;
  justify-content: center;
}
```

### 2. 两端对齐的导航

```css
.nav {
  display: flex;
  justify-content: space-between;
}
```

### 3. 子元素分散对齐

```css
.nav {
  display: flex;
  justify-content: space-around;
}
```

```css
  align-items: center;
}

.logo {

}

  flex: 1;
  display: flex;
  justify-content: center;
}

.actions {
  /* 右对齐 */
}
```
