# 25.CSS表格样式

## 基本表格

```css
table {
  width: 100%;
  border-collapse: collapse;
  border-spacing: 0;
}

th, td {
  padding: 10px;
  text-align: left;
  border: 1px solid #ddd;
}
```

## border-collapse

```css
/* 合并边框 */
table {
  border-collapse: collapse;
}

/* 分离边框 */
table {
  border-collapse: separate;
  border-spacing: 5px;
}
```

## 斑马条纹表格

```css
tbody tr:nth-child(odd) {
  background: #f9f9f9;
}

tbody tr:nth-child(even) {
  background: #fff;
}
```

## 悬停效果

```css
tbody tr:hover {
  background: #f5f5f5;
}
```

## 紧凑型表格

```css
.compact th,
.compact td {
  padding: 5px 8px;
  font-size: 14px;
}
```

## 带边框表格

```css
.bordered th,
.bordered td {
  border: 1px solid #ddd;
}

.bordered th {
  background: #f5f5f5;
}
```

## 圆角表格

```css
.rounded {
  border-radius: 8px;
  overflow: hidden;
}
```

## 响应式表格

```css
/* 滚动 */
.table-wrapper {
  overflow-x: auto;
}

/* 卡片式 */
@media (max-width: 768px) {
  .table-card {
    border: none;
  }
  
  .table-card thead {
    display: none;
  }
  
  .table-card tbody tr {
    display: block;
    margin-bottom: 20px;
    border: 1px solid #ddd;
    border-radius: 8px;
  }
  
  .table-card td {
    display: block;
    padding: 10px;
    text-align: right;
  }
  
  .table-card td::before {
    content: attr(data-label);
    float: left;
    font-weight: bold;
  }
}
```
