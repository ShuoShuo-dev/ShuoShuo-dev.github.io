# 23.CSS边框和圆角

## border 属性

```css
/* 边框样式 */
border-style: solid;    /* 实线 */
border-style: dashed;   /* 虚线 */
border-style: dotted;   /* 点线 */
border-style: double;   /* 双线 */
border-style: groove;   /* 3D凹槽 */
border-style: ridge;    /* 3D凸槽 */
border-style: inset;    /* 3D凹边 */
border-style: outset;   /* 3D凸边 */

/* 边框宽度 */
border-width: 1px;
border-top-width: 2px;

/* 边框颜色 */
border-color: red;
border-color: #3498db;
border-color: rgb(52, 152, 219);

/* 简写 */
border: 1px solid #333;
border-top: 2px solid red;
```

## 单边边框

```css
border-top: 2px solid #333;
border-bottom: 1px dashed #ccc;
border-left: 3px double #666;
border-right: none;
```

## border-radius 圆角

```css
/* 四角相同 */
border-radius: 10px;

/* 四个值：左上、右上、右下、左下 */
border-radius: 5px 10px 15px 20px;

/* 三个值：左上、右上左下、右下 */
border-radius: 5px 10px 15px;

/* 两个值：左上右下、右上左下 */
border-radius: 10px 20px;

/* 圆形 */
border-radius: 50%;
border-radius: 9999px;
```

## 复杂圆角

```css
/* 椭圆圆角 */
border-radius: 50% / 30%;

/* 分别设置 */
border-radius: 50% 30% 30% 50% / 60% 40% 40% 60%;
```

## 实用边框效果

### 1. 头像圆形
```css
.avatar {
  width: 60px;
  height: 60px;
  border-radius: 50%;
  object-fit: cover;
}
```

### 2. 按钮圆角
```css
.btn {
  border-radius: 4px;
}

.btn-pill {
  border-radius: 9999px;
}
```

### 3. 卡片圆角
```css
.card {
  border-radius: 8px;
}
```

### 4. 图片圆角
```css
img {
  border-radius: 4px;
}
```

### 5. 半圆
```css
.half-circle-top {
  width: 100px;
  height: 50px;
  border-radius: 100px 100px 0 0;
  background: #3498db;
}
```

### 6. 叶子形状
```css
.leaf {
  width: 100px;
  height: 100px;
  border-radius: 0 100% 0 100%;
  background: #2ecc71;
}
```

### 7. 胶囊形状
```css
.badge {
  display: inline-block;
  padding: 4px 12px;
  border-radius: 9999px;
  background: #e74c3c;
  color: white;
}
```
