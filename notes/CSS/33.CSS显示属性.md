# 33.CSS显示属性

## display 属性值

```css
display: block;       /* 块级元素 */
display: inline;      /* 行内元素 */
display: inline-block; /* 行内块 */
display: flex;        /* 弹性盒子 */
display: inline-flex;  /* 行内弹性盒子 */
display: grid;        /* 网格 */
display: inline-grid;  /* 行内网格 */
display: none;        /* 不显示 */
display: contents;    /* 幽灵元素 */
display: list-item;   /* 列表项 */
display: table;       /* 表格 */
```

## 块级元素

```css
/* 默认占满整行 */
div, p, h1-h6, ul, ol, form {
  display: block;
}
```

## 行内元素

```css
/* 不独占一行，不支持宽高 */
span, a, strong, em, img {
  display: inline;
}
```

## 行内块

```css
/* 结合两者特性 */
input, button, img {
  display: inline-block;
  /* 不独占一行，可以设置宽高 */
}
```

## none vs hidden

```css
/* 不显示，不占空间 */
.hidden {
  display: none;
}

/* 隐藏但占空间 */
.invisible {
  visibility: hidden;
  opacity: 0;
}
```

## contents 幽灵元素

```css
/* 元素本身不显示，但子元素显示 */
.wrapper {
  display: contents;
}
```

## 实用技巧

```css
/* 屏幕阅读器可见但视觉隐藏 */
.sr-only {
  position: absolute;
  width: 1px;
  height: 1px;
  overflow: hidden;
  clip: rect(0, 0, 0, 0);
}

/* 显示为行内 */
.inline {
  display: inline;
}

/* 显示为块 */
.block {
  display: block;
}

/* 显示为行内块 */
.inline-block {
  display: inline-block;
}
```
