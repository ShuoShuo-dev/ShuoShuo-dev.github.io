# 6.CSS浮动

## 什么是浮动？

浮动最初用于实现文字环绕图片的效果，现在更多用于布局。

```css
.left {
  float: left;
}

.right {
  float: right;
}
```

## 浮动的三个值

```css
.box {
  float: left;     /* 向左浮动 */
  float: right;    /* 向右浮动 */
  float: none;     /* 不浮动，默认 */
}
```

## 浮动的影响

### 1. 文字环绕效果
```html
<style>
img {
  float: left;
  margin-right: 10px;
}
</style>

<p>
  <img src="photo.jpg" alt="照片">
  这段文字会环绕在图片右侧...
</p>
```

### 2. 父元素高度塌陷
```html
<div class="parent">
  <div class="child" style="float: left;">子元素1</div>
  <div class="child" style="float: right;">子元素2</div>
  <!-- 父元素高度变成0！ -->
</div>
```

## 清除浮动

### 方法1：空元素清除
```html
<div class="parent">
  <div class="child" style="float: left;">1</div>
  <div class="child" style="float: left;">2</div>
  <div class="clear"></div>
</div>

<style>
.clear {
  clear: both; /* 清除左右浮动 */
}
</style>
```

### 方法2：伪元素清除（推荐）
```css
.clearfix::after {
  content: '';
  display: block;
  clear: both;
}
```

### 方法3：overflow清除
```css
.parent {
  overflow: hidden; /* 或 auto */
}
```

### 方法4：display: flow-root（现代）
```css
.parent {
  display: flow-root; /* 新的清除浮动方式 */
}
```

## 浮动布局示例

### 两栏布局
```html
<style>
.container {
  max-width: 1200px;
  margin: 0 auto;
}

.sidebar {
  float: left;
  width: 250px;
}

.content {
  margin-left: 250px; /* 为侧边栏留出空间 */
}

.clearfix::after {
  content: '';
  display: block;
  clear: both;
}
</style>

<div class="container clearfix">
  <aside class="sidebar">侧边栏</aside>
  <main class="content">主内容</main>
</div>
```

### 三栏布局
```html
<style>
.container {
  max-width: 1200px;
  margin: 0 auto;
}

.left {
  float: left;
  width: 200px;
}

.right {
  float: right;
  width: 200px;
}

.center {
  margin-left: 200px;
  margin-right: 200px;
}

.clearfix::after {
  content: '';
  display: block;
  clear: both;
}
</style>

<div class="container clearfix">
  <aside class="left">左侧栏</aside>
  <aside class="right">右侧栏</aside>
  <main class="center">中间内容</main>
</div>
```

### 网格布局
```html
<style>
.row {
  margin-left: -10px;
  margin-right: -10px;
}

.col {
  float: left;
  width: 33.333%;
  padding: 0 10px;
  box-sizing: border-box;
}

.row::after {
  content: '';
  display: block;
  clear: both;
}
</style>

<div class="row">
  <div class="col">第1列</div>
  <div class="col">第2列</div>
  <div class="col">第3列</div>
</div>
```

## 浮动的注意事项

1. **浮动元素会脱离文档流** - 不占据原来空间
2. **浮动元素会形成块格式化上下文**
3. **浮动元素会变成块级元素** - 可以设置宽高
4. **清除浮动要在浮动元素之后**

## 现代布局建议

虽然浮动可以实现布局，但现代更推荐使用 Flexbox 和 Grid：

```css
/* 代替浮动的两栏布局 */
.container {
  display: flex;
}

.sidebar {
  width: 250px;
  flex-shrink: 0;
}

.content {
  flex: 1; /* 自动占据剩余空间 */
}
```

## 完整示例

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
<style>

  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

  max-width: 1200px;
  margin: 0 auto;
  padding: 20px;
}

.clearfix::after {
  content: '';
  display: block;
  clear: both;
}

.header {
  background: #333;
  color: white;
  padding: 20px;
  text-align: center;

.nav {
  background: #555;
  overflow: hidden; /* 清除浮动 */

.nav a {
  float: left;
  padding: 15px 20px;
  color: white;
  text-decoration: none;
}

.nav a:hover {
  background: #777;
}

  margin-top: 20px;
}

</head>

        </div>
```
