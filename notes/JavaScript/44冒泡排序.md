## 什么是冒泡排序？

相邻两个数比较，大的往后放（或小的往前放）

```
原始：[5, 3, 8, 4, 2]

第一轮：相邻比较交换
3 5 8 4 2  →  3 5 8 4 2  (5<8不换)
3 5 8 4 2  →  3 5 4 8 2  (8>4交换)
3 5 4 8 2  →  3 5 4 2 8  (8>2交换)
第一轮结束：最大值8到最后

第二轮：...
```

## 图解过程

```
[5, 3, 8, 4, 2]

第1轮：比较4次 → [3, 5, 4, 2, 8]
第2轮：比较3次 → [3, 4, 2, 5, 8]
第3轮：比较2次 → [3, 2, 4, 5, 8]
第4轮：比较1次 → [2, 3, 4, 5, 8]

规律：第i轮，比较 (n-i) 次
```

## 代码实现

```javascript
let arr = [5, 3, 8, 4, 2];

// 外层：轮数
for (let i = 0; i < arr.length - 1; i++) {
    // 内层：比较次数
    for (let j = 0; j < arr.length - 1 - i; j++) {
        // 交换
        if (arr[j] > arr[j + 1]) {
            let temp = arr[j];
            arr[j] = arr[j + 1];
            arr[j + 1] = temp;
        }
    }
}

console.log(arr);  // [2, 3, 4, 5, 8]
```

## ES6 简化交换

```javascript
// 交换两数
[arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
```

## 优化：提前结束

```javascript
let arr = [5, 3, 8, 4, 2];

for (let i = 0; i < arr.length - 1; i++) {
    let flag = true;  // 假设已排序
    
    for (let j = 0; j < arr.length - 1 - i; j++) {
        if (arr[j] > arr[j + 1]) {
            [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
            flag = false;  // 有交换
        }
    }
    
    if (flag) break;  // 没交换，已有序
}
```

## 注意事项

1. **外层轮数**：`arr.length - 1`
2. **内层次数**：`arr.length - 1 - i`
3. **交换要借助临时变量**（除非用ES6）

## 记忆口诀

> **冒泡排序**：相邻比较交换，外层轮数内层次，大的往后跑，小的往前冒
