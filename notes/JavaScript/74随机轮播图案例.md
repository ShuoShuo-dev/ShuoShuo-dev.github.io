# 随机轮播图案例

## 需求

页面刷新时，**随机**显示一张图片 + 对应标题 + 对应小圆点高亮。

## 数据准备

```javascript
const sliderData = [
  { url: './images/slider01.jpg', title: '新年特惠', color: 'rgb(100, 167, 103)' },
  { url: './images/slider02.jpg', title: '春季上新', color: 'rgb(38, 168, 157)' },
  { url: './images/slider03.jpg', title: '周年庆典', color: 'rgb(215, 117, 73)' }
];
```

## 完整实现

```javascript
// 1. 生成随机下标
const random = Math.floor(Math.random() * sliderData.length);

// 2. 获取对应数据
const data = sliderData[random];

// 3. 修改图片
const img = document.querySelector('.slider-wrapper img');
img.src = data.url;

// 4. 修改标题
const title = document.querySelector('.slider-footer p');
title.innerHTML = data.title;

// 5. 修改背景色
const footer = document.querySelector('.slider-footer');
footer.style.backgroundColor = data.color;

// 6. 小圆点高亮（排他思想）
const lis = document.querySelectorAll('.slider-indicator li');

// 先移除所有 active
for (let i = 0; i < lis.length; i++) {
  lis[i].classList.remove('active');
}

// 再给当前添加 active
lis[random].classList.add('active');
```

## 排他思想（重点）

```
操作前：先清除所有元素的样式
操作中：再给当前元素设置样式
```

```javascript
// 经典模式：干掉所有人，复活我自己
for (let i = 0; i < lis.length; i++) {
  lis[i].classList.remove('active');  // 先全删
}
lis[random].classList.add('active');  // 再加当前
```

## 渲染要素清单

| 需修改 | 元素 | 用什么属性 |
|--------|------|-----------|
| 图片 | `<img>` | `src` |
| 文字 | `<p>` | `innerHTML` |
| 背景色 | `<div>` | `style.backgroundColor` |
| 小圆点 | `<li>` | `classList.add/remove` |
