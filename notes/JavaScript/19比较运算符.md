# 比较运算符

## 一、比较运算符一览

| 运算符 | 说明 | 示例 | 结果 |
|:------:|------|------|------|
| `>` | 大于 | `3 > 5` | `false` |
| `<` | 小于 | `3 < 5` | `true` |
| `>=` | 大于等于 | `3 >= 3` | `true` |
| `<=` | 小于等于 | `2 <= 1` | `false` |
| `==` | 双等（值相等） | `5 == '5'` | `true` |
| `===` | 三等（全等） | `5 === '5'` | `false` |
| `!=` | 不等 | `5 != 3` | `true` |
| `!==` | 不全等 | `5 !== '5'` | `true` |

---

## 二、`==` vs `===`

```javascript
console.log(5 == '5');   // true（值相等，自动转换类型）
console.log(5 === '5');  // false（值和类型都要相等）
```

### 建议：尽量用 `===` 三等

```javascript
// ❌ 双等有坑
console.log('' == false);   // true（容易出错）

// ✅ 三等更安全
console.log('' === false);  // false
```

---

## 三、隐式转换

比较运算符会**自动转换类型**：

```javascript
console.log('2' > 1);     // true（'2'转数字）
console.log('hello' > 5); // false
console.log(NaN == NaN);  // false（NaN不等于自己）
```

---

## 四、小数比较（精度问题）

```javascript
// ❌ 不要直接比较小数
console.log(0.1 + 0.2);    // 0.30000000000000004
console.log(0.1 + 0.2 === 0.3);  // false

// ✅ 正确做法
console.log(Math.abs(0.1 + 0.2 - 0.3) < 0.0001);  // true
```

---

## 五、字符串比较

按 ASCII 码比较：

```javascript
console.log('a' > 'b');   // false（a=97, b=98）
console.log('abc' > 'abd'); // false（第三位a<b）
console.log('10' > '2');  // false（'1'<'2'）
```

中文按拼音比较：

```javascript
console.log('张' > '李');  // true（zhang > li）
```
