## 为什么需要循环？

不用写重复代码：

```javascript
// ❌ 重复写法
document.write('月薪过万<br>');
document.write('月薪过万<br>');
document.write('月薪过万<br>');

// ✅ 循环写法
let i = 1;
while (i <= 3) {
    document.write('月薪过万<br>');
    i++;
}
```

## 语法

```javascript
while (条件) {
    // 循环体：满足条件时重复执行
}
```

## 执行流程

```
     ┌──────────────┐
     │  1.初始化变量  │
     └──────┬───────┘
            ↓
     ┌──────┴───────┐
     │  2.判断条件    │←────────┐
     └──────┬───────┘         │
            ↓                 │
     ┌──────┴───────┐         │
     │  true/false   │         │
     └──┬───────────┘         │
        ↓                    │
   ┌────┴────┐                │
   ↓         ↓                │
  true      false             │
   ↓         ↓                │
   ↓    ┌────┴────┐           │
   ↓    │ 结束循环 │           │
   ↓    └─────────┘           │
   ↓                          │
 ┌──┴───────────┐              │
 │ 3.执行循环体   │────────────┘
 │ 4.变量变化     │
 └──────────────┘
```

## 循环三要素

| 要素 | 作用 | 示例 |
|------|------|------|
| 起始值 | 从哪里开始 | `let i = 1` |
| 终止条件 | 什么时候停 | `i <= 3` |
| 变化量 | 逐渐接近结束 | `i++` |

**缺一不可！缺变化量会死循环**

## 案例：打印3句话

```javascript
let i = 1;
while (i <= 3) {
    document.write(`我要吃第${i}个包子<br>`);
    i++;
}
```

## while vs if

| 语句 | 区别 |
|------|------|
| if | 判断一次，为真执行一次 |
| while | 判断多次，为真反复执行 |

## 注意事项

1. **必须有变化量**：否则死循环
2. **起始值在循环外**：只执行一次
3. **先判断后执行**：条件不满足可能一次都不执行

## 记忆口诀

> **while循环**：while小括号放条件，大括号里写循环体，三要素缺一不可
