## 为什么需要立即执行函数？

避免全局变量污染：

```javascript
// ❌ 全局变量可能冲突
let num = 10;
let num = 20;  // 报错

// ✅ 立即执行函数：独立作用域
(function() {
    let num = 10;
})();
```

## 两种写法

```javascript
// 写法1：包裹后调用
(function() {
    console.log('写法1');
})();

// 写法2：直接包裹
(function() {
    console.log('写法2');
}());
```

## 带参数

```javascript
(function(a, b) {
    console.log(a + b);
})(1, 2);  // 3

(function(x) {
    console.log(x * 2);
}(5));  // 10
```

## 实际应用

```javascript
// 避免 i 变量污染
for (var i = 0; i < 3; i++) {
    (function(index) {
        setTimeout(function() {
            console.log(index);
        }, 1000);
    })(i);
}
// 输出：0, 1, 2（正确）
```

## 多个 IIFE 之间

必须加分号隔开：

```javascript
(function() {
    console.log('第一个');
})();  // ← 分号必须

(function() {
    console.log('第二个');
})();
```

## 作用

| 作用 | 说明 |
|------|------|
| 避免污染全局 | 变量都在函数内部 |
| 模块化 | 独立的功能块 |
| 闭包场景 | 保存变量状态 |

## 注意事项

1. **必须加分号**：多个 IIFE 之间
2. **无需调用**：创建即执行
3. **可有参数**：和普通函数一样

## 记忆口诀

> **立即执行**：function 加括号立即跑，变量封闭不干扰，分号隔开防报错
