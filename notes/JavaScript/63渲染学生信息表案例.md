# 渲染学生信息表案例

## 核心思路

**数组 + for循环 + innerHTML + 模板字符串 → 动态生成表格行**

## 数据结构

```javascript
// 数组对象：数组里放多个对象，开发中非常常见
let students = [
  { name: '张三', age: 18, gender: '男', hometown: '北京' },
  { name: '李四', age: 19, gender: '女', hometown: '上海' },
  { name: '王五', age: 20, gender: '男', hometown: '广州' }
];
```

## 完整实现

```html
<table border="1">
  <thead>
    <tr>
      <th>序号</th><th>姓名</th><th>年龄</th><th>性别</th><th>家乡</th>
    </tr>
  </thead>
  <tbody id="tbody">
    <!-- JS 动态生成 -->
  </tbody>
</table>

<script>
let students = [
  { name: '张三', age: 18, gender: '男', hometown: '北京' },
  { name: '李四', age: 19, gender: '女', hometown: '上海' },
  { name: '王五', age: 20, gender: '男', hometown: '广州' }
];

// 遍历数组，拼接HTML字符串
let html = '';
for (let i = 0; i < students.length; i++) {
  html += `
    <tr>
      <td>${i + 1}</td>
      <td>${students[i].name}</td>
      <td>${students[i].age}</td>
      <td>${students[i].gender}</td>
      <td>${students[i].hometown}</td>
    </tr>
  `;
}

// 渲染到页面
document.querySelector('#tbody').innerHTML = html;
</script>
```

## 关键步骤

```
数据(数组对象) → for循环遍历 → 模板字符串拼接HTML → innerHTML渲染
```

| 步骤 | 说明 | 代码 |
|------|------|------|
| 1. 遍历 | `students[i]` 是每个对象 | `for (let i = 0; i < students.length; i++)` |
| 2. 取值 | `students[i].name` 取对象属性 | `${students[i].name}` |
| 3. 渲染 | 一次性写入，性能好 | `innerHTML = html` |

> **数据驱动视图**：只需修改数组数据，页面自动更新，无需手动改HTML。
