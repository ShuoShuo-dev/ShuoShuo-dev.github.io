# 焦点事件 + 小米搜索框案例

## 焦点事件

| 事件 | 说明 | 触发时机 |
|------|------|----------|
| `focus` | 获得焦点 | 点击输入框，光标闪烁 |
| `blur` | 失去焦点 | 点击输入框外部 |

> 焦点事件主要用于**表单元素**（input、textarea、select）

```javascript
const input = document.querySelector('input');

input.addEventListener('focus', function() {
  console.log('获得焦点');  // 光标在输入框里
});

input.addEventListener('blur', function() {
  console.log('失去焦点');  // 点击了输入框外面
});
```

## 小米搜索框案例

### 需求
- 输入框获得焦点 → 显示搜索历史下拉框
- 输入框失去焦点 → 隐藏下拉框

### HTML 结构

```html
<div class="mi">
  <input type="search" placeholder="小米手机">
  <ul class="result-list" style="display: none;">
    <li>小米15</li>
    <li>小米平板</li>
    <li>小米手环</li>
  </ul>
</div>
```

### 完整代码

```javascript
// 1. 获取元素
const input = document.querySelector('input');
const list = document.querySelector('.result-list');

// 2. 获得焦点 → 显示
input.addEventListener('focus', function() {
  list.style.display = 'block';
  input.classList.add('search-active');  // 添加边框样式
});

// 3. 失去焦点 → 隐藏
input.addEventListener('blur', function() {
  list.style.display = 'none';
  input.classList.remove('search-active');
});
```

### CSS 样式

```css
.search-active {
  border-color: #ff6700;  /* 小米橙色 */
}

.result-list {
  display: none;
  border: 1px solid #ccc;
  border-top: none;
}

.result-list li {
  padding: 8px 12px;
  cursor: pointer;
}

.result-list li:hover {
  background-color: #f5f5f5;
}
```

## 小结

| 场景 | 事件 |
|------|------|
| 搜索框显示历史 | `focus` |
| 搜索框隐藏历史 | `blur` |
| 验证输入内容 | `blur` |
| 输入框高亮边框 | `focus` + `classList` |
