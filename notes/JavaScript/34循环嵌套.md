## 为什么需要循环嵌套？

现实中的重复有**层级关系**：

```
第一天：学5个单词
第二天：学5个单词
第三天：学5个单词

= 3天 × 5个单词 = 15个单词
```

## 语法

```javascript
for (let i = 1; i <= 3; i++) {      // 外层：控制天数
    for (let j = 1; j <= 5; j++) {  // 内层：控制单词数
        console.log(`第${j}个单词`);
    }
}
```

## 执行规律

```
外层 i=1 时：
    内层 j=1,2,3,4,5 → 完成5次
外层 i=2 时：
    内层 j=1,2,3,4,5 → 完成5次
外层 i=3 时：
    内层 j=1,2,3,4,5 → 完成5次

总计：3 × 5 = 15 次
```

## 流程图

```
外层 i=1 ─┬─ 内层 j=1 → 打印
          ├─ 内层 j=2 → 打印
          ├─ 内层 j=3 → 打印
          ├─ 内层 j=4 → 打印
          └─ 内层 j=5 → 打印
                 ↓
外层 i=2 ─┬─ 内层 j=1 → 打印
          └─ ...（重复）
```

## 案例：记单词

```javascript
for (let day = 1; day <= 3; day++) {
    document.write(`第${day}天<br>`);
    for (let word = 1; word <= 5; word++) {
        document.write(`  记第${word}个单词<br>`);
    }
}
```

**输出**：
```
第1天
  记第1个单词
  记第2个单词
  ...
第2天
  ...
```

## 注意事项

1. **内外变量不同**：`i` 和 `j`
2. **外层循环慢**：外层执行1次，内层执行全部
3. **内层变量先变**：内层循环完，外层才变

## 记忆口诀

> **循环嵌套**：外层天数，内层次数，外层一次内层全跑完
