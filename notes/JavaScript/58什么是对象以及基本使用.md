## 为什么需要对象？

描述一个事物需要多个属性：

```javascript
// ❌ 数组：只能按顺序
let student = ['小红', 18, '女', 90];
// 90是什么？成绩？身高？不清楚

// ✅ 对象：描述清晰
let student = {
    name: '小红',
    age: 18,
    gender: '女',
    score: 90
};
// 清晰知道每个值的含义
```

## 对象是什么？

```
对象 = 属性 + 方法
属性 = 名词（描述特征）
方法 = 函数（描述行为）
```

## 声明对象

```javascript
// 字面量（常用）
let obj = {};

// 带数据
let student = {
    name: '小红',
    age: 18,
    gender: '女',
    study: function() {
        console.log('学习中');
    }
};
```

## 访问属性

```javascript
let student = {
    name: '小红',
    age: 18
};

// 点语法
console.log(student.name);  // '小红'
console.log(student.age);   // 18

// 调用方法
student.study();  // '学习中'
```

## 对比数组和对象

| 特点 | 数组 | 对象 |
|------|------|------|
| 结构 | 有序 | 无序 |
| 访问 | 索引 [0] | 属性名 .name |
| 适用 | 同类数据列表 | 事物属性集合 |

## 注意事项

1. **属性名可加引号**：通常省略
2. **方法本质**：属性值是函数
3. **typeof 对象**：`object`

## 记忆口诀

> **对象**：大括号包属性，属性名和属性值，方法就是函数值，点语法来访问
