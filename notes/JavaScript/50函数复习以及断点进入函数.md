## 函数知识回顾

```javascript
// 声明
function 函数名(形参1, 形参2 = 默认值) {
    // 代码
    return 结果;
}

// 调用
let result = 函数名(实参1, 实参2);
```

## 调试：进入函数内部

```
F10：下一步（不进入函数）
F11：进入函数（能看到函数内部怎么执行）
```

## 断点调试函数

```javascript
function getSum(n, m) {
    let sum = 0;
    for (let i = n; i <= m; i++) {
        sum += i;  // ← 在这打断点
    }
    return sum;
}

let result = getSum(1, 10);
console.log(result);
```

**调试步骤**：
1. F12 打开开发者工具
2. 点击第4行行号打断点
3. 刷新页面
4. 按 F11 进入函数
5. 观察 sum 和 i 的变化

## 函数注意点

| 问题 | 说明 |
|------|------|
| 同名函数 | 后面的覆盖前面的 |
| 实参多于形参 | 多余实参忽略 |
| 实参少于形参 | 未传值为 undefined |
| return vs break | return 结束函数，break 结束循环 |

## arguments 对象

```javascript
function fn() {
    console.log(arguments);  // 伪数组，存所有实参
}

fn(1, 2, 3);  // [1, 2, 3]
fn(1, 2);     // [1, 2]
```
