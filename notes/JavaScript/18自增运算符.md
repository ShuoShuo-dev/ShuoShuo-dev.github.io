# 自增运算符

## 一、自增 `++`

让变量每次加 1：

```javascript
let num = 10;
num++;        // 等价于 num = num + 1
console.log(num);  // 11
```

## 二、自减 `--`

让变量每次减 1：

```javascript
let num = 10;
num--;        // 等价于 num = num - 1
console.log(num);  // 9
```

---

## 三、前置 `++i` vs 后置 `i++`

### 单独使用时：没区别

```javascript
let i = 1;
i++;     // 2
++i;     // 2
// ✅ 结果一样
```

### 参与运算时：有区别

| 写法 | 表达式值 | i 的最终值 |
|------|---------|-----------|
| `++i + 1` | 先加后用（2+1=3） | 2 |
| `i++ + 1` | 先用后加（1+1=2） | 2 |

```javascript
let i = 1;

// 前置：先加1，再参与运算
console.log(++i + 1);  // 3
console.log(i);        // 2

let j = 1;

// 后置：先参与运算，再加1
console.log(j++ + 1);  // 2
console.log(j);        // 2
```

---

## 四、记忆口诀

```
自增自减要记清
单独使用没差别
参与运算要分清
++在前先加后用
++在后先用后加
开发常用后置++
```

---

## 五、使用场景

常用于**计数**：

```javascript
// 统计点击次数
let count = 0;
button.onclick = function() {
  count++;
  console.log('点击了' + count + '次');
};

// 遍历数组
for (let i = 0; i < arr.length; i++) {
  console.log(arr[i]);
}
```
