## 为什么有三元运算符？

双分支写法较繁琐，三元运算符是**简化版的 if else**

## 语法

```javascript
条件 ? 值1 : 值2
```

- 条件为 `true` → 返回 `值1`
- 条件为 `false` → 返回 `值2`

## 对比 if else

```javascript
// if else 版本（3行）
if (3 > 5) {
    result = 3;
} else {
    result = 5;
}

// 三元运算符（1行）
let result = 3 > 5 ? 3 : 5;
```

## 基本示例

```javascript
// 比较数字
console.log(3 > 5 ? 3 : 5);  // 5

// 输出判断
3 < 5 ? console.log('真的') : console.log('假的');  // '真的'

// 赋值操作
let age = 20;
let type = age >= 18 ? '成年' : '未成年';
```

## 案例：求最大值

```javascript
// 方法1：if else
let num1 = +prompt('请输入第一个数：');
let num2 = +prompt('请输入第二个数：');

if (num1 > num2) {
    alert(num1);
} else {
    alert(num2);
}

// 方法2：三元运算符
alert(num1 > num2 ? num1 : num2);
```

## 嵌套三元运算符

```javascript
// 求三个数最大值
let a = 10, b = 5, c = 8;
let max = a > b ? (a > c ? a : c) : (b > c ? b : c);
```

**注意**：嵌套可读性差，慎用

## 使用场景

| 场景 | 推荐 |
|------|------|
| 简单二选一赋值 | ✅ 三元运算符 |
| 复杂条件判断 | ❌ 用 if else |
| 多选一判断 | ❌ 用 if else if |

## 记忆口诀
