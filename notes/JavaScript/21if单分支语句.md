## 表达式 vs 语句

| 类型 | 定义 | 示例 | 有值？ |
|------|------|------|--------|
| 表达式 | 可被求值的代码 | `3 + 4`、`x = 7`、`num++` | ✅ |
| 语句 | 可执行的代码片段 | `if`、`for`、`alert` | ❌ |

## 三大流程控制

```
顺序结构 → 分支结构 → 循环结构
  ↓          ↓           ↓
 依次执行   选择执行    反复执行
```

## if 单分支语法

```javascript
if (条件) {
    // 条件为 true 时执行
    满足条件要执行的代码
}
```

## 条件隐式转换

小括号内会自动转布尔值：

| 真 (true) | 假 (false) |
|-----------|------------|
| 非 0 数字 | 0 |
| 非空字符串 | 空字符串 `""` |
| 对象/数组 | null / undefined |

## 注意事项

- 大括号可省略，但不推荐
- 建议始终写大括号，结构更清晰

## 案例：高考成绩

```javascript
let score = +prompt('请输入高考成绩：');

if (score >= 700) {
    alert('恭喜考入黑马程序员！');
}
```

## 记忆口诀

> **单分支**：if 小括号放条件，大括号里写要做的事
