# 赋值运算符

## 一、赋值 `=`

把右边的值赋给左边：

```javascript
let age = 18;  // 把 18 存入 age
```

---

## 二、复合赋值运算符

### `+=` 加等于

```javascript
let num = 10;
num += 5;   // 等价于 num = num + 5
console.log(num);  // 15
```

### `-=` 减等于

```javascript
let num = 10;
num -= 3;   // 等价于 num = num - 3
console.log(num);  // 7
```

### `*=` 乘等于

```javascript
let num = 10;
num *= 2;   // 等价于 num = num * 2
console.log(num);  // 20
```

### `/=` 除等于

```javascript
let num = 10;
num /= 2;   // 等价于 num = num / 2
console.log(num);  // 5
```

### `%=` 取模等于

```javascript
let num = 10;
num %= 3;   // 等价于 num = num % 3
console.log(num);  // 1
```

---

## 三、对比表

| 写法 | 等价于 |
|------|--------|
| `num += 5` | `num = num + 5` |
| `num -= 5` | `num = num - 5` |
| `num *= 5` | `num = num * 5` |
| `num /= 5` | `num = num / 5` |
| `num %= 5` | `num = num % 5` |
