# client家族和offset家族

## 一、offset家族（偏移量）

### 1.1 什么是offset家族

`offset` 系列属性用于获取元素**自身**的位置信息，基于**已定位的父元素**计算。

### 1.2 常用属性

| 属性 | 说明 |
|------|------|
| `offsetParent` | 返回最近已定位的祖先元素 |
| `offsetTop` | 元素上边框到 `offsetParent` 顶部的距离 |
| `offsetLeft` | 元素左边框到 `offsetParent` 左侧的距离 |
| `offsetWidth` | 元素的宽度（width + padding + border） |
| `offsetHeight` | 元素的高度（height + padding + border） |

### 1.3 示例代码

```javascript
// 获取元素的位置和尺寸
const box = document.querySelector('.box');

console.log(box.offsetParent);    // 最近定位祖先元素
console.log(box.offsetTop);      // 上偏移量
console.log(box.offsetLeft);     // 左偏移量
console.log(box.offsetWidth);    // 包含border的宽度
console.log(box.offsetHeight);   // 包含border的高度
```

### 1.4 offsetWidth 与 style.width 的区别

| 特性 | offsetWidth | style.width |
|------|-------------|-------------|
| 类型 | 只读属性 | 可读写 |
| 值 | number（带单位） | string |
| 包含 | width + padding + border | 只有width |
| 来源 | 计算后的CSS | 行内样式 |

```javascript
const box = document.querySelector('.box');

console.log(box.offsetWidth);   // 200（包含padding和border）
console.log(box.style.width);   // "100px"（仅width）

// 修改宽度
box.style.width = '150px';
```

## 二、client家族（客户端）

### 2.1 什么是client家族

`client` 系列属性用于获取元素**可视区域**相关的信息，基于**浏览器可视窗口**计算。

### 2.2 常用属性

| 属性 | 说明 |
|------|------|
| `clientTop` | 元素上边框的宽度（border-top-width） |
| `clientLeft` | 元素左边框的宽度（border-left-width） |
| `clientWidth` | 元素可视宽度（width + padding，不含border） |
| `clientHeight` | 元素可视高度（height + padding，不含border） |

### 2.3 示例代码

```javascript
const box = document.querySelector('.box');

console.log(box.clientTop);      // 边框宽度
console.log(box.clientLeft);     // 左边框宽度
console.log(box.clientWidth);    // 可视宽度
console.log(box.clientHeight);   // 可视高度
```

### 2.4 clientWidth 与 offsetWidth 的区别

```javascript
/* 假设CSS：
.box {
  width: 100px;
  padding: 10px;
  border: 5px solid red;
}
*/

console.log(box.clientWidth);   // 120 (100 + 10*2) - 不含border
console.log(box.offsetWidth);   // 130 (100 + 10*2 + 5*2) - 含border
```

## 三、对比总结

### 3.1 尺寸对比

```
┌─────────────────────────────────────┐
│ border                               │
│  ┌────────────────────────────────┐  │
│  │ padding                         │  │
│  │  ┌────────────────────────────┐ │  │
│  │  │                            │ │  │
│  │  │        width/height        │ │  │
│  │  │                            │ │  │
│  │  └────────────────────────────┘ │  │
│  │                                  │  │
│  └────────────────────────────────┘  │
└─────────────────────────────────────┘

clientWidth = width + padding        （不含border）
offsetWidth  = width + padding + border
```

### 3.2 位置对比

| 属性 | 说明 | 计算基准 |
|------|------|----------|
| `offsetTop` | 到定位父元素顶部的距离 | offsetParent |
| `clientTop` | 边框宽度 | - |
| `scrollTop` | 滚动条隐藏的顶部距离 | 滚动容器 |

## 四、实战应用

### 4.1 模态框居中

```javascript
function centerModal(modal) {
  // 使用offsetWidth获取模态框尺寸
  const modalWidth = modal.offsetWidth;
  const modalHeight = modal.offsetHeight;

  // 使用clientWidth获取视口尺寸
  const viewWidth = document.documentElement.clientWidth;
  const viewHeight = document.documentElement.clientHeight;

  // 计算居中位置
  modal.style.left = (viewWidth - modalWidth) / 2 + 'px';
  modal.style.top = (viewHeight - modalHeight) / 2 + 'px';
}
```

### 4.2 拖拽效果

```javascript
const draggable = document.querySelector('.draggable');

draggable.addEventListener('mousedown', function(e) {
  // 记录鼠标按下时，元素相对于视口的位置
  const offsetX = e.clientX - this.offsetLeft;
  const offsetY = e.clientY - this.offsetTop;

  document.addEventListener('mousemove', move);
  document.addEventListener('mouseup', stop);

  function move(e) {
    // 使用clientX/clientY计算新位置
    draggable.style.left = e.clientX - offsetX + 'px';
    draggable.style.top = e.clientY - offsetY + 'px';
  }

  function stop() {
    document.removeEventListener('mousemove', move);
    document.removeEventListener('mouseup', stop);
  }
});
```

### 4.3 获取元素在页面中的位置

```javascript
function getElementPosition(element) {
  let left = 0;
  let top = 0;

  while (element) {
    // 累加offsetLeft和offsetTop
    left += element.offsetLeft;
    top += element.offsetTop;
    // 移动到offsetParent继续累加
    element = element.offsetParent;
  }

  return { left, top };
}

// 使用
const box = document.querySelector('.box');
const pos = getElementPosition(box);
console.log(`元素位置: left=${pos.left}px, top=${pos.top}px`);
```

## 五、注意事项

### 5.1 offsetParent 不一定是 parentNode

```javascript
<div class="parent">
  <div class="child">
    <div class="grandson"></div>
  </div>
</div>

const grandson = document.querySelector('.grandson');

// 如果 .parent 设置了 position，offsetParent 就是 .parent
// 如果 .child 设置了 position，offsetParent 就是 .child
// 如果都没有设置，offsetParent 就是 body
```

### 5.2 display:none 的元素

```javascript
const hidden = document.querySelector('.hidden');
hidden.style.display = 'none';

console.log(hidden.offsetWidth);  // 0
console.log(hidden.offsetHeight); // 0

// 临时显示再获取
hidden.style.visibility = 'hidden';
hidden.style.display = 'block';
const width = hidden.offsetWidth;
hidden.style.display = 'none';
hidden.style.visibility = 'visible';
```

## 七、记忆口诀

- **offset**：找"偏移"，基于定位父元素，**包含border**
- **client**：找"客户端"，基于浏览器窗口，**不含border**
- **scroll**：找"滚动"，滚动了多少
