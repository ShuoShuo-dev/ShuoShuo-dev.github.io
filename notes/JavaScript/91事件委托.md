# 91事件委托

## 一、什么是事件委托

**事件委托**：利用事件冒泡机制，将子元素的事件绑定到父元素身上，由父元素统一处理。

### 1.1 原理图示

```
        ┌─────────────────┐
        │   父元素 ul     │ ← 实际绑定在这里
        │  ┌─────────────┐ │
        │  │ li 1        │ │
        │  ├─────────────┤ │
        │  │ li 2        │ │ ← 点击任意 li
        │  ├─────────────┤ │
        │  │ li 3        │ │
        │  └─────────────┘ │
        └─────────────────┘
              ↑ 事件冒泡 ↑
```

### 1.2 为什么使用事件委托

| 方式 | 优点 | 缺点 |
|-----|------|------|
| 传统绑定 | 简单直观 | 大量绑定、动态元素无效 |
| 事件委托 | 性能高、支持动态元素 | 需要判断事件源 |

## 二、jQuery 事件委托

### 2.1 基本语法

```javascript
// 语法
$(父元素).on('事件名', '子元素选择器', 处理函数)

// 示例：ul 下所有 li 点击
$('ul').on('click', 'li', function() {
    console.log('点击了:', $(this).text());
});
```

### 2.2 this 指向

```javascript
$('ul').on('click', 'li', function() {
    // this 指向触发事件的子元素（li）
    console.log(this); // 指向具体的 li DOM
    
    // jQuery 对象
    console.log($(this));
});
```

### 2.3 获取事件源信息

```javascript
$('ul').on('click', 'li', function(event) {
    // 事件对象
    console.log(event.currentTarget); // 当前绑定事件的元素（ul）
    console.log(event.target);         // 实际触发事件的元素（li）
    console.log(event.delegateTarget); // 委托的父元素（ul）
});
```

## 三、事件委托的好处

### 3.1 性能优化

```javascript
// ❌ 错误：100个li绑定100个事件
$('li').on('click', function() {
    console.log('点击');
});

// ✅ 正确：只绑定1个事件
$('ul').on('click', 'li', function() {
    console.log('点击');
});
```

### 3.2 动态元素支持

```javascript
// ❌ 普通绑定：新增的 li 没有点击事件
$('li').on('click', function() {
    console.log('点击');
});

// 点击按钮添加新 li
$('#addBtn').on('click', function() {
    $('ul').append('<li>新添加的li</li>');
});

// ✅ 事件委托：新添加的 li 也有点击事件
$('ul').on('click', 'li', function() {
    console.log('点击');
});
```

### 3.3 完整示例

```javascript
$(function() {
    var $todoList = $('.todo-list');
    var $input = $('#todoInput');
    
    // 1. 添加任务（使用事件委托）
    $('#addBtn').on('click', function() {
        var value = $input.val().trim();
        if (!value) return;
        
        var $li = $('<li class="todo-item">' +
            '<input type="checkbox">' +
            '<span>' + value + '</span>' +
            '<button class="delete-btn">删除</button>' +
            '</li>');
        
        $todoList.append($li);
        $input.val('');
    });
    
    // 2. 删除任务（事件委托）
    // 注意：.deleted-btn 是动态添加的，必须用委托
    $todoList.on('click', '.delete-btn', function() {
        $(this).closest('li').remove();
    });
    
    // 3. 完成任务（事件委托）
    $todoList.on('change', 'input[type="checkbox"]', function() {
        $(this).next('span').toggleClass('completed');
    });
});
```

## 四、事件委托的注意事项

### 4.1 选择器范围

```javascript
// 只委托 li 和 span
$('ul').on('click', 'li, span', function() {
    console.log('点击了');
});

// 不委托链接点击
$('div').on('click', 'button:not(.no-delegate)', function() {
    console.log('按钮点击');
});
```

### 4.2 阻止冒泡

```javascript
$('ul').on('click', 'li', function(event) {
    console.log('li clicked');
    
    // 阻止冒泡
    event.stopPropagation();
});

$('ul').on('click', function() {
    console.log('ul clicked - 不会触发');
});
```

### 4.3 阻止默认行为

```javascript
$('ul').on('click', 'a', function(event) {
    event.preventDefault();
    console.log('链接被阻止了');
});
```

## 五、原生 JS 事件委托

```javascript
// 原生实现
document.querySelector('ul').addEventListener('click', function(event) {
    // 判断是否是 li
    if (event.target.tagName === 'LI') {
        console.log('点击了:', event.target.textContent);
    }
    
    // 或者使用 matches
    if (event.target.matches('li')) {
        console.log('点击了:', event.target.textContent);
    }
});
```

## 六、面试要点

1. **事件委托原理**：利用事件冒泡，将子元素事件绑定到父元素
2. **jQuery 语法**：`$(父).on('事件', '子选择器', 处理函数)`
3. **优势**：性能优化 + 支持动态元素
4. **this 指向**：回调函数中 this 指向触发事件的子元素
5. **原生实现**：`addEventListener` + `event.target` 判断

## 七、综合练习

```javascript
// 表格行点击高亮
$('table').on('click', 'tr', function() {
    // 排除表头
    if ($(this).find('th').length) return;
    
    // 切换选中状态
    $(this).toggleClass('active')
           .siblings().removeClass('active');
});

// 表格排序点击
$('table').on('click', 'th', function() {
    var index = $(this).index();
    console.log('按第' + (index + 1) + '列排序');
});
```
