# 模板字符串

## 一、为什么需要模板字符串

### 传统拼接（麻烦）

```javascript
let name = '张三';
let age = 18;

document.write('我叫' + name + '，今年' + age + '岁');
```

问题：引号多，容易出错

---

## 二、模板字符串语法

### 基本用法

```javascript
let name = '张三';
let age = 18;

// 使用反引号 + ${变量名}
document.write(`我叫${name}，今年${age}岁`);
```

### 语法要点

| 要点 | 说明 |
|------|------|
| 外层 | 必须用**反引号** `` ` `` |
| 变量 | 用 `${变量名}` 包裹 |

---

## 三、反引号位置

```
键盘左上角，ESC下面
英文状态下，打出来是 ` （反斜点）
```

---

## 四、模板字符串优势

| 对比 | 传统拼接 | 模板字符串 |
|------|---------|-----------|
| 引号 | 容易混淆 | 统一用反引号 |
| 换行 | 需要 + '\n' + | 直接回车 |
| 可读性 | 差 | 好 |

---

## 五、综合示例

```javascript
let uname = prompt('请输入姓名');
let age = prompt('请输入年龄');

// 模板字符串输出
document.write(`
  <div>
    <h2>用户信息</h2>
    <p>姓名：${uname}</p>
    <p>年龄：${age}</p>
  </div>
`);
```
