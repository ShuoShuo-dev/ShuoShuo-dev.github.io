## 为什么需要函数？

避免重复代码：

```javascript
// ❌ 重复
alert('你好');
alert('再见');
alert('欢迎');

// ✅ 函数
function sayHi() {
    alert('你好');
}
sayHi();
sayHi();
sayHi();
```

## 声明函数

```javascript
function 函数名() {
    // 函数体：要执行的代码
}
```

## 调用函数

```javascript
function sayHi() {
    console.log('你好');
}

sayHi();  // 调用一次执行一次
sayHi();
sayHi();
```

## 命名规范

| 规范 | 正确 | 错误 |
|------|------|------|
| 小驼峰 | `getSum` | `getsum` |
| 动词开头 | `getName` | `name` |
| 见名知意 | `calcAge` | `fn` |

### 常用动词前缀

```
get  获取    set 设置
has  是否有  is  是否
add  添加    del 删除
show 显示    hide 隐藏
```

## 封装案例

```javascript
// 封装：计算1-100之和
function getSum() {
    let sum = 0;
    for (let i = 1; i <= 100; i++) {
        sum += i;
    }
    console.log(sum);
}

getSum();  // 5050
```

## 函数 vs 循环

| 特点 | 循环 | 函数 |
|------|------|------|
| 执行时机 | 写完立即执行 | 随时调用 |
| 控制位置 | 不能 | 能 |
| 重复使用 | 不能 | 能 |

## 记忆口诀

> **函数**：function 定函数，函数名后跟括号，代码写在大括号，调用函数名()
