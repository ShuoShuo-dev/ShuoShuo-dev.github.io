# 逻辑运算符以及优先级

## 一、逻辑与 `&&`（并且）

**口诀**：一假则假（全真才真）

```javascript
console.log(true && true);   // true
console.log(true && false);   // false
console.log(false && true);   // false
console.log(false && false);  // false
```

### 实际应用

```javascript
let age = 20;
// 判断年龄是否在 18-60 之间
console.log(age >= 18 && age <= 60);  // true
```

---

## 二、逻辑或 `||`（或者）

**口诀**：一真则真（有真就真）

```javascript
console.log(true || true);   // true
console.log(true || false);  // true
console.log(false || true);  // true
console.log(false || false); // false
```

### 实际应用

```javascript
let vip = false;
let money = 100;
// VIP用户或者有100元以上可以进入
console.log(vip || money >= 100);  // true
```

---

## 三、逻辑非 `!`（取反）

**口诀**：真假颠倒

```javascript
console.log(!true);   // false
console.log(!false);  // true
console.log(!!true);  // true（双重取反变原值）
```

### 实际应用

```javascript
let isLogin = false;
// 取反：未登录时显示登录按钮
console.log(!isLogin);  // true
```

---

## 四、逻辑短路

### `&&` 短路

```javascript
false && console.log('不会执行');
true && console.log('会执行');
```

### `||` 短路

```javascript
true || console.log('不会执行');
false || console.log('会执行');
```

---

## 五、运算符优先级

从高到低：

```
最高：小括号 ()
其次：一元运算符 ! ++ --
其次：算术运算符 * / % + -
其次：比较运算符 > < >= <= == === != !==
其次：逻辑与 &&
其次：逻辑或 ||
最低：赋值运算符 =
```

### 记忆口诀

```
先括号，再非
乘除加减算术运算
比较运算排第三
逻辑运算在后边
赋值最低最后算
```

---

## 六、综合示例

```javascript
let age = 25;
let score = 90;

// 判断是否成年且及格
console.log(age >= 18 && score >= 60);  // true

// 判断是否未成年或不及格
console.log(age < 18 || score < 60);   // false
```
