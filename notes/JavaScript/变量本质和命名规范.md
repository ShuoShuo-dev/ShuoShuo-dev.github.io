# 变量本质和命名规范

## 一、变量的本质

### 内存比喻

```
内存 = 超市的储物柜
变量 = 柜子
变量名 = 柜子号码（用来找到柜子）
变量值 = 柜子里的东西
```

```javascript
let age = 18;
// 相当于在内存中开辟一个叫 "age" 的柜子，里面放 18
```

### 关键点

- 变量存在**内存**中（不是硬盘）
- 关机或刷新页面，变量会消失
- 变量名是柜子的"名字"，变量值是柜子里的"东西"

---

## 二、命名规则（必须遵守）

### 1. 不能用关键字

```javascript
let let = 1;    // ❌ let 是关键字
let if = 2;     // ❌ if 是关键字
let for = 3;    // ❌ for 是关键字
```

常见关键字：`let`、`var`、`if`、`else`、`for`、`function` 等

### 2. 组成规则

```javascript
// ✅ 可以：字母、数字、下划线、$
let name1 = '张三';
let _age = 18;
let $box = '盒子';

// ❌ 不行：数字开头
let 2name = '李四';

// ❌ 不行：短横线、空格等
let user-name = '王五';
let user name = '赵六';
```

### 3. 严格区分大小写

```javascript
let pink = '粉色';
let Pink = '粉色';  // ⚠️ 这是另一个变量！
```

---

## 三、命名规范（推荐遵守）

### 1. 名字要有意义

```javascript
// ❌ 无意义
let a = 18;
let b = '男';

// ✅ 有意义
let age = 18;       // 年龄
let gender = '男'; // 性别
```

### 2. 使用小驼峰命名法

多个单词组成的变量名：

```javascript
// ❌ 不规范
let username = '张三';        // 单词连在一起
let user_name = '李四';      // 下划线分隔
let UserName = '王五';       // 大驼峰

// ✅ 规范：小驼峰
let userName = '张三';       // 第一个单词小写，后面单词首字母大写
let myAge = 25;
let totalPrice = 99.9;
```

### 驼峰命名对比

| 类型 | 格式 | 示例 |
|------|------|------|
| 小驼峰 | userName | 变量名常用 |
| 大驼峰 | UserName | 类名常用 |
| 下划线 | user_name | 老项目/SQL常用 |

## 四、合法性判断

```javascript
// ❌ 不合法
21age      // 数字开头
let        // 关键字
user-name  // 短横线
user@name  // 特殊符号

// ✅ 合法但不规范
_age       // 可以但不推荐
userName   // ✅ 小驼峰，最推荐
```
