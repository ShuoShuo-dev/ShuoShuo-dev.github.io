# 用户倒计时效果（注册协议）

## 需求

- 按钮初始**禁用**，显示倒计时 "我已阅读用户协议 (5s)"
- 每秒减 1，到 0 时按钮**启用**，文字变为 "同意"

## 完整代码

```javascript
// 1. 获取按钮
const btn = document.querySelector('.btn');

// 2. 声明倒计时秒数
let i = 5;

// 3. 开启定时器
const timerId = setInterval(function() {
  i--;

  // 4. 更新按钮文字
  btn.innerHTML = `我已阅读用户协议 (${i}s)`;

  // 5. 判断是否到时间
  if (i === 0) {
    clearInterval(timerId);    // 停止定时器
    btn.disabled = false;       // 启用按钮
    btn.innerHTML = '同意';     // 修改文字
  }
}, 1000);
```

## 执行流程

```
页面加载
  → 按钮：disabled + "我已阅读用户协议 (5s)"
  → 1秒后 → "(4s)"
  → 1秒后 → "(3s)"
  → 1秒后 → "(2s)"
  → 1秒后 → "(1s)"
  → 1秒后 → disabled=false, 文字="同意" ✅
```

## 关键知识点

| 步骤 | 代码 | 说明 |
|------|------|------|
| 获取按钮 | `querySelector('.btn')` | 事件源 |
| 更新文字 | `btn.innerHTML = ...` | 按钮是双标签 |
| 禁用按钮 | `btn.disabled = true/false` | 布尔属性 |
| 停止定时器 | `clearInterval(timerId)` | 防止无限执行 |

## HTML 结构参考

```html
<button class="btn" disabled>我已阅读用户协议 (5s)</button>
```

> **注意**：`<button>` 是双标签，修改文字用 `innerHTML`，不用 `value`。
