## 查：访问元素

```javascript
let arr = ['小红', '小明', '小李'];
console.log(arr[0]);  // '小红'
console.log(arr[2]);  // '小李'
```

## 改：修改元素

```javascript
let arr = ['小红', '小明', '小李'];

// 修改单个
arr[1] = '小王';
console.log(arr);  // ['小红', '小王', '小李']

// 修改多个：遍历
for (let i = 0; i < arr.length; i++) {
    arr[i] = arr[i] + '老师';  // 拼接
}
console.log(arr);  // ['小红老师', '小王老师', '小李老师']
```

## 遍历修改元素

```javascript
let nums = [1, 2, 3, 4, 5];

// 每个数翻倍
for (let i = 0; i < nums.length; i++) {
    nums[i] *= 2;
}
console.log(nums);  // [2, 4, 6, 8, 10]
```

## 简化写法

```javascript
let arr = [85, 90, 78, 92];
let max = arr[0];

for (let i = 1; i < arr.length; i++) {
    // 三元运算符简化
    max < arr[i] ? max = arr[i] : max;
}
```

## 对比：查 vs 改

| 操作 | 语法 | 说明 |
|------|------|------|
| 查 | `arr[i]` | 读取值 |
| 改 | `arr[i] = 值` | 赋值覆盖 |

## 注意事项

1. **直接赋值覆盖**：不需要push等操作
2. **索引必须在范围内**：否则新增而不是修改
3. **遍历修改**：用 `arr[i] = ...` 形式

## 记忆口诀

> **数组修改**：中括号加索引，赋值号来修改，直接覆盖原值
