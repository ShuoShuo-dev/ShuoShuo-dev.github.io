## 删除方法一览

| 方法 | 删除位置 | 返回值 | 示例 |
|------|----------|--------|------|
| pop() | 末尾 | 删除的元素 | `[1,2,3].pop()` → 3 |
| shift() | 开头 | 删除的元素 | `[1,2,3].shift()` → 1 |
| splice() | 指定位置 | 删除的元素数组 | `[1,2,3].splice(1,1)` → [2] |

## pop：删除末尾

```javascript
let arr = ['red', 'green', 'blue'];

// 删除并返回最后一个
let deleted = arr.pop();
console.log(deleted);  // 'blue'
console.log(arr);     // ['red', 'green']
```

## shift：删除开头

```javascript
let arr = ['red', 'green', 'blue'];

// 删除并返回第一个
let deleted = arr.shift();
console.log(deleted);  // 'red'
console.log(arr);     // ['green', 'blue']
```

## splice：指定位置删除

```javascript
let arr = ['red', 'green', 'blue'];

// 从索引1开始，删除1个
let deleted = arr.splice(1, 1);
console.log(deleted);  // ['green']
console.log(arr);     // ['red', 'blue']
```

### splice 参数说明

```
array.splice(起始位置, 删除个数, 插入的元素...)
```

```javascript
let arr = ['a', 'b', 'c', 'd'];

// 删除并插入
arr.splice(1, 2, 'x', 'y');
console.log(arr);  // ['a', 'x', 'y', 'd']
```

## 增删改对比

| 操作 | 方法 | 说明 |
|------|------|------|
| 增末尾 | push() | 返回新长度 |
| 增开头 | unshift() | 返回新长度 |
| 删末尾 | pop() | 返回删除元素 |
| 删开头 | shift() | 返回删除元素 |
| 删/插入 | splice() | 返回删除元素 |

## 记忆口诀

> **pop**：末尾出去一个
> **shift**：开头出去一个
> **splice**：想删哪里删哪里
