## 打擂台算法

就像比武大赛，**两两比较**，留下最强的

```
擂主 vs 挑战者
   ↓
擂主更强 → 擂主留下
   ↓
挑战者更强 → 挑战者成为新擂主
   ↓
重复直到没人挑战 → 擂主就是最大值
```

## 求最大值

```javascript
let arr = [85, 90, 78, 92, 88];

// 擂主默认第一个
let max = arr[0];

// 依次比较
for (let i = 1; i < arr.length; i++) {
    if (arr[i] > max) {
        max = arr[i];  // 遇到更强的，更新擂主
    }
}

console.log('最大值：' + max);  // 92
```

## 求最小值

```javascript
let arr = [85, 90, 78, 92, 88];

// 擂主默认第一个
let min = arr[0];

// 依次比较
for (let i = 1; i < arr.length; i++) {
    if (arr[i] < min) {
        min = arr[i];  // 遇到更弱的，更新擂主
    }
}

console.log('最小值：' + min);  // 78
```

## 图解过程

```
arr = [85, 90, 78, 92, 88]

步骤1：max = 85（默认）
步骤2：90 > 85 → max = 90
步骤3：78 < 90 → 不变
步骤4：92 > 90 → max = 92
步骤5：88 < 92 → 不变

结果：max = 92
```

## 同时求最大最小值

```javascript
let arr = [85, 90, 78, 92, 88];
let max = arr[0];
let min = arr[0];

for (let i = 1; i < arr.length; i++) {
    if (arr[i] > max) max = arr[i];
    if (arr[i] < min) min = arr[i];
}

console.log(`最大值：${max}，最小值：${min}`);
```

## 注意事项

1. **默认第一个**：擂主从 arr[0] 开始
2. **从第二个开始比**：i = 1
3. **注意比较符号**：最大值用 `>`，最小值用 `<`

## 记忆口诀

> **最值**：擂主默认第一个，依次比较来挑战，大于擂主就替换，遍历结束出答案
