# 94.页面加载事件和页面滚动事件

## 一、页面加载事件

### 1. DOMContentLoaded

当 HTML 文档完全解析完成，**DOM 树构建完毕**时触发，不等待图片、样式表、iframe 加载完成。

```javascript
document.addEventListener('DOMContentLoaded', function() {
    console.log('DOM 已加载完成，可以操作 DOM 了！');
    const btn = document.getElementById('btn'); // 此时可以获取元素
});
```

**特点：**
- 不需要等待外部资源（图片、CSS等）加载完成
- 比 `load` 事件更早触发
- 推荐在大多数场景使用

### 2. load

当页面**所有资源**（图片、CSS、JS、iframe等）全部加载完成后触发。

```javascript
window.addEventListener('load', function() {
    console.log('页面所有资源加载完成！');
    const img = document.querySelector('img');
    console.log(img.width); // 此时图片尺寸已可用
});
```

### 3. beforeunload

用户**即将离开页面**时触发，常用于提示用户保存未保存的内容。

```javascript
window.addEventListener('beforeunload', function(e) {
    // 如果有未保存的更改
    if (hasUnsavedChanges) {
        e.preventDefault();
        e.returnValue = ''; // 兼容旧版浏览器
        return ''; // 兼容新版浏览器
    }
});
```

### 4. unload

页面**完全关闭**时触发（现代浏览器支持有限）。

```javascript
window.addEventListener('unload', function() {
    // 发送统计数据
    navigator.sendBeacon('/analytics', JSON.stringify(data));
});
```

### 加载事件对比

| 事件 | 触发时机 | 使用场景 |
|------|----------|----------|
| DOMContentLoaded | DOM树构建完成 | **推荐首选**，操作 DOM |
| load | 所有资源加载完成 | 需要获取图片尺寸等 |
| beforeunload | 即将离开页面 | 提示保存、发送数据 |
| readystatechange | 文档状态变化 | 兼容性场景 |

### 完整示例

```javascript
document.addEventListener('DOMContentLoaded', function() {
    console.log('1. DOMContentLoaded - DOM已就绪');
    initApp(); // 初始化应用
});

window.addEventListener('load', function() {
    console.log('2. load - 所有资源加载完成');
    hideLoadingScreen();
});

window.addEventListener('beforeunload', function(e) {
    if (app.hasUnsavedData()) {
        e.preventDefault();
        e.returnValue = '您有未保存的内容，确定要离开吗？';
    }
});
```

## 二、页面滚动事件

### 1. scroll 事件

当用户滚动页面或元素时持续触发。

```javascript
// 监听整个页面滚动
window.addEventListener('scroll', function() {
    console.log('页面滚动了！');
    console.log('滚动距离:', window.scrollY);
});

// 监听某个元素滚动
const container = document.querySelector('.scroll-container');
container.addEventListener('scroll', function() {
    console.log('元素滚动了！');
    console.log('垂直滚动位置:', this.scrollTop);
    console.log('水平滚动位置:', this.scrollLeft);
});
```

### 2. 获取滚动位置

```javascript
// 现代浏览器推荐
const scrollY = window.scrollY;
const scrollX = window.scrollX;

// 兼容旧版浏览器
const scrollY = window.pageYOffset;
const scrollX = window.pageXOffset;

// 获取元素内部滚动位置
const elementScrollTop = element.scrollTop;
const elementScrollLeft = element.scrollLeft;
```

### 3. scrollTo() - 滚动到指定位置

```javascript
// 滚动到页面顶部
window.scrollTo(0, 0);

// 滚动到指定元素
const target = document.getElementById('target');
target.scrollIntoView();

// 滚动到指定位置（带平滑动画）
window.scrollTo({
    top: 500,
    left: 0,
    behavior: 'smooth' // 平滑滚动
});

// 相对当前滚动位置滚动
window.scrollBy({
    top: 100, // 向下滚动100px
    left: 0,
    behavior: 'smooth'
});
```

### 4. scrollByPages() - 按页滚动

```javascript
// 向下滚动一页
window.scrollByPages(1);

// 向上滚动一页
window.scrollByPages(-1);
```

### 5. scrollIntoView() - 滚动到元素可见

```javascript
const element = document.getElementById('target');

// 默认：滚动到元素顶部与视口顶部对齐
element.scrollIntoView();

// 滚动到元素底部与视口底部对齐
element.scrollIntoView(false);

// 带平滑动画
element.scrollIntoView({ behavior: 'smooth' });

// 自定义对齐方式
element.scrollIntoView({
    behavior: 'smooth',
    block: 'center',    // 垂直居中: start/center/end/nearest
    inline: 'nearest'  // 水平对齐: start/center/end/nearest
});
```

## 三、滚动事件节流

`scroll` 事件会高频触发，需要节流优化：

### 1. 使用 setTimeout 节流

```javascript
let scrollTimer = null;

window.addEventListener('scroll', function() {
    if (scrollTimer) clearTimeout(scrollTimer);

    scrollTimer = setTimeout(function() {
        // 实际处理逻辑
        handleScroll();
    }, 100); // 100ms 后执行
});
```

### 2. 使用 requestAnimationFrame

```javascript
let ticking = false;

window.addEventListener('scroll', function() {
    if (!ticking) {
        window.requestAnimationFrame(function() {
            handleScroll();
            ticking = false;
        });
        ticking = true;
    }
});

function handleScroll() {
    console.log('滚动位置:', window.scrollY);
}
```

### 3. 防抖 vs 节流

```javascript
// 防抖：滚动停止后才执行
function debounce(func, wait) {
    let timeout;
    return function() {
        clearTimeout(timeout);
        timeout = setTimeout(func, wait);
    };
}

// 节流：固定间隔执行
function throttle(func, wait) {
    let previous = 0;
    return function() {
        const now = Date.now();
        if (now - previous > wait) {
            func();
            previous = now;
        }
    };
}

window.addEventListener('scroll', debounce(handleScroll, 200));
window.addEventListener('scroll', throttle(handleScroll, 100));
```

## 四、滚动相关 CSS 属性

### 1. overflow 属性

```css
/* 隐藏滚动条但保留滚动功能 */
.scroll-container {
    overflow: auto;
    scrollbar-width: none; /* Firefox */
}

.scroll-container::-webkit-scrollbar {
    display: none; /* Chrome/Safari */
}

/* 强制显示滚动条 */
.scroll-container {
    overflow: scroll;
}
```

### 2. scroll-behavior 平滑滚动

```css
/* 全局启用平滑滚动 */
html {
    scroll-behavior: smooth;
}

/* 禁用平滑滚动 */
.no-smooth {
    scroll-behavior: auto;
}
```

### 3. scroll-snap 滚动吸附

```css
.container {
    display: flex;
    overflow-x: auto;
    scroll-snap-type: x mandatory;
}

.item {
    scroll-snap-align: start;
    flex-shrink: 0;
}
```

## 五、实用滚动案例

### 1. 返回顶部按钮

```javascript
const backToTop = document.getElementById('backToTop');

// 监听滚动显示/隐藏按钮
window.addEventListener('scroll', throttle(function() {
    if (window.scrollY > 300) {
        backToTop.style.display = 'block';
    } else {
        backToTop.style.display = 'none';
    }
}, 100));

// 点击返回顶部
backToTop.addEventListener('click', function() {
    window.scrollTo({
        top: 0,
        behavior: 'smooth'
    });
});
```

### 2. 滚动进度条

```javascript
const progressBar = document.getElementById('progressBar');

window.addEventListener('scroll', throttle(function() {
    const scrollTop = window.scrollY;
    const docHeight = document.documentElement.scrollHeight - window.innerHeight;
    const progress = (scrollTop / docHeight) * 100;

    progressBar.style.width = progress + '%';
}, 50));
```

### 3. 懒加载图片

```javascript
const images = document.querySelectorAll('img[data-src]');

const observer = new IntersectionObserver((entries) => {
    entries.forEach(entry => {
        if (entry.isIntersecting) {
            const img = entry.target;
            img.src = img.dataset.src;
            img.removeAttribute('data-src');
            observer.unobserve(img);
        }
    });
}, {
    rootMargin: '50px'
});

images.forEach(img => observer.observe(img));
```

### 4. 导航栏滚动时固定

```javascript
const navbar = document.querySelector('.navbar');
const offset = navbar.offsetTop;

window.addEventListener('scroll', throttle(function() {
    if (window.scrollY > offset) {
        navbar.classList.add('fixed');
    } else {
        navbar.classList.remove('fixed');
    }
}, 100));
```

### 5. 滚动时执行动画

```javascript
const animatedElements = document.querySelectorAll('.animate-on-scroll');

const observer = new IntersectionObserver((entries) => {
    entries.forEach(entry => {
        if (entry.isIntersecting) {
            entry.target.classList.add('visible');
        }
    });
}, {
    threshold: 0.1
});

animatedElements.forEach(el => observer.observe(el));
```

## 七、注意事项

1. **滚动事件节流**：scroll 事件触发频率很高，记得使用节流（throttle）或 requestAnimationFrame 优化性能。

2. **加载事件顺序**：`DOMContentLoaded` → `load` → `beforeunload` / `unload`

3. **兼容性**：现代浏览器推荐使用 `window.scrollY`，旧版浏览器用 `window.pageYOffset`

4. **移动端滚动**：在移动端可以使用 `touchmove` 事件配合节流使用

5. **懒加载优先使用 IntersectionObserver**：比监听 scroll 事件更高效
