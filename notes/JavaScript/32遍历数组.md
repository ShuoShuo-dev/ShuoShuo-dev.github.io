## 为什么需要遍历？

数组有多个元素，需要**依次访问每个元素**

```
数组：['小红', '小明', '小李', '小张']
索引：    0       1       2       3
         ↓       ↓       ↓       ↓
       第1个人  第2个人  第3个人  第4个人
```

## 遍历数组

```javascript
let arr = ['小红', '小明', '小李', '小张'];

// 从0开始，到 length-1 结束
for (let i = 0; i < arr.length; i++) {
    console.log(arr[i]);
}
```

**输出**：
```
小红
小明
小李
小张
```

## 遍历图解

```
arr = ['A', 'B', 'C', 'D']
索引：  0    1    2    3
长度：4

i=0 → arr[0] → 'A' → i++
i=1 → arr[1] → 'B' → i++
i=2 → arr[2] → 'C' → i++
i=3 → arr[3] → 'D' → i++
i=4 → 退出循环（4 < 4 为 false）
```

## 遍历求和

```javascript
let scores = [85, 90, 78, 92, 88];
let sum = 0;

for (let i = 0; i < scores.length; i++) {
    sum += scores[i];
}

let avg = sum / scores.length;
console.log(`总和：${sum}，平均分：${avg}`);
```

## 遍历改值

```javascript
let arr = [1, 2, 3, 4, 5];

for (let i = 0; i < arr.length; i++) {
    arr[i] = arr[i] * 2;  // 每个元素翻倍
}

console.log(arr);  // [2, 4, 6, 8, 10]
```

## 注意事项

1. **索引从0开始**：不是从1
2. **条件用 `<`**：不用 `<=`
3. **`arr.length`**：自动获取数组长度，数组变化时代码不用改

## 记忆口诀

> **遍历数组**：i 从0开始，i 小于数组长度，arr[i] 取元素
