# 输入输出语句和字面量

## 一、输出语句

### 1. document.write()

向网页（body）输出内容：

```javascript
document.write('Hello World');
document.write('<h1>标题</h1>');  // 可以输出HTML标签
```

### 2. alert()

弹出警告框：

```javascript
alert('你好');
```

### 3. console.log()

向**控制台**输出（按F12查看）：

```javascript
console.log('调试信息');
```

| 语句 | 输出位置 | 使用场景 |
|------|---------|---------|
| document.write() | 网页 | 页面渲染内容 |
| alert() | 弹窗 | 重要提示信息 |
| console.log() | 控制台 | 程序员调试 |

---

## 二、输入语句

### prompt()

弹出输入框，用户可以输入内容：

```javascript
prompt('请输入你的名字');
```

```javascript
// 带变量的完整示例
let name = prompt('请输入你的名字');
document.write('欢迎 ' + name);
```

---

## 三、执行顺序

### 正常顺序

代码从上往下执行：

```html
<script>
  document.write('第一');  // 先执行
  document.write('第二');  // 后执行
</script>
```

### 特殊情况

`prompt` 和 `alert` 会**优先弹窗**，跳过页面渲染：

```javascript
alert('先弹出这个');      // 代码暂停，等待用户点击
document.write('后渲染'); // 弹窗关闭后才执行
```

## 四、字面量

字面量是代码中**直接写出来的值**。

### 常见类型

| 类型 | 示例 |
|------|------|
| 数字字面量 | `100`、`3.14`、`-20` |
| 字符串字面量 | `'你好'`、`"Hello"` |
| 数组字面量 | `[1, 2, 3]` |
| 对象字面量 | `{name: '张三', age: 18}` |
| 布尔字面量 | `true`、`false` |

### 示例

```javascript
// 数字字面量
let score = 100;

// 字符串字面量
let name = '张三';

// 数组字面量
let arr = [1, 2, 3];

// 对象字面量
let person = { name: '李四', age: 20 };
```
