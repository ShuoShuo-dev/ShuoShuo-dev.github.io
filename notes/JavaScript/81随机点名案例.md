# 随机点名案例

## 需求

- 点击「开始」→ 名字快速闪动
- 点击「停止」→ 停在当前名字，并从数组中**删除**
- 数组只剩1人时，**禁用**按钮

## 完整代码

```javascript
// 1. 数据
const arr = ['张三', '李四', '王五', '赵六', '钱七', '孙八'];

// 2. 获取元素
const startBtn = document.querySelector('.start');
const endBtn = document.querySelector('.end');
const box = document.querySelector('.box');

// 3. 全局变量（定时器ID和随机数）
let timerId = null;
let random = 0;

// 4. 开始按钮
startBtn.addEventListener('click', function() {
  timerId = setInterval(function() {
    random = Math.floor(Math.random() * arr.length);
    box.innerHTML = arr[random];  // 快速闪动
  }, 30);  // 30ms 间隔，闪动很快

  // 禁用开始按钮
  startBtn.disabled = true;
  endBtn.disabled = false;
});

// 5. 结束按钮
endBtn.addEventListener('click', function() {
  clearInterval(timerId);  // 停止定时器

  arr.splice(random, 1);   // 删除被选中的名字

  // 判断是否只剩1人
  if (arr.length === 1) {
    startBtn.disabled = true;
    endBtn.disabled = true;
  }

  startBtn.disabled = false;
});
```

## 关键点

| 问题 | 解决方案 |
|------|----------|
| 定时器变量作用域 | 声明为**全局变量** `let timerId` |
| 随机数也要跨函数用 | 声明为**全局变量** `let random` |
| 防止重复抽取 | `splice(random, 1)` 从数组删除 |
| 只剩1人禁用按钮 | `arr.length === 1` 判断 |

## 为什么用全局变量？

```
startBtn 点击 → 开启定时器（timerId 在这里赋值）
endBtn 点击 → 关闭定时器（需要用到同一个 timerId）

如果 timerId 声明在 startBtn 的回调里，
endBtn 的回调就访问不到 → 必须用全局变量
```

## 流程图

```
点击开始 → setInterval(30ms) → 名字快速切换
                                    ↓
点击停止 → clearInterval → 名字停止 → splice删除 → 判断剩余人数
                                                      ↓
                                              剩1人 → 禁用按钮
```
