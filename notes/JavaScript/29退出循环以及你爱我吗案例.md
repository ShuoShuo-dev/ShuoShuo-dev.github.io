## break vs continue

| 关键字 | 作用 | 类比 |
|--------|------|------|
| break | 退出整个循环 | 吃包子发现虫子，直接不吃了 |
| continue | 跳过本次循环 | 吃包子发现虫子，跳过这个继续吃 |

## break 终止循环

```javascript
// 吃5个包子，吃到第3个发现虫子，后面不吃了
for (let i = 1; i <= 5; i++) {
    if (i === 3) {
        console.log('第3个有虫子，不吃了！');
        break;
    }
    console.log(`吃掉第${i}个包子`);
}
// 输出：吃掉第1个、吃掉第2个、第3个有虫子，不吃了！
```

## continue 跳过本次

```javascript
// 吃5个包子，跳过第3个
for (let i = 1; i <= 5; i++) {
    if (i === 3) {
        continue;  // 跳过这次，继续下次
    }
    console.log(`吃掉第${i}个包子`);
}
// 输出：1、2、4、5（跳过了3）
```

## 区别图解

```
break:    ██ ██ ██[BRK] ░░ ░░  →  立即结束

continue: ██ ██ ██[SKP] ██ ██  →  跳过本次，继续
```

## 案例：你爱我吗

```javascript
while (true) {
    let answer = prompt('你爱我吗？');
    if (answer === '爱') {
        alert('我也爱你！');
        break;  // 答对了，退出循环
    }
}
```

**注意**：
- `while(true)` 是死循环
- 必须有 break 退出，否则永远弹框

## 注意事项

1. **break**：用于结果已知，提前结束循环
2. **continue**：用于排除某一项，跳过继续
3. **小心死循环**：continue 后的代码不执行，注意变化量位置

## 记忆口诀

> **break**：break 直接终止，整个循环结束
> **continue**：continue 跳过本次，剩下的继续
