# 对象的方法

方法是存储在对象属性中的**函数**，用于描述对象的行为。

## 定义方法

### ES5 写法

```javascript
let person = {
  name: '张三',
  sayHello: function() {
    console.log('你好，我是' + this.name);
  }
};
```

### ES6 简写（推荐）

```javascript
let person = {
  name: '张三',
  // 方法名直接写，省略 function 关键字
  sayHello() {
    console.log('你好，我是' + this.name);
  }
};

person.sayHello(); // '你好，我是张三'
```

## this 指向

> 方法中的 `this` 指向**调用该方法的对象**

```javascript
let obj = {
  name: ' calculator',
  num1: 10,
  num2: 5,
  add() {
    return this.num1 + this.num2; // this → obj
  }
};
console.log(obj.add()); // 15
```

## 方法的动态调用

```javascript
let obj = {
  name: '测试',
  fn() { console.log('方法被调用'); }
};

// 通过变量名调用（场景：根据用户选择执行不同方法）
let methodName = 'fn';
obj[methodName](); // '方法被调用'
```

## 常用内置对象方法

| 方法 | 作用 | 返回值 |
|------|------|--------|
| `Object.keys(obj)` | 获取所有**属性名** | `['name', 'age']` |
| `Object.values(obj)` | 获取所有**属性值** | `['张三', 18]` |
| `Object.entries(obj)` | 获取**键值对数组** | `[['name','张三'], ['age',18]]` |

```javascript
let person = { name: '张三', age: 18 };
console.log(Object.keys(person));    // ['name', 'age']
console.log(Object.values(person));  // ['张三', 18]
```

## 小结

| 要点 | 说明 |
|------|------|
| 方法的本质 | 属性值是函数 |
| this 指向 | 谁调用，指向谁 |
| ES6 简写 | 省略 `function` 关键字 |
| 动态调用 | `obj[变量名]()` |
