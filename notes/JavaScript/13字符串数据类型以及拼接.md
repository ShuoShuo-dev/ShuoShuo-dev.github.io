# 字符串数据类型以及拼接

## 一、字符串定义

用引号包裹的内容就是字符串：

```javascript
let str1 = 'hello';    // 单引号
let str2 = "world";    // 双引号
let str3 = `你好`;     // 反引号
```

### 推荐使用

- HTML 属性用双引号 → JS 字符串推荐用**单引号**
- 学 JSON 时必须用双引号

---

## 二、引号注意事项

### 1. 数字加引号变字符串

```javascript
console.log(1);        // 数字型（蓝色）
console.log('1');      // 字符串型（黑色）
```

### 2. 变量不加引号

```javascript
let name = '张三';
console.log(name);     // ✅ 张三（输出变量值）
console.log('name');   // ❌ name（输出字符串）
```

### 3. 引号嵌套规则

```javascript
// 外单内双
let str1 = '他说"你好"';

// 外双内单
let str2 = "她说'hello'";

// 不能混搭
// let str3 = 'hello";   // ❌ 错误
```

### 4. 转义字符

```javascript
let str = '我\'m 程序员';   // 转义单引号
let path = "C:\\Users";     // 转义反斜杠
```

### 5. 空字符串

```javascript
let empty = '';  // 有引号但没内容
```

## 三、字符串拼接

### 核心口诀

```
数字相加，字符相连
```

### 基础示例

```javascript
console.log('hello' + 'world');     // helloworld
console.log('pink' + '老师');        // pink老师
console.log('年龄：' + 18);          // 年龄：18
```

### 拼接变量

```javascript
let name = '张三';
let age = 18;

console.log('我叫' + name + '，今年' + age + '岁');
// 输出：我叫张三，今年18岁
```

## 四、综合案例

### 用户信息展示

```javascript

document.write('姓名：' + uname + '<br>');
document.write('年龄：' + age + '<br>');

```

## 六、对比速查

| 写法 | 结果 |
|------|------|
| `1 + 1` | `2`（数字相加） |
| `'1' + '1'` | `'11'`（字符相连） |
| `'1' + 1` | `'11'`（转字符串） |
| `1 + '1'` | `'11'`（转字符串） |
