## 转换规则

| 值 | 转为 |
|----|------|
| 0 | false |
| '' | false |
| null | false |
| undefined | false |
| NaN | false |
| 其他 | true |

```javascript
Boolean(0)       // false
Boolean('')      // false
Boolean(null)    // false
Boolean(undefined) // false
Boolean(NaN)     // false

Boolean(1)       // true
Boolean('hello') // true
Boolean([])      // true
Boolean({})      // true
```

## 隐式转换

if 判断时自动转换：

```javascript
// 这6个为假，不进入if
if (0) {}      // 不执行
if ('') {}     // 不执行
if (null) {}   // 不执行
if (undefined) {} // 不执行
if (NaN) {}    // 不执行
if (false) {}  // 不执行

// 其他都执行
if (1) {}      // 执行
if ('hello') {} // 执行
```

## 快速转换

```javascript
// 方式1：Boolean()
let result = Boolean(0);  // false

// 方式2：!!
let result = !!0;  // false
```

## 实际应用

```javascript
// 判断是否有值
let name = '';
if (name) {
    console.log('有名字');
} else {
    console.log('名字为空');  // 执行
}

// 取反判断
let isLogin = false;
if (!isLogin) {
    console.log('未登录，请登录');  // 执行
}
```

## 数据类型转换总览

```
转数字：
  Number('123')  → 123
  parseInt('123') → 123
  parseFloat('1.5') → 1.5
  '6' - '2' → 4

转字符串：
  String(123)  → '123'
  123 + ''  → '123'

转布尔：
  Boolean(x) → true/false
  !!x        → true/false
```
