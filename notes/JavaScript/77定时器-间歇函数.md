# 定时器 — 间歇函数

## setInterval 开启

> 每隔一段时间，**重复执行**函数

```javascript
// 语法
setInterval(函数, 间隔时间);

// 匿名函数写法
setInterval(function() {
  console.log('每秒执行一次');
}, 1000);

// 有名函数写法（函数名不加括号！）
function fn() {
  console.log('每秒执行一次');
}
setInterval(fn, 1000);
```

> **注意**：间隔时间单位是**毫秒**（1000ms = 1秒）

### ⚠️ 函数名不加括号

```javascript
setInterval(fn, 1000);   // ✅ 传函数引用，由定时器调用
setInterval(fn(), 1000);  // ❌ 立即执行fn，把返回值传给定时器
```

## clearInterval 关闭

```javascript
// setInterval 返回一个 ID（数字），用于关闭
const timerId = setInterval(function() {
  console.log('执行中');
}, 1000);

// 关闭定时器
clearInterval(timerId);
```

## 完整示例：倒计时

```javascript
let count = 5;
const timerId = setInterval(function() {
  console.log(count);
  count--;
  if (count < 0) {
    clearInterval(timerId);  // 停止定时器
    console.log('结束！');
  }
}, 1000);
```

## 小结

| 操作 | 代码 |
|------|------|
| 开启 | `setInterval(fn, 毫秒)` |
| 关闭 | `clearInterval(定时器ID)` |
| 返回值 | 数字 ID，每个定时器唯一 |
| 存 ID 的变量 | 用 `let`（需要重新赋值） |

## 使用场景

- ⏱ 倒计时
- 🎠 轮播图自动播放
- 📊 实时刷新数据
- ✅ 验证码倒计时
