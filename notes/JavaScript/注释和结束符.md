# 注释和结束符

## 一、注释

注释是代码中**不执行的说明文字**，用于解释代码。

### 1. 单行注释

```javascript
// 这是单行注释
let name = '张三';  // 这也是单行注释
```

**快捷键**：`Ctrl + /`

### 2. 多行注释

```javascript
/*
  这是多行注释
  可以写很多行
*/
let age = 18;
```

**快捷键**：`Shift + Alt + A`

---

## 二、结束符

JavaScript 使用 **`;`** 作为语句结束符。

### 可省略

```javascript
// 都可以，不加分号也能运行
let a = 1;
let b = 2
```

### 规范建议

| 方式 | 说明 |
|------|------|
| 全加分号 | 风格统一 |
| 全不加分号 | 更简洁 |

**原则**：要么都写，要么都不写，保持一致。

---

## 三、注释使用技巧

### 好注释

```javascript
// 计算总价
let total = price * count;

// 处理用户登录
function login(username, password) { }

// TODO: 待优化
function slowFunction() { }
```

### 避免过度注释

```javascript
// ❌ 过度注释，代码本身已经很清楚
let x = 10;  // 定义变量x为10

// ✅ 简洁注释，只解释为什么
let x = 10;  // 兼容旧浏览器
```
