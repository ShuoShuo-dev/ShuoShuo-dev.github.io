## 需求

把秒数转换为 时:分:秒 格式：

```
65秒 → 0小时0分5秒
3665秒 → 1小时1分5秒
```

## 公式

```
小时 = 总秒数 ÷ 3600
分钟 = (总秒数 % 3600) ÷ 60  
秒数 = 总秒数 % 60
```

## 代码实现

```javascript
function getTime(second = 0) {
    // 计算时分秒
    let h = Math.floor(second / 3600);
    let m = Math.floor((second % 3600) / 60);
    let s = second % 60;
    
    // 补零
    h = h < 10 ? '0' + h : h;
    m = m < 10 ? '0' + m : m;
    s = s < 10 ? '0' + s : s;
    
    return `${h}:${m}:${s}`;
}

// 测试
console.log(getTime(65));      // 00:00:05
console.log(getTime(3665));    // 01:01:05
console.log(getTime(0));       // 00:00:00
```

## 用户输入版

```javascript
function getTime(second) {
    let h = Math.floor(second / 3600);
    let m = Math.floor((second % 3600) / 60);
    let s = second % 60;
    
    return `${h < 10 ? '0' + h : h}:${m < 10 ? '0' + m : m}:${s < 10 ? '0' + s : s}`;
}

let input = +prompt('请输入总秒数：');
console.log(getTime(input));
```

## 公式图解

```
总秒数 = 3665

3665 ÷ 3600 = 1 余 65
       ↓
     小时 = 1

65 ÷ 60 = 1 余 5
       ↓
    分钟 = 1

65 % 60 = 5
       ↓
    秒数 = 5

结果：01:01:05
```

## 注意事项

1. **Math.floor**：向下取整
2. **补零函数**：可单独封装
3. **默认参数**：防 NaN

## 记忆口诀

> **时分秒**：除3600得小时，余下除60得分钟，再余60就是秒，补零对齐更美观
