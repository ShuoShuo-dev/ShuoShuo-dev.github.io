# 变量更新和输入名字案例

## 一、变量更新

变量可以被重新赋值：

```javascript
let age = 18;
age = 19;  // 旧值被新值覆盖
console.log(age);  // 19
```

---

## 二、let vs var

| 特性 | let | var |
|------|-----|-----|
| 重复声明 | ❌ 报错 | ✅ 允许（不推荐） |
| 作用域 | 块级作用域 | 函数作用域 |

```javascript
// let - 不能重复声明
let age = 18;
let age = 20;  // ❌ 报错

// var - 可以重复声明（不推荐）
var name = '张三';
var name = '李四';  // ⚠️ 不报错，但不推荐
```

**结论**：统一使用 `let`

---

## 三、声明多个变量

### 不推荐写法

```javascript
let age = 18, name = '张三', score = 100;
```

### 推荐写法

```javascript
let age = 18;
let name = '张三';
let score = 100;
```

**原因**：分开写可读性更好

## 四、输入名字案例

### 效果
1. 弹出输入框，让用户输入姓名
2. 将姓名显示在页面上

### 代码

```javascript
// 1. 输入：使用 prompt 弹出输入框
let uname = prompt('请输入你的姓名');

// 2. 输出：使用 document.write 显示到页面
document.write('你的姓名是：' + uname);
```

### 步骤拆解

```
用户交互流程
├── 1. prompt() → 弹出输入框
├── 2. 用户输入 → 保存到变量
└── 3. document.write() → 显示结果
```

### 完整示例

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <title>输入名字</title>
</head>
<body>
  <script>
    let uname = prompt('请输入你的姓名');
    document.write('欢迎 ' + uname + '！');
  </script>
</body>
</html>
```

---

## 五、记忆口诀

- **变量值会变，用等号重新赋值**
- **let 不重复声明，var 可以但不推荐**
- **多个变量分开写，代码清晰顶呱呱**
- **输入用 prompt，输出用 document.write**
