## 查：访问属性

```javascript
let student = {
    name: '小红',
    age: 18,
    score: 90
};

// 点语法
console.log(student.name);  // '小红'

// 中括号
console.log(student['name']);  // '小红'
```

## 改：修改属性

```javascript
let student = {
    name: '小红',
    age: 18
};

// 修改已有属性
student.age = 20;
console.log(student.age);  // 20

// 不能加 let，已经声明了
```

## 增：添加属性

```javascript
let student = {
    name: '小红'
};

// 属性不存在就是新增
student.gender = '女';
student.score = 90;

console.log(student);
// { name: '小红', gender: '女', score: 90 }
```

## 删：删除属性

```javascript
let student = {
    name: '小红',
    age: 18
};

// 使用 delete
delete student.age;
console.log(student);  // { name: '小红' }
```

## 一张图总结

```
              ┌─ 存在 ─→ 改（覆盖）
              │
对象.属性 = 值 ─┤
              │
              └─ 不存在 → 增（添加新属性）
```

## 完整示例

```javascript
let product = {
    name: '手机',
    price: 2999
};

// 改
product.price = 1999;

// 增
product.color = '黑色';

// 查
console.log(product.name, product.price, product.color);
// 手机 1999 黑色

// 删
delete product.price;
```

## 注意事项

1. **改和增语法相同**：看属性存不存在
2. **delete 少用**：了解即可
3. **不用 let**：直接赋值

## 记忆口诀

> **增删改**：点语法来赋值，存在就是改，不存在就是增，delete删属性
