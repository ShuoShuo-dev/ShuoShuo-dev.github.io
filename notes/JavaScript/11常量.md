# 常量

## 一、什么是常量

常量是**值不能改变**的数据，使用 `const` 关键字声明。

### 使用场景

当某个值确定后**不会再改变**，使用常量：

```javascript
const PI = 3.14159;      // 圆周率
const SITE_NAME = '博客'; // 网站名称
const TAX_RATE = 0.13;   // 税率
```

---

## 二、常量的声明

### 基本语法

```javascript
const 常量名 = 值;
```

### 示例

```javascript
const PI = 3.14;
const URL = 'https://example.com';
const MAX_COUNT = 100;
```

---

## 三、常量 vs 变量

| 特性 | 变量 (let) | 常量 (const) |
|------|-----------|-------------|
| 值是否可变 | ✅ 可以修改 | ❌ 不能修改 |
| 声明时必须赋值 | ❌ 可以不赋值 | ✅ 必须赋值 |
| 语法 | `let name;` | `const name = value;` |

### 对比示例

```javascript
// 变量 - 可以先声明后赋值
let name;
name = '张三';
name = '李四';  // ✅ 允许修改

// 常量 - 必须声明时赋值
const PI = 3.14;
PI = 3.15;  // ❌ 报错！常量不能重新赋值
```

---

## 四、注意事项

### 1. 必须初始化

```javascript
const PI;  // ❌ 报错：Missing initializer in const declaration
const PI = 3.14;  // ✅ 正确
```

### 2. 不能重新赋值

```javascript
const TAX = 0.1;
TAX = 0.15;  // ❌ 报错
```

### 3. 命名规范

与变量相同：

```javascript
const MAX_VALUE = 100;      // ✅ 全大写下划线分隔
const siteName = '博客';    // ✅ 小写下划线
const TAX_RATE = 0.13;     // ✅ 大写下划线
```

---

## 五、const优先原则

**推荐：优先使用 const，只有需要重新赋值时才用 let**

```javascript
// ✅ 推荐：值不需改变就用 const
const name = '张三';
const PI = 3.14;

// ❌ 不推荐：能用 const 就不要用 let
let count = 0;  // 如果后面不改变值，应该用 const
```

### 为什么？

1. **代码更安全**：防止意外修改
2. **意图更清晰**：让其他人知道这个值不会变
3. **性能更好**：JavaScript引擎可以做更多优化
