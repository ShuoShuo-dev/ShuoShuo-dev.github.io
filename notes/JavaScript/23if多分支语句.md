## 为什么需要多分支？

双分支只能处理**二选一**，但实际场景需要**多选一**：

```
成绩等级：优秀(≥90) / 良好(70-89) / 及格(60-69) / 不及格(<60)
月份季节：春 / 夏 / 秋 / 冬
```

## 语法结构

```javascript
if (条件1) {
    // 条件1满足时执行
} else if (条件2) {
    // 条件2满足时执行
} else if (条件3) {
    // 条件3满足时执行
} else {
    // 所有条件都不满足时执行
}
```

**特点**：多选一，只执行其中一个分支

## 流程图

```
     ┌─────────┐
     │ 条件1   │
     └───┬─────┘
        ↓
    ┌───┴───┐
    ↓       ↓
  true    false → 检查条件2 → ...
    ↓
  代码1
    ↓
   结束
```

## 案例：成绩评级

```javascript
let score = +prompt('请输入成绩：');

if (score >= 90) {
    alert('成绩优秀');
} else if (score >= 70) {
    alert('成绩良好');
} else if (score >= 60) {
    alert('成绩及格');
} else {
    alert('成绩不及格');
}
```

**注意**：条件已按从大到小排列，`score >= 70` 时必然 `< 90`

## 多分支 vs 双分支

| 类型 | 语法 | 选择情况 |
|------|------|----------|
| 双分支 | `if () {} else {}` | 二选一 |
| 多分支 | `if () {} else if () {} ... else {}` | 多选一 |

## 注意事项

1. **else if 中间有空格**：`else if`，不是 `elseif`
2. **else if 可以有多个**：根据需求添加
3. **最后的 else 可省略**：如果没有默认情况
4. **顺序很重要**：条件要从严到松或从宽到严

## 记忆口诀

> **多分支**：else if 层层把关，最后 else 默认担当
