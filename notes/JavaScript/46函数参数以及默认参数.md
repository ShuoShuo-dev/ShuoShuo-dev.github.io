## 为什么需要参数？

让函数更灵活：

```javascript
// ❌ 只能算1-100
function getSum() {
    let sum = 0;
    for (let i = 1; i <= 100; i++) {
        sum += i;
    }
}

// ✅ 想算多少就多少
function getSum(end) {
    let sum = 0;
    for (let i = 1; i <= end; i++) {
        sum += i;
    }
}
getSum(50);   // 1+2+...+50
getSum(100);  // 1+2+...+100
```

## 形参 vs 实参

| 类型 | 位置 | 说明 |
|------|------|------|
| 形参 | 函数声明时 | 形式上的参数，类似变量声明 |
| 实参 | 函数调用时 | 实际传递的值 |

```javascript
// 形参：start, end
function getSum(start, end) {
    let sum = 0;
    for (let i = start; i <= end; i++) {
        sum += i;
    }
    console.log(sum);
}

// 实参：1, 50
getSum(1, 50);   // 1275
getSum(100, 200); // 15150
```

## 默认参数

```javascript
function getSum(start = 0, end = 0) {
    let sum = 0;
    for (let i = start; i <= end; i++) {
        sum += i;
    }
    console.log(sum);
}

getSum();        // 0（不传参用默认值）
getSum(1, 50);   // 1275（传参用实参）
```

## 图解

```
调用 getSum(1, 50)

实参 1, 50 →→→→→→→→→→→→ 形参 start, end
                              ↓
                          start = 1
                          end = 50
```

## 注意事项

1. **形参是局部变量**：只在函数内有效
2. **实参多于形参**：多余实参忽略
3. **实参少于形参**：未传值为 undefined
4. **默认值语法**：`param = defaultValue`

## 记忆口诀

> **参数**：形参函数声明里，实参调用传递值，默认参数防NaN
