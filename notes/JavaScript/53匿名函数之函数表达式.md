## 匿名函数

没有名字的函数：

```javascript
// 普通函数
function sayHi() {
    console.log('hi');
}

// 匿名函数（不能直接用）
function() {
    console.log('hi');
}
```

## 函数表达式

把匿名函数赋值给变量：

```javascript
let fn = function() {
    console.log('hi');
};

// 通过变量调用
fn();  // 'hi'
```

## 对比具名函数

| 类型 | 声明 | 调用位置 |
|------|------|----------|
| 具名函数 | `function fn(){}` | 任意位置 |
| 函数表达式 | `let fn = function(){}` | 必须在声明后 |

```javascript
// 具名函数：可以在声明前调用
fn1();  // 可以
function fn1() {}

// 函数表达式：必须在声明后调用
fn2();  // 报错！
let fn2 = function() {}
```

## 带参数和返回值

```javascript
// 带参数
let add = function(a, b) {
    return a + b;
};
console.log(add(1, 2));  // 3

// 带返回值
let getMax = function(arr) {
    return Math.max(...arr);
};
console.log(getMax([1, 5, 3]));  // 5
```

## 使用场景

```javascript
// 事件处理
btn.onclick = function() {
    alert('点击了');
};

// 回调函数
arr.forEach(function(item) {
    console.log(item);
});
```

## 注意事项

1. **必须有 let**：变量声明
2. **调用在声明后**：遵循变量提升规则
3. **函数表达式是赋值**：不是声明

## 记忆口诀

> **函数表达式**：匿名函数赋值给变量，调用用变量名，必须声明后才能用
