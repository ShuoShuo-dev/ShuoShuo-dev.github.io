# 键盘事件 + 发表评论案例

## 键盘事件

| 事件 | 说明 | 触发时机 |
|------|------|----------|
| `keydown` | 键盘按下 | 按下瞬间（持续按会重复触发） |
| `keyup` | 键盘抬起 | 松开瞬间 |

> **推荐用 `keyup`**：`keydown` 在按住时会重复触发，`keyup` 只在松开时触发一次

## 事件对象 e 的属性

| 属性 | 说明 | 示例 |
|------|------|------|
| `e.key` | 按键的值 | `'Enter'` `'a'` `'ArrowUp'` |
| `e.code` | 物理键位码 | `'KeyA'` `'Enter'` |

```javascript
document.addEventListener('keyup', function(e) {
  console.log(e.key);   // 'Enter'  'a'  'Escape'
  console.log(e.code);  // 'Enter'  'KeyA'  'Escape'
});
```

## 常用 key 值

```javascript
'Enter'       // 回车

```

## 案例：发表评论

### 需求
在文本框输入内容，按**回车**或点击按钮发布评论。

### 完整代码

```javascript
// 1. 获取元素
const textarea = document.querySelector('textarea');
const btn = document.querySelector('.btn');
const ul = document.querySelector('ul');

// 2. 发布评论函数
function publish() {
  const content = textarea.value.trim();
  if (!content) return alert('评论不能为空！');

  // 创建 li，添加到 ul 最前面
  const li = document.createElement('li');
  li.innerHTML = content;
  ul.insertBefore(li, ul.firstChild);

  // 清空输入框
  textarea.value = '';
}

// 3. 点击按钮发布
btn.addEventListener('click', publish);

// 4. 回车发布
textarea.addEventListener('keyup', function(e) {
  if (e.key === 'Enter') {
    publish();
  }
});
```

## 小结

| 要点 | 说明 |
|------|------|
| 回车判断 | `e.key === 'Enter'` |
| 阻止默认回车换行 | `e.preventDefault()` |
| 获取输入 | `textarea.value` |
| 插入到最前面 | `ul.insertBefore(li, ul.firstChild)` |
