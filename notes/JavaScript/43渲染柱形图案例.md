## 需求分析

输入4个季度数据，生成柱形图显示：

```
输入：200, 450, 300, 150

效果：
      ██  450
  ██  ██  300
██  ██  ██  150
Q1  Q2  Q3  Q4
```

## CSS 布局

```css
.box {
    display: flex;
    align-items: flex-end;  /* 底部对齐 */
    justify-content: space-around;
    height: 300px;
    width: 400px;
    border: 1px solid #333;
}

.bar {
    width: 50px;
    height: 100px;  /* JS动态设置 */
    background-color: skyblue;
}
```

## JS 实现

```javascript
// 1. 输入数据
let data = [];
for (let i = 0; i < 4; i++) {
    data.push(+prompt(`第${i + 1}季度收入：`));
}

// 2. 渲染柱形图
let html = '<div class="box">';
for (let i = 0; i < data.length; i++) {
    html += `
        <div class="bar" style="height: ${data[i]}px;">
            <span>${data[i]}</span>
        </div>
    `;
}
html += '</div>';

document.write(html);
```

## 进阶：带标签

```javascript
let data = [
    { quarter: 'Q1', value: 200 },
    { quarter: 'Q2', value: 450 },
    { quarter: 'Q3', value: 300 },
    { quarter: 'Q4', value: 150 }
];

let html = '<div class="box">';
for (let i = 0; i < data.length; i++) {
    html += `
        <div class="bar" style="height: ${data[i].value}px;">
            <span>${data[i].value}</span>
            <div class="label">${data[i].quarter}</div>
        </div>
    `;
}
html += '</div>';

document.write(html);
```

## 关键点

| 要点 | 说明 |
|------|------|
| flex 布局 | `align-items: flex-end` 底部对齐 |
| 动态高度 | `style="height: ${data[i]}px"` |
| 循环渲染 | 用 for 循环拼接 HTML |
