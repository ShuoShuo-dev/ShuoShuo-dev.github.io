## 分析规律

```
1×1=1
1×2=2  2×2=4
1×3=3  2×3=6  3×3=9
...

规律：第i行，有i列，j×i=ji
```

## 核心代码

```javascript
for (let i = 1; i <= 9; i++) {        // 行
    for (let j = 1; j <= i; j++) {   // 列（最大为当前行数）
        document.write(`${j}×${i}=${j*i} `);
    }
    document.write('<br>');
}
```

## 输出结果

```
1×1=1 
1×2=2  2×2=4 
1×3=3  2×3=6  3×3=9 
1×4=4  2×4=8  3×4=12 4×4=16 
...
```

## 注意事项

1. **列数 ≤ 行数**：`j <= i`
2. **格式化输出**：空格分隔
3. **j 在前**：表达式是 `j×i`，不是 `i×j`

## 记忆口诀

> **乘法表**：外层控制行，内层列数等于行数，乘积等于j×i
