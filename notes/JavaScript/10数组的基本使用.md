# 数组的基本使用

## 一、为什么需要数组

### 普通变量的局限

```javascript
// 存3个学生姓名，需要3个变量
let name1 = '张三';
let name2 = '李四';
let name3 = '王五';
```

### 数组的优势

```javascript
// 一个数组搞定！
let names = ['张三', '李四', '王五'];
```

---

## 二、声明数组

### 方式一：字面量（推荐）

```javascript
let arr = [10, 20, 30];
let names = ['小明', '小刚', '小红'];
let mixed = [1, 'hello', true];
```

### 方式二：new Array()

```javascript
let arr = new Array(10, 20, 30);
let empty = new Array(3);  // 创建长度为3的空数组
```

---

## 三、访问数组元素

数组元素有**索引号**（从0开始）：

```javascript
let fruits = ['苹果', '香蕉', '橙子'];

console.log(fruits[0]);  // 苹果
console.log(fruits[1]);  // 香蕉
console.log(fruits[2]);  // 橙子
console.log(fruits[3]);  // undefined（不存在）
```

### 索引号示意图

```
数组: ['苹果', '香蕉', '橙子']
索引:    [0]      [1]      [2]
```

---

## 四、数组长度

使用 `.length` 获取长度：

```javascript
let fruits = ['苹果', '香蕉', '橙子'];

console.log(fruits.length);  // 3
```

### 索引与长度的关系

```
length = 最大索引 + 1
fruits.length = 2 + 1 = 3
```

---

## 五、遍历数组

```javascript
let arr = ['a', 'b', 'c'];

// for循环
for (let i = 0; i < arr.length; i++) {
  console.log(arr[i]);
}
```

## 六、相关术语

| 术语 | 说明 |
|------|------|
| 元素 | 数组中的每个数据 |
| 索引/下标 | 元素的编号，从0开始 |
| 长度 | 数组中元素的数量，用 `.length` 获取 |

---

## 七、注意事项

### 中文要加引号

```javascript
// ✅ 正确
let names = ['小明', '小刚'];

// ❌ 错误（老浏览器会报错）
let names = [小明, 小刚];
```
