# 表单全选反选案例

## 需求

- 全选框**控制**所有小复选框
- 所有小复选框**控制**全选框

## HTML 结构

```html
<input type="checkbox" id="checkAll"> 全选
<input type="checkbox" class="ck"> 选项1
<input type="checkbox" class="ck"> 选项2
<input type="checkbox" class="ck"> 选项3
```

## 完整代码

```javascript
// 1. 获取元素
const checkAll = document.querySelector('#checkAll');
const cks = document.querySelectorAll('.ck');

// ========== 模块一：全选控制小复选框 ==========
checkAll.addEventListener('click', function() {
  // 全选的 checked 状态 → 赋值给所有小复选框
  for (let i = 0; i < cks.length; i++) {
    cks[i].checked = this.checked;
  }
});

// ========== 模块二：小复选框控制全选 ==========
for (let i = 0; i < cks.length; i++) {
  cks[i].addEventListener('click', function() {
    // 统计被勾选的小复选框数量
    const checkedCount = document.querySelectorAll('.ck:checked').length;
    // 勾选数 === 总数 → 全选框勾选
    checkAll.checked = (checkedCount === cks.length);
  });
}
```

## 模块一：全选 → 小复选框

```
点击全选框
  → this.checked 为 true 或 false
  → 遍历所有小复选框
  → 每个小复选框.checked = 全选框.checked
```

```javascript
// this 指向 checkAll
cks[i].checked = this.checked;
```

## 模块二：小复选框 → 全选

```
任意小复选框点击
  → 统计 .ck:checked 的数量
  → 数量 === cks.length → checkAll.checked = true
  → 数量 !== cks.length → checkAll.checked = false
```

```javascript
// CSS 伪类选择器 :checked 可在 JS 中使用
document.querySelectorAll('.ck:checked').length;
```

## 关键点

| 要点 | 说明 |
|------|------|
| 全选控制小 | `cks[i].checked = this.checked` |
| 小控制全选 | 勾选数 === 总数 |
| 统计勾选数 | `.ck:checked` 伪类选择器 |
| `checked` 值 | `true`（勾选）/ `false`（未勾选） |
