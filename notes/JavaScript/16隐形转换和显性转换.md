# 隐式转换和显性转换

## 一、为什么需要类型转换

```javascript
let price = prompt('请输入价格');  // 字符串 "1000"
let count = prompt('请输入数量'); // 字符串 "2"

console.log(price + count);  // "10002" ❌ 字符串拼接
console.log(price - count);  // 998    ✅ 自动转数字
```

---

## 二、隐式转换（自动转换）

### 加号 `+`

```javascript
// 有字符串 → 拼接
console.log('hello' + 123);    // "hello123"
console.log(123 + 'hello');    // "123hello"

// 无字符串 → 数字运算
console.log(1 + 2);           // 3
console.log(+'123');          // 123（转数字）
```

### 其他运算符 `-` `*` `/`

```javascript
console.log('6' - '3');  // 3（转数字）
console.log('6' * '2');   // 12
console.log('6' / '2');   // 3
```

---

## 三、显性转换（手动转换）

### 1. Number() - 转数字

```javascript
console.log(Number('123'));     // 123
console.log(Number('12.34'));   // 12.34
console.log(Number('hello'));   // NaN
console.log(Number(''));        // 0
```

### 2. parseInt() - 转整数

```javascript
console.log(parseInt('12'));    // 12
console.log(parseInt('12.34')); // 12（只取整数）
console.log(parseInt('12px'));  // 12（去掉单位）
console.log(parseInt('px12'));  // NaN
```

### 3. parseFloat() - 转浮点数

```javascript
console.log(parseFloat('12'));    // 12
console.log(parseFloat('12.34')); // 12.34
console.log(parseFloat('12.34px')); // 12.34
```

---

## 四、转换方法对比

| 方法 | 转整数 | 去单位 | 转NaN |
|------|:------:|:------:|:-----:|
| Number() | ✅ | ❌ | ✅ |
| parseInt() | ✅ | ✅ | ✅ |
| parseFloat() | ❌ | ✅ | ✅ |

## 五、综合案例：计算两数之和

```javascript
let num1 = +prompt('请输入第一个数');
let num2 = +prompt('请输入第二个数');

document.write(`两数之和是：${num1 + num2}`);
```
