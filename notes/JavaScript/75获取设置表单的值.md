# 获取设置表单的值

## 获取输入内容

```javascript
const input = document.querySelector('input');
input.value;           // 获取用户输入的值
input.placeholder;     // 获取占位文字
```

> **注意**：表单用 `value`，不用 `innerHTML`！

```javascript
// ❌ innerHTML 获取不到表单值
console.log(input.innerHTML);  // 空

// ✅ 用 value
console.log(input.value);      // '用户输入的内容'
```

## 设置值和类型

```javascript
const input = document.querySelector('input');

// 设置值
input.value = '默认内容';

// 切换类型（密码显示/隐藏）
input.type = 'password';  // 密码框
input.type = 'text';      // 文本框
```

## 布尔属性

布尔属性：`true` = 添加该属性，`false` = 移除该属性

| 属性 | 作用 | 示例 |
|------|------|------|
| `disabled` | 禁用 | `btn.disabled = true` |
| `checked` | 勾选 | `checkbox.checked = true` |
| `selected` | 选中（下拉框） | `option.selected = true` |
| `readonly` | 只读 | `input.readonly = true` |

### 案例：全选复选框

```javascript
const allCheck = document.querySelector('.all');
const checks = document.querySelectorAll('.ck');

// 全选
allCheck.addEventListener('click', function() {
  for (let i = 0; i < checks.length; i++) {
    checks[i].checked = this.checked;  // this 指向全选框
  }
});
```

### 案例：禁用按钮

```javascript
const btn = document.querySelector('button');

// 禁用
btn.disabled = true;

// 启用
btn.disabled = false;
```

## 小结

| 操作 | 语法 |
|------|------|
| 获取值 | `input.value` |
| 设置值 | `input.value = '新值'` |
| 改类型 | `input.type = 'password'` |
| 禁用 | `btn.disabled = true` |
| 勾选 | `checkbox.checked = true` |
