## 语法结构

```javascript
if (条件) {
    // 条件为 true 时执行
} else {
    // 条件为 false 时执行
}
```

**特点**：二选一，必定执行其中之一

## 流程图

```
     ┌─────────┐
     │ 条件判断 │
     └────┬────┘
          │
    ┌─────┴─────┐
    ↓           ↓
  true        false
    ↓           ↓
  if分支      else分支
    ↓           ↓
  执行代码     执行代码
    └─────┬─────┘
          ↓
       结束
```

## 案例1：简单判断

```javascript
let age = +prompt('请输入年龄：');

if (age >= 18) {
    alert('已成年');
} else {
    alert('未成年');
}
```

## 案例2：登录验证

```javascript
let username = prompt('请输入用户名：');
let password = prompt('请输入密码：');

if (username === 'pink' && password === '123456') {
    alert('登录成功');
} else {
    alert('用户名或密码错误');
}
```

**注意**：字符串比较用 `===` 全等，确保值和类型都相等

## 案例3：闰年判断

```javascript
let year = +prompt('请输入年份：');

if ((year % 4 === 0 && year % 100 !== 0) || year % 400 === 0) {
    alert(`${year}年是闰年`);
} else {
    alert(`${year}年是平年`);
}
```

**闰年条件**：
- 能被 4 整除 且 不能被 100 整除
- 或者 能被 400 整除

## 单分支 vs 双分支

| 类型 | 语法 | 执行情况 |
|------|------|----------|
| 单分支 | `if () {}` | 条件满足才执行 |
| 双分支 | `if () {} else {}` | 二选一，必定执行一个 |

## 记忆口诀

> **双分支**：if 满足做这事，else 否则做那事
