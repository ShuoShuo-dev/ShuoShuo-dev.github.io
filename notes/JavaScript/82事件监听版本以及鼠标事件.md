# 鼠标事件

## 事件监听 vs 传统方式

```javascript
const btn = document.querySelector('button');

// ✅ addEventListener（推荐）：可绑定多个同名事件
btn.addEventListener('click', function() {
  console.log('事件1');
});
btn.addEventListener('click', function() {
  console.log('事件2');  // 两个都执行
});

// ❌ onclick（不推荐）：后写的会覆盖前面的
btn.onclick = function() { console.log('事件1'); };
btn.onclick = function() { console.log('事件2'); };  // 只执行这个
```

## 常用鼠标事件

| 事件 | 说明 | 触发时机 |
|------|------|----------|
| `click` | 点击 | 按下并松开 |
| `dblclick` | 双击 | 快速按两次 |
| `mouseenter` | 鼠标移入 | 进入元素区域 |
| `mouseleave` | 鼠标移出 | 离开元素区域 |
| `mousemove` | 鼠标移动 | 在元素区域内移动 |
| `mousedown` | 鼠标按下 | 按下瞬间 |
| `mouseup` | 鼠标松开 | 松开瞬间 |
| `contextmenu` | 右键菜单 | 右键点击 |

## ⚠️ mouseenter vs mouseover 区别

| 对比 | `mouseenter` | `mouseover` |
|------|-------------|-------------|
| 冒泡 | ❌ 不冒泡 | ✅ 冒泡 |
| 子元素触发 | ❌ 不会 | ✅ 会 |
| 推荐 | ✅ 更符合预期 | 需要冒泡时用 |

```javascript
const box = document.querySelector('.box');

box.addEventListener('mouseenter', function() {
  this.style.backgroundColor = 'hotpink';  // 移入变色
});

box.addEventListener('mouseleave', function() {
  this.style.backgroundColor = '';         // 移出恢复
});
```

## 事件对象 e

回调函数可以接收一个参数 `e`（事件对象），包含事件相关信息：

```javascript
btn.addEventListener('click', function(e) {
  console.log(e.target);    // 触发事件的元素
  console.log(e.type);      // 事件类型 'click'
  console.log(e.clientX);   // 鼠标X坐标（相对视口）
  console.log(e.clientY);   // 鼠标Y坐标（相对视口）
});
```
