# 轮播图定时版

## 需求

在随机版基础上，增加**定时自动播放**，每隔一段时间自动切换下一张。

## 完整代码

```javascript
// 1. 数据
const sliderData = [
  { url: './images/slider01.jpg', title: '新年特惠', color: 'rgb(100, 167, 103)' },
  { url: './images/slider02.jpg', title: '春季上新', color: 'rgb(38, 168, 157)' },
  { url: './images/slider03.jpg', title: '周年庆典', color: 'rgb(215, 117, 73)' }
];

// 2. 获取元素
const img = document.querySelector('.slider-wrapper img');
const title = document.querySelector('.slider-footer p');
const footer = document.querySelector('.slider-footer');

// 3. 声明计数器
let i = 0;

// 4. 开启定时器
const timerId = setInterval(function() {
  i++;

  // 5. 无缝滚动（到末尾重置为0）
  if (i >= sliderData.length) {
    i = 0;
  }

  // 6. 更新数据
  const data = sliderData[i];
  img.src = data.url;
  title.innerHTML = data.title;
  footer.style.backgroundColor = data.color;

  // 7. 小圆点排他
  // 先移除所有 active
  document.querySelector('.slider-indicator .active')
    .classList.remove('active');
  // 再给当前添加 active
  document.querySelector(
    `.slider-indicator li:nth-child(${i + 1})`
  ).classList.add('active');

}, 1000);
```

## 无缝滚动原理

```
i = 0 → 显示第1张
i = 1 → 显示第2张
i = 2 → 显示第3张
i = 3 → i >= 3, 重置 i = 0 → 显示第1张（回到开头）
```

## 排他思想

```
每次切换：
1. 找到当前有 active 的圆点 → 移除 active
2. 给目标圆点 → 添加 active
```

## vs 随机版对比

| 对比 | 随机版 | 定时版 |
|------|--------|--------|
| 触发时机 | 页面刷新时 | 每隔1秒 |
| 索引获取 | `Math.random()` | 变量 `i++` |
| 边界处理 | 无需 | 到末尾重置为0 |
| 循环方式 | 不循环 | **无缝循环** |
