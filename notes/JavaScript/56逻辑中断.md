## 什么是逻辑中断？

满足条件时，右边不执行：

```
短路求值：左边已经能确定结果，右边就不执行了
```

## 逻辑与 `&&`

| 左边 | 结果 | 右边 |
|------|------|------|
| 假 | 左边 | 不执行 |
| 真 | 右边 | 执行 |

```javascript
// 左边为0（假），右边不执行
let result = 0 && console.log('不执行');
console.log(result);  // 0

// 左边为1（真），执行右边
let result = 1 && console.log('执行');
// 输出：执行，result = undefined
```

## 逻辑或 `||`

| 左边 | 结果 | 右边 |
|------|------|------|
| 真 | 左边 | 不执行 |
| 假 | 右边 | 执行 |

```javascript
// 左边为1（真），右边不执行
let result = 1 || console.log('不执行');
console.log(result);  // 1

// 左边为0（假），执行右边
let result = 0 || 100;
console.log(result);  // 100
```

## 实际应用

### 1. 默认值

```javascript
function add(a, b) {
    a = a || 0;  // 如果a是假的，用0
    b = b || 0;
    return a + b;
}
console.log(add());        // 0
console.log(add(1, 2));   // 3
```

### 2. 简化判断

```javascript
let name = '';
name = name || '默认名字';  // 用默认值
console.log(name);  // '默认名字'
```

## 对比总结

| 运算符 | 中断条件 | 返回值 |
|--------|----------|--------|
| `&&` | 遇到假 | 那个假值 |
| `\|\|` | 遇到真 | 那个真值 |

## 注意事项

1. **逻辑与**：一假则假，遇到假就停
2. **逻辑或**：一真则真，遇到真就停
3. **非运算没有中断**

## 记忆口诀

> **逻辑中断**：&& 遇到假就停，|| 遇到真就停，左边能定结果，右边不用执行
