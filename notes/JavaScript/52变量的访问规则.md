## 就近原则

变量相同时，先找近的：

```javascript
let num = 10;  // 全局

function fn() {
    let num = 20;  // 局部
    console.log(num);  // 20
}
fn();
```

## 查找规则

```
要访问变量 → 先看当前作用域有没有
                   ↓
              有 → 使用当前作用域的
              ↓没有
          往外层找
              ↓
          找到 → 使用
          找不到
              ↓
          继续往外层找
              ↓
          直到全局都没有 → 报错
```

## 图解

```
┌─────────────────────────────┐
│ 全局作用域                    │
│ let a = 1                   │
│ ┌─────────────────────────┐ │
│ │ 函数fn1作用域            │ │
│ │ let a = 10              │ │
│ │ ┌─────────────────────┐ │ │
│ │ │ 函数fn2作用域        │ │ │
│ │ │ console.log(a) → 100│ │ │
│ │ │ 找到本层的 a=100     │ │ │
│ │ └─────────────────────┘ │ │
│ └─────────────────────────┘ │
└─────────────────────────────┘
```

## 特殊情况

### 1. 函数内不声明直接赋值

```javascript
function fn() {
    num = 100;  // 没let，修改全局变量（强烈不推荐）
}
fn();
console.log(num);  // 100
```

### 2. 函数形参是局部变量

```javascript
function fn(a) {  // 形参a是局部变量
    console.log(a);
}
fn(10);  // 10
console.log(a);  // 报错
```

## 注意事项

1. **变量访问从近到远**
2. **不会反向查找**：子函数找不到父函数的变量
3. **不要省略 let**：不声明直接赋值很危险

## 记忆口诀

> **访问规则**：变量访问就近原则，函数内部先找函数内，找不到往外层找
