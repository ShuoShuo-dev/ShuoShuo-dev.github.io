# Math 数学内置对象

`Math` 是 JS 内置对象，**不需要 new**，直接使用。

## 常用属性

```javascript
Math.PI  // 3.141592653589793（圆周率）
```

## 取整方法（重点）

| 方法 | 含义 | 示例 | 结果 |
|------|------|------|------|
| `Math.ceil()` | 向上取整（天花板） | `Math.ceil(1.1)` | **2** |
| `Math.floor()` | 向下取整（地板） | `Math.floor(1.9)` | **1** |
| `Math.round()` | 四舍五入 | `Math.round(1.5)` | **2** |
| `Math.round()` | 四舍五入（注意负数） | `Math.round(-1.5)` | **-1** |

> **`Math.floor` vs `parseInt`**：`Math.floor` 只能处理数字，`parseInt` 可以处理字符串 `'12.5px'` → `12`

## 其他常用方法

| 方法 | 说明 | 示例 |
|------|------|------|
| `Math.max()` | 最大值 | `Math.max(1, 2, 3)` → `3` |
| `Math.min()` | 最小值 | `Math.min(1, 2, 3)` → `1` |
| `Math.abs()` | 绝对值 | `Math.abs(-5)` → `5` |
| `Math.pow()` | 幂运算 | `Math.pow(2, 3)` → `8` |
| `Math.sqrt()` | 平方根 | `Math.sqrt(9)` → `3` |
| `Math.random()` | 随机数 | `0 ≤ x < 1` 的随机小数 |

## 🔥 随机数公式（必背）

```javascript
// 生成 [min, max] 之间的随机整数
function getRandom(min, max) {
  return Math.floor(Math.random() * (max - min + 1)) + min;
}

getRandom(1, 10)  // 1~10 的随机整数
getRandom(1, 6)   // 模拟掷骰子
getRandom(0, 255) // 随机RGB颜色
```

**公式拆解：**

```
Math.random()           →  0 ≤ x < 1
* (max - min + 1)       →  0 ≤ x < max-min+1
Math.floor(...)         →  0 ≤ x ≤ max-min  （取整）
+ min                   →  min ≤ x ≤ max    （平移）
```

## 小结

| 场景 | 用什么 |
|------|--------|
| 圆相关计算 | `Math.PI` |
| 随机抽取 | `Math.floor(Math.random() * N)` |
| 求 N 个数最值 | `Math.max(...arr)` / `Math.min(...arr)` |
