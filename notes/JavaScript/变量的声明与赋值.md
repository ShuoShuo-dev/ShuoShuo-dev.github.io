# 变量的声明与赋值

## 一、什么是变量

变量是计算机**存储数据的容器**，相当于一个盒子。

```javascript
let name = '张三';  // 声明一个叫 name 的盒子，里面放 '张三'
```

---

## 二、声明变量

使用 `let` 关键字：

```javascript
let age;        // 声明变量
let name;       // 声明变量
let score, sum; // 同时声明多个变量（不推荐）
```

---

## 三、赋值

使用 `=` 进行赋值：

```javascript
let age;
age = 18;        // 把 18 存入 age 盒子
```

---

## 四、初始化

声明的同时赋值（推荐）：

```javascript
let age = 18;        // 数字
let name = '张三';    // 字符串
let isVip = true;    // 布尔
```

---

## 五、修改变量值

变量可以多次赋值，后面的值会覆盖前面的：

```javascript
let age = 18;
age = 20;      // 现在 age 是 20
age = 25;      // 现在 age 是 25
```

---

## 六、注意事项

### 1. 等号两边加空格

```javascript
// ✅ 推荐
let age = 18;

// ❌ 不推荐
let age=18;
```

### 2. 字符串加引号

```javascript
let name = '张三';  // ✅ 字符串要加引号
let name = 张三;    // ❌ 会报错
```

### 3. 输出时不加引号

```javascript
let name = '张三';
console.log(name);      // ✅ 输出变量值：张三
console.log('name');    // ❌ 输出字符串：name
```

### 4. 用 let，不用 var

```javascript
// ✅ 推荐
let age = 18;

// ❌ 老旧写法
var age = 18;
```

---

## 七、记忆口诀

```
变量变量，存储数据
let 声明，等号赋值
声明同时赋值，叫初始化
变量值可变，常量值不变
```
