## 什么是作用域？

变量能使用的范围：

```
全局作用域 → 整个页面都能用
局部作用域 → 只有函数内能用
```

## 全局变量

```javascript
let num = 10;  // 全局变量

function fn() {
    console.log(num);  // 10，能用
}
fn();
console.log(num);      // 10，能用
```

## 局部变量

```javascript
function fn() {
    let num = 10;  // 局部变量
    console.log(num);  // 10
}
fn();
console.log(num);      // 报错！访问不到
```

## 对比

| 类型 | 声明位置 | 作用范围 | 生命周期 |
|------|----------|----------|----------|
| 全局变量 | 函数外部 | 全局可用 | 页面关闭消失 |
| 局部变量 | 函数内部 | 函数内可用 | 函数结束消失 |

## 作用域链

函数可以嵌套，形成作用域链：

```javascript
let a = 1;

function fn1() {
    let a = 10;
    
    function fn2() {
        let a = 100;
        console.log(a);  // 100（就近原则）
    }
    fn2();
}
fn1();
```

## 注意事项

1. **函数外的变量**：全局变量
2. **函数内的变量**：局部变量
3. **函数形参**：也是局部变量
4. **强烈不推荐**：函数内不声明直接赋值（变成全局变量）

## 记忆口诀

> **作用域**：全局变量到处跑，局部变量函数里找，就近原则用变量
