# 数据类型和算术运算符

## 一、数据类型分类

| 类型 | 说明 | 示例 |
|------|------|------|
| number | 数字型 | `18`, `3.14`, `-10` |
| string | 字符串型 | `'hello'`, `"world"` |
| boolean | 布尔型 | `true`, `false` |
| undefined | 未定义 | `let a;` |
| null | 空值 | `let b = null;` |
| object | 对象 | `{name: '张三'}` |

### 两大类

```
数据类型
├── 基本类型：number, string, boolean, undefined, null
└── 引用类型：object（对象）
```

---

## 二、数字型 (number)

### 特点

JS中所有数字都是 number 类型：

```javascript
let age = 18;        // 整数
let price = 3.14;    // 小数
let negative = -5;   // 负数
```

### JS是弱数据类型

```javascript
// JS：赋值后才确定类型
let a = 18;      // number
let b = 'hello'; // string

// Java（强类型）：声明时必须指定类型
// int age = 18;
// String name = "hello";
```

## 三、算术运算符

| 运算符 | 说明 | 示例 |
|:------:|------|------|
| `+` | 加 | `10 + 5` → `15` |
| `-` | 减 | `10 - 5` → `5` |
| `*` | 乘 | `10 * 5` → `50` |
| `/` | 除 | `10 / 5` → `2` |
| `%` | 取模(余) | `10 % 3` → `1` |

### 优先级

```
最高：() 括号
其次：* / %
最低：+ -
```

```javascript
console.log(10 + 5 * 2);      // 20（先乘后加）
console.log((10 + 5) * 2);   // 30（先算括号）
```

---

## 四、取模 `%` 的应用

取模 = 取余数

```javascript
console.log(10 % 3);   // 1（10除3余1）
console.log(4 % 2);    // 0（4除2余0）
```

### 判断技巧

| 判断 | 条件 | 用途 |
|------|------|------|
| 能否被整除 | `n % 2 === 0` | 偶数 |
| 是否奇数 | `n % 2 !== 0` | 奇数 |
| 是否闰年 | `year % 4 === 0` | 闰年 |

## 五、特殊值：NaN

### 含义

`NaN` = Not a Number（非数字）

### 出现情况

```javascript
console.log('hello' - 5);  // NaN
console.log('abc' * 2);    // NaN
console.log(0 / 0);       // NaN
```

### 特性

```javascript
console.log(NaN + 5);      // NaN
console.log(NaN - 10);     // NaN
console.log(NaN === NaN);   // false（NaN不等于自己）
```

### 判断 NaN

```javascript
// 使用 isNaN()
console.log(isNaN('hello'));  // true
console.log(isNaN(123));       // false
```

## 六、综合示例：计算圆面积

```javascript
// 1. 输入半径
let r = prompt('请输入圆的半径');

// 2. 计算面积

// 3. 输出结果
document.write('圆的面积是：' + area);
```
