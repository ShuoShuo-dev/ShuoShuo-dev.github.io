# 事件监听

## 什么是事件

> 事件是用户或浏览器执行的**动作**，如点击、鼠标移入、按键等。

```javascript
// 事件三要素
// 1. 事件源：谁被触发 → 按钮
// 2. 事件类型：什么方式 → click（点击）
// 3. 事件处理函数：做什么 → 弹出对话框

const btn = document.querySelector('button');
btn.addEventListener('click', function() {
  alert('按钮被点击了！');
});
```

## 语法

```javascript
事件源.addEventListener('事件类型', 事件处理函数);
```

## 常见事件类型

| 类别 | 事件类型 | 说明 |
|------|----------|------|
| 鼠标事件 | `click` | 点击 |
| | `dblclick` | 双击 |
| | `mouseenter` | 鼠标移入 |
| | `mouseleave` | 鼠标移出 |
| 焦点事件 | `focus` | 获得焦点 |
| | `blur` | 失去焦点 |
| 键盘事件 | `keydown` | 键盘按下 |
| | `keyup` | 键盘抬起 |
| 表单事件 | `input` | 输入内容 |
| | `change` | 值改变 |

## ⚠️ 注意事项

```javascript
// 1. 事件类型是字符串，必须加引号
btn.addEventListener('click', fn);    // ✅
btn.addEventListener(click, fn);      // ❌ click 未定义

// 2. 函数不加括号！由浏览器在事件触发时调用
btn.addEventListener('click', fn);    // ✅ 传函数引用
btn.addEventListener('click', fn());  // ❌ 立即执行

// 3. 函数不会立即执行，等事件触发才执行
```

## 案例：关闭广告

```javascript
// 点击 × 关闭广告
document.querySelector('.close-btn').addEventListener('click', function() {
  document.querySelector('.ad').style.display = 'none';
});
```

## 案例：点击按钮显示/隐藏

```javascript
const btn = document.querySelector('button');
const box = document.querySelector('.box');

btn.addEventListener('click', function() {
  box.classList.toggle('hidden');  // toggle 切换
});
```

## 事件执行特点

> 事件触发一次 → 函数执行一次（类似定时器里的回调函数）
