# 交换两个变量案例

## 一、案例需求

有两个变量：
- `num1 = 10`
- `num2 = 20`

交换后：
- `num1 = 20`
- `num2 = 10`

---

## 二、为什么需要临时变量

直接交换不行：

```javascript
num1 = num2;  // num1 = 20，num1 原来的值 10 丢失了
num2 = num1;  // num2 = 20，得到的是被覆盖后的值
// ❌ 结果：num1 = 20, num2 = 20（都变成20了）
```

---

## 三、解题思路：借用第三个变量

就像交换两个杯子的水，需要**第三个杯子**：

```
初始：杯子A装可乐，杯子B装雪碧，杯子C空
1. 把A倒入C → A空，C有可乐
2. 把B倒入A → B空，A有雪碧
3. 把C倒入B → C空，B有可乐
结果：A=雪碧，B=可乐
```

---

## 四、代码实现

```javascript
// 1. 准备两个变量
let num1 = 10;
let num2 = 20;

// 2. 声明临时变量
let temp;

// 3. 交换
temp = num1;   // temp = 10
num1 = num2;   // num1 = 20
num2 = temp;   // num2 = 10

// 4. 查看结果
console.log(num1);  // 20
console.log(num2);  // 10
```

### 步骤图解

```
步骤        num1    num2    temp
---------------------------------------
初始值       10      20      -
temp=num1   10      20      10
num1=num2   20      20      10
num2=temp   20      10      10
```

---

## 五、变形：不借助临时变量

### 方法：利用加减法

```javascript
let a = 10;
let b = 20;

a = a + b;  // a = 30
b = a - b;  // b = 10
a = a - b;  // a = 20

console.log(a, b);  // 20, 10
```

### 方法：利用数组

```javascript
let a = 10;
let b = 20;

[a, b] = [b, a];  // 解构赋值

console.log(a, b);  // 20, 10
```
