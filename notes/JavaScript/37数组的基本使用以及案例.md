## 为什么需要数组？

不用多个变量存储：

```javascript
// ❌ 麻烦
let student1 = '张三';
let student2 = '李四';
let student3 = '王五';

// ✅ 简洁
let students = ['张三', '李四', '王五'];
```

## 声明数组

```javascript
// 方式1：字面量（推荐）
let arr = [];

// 方式2：构造函数（了解）
let arr = new Array();
```

## 数组组成

```
数组：['小红', '小明', '小李']
索引：   0      1      2
长度：3
```

| 术语 | 说明 |
|------|------|
| 元素 | 数组中的每个数据 |
| 索引 | 从0开始的编号 |
| 长度 | 元素个数（arr.length） |

## 基本操作

```javascript
let arr = ['小红', '小明', '小李'];

// 取值
console.log(arr[0]);  // '小红'
console.log(arr[2]);   // '小李'

// 存值
arr[1] = '小王';  // 替换
console.log(arr); // ['小红', '小王', '小李']

// 获取长度
console.log(arr.length);  // 3
```

## 遍历数组

```javascript
let scores = [85, 90, 78, 92];

for (let i = 0; i < scores.length; i++) {
    console.log(`第${i+1}个成绩：${scores[i]}`);
}
```

## 求和与平均值

```javascript
let scores = [85, 90, 78, 92, 88];
let sum = 0;

for (let i = 0; i < scores.length; i++) {
    sum += scores[i];
}

let avg = sum / scores.length;
console.log(`总分：${sum}，平均分：${avg}`);
```

## 注意事项

1. **索引从0开始**
2. **长度用 length**：`arr.length`
3. **可以存任意类型**：数字、字符串、布尔、数组等

## 记忆口诀

> **数组**：中括号包起来，逗号隔开每个元素，索引从0开始
