## 为什么需要补0？

时钟、计数器等场景需要**两位数显示**：

```
9:05:08   而不是   9:5:8
09:05:08  补0后   更美观规范
```

## 核心原理

```javascript
// 核心公式
num < 10 ? '0' + num : num

// 完整函数
function padZero(num) {
    return num < 10 ? '0' + num : num;
}
```

## 案例：数字时钟

```javascript
function padZero(num) {
    return num < 10 ? '0' + num : num;
}

let hour = 9;
let minute = 5;
let second = 8;

let time = `${padZero(hour)}:${padZero(minute)}:${padZero(second)}`;
console.log(time);  // "09:05:08"
```

## 案例：倒计时秒数显示

```javascript
function padZero(num) {
    return num < 10 ? '0' + num : num;
}

let totalSeconds = 65;
let minutes = Math.floor(totalSeconds / 60);
let seconds = totalSeconds % 60;

console.log(`${padZero(minutes)}:${padZero(seconds)}`);  // "01:05"
```

## 案例：日期格式化

```javascript
function padZero(num) {
    return num < 10 ? '0' + num : num;
}

let now = new Date();
let year = now.getFullYear();
let month = now.getMonth() + 1;
let day = now.getDate();

console.log(`${year}-${padZero(month)}-${padZero(day)}`);
// "2026-04-15"
```

## 封装为工具函数

```javascript
// 补0函数
function padZero(num) {
    return num < 10 ? '0' + num : '' + num;
}

// 或使用字符串方法
function padZero(num) {
    return String(num).padStart(2, '0');
}
```

## 原理解析

```
num = 5:
5 < 10 ? '0' + 5 : 5
→ '0' + '5' → '05'

num = 12:
12 < 10 ? '0' + 12 : 12
→ 12 (条件为false，直接返回)
```

## 记忆口诀

> **补0**：小于10前面补0，大于等于原样返回
