## 两种查询语法

```javascript
let person = {
    name: '小红',
    age: 18,
    'first-name': '张'  // 特殊属性名
};

// 方式1：点语法（常用）
console.log(person.name);    // '小红'
console.log(person.age);     // 18

// 方式2：中括号
console.log(person['name']); // '小红'
console.log(person['age']);  // 18
```

## 必须用中括号的情况

### 1. 属性名是变量

```javascript
let person = { name: '小红', age: 18 };

let key = 'name';
console.log(person.key);      // undefined（找key属性）
console.log(person[key]);      // '小红'（找name属性）
```

### 2. 属性名含特殊字符

```javascript
let person = {
    'first-name': '张',
    'my age': 18
};

console.log(person.first-name);  // 报错
console.log(person['first-name']); // '张'
console.log(person['my age']);    // 18
```

## 对比

| 场景 | 点语法 | 中括号 |
|------|--------|--------|
| 普通属性 | ✅ | ✅ |
| 变量作属性名 | ❌ | ✅ |
| 含特殊字符 | ❌ | ✅ |
| 代码简洁 | ✅ 更简洁 | ❌ 稍繁琐 |

## 实际应用

```javascript
let config = {
    'background-color': '#fff',
    'font-size': '14px'
};

// 只能用中括号
let style = config['background-color'];
```

## 注意事项

1. **优先用点语法**：更简洁
2. **变量或特殊字符**：必须用中括号
3. **中括号里是字符串**：引号不能省

## 记忆口诀

> **查属性**：普通属性用点语法，变量或特殊字符用中括号
