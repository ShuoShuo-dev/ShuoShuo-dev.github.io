## 练习1：五行五列星星

```javascript
// 外层控制行，内层控制列
for (let i = 1; i <= 5; i++) {
    for (let j = 1; j <= 5; j++) {
        document.write('★');
    }
    document.write('<br>');  // 换行
}
```

**输出**：
```
★★★★★
★★★★★
★★★★★
★★★★★
★★★★★
```

## 练习2：用户自定义

```javascript
let rows = +prompt('行数：');
let cols = +prompt('列数：');

for (let i = 1; i <= rows; i++) {
    for (let j = 1; j <= cols; j++) {
        document.write('★');
    }
    document.write('<br>');
}
```

## 练习3：直角三角形

```javascript
for (let i = 1; i <= 5; i++) {      // 行
    for (let j = 1; j <= i; j++) {  // 列 = 行数
        document.write('★');
    }
    document.write('<br>');
}
```

**输出**：
```
★
★★
★★★
★★★★
★★★★★
```

## 规律总结

| 形状 | 外层 | 内层条件 |
|------|------|----------|
| 正方形 | `i <= n` | `j <= n` |
| 三角形 | `i <= n` | `j <= i` |
| 倒三角 | `i <= n` | `j <= n-i+1` |

## 倒三角形

```javascript
for (let i = 1; i <= 5; i++) {
    for (let j = 1; j <= 6 - i; j++) {
        document.write('★');
    }
    document.write('<br>');
}
```

**输出**：
```
★★★★★
★★★★
★★★
★★
★
```

## 关键技巧

1. **换行位置**：内层循环结束后、外层循环内
2. **找规律**：列数与行数的关系
3. **画出来**：先画图再写代码
