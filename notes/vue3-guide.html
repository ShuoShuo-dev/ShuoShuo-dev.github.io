Vue 3 入门指南

发布日期：2026-03-20

1. Vue 3 简介

Vue 3 是 Vue.js 的最新版本，引入了组合式 API（Composition API），提供了更好的逻辑复用和类型支持。

2. 创建项目

# 使用 Vite 创建项目
npm create vite@latest my-vue-app -- --template vue

# 进入项目目录
cd my-vue-app

# 安装依赖
npm install

# 启动开发服务器
npm run dev

3. 组合式 API 基础

3.1 setup 函数

<script setup>
import { ref, computed } from 'vue'

// 响应式数据
const count = ref(0)

// 计算属性
const doubleCount = computed(() => count.value * 2)

// 方法
function increment() {
  count.value++
}
</script>

3.2 生命周期钩子

import { onMounted, onUnmounted } from 'vue'

onMounted(() => {
  console.log('组件已挂载')
})

onUnmounted(() => {
  console.log('组件已卸载')
})

4. 常用指令

  • v-bind / : - 绑定属性
  • v-model - 双向绑定
  • v-if / v-show - 条件渲染
  • v-for - 列表渲染
  • @click - 事件监听
