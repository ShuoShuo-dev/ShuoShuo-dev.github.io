Java 基础学习笔记

发布日期：2026-03-25

1. Java 简介

Java 是一种面向对象的编程语言，具有跨平台、安全性高、稳定性好等特点。

2. 基础语法

2.1 Hello World

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

2.2 数据类型

  • 基本类型：int, long, double, float, boolean, char, byte, short
  • 引用类型：String, 数组, 类, 接口

3. 面向对象

3.1 三大特性

  • 封装：将数据和方法封装在类中
  • 继承：子类继承父类的属性和方法
  • 多态：同一接口，不同实现

4. 常用集合

// List
List<String> list = new ArrayList<>();
list.add("Java");
list.add("Python");

// Map
Map<String, Integer> map = new HashMap<>();
map.put("age", 20);

// Set
Set<String> set = new HashSet<>();
set.add("A");
set.add("B");

5. 学习心得

Java 作为一门经典的编程语言，语法严谨，适合大型项目开发。学习过程中要多动手实践，理解面向对象的思想。
