# 📚 Shuo's Study Blog - 使用说明

> 专为本地学习笔记管理打造的个人博客系统

---

## 🚀 快速启动

1. **双击** `start-server.bat` 文件
2. 浏览器会自动打开 `http://localhost:8080`
3. 开始使用你的学习博客！

---

## 📁 目录结构

```
Shuo.Study/
├── 📄 index.html           ← 博客主页
├── ⚙️  config.json          ← 配置文件（个人信息/分类）
├── 🔧 generate-index.py    ← 笔记索引生成器
├── 🚀 start-server.bat     ← 一键启动服务器
├── 📝 notes/               ← 笔记存放目录
│   ├── Web前端/
│   │   ├── HTML5语义标签.md
│   │   └── CSS3弹性布局Flexbox.md
│   ├── Java后端/
│   │   └── Java面向对象核心概念.md
│   ├── 面试题/
│   │   └── 前端面试高频题精选.md
│   └── 算法与数据结构/
└── assets/                 ← 静态资源
```

---

## ✍️ 如何新增笔记

### 方法 1：直接创建文件（推荐）

1. 在 `notes/对应分类/` 文件夹下新建 `.md` 文件
2. 在文件顶部添加 frontmatter：

```markdown
---
title: 你的笔记标题
category: Web前端
tags: [HTML5, CSS3]
date: 2026-04-06
description: 笔记简介
---

# 正文开始...
```

3. 运行 `generate-index.py` 更新索引
4. 刷新浏览器即可看到新笔记

### 方法 2：通过后台管理

1. 点击导航栏 **「后台管理」**
2. 点击 **「新建笔记」**
3. 填写信息后生成模板
4. 复制模板内容，保存为 `.md` 文件

---

## ⚙️ 个性化配置

编辑 `config.json` 修改：
- 个人信息（名字、介绍、技能标签）
- 分类列表（名称、图标、颜色）
- 主题颜色

---

## 🎨 主题切换

点击右上角 **🌙 / ☀️** 按钮切换亮色/暗色模式

---

## 🔍 全文搜索

在顶部搜索框输入关键词，支持搜索：
- 笔记标题
- 笔记内容
- 标签
- 分类

---

## 💾 数据备份

直接复制整个 `Shuo.Study` 文件夹即可完成备份！
笔记全部保存为 Markdown 文件，永久可读。
